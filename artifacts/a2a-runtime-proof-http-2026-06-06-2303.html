硬性守卫规则

  • 未托管的订单不能进入 Agent 执行。
  • Agent 只能在声明的执行范围和工具权限内交付。
  • 买方不能绕过验收或仲裁直接释放资金。
  • A2A 会话角色必须先切换并保存，不能只靠 URL role 参数执行敏感动作。
