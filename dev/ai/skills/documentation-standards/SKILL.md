# documentation-standards（极简版）

## 何时使用

- 用户明确要求文档时
- 新增用户可见功能需使用说明
- 功能使用方式重大变更

## 必做

- 使用文档：功能是什么、如何使用、参数、示例
- 开发文档：架构、设计、扩展点
- 变更记录：日期、版本、变更内容
- 文档放模块 doc/ 目录

## 最小示例

```markdown
# 功能使用指南
## 功能说明
简述功能
## 使用步骤
1. 步骤1
2. 步骤2
```

## 禁止

- 用户未要求时主动创建文档
- 文档冗长、重复代码能表达的内容
