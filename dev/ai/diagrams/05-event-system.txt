┌─────────────────────────────────────────────────────────────────────────────┐
│ Three Extension Mechanisms Comparison                                         │
└─────────────────────────────────────────────────────────────────────────────┘

┌──────────┬────────────────────┬────────────────────┬──────────────────┐
│ Mechanism│ Event              │ Hook               │ Extends          │
├──────────┼────────────────────┼────────────────────┼──────────────────┤
│ Purpose  │ Business logic     │ View layer         │ Class method     │
│          │ decoupling         │ extension          │ extension        │
│ Trigger  │ dispatch()         │ Auto on render     │ Auto on call     │
│ Config   │ etc/event.xml      │ hook.php           │ extends.php      │
│ Scenario │ Order→send email   │ Insert ad slot     │ Price calc ext   │
└──────────┴────────────────────┴────────────────────┴──────────────────┘

Event: dispatch(event, $payload) → EventManager → Observer::execute()
Naming: {ModuleName}::{event-name}  Example: Weline_Order::order-place-after

Hook: {Module}::{area}::{type}::{component}::{position}
area=frontend|backend, type=partials|layouts, lowercase+hyphens only

Extends: Extend other modules' methods (before/after/around interceptors)

Cross-module query: Use w_query() + Provider pattern (NO events for queries)

┌─────────────────────────────────────────────────────────────────────────────┐
│ Event Example                                                                 │
└─────────────────────────────────────────────────────────────────────────────┘

1. Trigger: $data = ['order' => $this];
            $this->eventManager->dispatch('Weline_Order::order-place-after', $data);

2. Observer: class SendOrderEmail implements ObserverInterface {
                 public function execute(array $data) { ... }
             }

3. Register: etc/event.xml
             <event name="Weline_Order::order-place-after">
                 <observer name="send_order_email" class="..."/>
             </event>

Constraints:
  ✗ NO inline array: dispatch('event', ['key' => $val])
  ✓ MUST use variable: $data = [...]; dispatch('event', $data);
  ✗ NO events for data queries (use w_query())

┌─────────────────────────────────────────────────────────────────────────────┐
│ Hook Example                                                                  │
└─────────────────────────────────────────────────────────────────────────────┘

1. Define: hook.php
           return ['Weline_Catalog::frontend::partials::product-list::before' => [...]];

2. Use: <?= $this->hook('Weline_Catalog::frontend::partials::product-list::before') ?>

3. Implement: view/hooks/product-list-before.phtml

Constraints:
  ✗ NO underscores: product_list
  ✓ MUST hyphens: product-list
  ✗ NO uppercase: Product-List
  ✓ MUST lowercase: product-list

┌─────────────────────────────────────────────────────────────────────────────┐
│ Extends Example                                                               │
└─────────────────────────────────────────────────────────────────────────────┘

1. Define: extends.php
           return ['Weline\Catalog\Model\Product::getPrice' => [...]];

2. Implement: extends/module/Weline_Catalog/ProductPrice/ApplyDiscount.php
              public function aroundGetPrice($subject, callable $proceed) {
                  $price = $proceed();  // Call original
                  return $this->hasDiscount($subject) ? $price * 0.8 : $price;
              }

Method types: before{Method}, after{Method}, around{Method}

┌─────────────────────────────────────────────────────────────────────────────┐
│ Cross-module Query (w_query)                                                  │
└─────────────────────────────────────────────────────────────────────────────┘

Provider: class ProductQueryProvider {
              public function getProductsBySku(array $skus): array {
                  return $this->productModel->where('sku', $skus, 'IN')->select()->fetch();
              }
          }

Usage: $products = w_query('Weline\Catalog\Provider\ProductQueryProvider::getProductsBySku',
                            [['SKU001', 'SKU002']]);

┌─────────────────────────────────────────────────────────────────────────────┐
│ Selection Guide                                                               │
└─────────────────────────────────────────────────────────────────────────────┘

Use Event: Business notifications (order created, user login), side effects
Use Hook: Insert content in page (ads, banners), view layer extension
Use Extends: Modify method behavior (price calc, stock check), intercept calls
Use w_query(): Query other modules' data, cross-module aggregation
