┌─────────────────────────────────────────────────────────────────────────────┐
│ Core Principle: Attribute-driven Schema (#[Table]/#[Col]/#[Index])           │
└─────────────────────────────────────────────────────────────────────────────┘

Schema change flow: Modify Model attributes → php bin/w setup:upgrade → Auto DDL

┌─────────────────────────────────────────────────────────────────────────────┐
│ Model Definition Example                                                      │

┌─────────────────────────────────────────────────────────────────────────────┐
│ ORM Query Chain (MUST end with fetch/fetchArray)                             │
└─────────────────────────────────────────────────────────────────────────────┘

Query builder flow:
  Model → select() → where() → order() → limit() → fetch()
                                                    ↑
                                            MUST call to execute

Examples:
┌────────────────────────────────────────────────────────────────────────┐
│ // ✓ Correct: Ends with fetch()                                        │
│ $products = $productModel                                               │
│     ->select()                                                          │
│     ->where('status', 1)                                                │
│     ->order('created_at', 'DESC')                                       │
│     ->limit(10)                                                         │
│     ->fetch();  // Returns Model[] array                                │
│                                                                         │
│ // ✓ Correct: fetchArray() returns associative array                   │
│ $data = $productModel                                                   │
│     ->select(['product_id', 'sku', 'name'])                             │
│     ->where('status', 1)                                                │
│     ->fetchArray();  // Returns array[] array                           │
│                                                                         │
│ // ✗ Wrong: No fetch(), query won't execute                            │
│ $products = $productModel->select()->where('status', 1);                │
│                                                                         │
│ // ✓ Exception: save() doesn't need fetch()                            │
│ $product->setData('name', 'New Name')->save();                          │
└────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ Query Methods Quick Reference                                                 │
└─────────────────────────────────────────────────────────────────────────────┘

Query:
  ->select()                      Select all fields
  ->select(['id', 'name'])        Select specific fields
  ->where('status', 1)            Equal condition
  ->where('price', 100, '>')      Comparison condition
  ->where('id', [1,2,3], 'IN')    IN condition (NO whereIn())
  ->where('name', '%test%', 'LIKE') LIKE condition
  ->order('created_at', 'DESC')   Order by
  ->limit(10)                     Limit
  ->pagination($page, $size)      Pagination (auto-calculates offset)
  ->fetch()                       Execute and return Model[]
  ->fetchArray()                  Execute and return array[]

Find:
  ->find($id)                     Find by primary key (returns Query, needs fetch())
  ->find($id)->fetch()            Returns single Model or null

Insert:
  ->insert(['sku' => 'ABC', 'name' => 'Test'])->fetch()
  ->setData('sku', 'ABC')->save()  // Auto-detects insert/update

Update:
  ->update(['status' => 0])->where('id', 1)->fetch()
  ->load($id)->setData('status', 0)->save()

Delete:
  ->delete()->where('id', 1)->fetch()
  ->load($id)->delete()

└─────────────────────────────────────────────────────────────────────────────┘

php bin/w setup:upgrade
     │
     ▼
┌──────────────────────────────────────────────────────────────────────────┐
│ SchemaDiffStage                                                           │
│   1. Scan all Model classes for #[Table]/#[Col]/#[Index] attributes      │
│   2. Read current database schema                                         │
│   3. Compare differences (new tables/fields, modify fields, new indexes)  │
│   4. Generate DDL statements (CREATE TABLE, ALTER TABLE, CREATE INDEX)    │
│   5. Execute DDL                                                           │
└──────────────────────────────────────────────────────────────────────────┘

Supported changes:
  ✓ Add table
  ✓ Add field
  ✓ Modify field type/length/default
  ✓ Add index
  ✗ Drop field (manual SQL required)
  ✗ Drop index (manual SQL required)

┌─────────────────────────────────────────────────────────────────────────────┐
│ Connection Pool Architecture (WLS Mode)                                      │
└─────────────────────────────────────────────────────────────────────────────┘

Traditional mode (create connection per request):
  Request → new PDO() → Query → close PDO()

WLS mode (connection reuse):
  Request-1 → ConnectionPool::get() → Query → ConnectionPool::release()
                      ↓
  Request-2 → Reuse connection → Query → release()

ConnectionPoolManager:
  • Maintains connection pool (max connections, idle timeout)
  • Auto-reconnect (detects disconnection)
  • Health check (periodic ping)

┌─────────────────────────────────────────────────────────────────────────────┐
│ Critical Constraints                                                          │
└─────────────────────────────────────────────────────────────────────────────┘

✗ NEVER manually DDL in Setup/Upgrade.php (use #[Col] attributes)
✗ NEVER select/find/update/delete without fetch() (won't execute)
✗ NEVER use fetchOne() (doesn't exist, use find()->fetch())
✗ NEVER use whereIn() (doesn't exist, use where('id', [1,2], 'IN'))
✓ Schema changes MUST run setup:upgrade
✓ Primary key field MUST mark PRIMARY KEY AUTO_INCREMENT
✓ Indexes use #[Index] attribute, NO manual CREATE INDEX
✓ Transactions MUST catch exceptions and rollBack()

┌─────────────────────────────────────────────────────────────────────────────┐
│ Common Errors                                                                 │
└─────────────────────────────────────────────────────────────────────────────┘

Error 1: Forgot fetch()
  $products = $model->select()->where('status', 1);  // Returns Query object
  foreach ($products as $p) { ... }  // Error: Query not iterable

Correct:
  $products = $model->select()->where('status', 1)->fetch();

Error 2: Using non-existent methods
  $model->whereIn('id', [1,2,3]);  // Error: Method doesn't exist

Correct:
  $model->where('id', [1,2,3], 'IN');

Error 3: Manually editing generated/
  Edit generated/db/schema.php  // Will be overwritten

Correct:
  Modify Model #[Col] attributes, run setup:upgrade
