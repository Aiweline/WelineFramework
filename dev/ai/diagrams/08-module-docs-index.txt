┌─────────────────────────────────────────────────────────────────────────────┐
│ Workflow: How to Use Module Docs                                             │
└─────────────────────────────────────────────────────────────────────────────┘

Step 1: Identify the module you need to work with
  Example: "Add payment method" → Payment module

Step 2: Read module's doc/README.md FIRST
  php: Read app/code/Weline/Payment/doc/README.md

Step 3: Check specific subdirectories as needed
  • Need to listen to events? → doc/event/
  • Need to add UI hooks? → doc/hook/
  • Need to extend classes? → doc/extends/
  • Need API reference? → doc/API/

Step 4: Only read source code if docs are insufficient

✅ With docs:

  2. Read Payment/doc/API/API.md (~400 tokens)
  Total: ~1200 tokens

Savings: 80% tokens, 5x faster understanding
