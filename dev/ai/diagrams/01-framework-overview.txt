1. Modular: All features encapsulated in independent modules via register.php
2. Convention over Configuration: Auto-discovered routing, no routes.xml
3. Attribute-driven: Schema via #[Col], ACL via #[Acl]
4. Event-driven: Modules decoupled via events, NO direct dependencies
5. Dependency Injection: Constructor auto-injection
6. PSR-4 Autoloading: Namespace maps to directory structure
7. Long-running Optimization: WLS mode reuses processes, avoids re-init
