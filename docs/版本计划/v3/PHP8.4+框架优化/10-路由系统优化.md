# 10 - 路由系统优化 ⭐⭐⭐⭐

[← 索引](README.md)

**性能影响**：路由树匹配 O(n)→O(log n)，增量更新缩短构建时间。

---

## 目标

- 支持 `#[Route]` 注解路由
- 优化路由匹配算法（树结构）
- 增量更新（只更新变更模块）

## 路由树结构

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Router;

class RouteTree
{
    private array $tree = [];
    private array $namedRoutes = [];
    private array $methodIndex = [];
    
    public function addRoute(Route $route): void
    {
        $segments = explode('/', trim($route->path, '/'));
        $node = &$this->tree;
        
        foreach ($segments as $segment) {
            if (str_starts_with($segment, '{')) {
                // 参数节点
                $paramName = trim($segment, '{}');
                $segment = ':param';
                $node[$segment]['_param'] = $paramName;
            }
            
            if (!isset($node[$segment])) {
                $node[$segment] = [];
            }
            $node = &$node[$segment];
        }
        
        // 存储路由信息
        foreach ($route->methods as $method) {
            $node['_routes'][$method] = $route;
            $this->methodIndex[$method][] = $route;
        }
        
        if ($route->name) {
            $this->namedRoutes[$route->name] = $route;
        }
    }
    
    public function match(string $uri, string $method): ?MatchResult
    {
        $segments = explode('/', trim($uri, '/'));
        $params = [];
        $node = $this->tree;
        
        foreach ($segments as $segment) {
            if (isset($node[$segment])) {
                $node = $node[$segment];
            } elseif (isset($node[':param'])) {
                $params[$node[':param']['_param']] = $segment;
                $node = $node[':param'];
            } else {
                return null;
            }
        }
        
        $route = $node['_routes'][$method] ?? null;
        return $route ? new MatchResult($route, $params) : null;
    }
}
```
