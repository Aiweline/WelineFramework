# 13 - 实施计划（按优先级从高星到低星）

[← 索引](README.md)

实施顺序按 **优先级从高星到低星**，优先落地对性能有极大影响的项。

---

## 阶段划分（按性能优先级）

### 阶段 1：极大突破 ⭐⭐⭐⭐⭐（v3.0.0-alpha）

- [ ] PHP 8.4 最低版本要求
- [ ] **DI 容器 Lazy Objects 支持**（启动 -40%、内存 -30%）
- [ ] **ObjectManager 延迟加载**（getInstance(..., lazy: true)）
- [ ] 基础性能测试框架

### 阶段 2：重大提升 ⭐⭐⭐⭐（v3.0.0-alpha / beta）

- [ ] **新数组函数全面替换**（array_find / array_any 等，热点路径加速）
- [ ] **路由树结构**（匹配 O(n)→O(log n)）
- [ ] **路由增量更新**（只更新变更模块）
- [ ] **Block 延迟渲染**（Lazy Objects，按需初始化）
- [ ] 模板引擎 BlockFactory 集成 Lazy Ghost

### 阶段 3：明显优化 ⭐⭐⭐（v3.0.0-beta）

- [ ] `#[Route]` 属性注解
- [ ] 路由收集与存储优化（命名路由、预编译正则）
- [ ] `#[Validate]` / `#[Cache]` / `#[EventListener]` / `#[Cron]` 注解

### 阶段 4：代码质量 ⭐⭐（v3.0.0-beta / rc）

- [ ] AbstractModel Property Hooks
- [ ] Asymmetric Visibility（Model ID、Config 等）
- [ ] 字段属性注解、自动类型转换
- [ ] HTML5 解析器、编译时优化

### 阶段 5：稳定化（v3.0.0）

- [ ] 全面性能测试与基准对比
- [ ] 文档完善
- [ ] 迁移工具（Model / 路由）
- [ ] 发布
