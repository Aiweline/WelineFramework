## 破坏性变更

| 变更 | 影响 | 迁移方案 |
|------|------|---------|
| PHP 8.4 最低版本 | 所有用户 | 升级 PHP |
| Model API 变更 | Model 使用者 | 自动迁移工具 |
| 路由注册方式 | 控制器开发 | 支持两种方式过渡 |

## 迁移工具

```bash
# Model 迁移
bin/m v3:migrate:models --module=Weline_User

# 路由迁移
bin/m v3:migrate:routes --module=Weline_User

# 全量检查
bin/m v3:check:compatibility
```

## 兼容层

v3 将提供兼容层，允许渐进式迁移：

```php
// 兼容层：getData/setData 仍可用，但标记为废弃
#[\Deprecated('Use property access instead')]
public function getData(string $key): mixed
{
    return $this->$key ?? $this->data[$key] ?? null;
}
```
