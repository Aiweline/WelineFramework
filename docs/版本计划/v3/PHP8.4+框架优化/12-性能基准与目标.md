## 测试场景

| 场景 | 描述 |
|------|------|
| 冷启动 | 首次请求，无缓存 |
| 热启动 | OPcache 预热后 |
| API 请求 | JSON 响应 |
| 页面渲染 | 完整 HTML 页面 |
| 数据库操作 | CRUD 100 条记录 |

## 性能目标

| 指标 | v2 基准 | v3 目标 | 提升 |
|------|---------|---------|------|
| 冷启动时间 | 150ms | <100ms | -33% |
| 热启动时间 | 50ms | <30ms | -40% |
| API 响应（简单） | 20ms | <12ms | -40% |
| 页面渲染（10 Block） | 100ms | <60ms | -40% |
| 内存占用（基础） | 10MB | <7MB | -30% |
| 内存占用（复杂页面） | 50MB | <35MB | -30% |

## 基准测试计划

```bash
# 使用 phpbench 进行基准测试
vendor/bin/phpbench run --report=aggregate

# 测试项目
- ObjectManager::getInstance() 性能
- 路由匹配性能
- Model 属性访问性能
- 模板渲染性能
```
