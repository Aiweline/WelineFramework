# 11 - 模板引擎优化 ⭐⭐⭐⭐

**性能影响**：Block 延迟渲染，按需初始化，减少无用对象创建与 I/O。

---

## 目标

- Block 延迟渲染（Lazy Objects）
- 编译时优化
- 新 HTML5 解析器

## Block 延迟渲染

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\View;

class BlockFactory
{
    public function create(string $class, array $data = [], bool $lazy = true): Block
    {
        if ($lazy) {
            $reflector = new \ReflectionClass($class);
            return $reflector->newLazyGhost(function ($block) use ($class, $data) {
                $this->initializeBlock($block, $data);
            });
        }
        
        return $this->createImmediate($class, $data);
    }
}

// 模板中
// Block 只有在实际渲染时才会初始化
<block class="Weline\Catalog\Block\ProductList" lazy="true" />
```
