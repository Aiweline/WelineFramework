# 06 - 新数组函数 ⭐⭐⭐⭐

[← 索引](README.md)

**性能影响**：路由/ACL 等热点路径短路求值，匹配可快数十倍；替代 `array_filter` 时 O(n)→O(k)。

---

## 函数列表

| 函数 | 功能 | 性能特点 |
|------|------|---------|
| `array_find($array, $callback)` | 查找第一个匹配元素 | 短路求值 |
| `array_find_key($array, $callback)` | 查找第一个匹配的键 | 短路求值 |
| `array_any($array, $callback)` | 任一元素满足条件 | 短路求值 |
| `array_all($array, $callback)` | 所有元素满足条件 | 短路求值 |

## 性能优势

```php
// 测试数据：10000 个元素，目标在第 100 个位置

// ❌ array_filter - 遍历全部 10000 个
$found = current(array_filter($users, fn($u) => $u['id'] === 100));
// 耗时：~2.5ms

// ✅ array_find - 在第 100 个停止
$found = array_find($users, fn($u) => $u['id'] === 100);
// 耗时：~0.025ms（快 100 倍）

// ❌ 检查存在性 - 传统方式
$exists = !empty(array_filter($users, fn($u) => $u['role'] === 'admin'));
// 耗时：~2.5ms

// ✅ array_any - 短路
$exists = array_any($users, fn($u) => $u['role'] === 'admin');
// 耗时：取决于第一个 admin 的位置
```

## 框架应用

### 路由匹配

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Router;

class RouterMatcher
{
    private array $routes = [];
    
    /**
     * 匹配路由 - 使用 array_find
     */
    public function match(string $uri, string $method): ?array
    {
        // ✅ 找到即返回，不继续遍历
        return array_find(
            $this->routes,
            fn($route) => $this->matchRoute($route, $uri, $method)
        );
    }
    
    /**
     * 检查是否有匹配的路由 - 使用 array_any
     */
    public function hasRoute(string $uri): bool
    {
        return array_any(
            $this->routes,
            fn($route) => $this->uriMatches($route['pattern'], $uri)
        );
    }
    
    /**
     * 验证所有路由有效 - 使用 array_all
     */
    public function validateRoutes(): bool
    {
        return array_all(
            $this->routes,
            fn($route) => $this->isValidRoute($route)
        );
    }
    
    /**
     * 查找路由名称对应的键 - 使用 array_find_key
     */
    public function findRouteKey(string $name): ?string
    {
        return array_find_key(
            $this->routes,
            fn($route) => ($route['name'] ?? '') === $name
        );
    }
}
```

### ACL 检查

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Acl;

class AclChecker
{
    /**
     * 检查是否有任一权限 - array_any
     */
    public function hasAnyPermission(array $userPermissions, array $requiredPermissions): bool
    {
        return array_any(
            $requiredPermissions,
            fn($permission) => in_array($permission, $userPermissions, true)
        );
    }
    
    /**
     * 检查是否有全部权限 - array_all
     */
    public function hasAllPermissions(array $userPermissions, array $requiredPermissions): bool
    {
        return array_all(
            $requiredPermissions,
            fn($permission) => in_array($permission, $userPermissions, true)
        );
    }
    
    /**
     * 查找第一个缺失的权限 - array_find
     */
    public function findMissingPermission(array $userPermissions, array $requiredPermissions): ?string
    {
        return array_find(
            $requiredPermissions,
            fn($permission) => !in_array($permission, $userPermissions, true)
        );
    }
}
```

## 替换清单

以下代码模式应在 v3 中替换：

```php
// 模式 1：查找第一个匹配
// ❌ 旧
$found = null;
foreach ($array as $item) {
    if ($condition($item)) {
        $found = $item;
        break;
    }
}
// ✅ 新
$found = array_find($array, $condition);

// 模式 2：检查存在性
// ❌ 旧
$exists = !empty(array_filter($array, $condition));
// ✅ 新
$exists = array_any($array, $condition);

// 模式 3：检查全部满足
// ❌ 旧
$allValid = count(array_filter($array, $condition)) === count($array);
// ✅ 新
$allValid = array_all($array, $condition);

// 模式 4：查找键
// ❌ 旧
$key = array_search(true, array_map($condition, $array));
// ✅ 新
$key = array_find_key($array, $condition);
```
