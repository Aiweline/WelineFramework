## B. 术语表

| 术语 | 说明 |
|------|------|
| Property Hooks | PHP 8.4 属性钩子，允许定义 get/set 行为 |
| Lazy Objects | 延迟对象，首次访问时才初始化 |
| Ghost Objects | 延迟对象的一种，对象框架已创建但属性未初始化 |
| Proxy Objects | 延迟对象的一种，完全代理到另一个对象 |
| Short-circuit | 短路求值，满足条件即停止遍历 |
