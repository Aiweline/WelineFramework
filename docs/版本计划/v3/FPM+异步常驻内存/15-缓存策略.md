# 15 - 缓存策略

> **优先级**: ⭐⭐⭐⭐  
> **依赖**: 02-运行时抽象层, 10-WeAsync异步引擎  
> **预计工作量**: 2-3 天

---

## 0. 当前框架缓存淘汰策略调研

### 0.1 当前实现分析

**文件缓存驱动** (`Weline\Framework\Cache\Driver\File`):

```php
// 过期机制：使用文件修改时间（mtime）标记过期时间
public function set(string $key, mixed $value, int $duration = 1800): bool
{
    file_put_contents($cacheFile, serialize($value), LOCK_EX);
    touch($cacheFile, $duration + time()); // 用 mtime 标记过期
}

public function get(string $key): mixed
{
    if (filemtime($cacheFile) > time()) {
        return unserialize(file_get_contents($cacheFile));
    }
    return false; // 过期返回 false，但不删除文件！
}
```

**淘汰方式**：Cron 定时任务 (`Weline\CacheManager\Cron\CacheCleanup`)

```php
// 每天凌晨 2 点执行
public function cron_time(): string { return '0 2 * * *'; }

// 清理超过 24 小时的文件（硬编码 86400 秒）
if (time() - $file->getMTime() > 86400) {
    unlink($file->getPathname());
}
```

### 0.2 当前问题

| 问题 | 影响 | 严重程度 |
|------|------|----------|
| **过期文件不删除** | `get()` 返回 false 但文件仍占用磁盘 | ⚠️ 中 |
| **清理间隔太长** | 24 小时才清理一次，过期文件堆积 | ⚠️ 中 |
| **无容量限制** | 磁盘可能被撑满 | 🔴 高 |
| **无 LRU 淘汰** | 热点数据可能被冷数据挤占 | ⚠️ 中 |
| **WeAsync 阻塞** | 文件 I/O 阻塞事件循环 | 🔴 高 |

### 0.3 结论

当前框架的缓存淘汰**依赖 Cron**，但存在以下问题：
1. **被动清理** - 只能定时批量清理，不能实时淘汰
2. **无容量控制** - 没有内存/磁盘使用上限
3. **策略单一** - 只有 TTL 过期，没有 LRU/LFU

## 1. 核心决策

### 1.1 WeAsync 模式自动切换内存缓存

| 问题 | FPM 模式 | WeAsync 模式 |
|------|---------|-------------|
| **文件 I/O** | 每个请求独立，问题不大 | 高并发下成为瓶颈 |
| **文件锁** | 进程独立，锁竞争少 | 多请求共享进程，锁竞争严重 |
| **磁盘延迟** | 可接受 | 破坏事件循环的非阻塞特性 |

### 1.2 支持可配置的淘汰策略

```php
// config/cache.php
return [
    'eviction' => [
        // 淘汰策略
        'policy' => 'lru', // lru, lfu, ttl, random
        
        // 容量限制
        'max_items' => 10000,          // 最大缓存项数
        'max_memory' => 64 * 1024 * 1024, // 64MB
        
        // TTL 设置
        'default_ttl' => 3600,         // 默认 TTL（秒）
        'max_ttl' => 86400,            // 最大 TTL（秒）
        
        // GC 设置
        'gc_probability' => 100,       // GC 触发概率（1/N）
        'gc_divisor' => 10000,         // GC 触发除数
    ],
];
```

## 1. 缓存层级设计

```
┌─────────────────────────────────────────────────────────────────┐
│                        缓存层级（由快到慢）                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  L1: 进程内存缓存 (ProcessMemoryCache)                           │
│      ├─ 最快：无任何 I/O                                         │
│      ├─ 缺点：仅当前 Worker 可见，重启丢失                        │
│      └─ 适用：热点数据、计算结果、配置                            │
│                                                                  │
│  L2: 共享内存缓存 (APCu / Shared Memory)                         │
│      ├─ 快：同机器所有 Worker 共享                                │
│      ├─ 缺点：单机限制，需要扩展                                  │
│      └─ 适用：Session、频繁访问的数据                             │
│                                                                  │
│  L3: Redis 缓存                                                  │
│      ├─ 较快：网络 I/O，但支持分布式                              │
│      ├─ 缺点：需要额外服务                                        │
│      └─ 适用：分布式缓存、需要持久化的数据                        │
│                                                                  │
│  L4: 文件缓存 (仅 FPM 模式)                                       │
│      ├─ 慢：磁盘 I/O                                             │
│      ├─ 缺点：WeAsync 下有锁竞争                                  │
│      └─ 适用：FPM 模式、冷启动预热                                │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 CacheManager（自动切换）

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Cache;

use Weline\Framework\Runtime\RuntimeFactory;
use Weline\Framework\Runtime\RuntimeType;

/**
 * 缓存管理器
 * 
 * 根据运行时自动选择最优缓存驱动
 */
class CacheManager implements CacheInterface
{
    /**
     * 缓存驱动
     */
    private CacheInterface $driver;
    
    /**
     * L1 进程内存缓存（始终启用）
     */
    private ProcessMemoryCache $l1Cache;
    
    /**
     * 配置
     */
    private array $config;
    
    public function __construct(array $config = [])
    {
        $this->config = array_merge([
            // L1 配置
            'l1_enabled' => true,
            'l1_max_items' => 10000,
            'l1_max_memory' => 64 * 1024 * 1024, // 64MB
            
            // L2 配置（APCu）
            'l2_driver' => 'apcu', // apcu, redis, file
            
            // Redis 配置
            'redis' => [
                'host' => '127.0.0.1',
                'port' => 6379,
                'database' => 0,
                'prefix' => 'weline:cache:',
            ],
            
            // 文件缓存配置（仅 FPM）
            'file' => [
                'path' => 'var/cache',
            ],
        ], $config);
        
        $this->initDriver();
    }
    
    /**
     * 根据运行时初始化驱动
     */
    private function initDriver(): void
    {
        // L1 进程内存缓存（始终启用）
        if ($this->config['l1_enabled']) {
            $this->l1Cache = new ProcessMemoryCache(
                maxItems: $this->config['l1_max_items'],
                maxMemory: $this->config['l1_max_memory']
            );
        }
        
        $runtime = RuntimeFactory::detectType();
        
        if ($runtime === RuntimeType::WEASYNC) {
            // WeAsync 模式：使用内存/Redis 缓存
            $this->driver = $this->createAsyncDriver();
        } else {
            // FPM 模式：使用文件缓存
            $this->driver = $this->createFpmDriver();
        }
    }
    
    /**
     * 创建 WeAsync 模式驱动
     */
    private function createAsyncDriver(): CacheInterface
    {
        $l2Driver = $this->config['l2_driver'];
        
        return match($l2Driver) {
            'apcu' => extension_loaded('apcu') 
                ? new ApcuCache() 
                : new ProcessMemoryCache(), // fallback
            'redis' => new RedisCache($this->config['redis']),
            default => new ProcessMemoryCache(),
        };
    }
    
    /**
     * 创建 FPM 模式驱动
     */
    private function createFpmDriver(): CacheInterface
    {
        // FPM 模式仍可使用 Redis（如果配置了）
        if ($this->config['l2_driver'] === 'redis') {
            return new RedisCache($this->config['redis']);
        }
        
        return new FileCache($this->config['file']['path']);
    }
    
    /**
     * 获取缓存（L1 优先）
     */
    public function get(string $key, mixed $default = null): mixed
    {
        // L1: 进程内存
        if ($this->config['l1_enabled']) {
            $value = $this->l1Cache->get($key);
            if ($value !== null) {
                return $value;
            }
        }
        
        // L2: 驱动缓存
        $value = $this->driver->get($key, $default);
        
        // 回填 L1
        if ($value !== $default && $this->config['l1_enabled']) {
            $this->l1Cache->set($key, $value);
        }
        
        return $value;
    }
    
    /**
     * 设置缓存（同时写入 L1 和 L2）
     */
    public function set(string $key, mixed $value, null|int|\DateInterval $ttl = null): bool
    {
        // L1
        if ($this->config['l1_enabled']) {
            $this->l1Cache->set($key, $value, $ttl);
        }
        
        // L2
        return $this->driver->set($key, $value, $ttl);
    }
    
    /**
     * 删除缓存
     */
    public function delete(string $key): bool
    {
        if ($this->config['l1_enabled']) {
            $this->l1Cache->delete($key);
        }
        
        return $this->driver->delete($key);
    }
    
    /**
     * 清空缓存
     */
    public function clear(): bool
    {
        if ($this->config['l1_enabled']) {
            $this->l1Cache->clear();
        }
        
        return $this->driver->clear();
    }
    
    /**
     * 检查缓存是否存在
     */
    public function has(string $key): bool
    {
        if ($this->config['l1_enabled'] && $this->l1Cache->has($key)) {
            return true;
        }
        
        return $this->driver->has($key);
    }
    
    // ... getMultiple, setMultiple, deleteMultiple 类似实现
}
```

### 2.3 ProcessMemoryCache（L1 进程内存缓存，支持多种淘汰策略）

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Cache\Driver;

use Weline\Framework\Cache\CacheInterface;

/**
 * 进程内存缓存
 * 
 * 特点：
 * - 最快：无任何 I/O
 * - 仅当前 Worker 进程可见
 * - Worker 重启后丢失
 * - 支持多种淘汰策略（LRU/LFU/TTL/Random）
 * - 支持 TTL 过期
 * - 内存使用量限制
 */
class ProcessMemoryCache implements CacheInterface
{
    // 淘汰策略常量
    public const EVICTION_LRU = 'lru';      // 最近最少使用
    public const EVICTION_LFU = 'lfu';      // 最不经常使用
    public const EVICTION_TTL = 'ttl';      // 最快过期优先
    public const EVICTION_RANDOM = 'random'; // 随机淘汰
    
    /**
     * 缓存数据
     * @var array<string, array{value: mixed, expire: int|null, size: int, hits: int, created: int}>
     */
    private array $data = [];
    
    /**
     * LRU 访问顺序
     * @var array<string, int>
     */
    private array $accessOrder = [];
    
    /**
     * 当前内存使用
     */
    private int $currentMemory = 0;
    
    /**
     * 访问计数器
     */
    private int $accessCounter = 0;
    
    /**
     * 操作计数器（用于触发 GC）
     */
    private int $operationCounter = 0;
    
    /**
     * 统计信息
     */
    private array $stats = [
        'hits' => 0,
        'misses' => 0,
        'evictions' => 0,
        'gc_runs' => 0,
        'gc_cleaned' => 0,
    ];
    
    /**
     * 配置
     */
    private array $config;
    
    public function __construct(array $config = [])
    {
        $this->config = array_merge([
            // 容量限制
            'max_items' => 10000,
            'max_memory' => 64 * 1024 * 1024, // 64MB
            
            // 淘汰策略
            'eviction_policy' => self::EVICTION_LRU,
            
            // 批量淘汰数量（触发淘汰时一次淘汰多少个）
            'eviction_batch' => 100,
            
            // GC 设置
            'gc_probability' => 1,      // GC 触发概率
            'gc_divisor' => 100,        // GC 触发除数（1/100 概率）
            'gc_max_scan' => 1000,      // GC 单次最大扫描数
            
            // TTL 设置
            'default_ttl' => 3600,      // 默认 TTL（秒）
            'max_ttl' => 86400,         // 最大 TTL（秒）
        ], $config);
    }
    
    public function get(string $key, mixed $default = null): mixed
    {
        // 概率触发 GC
        $this->maybeGc();
        
        // 检查是否存在
        if (!isset($this->data[$key])) {
            $this->stats['misses']++;
            return $default;
        }
        
        $item = &$this->data[$key];
        
        // 检查是否过期
        if ($item['expire'] !== null && $item['expire'] < time()) {
            $this->delete($key);
            $this->stats['misses']++;
            return $default;
        }
        
        // 更新访问统计（用于 LRU/LFU）
        $this->accessOrder[$key] = ++$this->accessCounter;
        $item['hits']++;
        $this->stats['hits']++;
        
        return $item['value'];
    }
    
    public function set(string $key, mixed $value, null|int|\DateInterval $ttl = null): bool
    {
        // 计算过期时间
        $expire = $this->calculateExpire($ttl);
        
        // 计算大小
        $size = $this->estimateSize($value);
        
        // 如果已存在，先删除旧的
        if (isset($this->data[$key])) {
            $this->currentMemory -= $this->data[$key]['size'];
        }
        
        // 检查是否需要淘汰
        $this->evictIfNeeded($size);
        
        // 存储
        $this->data[$key] = [
            'value' => $value,
            'expire' => $expire,
            'size' => $size,
            'hits' => 0,
            'created' => time(),
        ];
        
        $this->accessOrder[$key] = ++$this->accessCounter;
        $this->currentMemory += $size;
        
        return true;
    }
    
    public function delete(string $key): bool
    {
        if (isset($this->data[$key])) {
            $this->currentMemory -= $this->data[$key]['size'];
            unset($this->data[$key], $this->accessOrder[$key]);
            return true;
        }
        
        return false;
    }
    
    public function clear(): bool
    {
        $this->data = [];
        $this->accessOrder = [];
        $this->currentMemory = 0;
        return true;
    }
    
    public function has(string $key): bool
    {
        if (!isset($this->data[$key])) {
            return false;
        }
        
        $item = $this->data[$key];
        if ($item['expire'] !== null && $item['expire'] < time()) {
            $this->delete($key);
            return false;
        }
        
        return true;
    }
    
    /**
     * 计算过期时间
     */
    private function calculateExpire(null|int|\DateInterval $ttl): ?int
    {
        if ($ttl === null) {
            $ttl = $this->config['default_ttl'];
        }
        
        if ($ttl instanceof \DateInterval) {
            $seconds = (new \DateTime())->add($ttl)->getTimestamp() - time();
        } else {
            $seconds = $ttl;
        }
        
        // 限制最大 TTL
        $seconds = min($seconds, $this->config['max_ttl']);
        
        return $seconds > 0 ? time() + $seconds : null;
    }
    
    /**
     * 检查并执行淘汰
     */
    private function evictIfNeeded(int $newSize): void
    {
        $evicted = 0;
        $batchSize = $this->config['eviction_batch'];
        
        // 检查项数限制
        while (count($this->data) >= $this->config['max_items'] && $evicted < $batchSize) {
            $this->evictOne();
            $evicted++;
        }
        
        // 检查内存限制
        while ($this->currentMemory + $newSize > $this->config['max_memory'] && !empty($this->data) && $evicted < $batchSize) {
            $this->evictOne();
            $evicted++;
        }
    }
    
    /**
     * 根据策略淘汰一个项
     */
    private function evictOne(): void
    {
        if (empty($this->data)) {
            return;
        }
        
        $keyToEvict = match($this->config['eviction_policy']) {
            self::EVICTION_LRU => $this->findLruKey(),
            self::EVICTION_LFU => $this->findLfuKey(),
            self::EVICTION_TTL => $this->findTtlKey(),
            self::EVICTION_RANDOM => $this->findRandomKey(),
            default => $this->findLruKey(),
        };
        
        if ($keyToEvict !== null) {
            $this->delete($keyToEvict);
            $this->stats['evictions']++;
        }
    }
    
    /**
     * LRU: 找到最久未访问的 key
     */
    private function findLruKey(): ?string
    {
        if (empty($this->accessOrder)) {
            return array_key_first($this->data);
        }
        return array_keys($this->accessOrder, min($this->accessOrder))[0] ?? null;
    }
    
    /**
     * LFU: 找到访问次数最少的 key
     */
    private function findLfuKey(): ?string
    {
        $minHits = PHP_INT_MAX;
        $minKey = null;
        
        foreach ($this->data as $key => $item) {
            if ($item['hits'] < $minHits) {
                $minHits = $item['hits'];
                $minKey = $key;
            }
        }
        
        return $minKey;
    }
    
    /**
     * TTL: 找到最快过期的 key
     */
    private function findTtlKey(): ?string
    {
        $minExpire = PHP_INT_MAX;
        $minKey = null;
        
        foreach ($this->data as $key => $item) {
            $expire = $item['expire'] ?? PHP_INT_MAX;
            if ($expire < $minExpire) {
                $minExpire = $expire;
                $minKey = $key;
            }
        }
        
        return $minKey;
    }
    
    /**
     * Random: 随机选择一个 key
     */
    private function findRandomKey(): ?string
    {
        $keys = array_keys($this->data);
        return $keys[array_rand($keys)] ?? null;
    }
    
    /**
     * 概率触发 GC
     */
    private function maybeGc(): void
    {
        $this->operationCounter++;
        
        // 根据概率决定是否触发 GC
        if (mt_rand(1, $this->config['gc_divisor']) <= $this->config['gc_probability']) {
            $this->gc();
        }
    }
    
    /**
     * 垃圾回收（清理过期项）
     */
    public function gc(): int
    {
        $now = time();
        $cleaned = 0;
        $scanned = 0;
        $maxScan = $this->config['gc_max_scan'];
        
        foreach ($this->data as $key => $item) {
            if (++$scanned > $maxScan) {
                break; // 限制单次扫描数量，避免阻塞
            }
            
            if ($item['expire'] !== null && $item['expire'] < $now) {
                $this->delete($key);
                $cleaned++;
            }
        }
        
        $this->stats['gc_runs']++;
        $this->stats['gc_cleaned'] += $cleaned;
        
        return $cleaned;
    }
    
    /**
     * 强制 GC（扫描所有项）
     */
    public function forceGc(): int
    {
        $now = time();
        $cleaned = 0;
        
        foreach ($this->data as $key => $item) {
            if ($item['expire'] !== null && $item['expire'] < $now) {
                $this->delete($key);
                $cleaned++;
            }
        }
        
        $this->stats['gc_runs']++;
        $this->stats['gc_cleaned'] += $cleaned;
        
        return $cleaned;
    }
    
    /**
     * 估算值大小
     */
    private function estimateSize(mixed $value): int
    {
        if (is_string($value)) {
            return strlen($value) + 64; // 字符串 + 元数据开销
        }
        if (is_array($value) || is_object($value)) {
            return strlen(serialize($value)) + 64;
        }
        if (is_int($value) || is_float($value)) {
            return 72; // 8 字节值 + 64 元数据
        }
        if (is_bool($value)) {
            return 65;
        }
        return 128; // 默认
    }
    
    /**
     * 获取统计信息
     */
    public function getStats(): array
    {
        $totalRequests = $this->stats['hits'] + $this->stats['misses'];
        
        return [
            ...$this->stats,
            'items' => count($this->data),
            'memory' => $this->currentMemory,
            'memory_formatted' => $this->formatBytes($this->currentMemory),
            'max_items' => $this->config['max_items'],
            'max_memory' => $this->config['max_memory'],
            'max_memory_formatted' => $this->formatBytes($this->config['max_memory']),
            'usage_items_percent' => count($this->data) / $this->config['max_items'] * 100,
            'usage_memory_percent' => $this->currentMemory / $this->config['max_memory'] * 100,
            'hit_rate' => $totalRequests > 0 ? $this->stats['hits'] / $totalRequests : 0,
            'eviction_policy' => $this->config['eviction_policy'],
        ];
    }
    
    /**
     * 格式化字节
     */
    private function formatBytes(int $bytes): string
    {
        $units = ['B', 'KB', 'MB', 'GB'];
        $pow = floor(($bytes ? log($bytes) : 0) / log(1024));
        $pow = min($pow, count($units) - 1);
        return round($bytes / pow(1024, $pow), 2) . ' ' . $units[$pow];
    }
    
    /**
     * 设置淘汰策略（运行时可切换）
     */
    public function setEvictionPolicy(string $policy): void
    {
        $this->config['eviction_policy'] = $policy;
    }
    
    /**
     * 获取当前配置
     */
    public function getConfig(): array
    {
        return $this->config;
    }
}
```

### 2.4 ApcuCache（共享内存）

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Cache\Driver;

use Weline\Framework\Cache\CacheInterface;

/**
 * APCu 缓存
 * 
 * 特点：
 * - 所有 Worker 进程共享
 * - 内存存储，速度快
 * - 需要 apcu 扩展
 */
class ApcuCache implements CacheInterface
{
    private string $prefix;
    
    public function __construct(string $prefix = 'weline:')
    {
        if (!extension_loaded('apcu')) {
            throw new \RuntimeException('APCu extension is not loaded');
        }
        
        $this->prefix = $prefix;
    }
    
    public function get(string $key, mixed $default = null): mixed
    {
        $success = false;
        $value = apcu_fetch($this->prefix . $key, $success);
        
        return $success ? $value : $default;
    }
    
    public function set(string $key, mixed $value, null|int|\DateInterval $ttl = null): bool
    {
        $ttlSeconds = 0;
        if ($ttl !== null) {
            if ($ttl instanceof \DateInterval) {
                $ttlSeconds = (new \DateTime())->add($ttl)->getTimestamp() - time();
            } else {
                $ttlSeconds = $ttl;
            }
        }
        
        return apcu_store($this->prefix . $key, $value, $ttlSeconds);
    }
    
    public function delete(string $key): bool
    {
        return apcu_delete($this->prefix . $key);
    }
    
    public function clear(): bool
    {
        // 只清除带前缀的 key
        $iterator = new \APCUIterator('/^' . preg_quote($this->prefix, '/') . '/');
        return apcu_delete($iterator);
    }
    
    public function has(string $key): bool
    {
        return apcu_exists($this->prefix . $key);
    }
    
    public function getMultiple(iterable $keys, mixed $default = null): iterable
    {
        $prefixedKeys = [];
        foreach ($keys as $key) {
            $prefixedKeys[$key] = $this->prefix . $key;
        }
        
        $values = apcu_fetch(array_values($prefixedKeys));
        
        $result = [];
        foreach ($prefixedKeys as $originalKey => $prefixedKey) {
            $result[$originalKey] = $values[$prefixedKey] ?? $default;
        }
        
        return $result;
    }
    
    public function setMultiple(iterable $values, null|int|\DateInterval $ttl = null): bool
    {
        $ttlSeconds = 0;
        if ($ttl !== null) {
            if ($ttl instanceof \DateInterval) {
                $ttlSeconds = (new \DateTime())->add($ttl)->getTimestamp() - time();
            } else {
                $ttlSeconds = $ttl;
            }
        }
        
        $prefixedValues = [];
        foreach ($values as $key => $value) {
            $prefixedValues[$this->prefix . $key] = $value;
        }
        
        $errors = apcu_store($prefixedValues, null, $ttlSeconds);
        return empty($errors);
    }
    
    public function deleteMultiple(iterable $keys): bool
    {
        $prefixedKeys = [];
        foreach ($keys as $key) {
            $prefixedKeys[] = $this->prefix . $key;
        }
        
        return apcu_delete($prefixedKeys) !== false;
    }
}
```

### 2.5 RedisCache

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Cache\Driver;

use Weline\Framework\Cache\CacheInterface;

/**
 * Redis 缓存
 * 
 * 特点：
 * - 分布式共享
 * - 支持持久化
 * - 需要 Redis 服务
 */
class RedisCache implements CacheInterface
{
    private \Redis $redis;
    private string $prefix;
    
    public function __construct(array $config)
    {
        $this->redis = new \Redis();
        $this->redis->connect(
            $config['host'] ?? '127.0.0.1',
            $config['port'] ?? 6379,
            $config['timeout'] ?? 2.0
        );
        
        if (!empty($config['password'])) {
            $this->redis->auth($config['password']);
        }
        
        if (!empty($config['database'])) {
            $this->redis->select($config['database']);
        }
        
        $this->prefix = $config['prefix'] ?? 'weline:cache:';
    }
    
    public function get(string $key, mixed $default = null): mixed
    {
        $value = $this->redis->get($this->prefix . $key);
        
        if ($value === false) {
            return $default;
        }
        
        return unserialize($value);
    }
    
    public function set(string $key, mixed $value, null|int|\DateInterval $ttl = null): bool
    {
        $serialized = serialize($value);
        
        if ($ttl === null) {
            return $this->redis->set($this->prefix . $key, $serialized);
        }
        
        $ttlSeconds = $ttl instanceof \DateInterval
            ? (new \DateTime())->add($ttl)->getTimestamp() - time()
            : $ttl;
        
        return $this->redis->setex($this->prefix . $key, $ttlSeconds, $serialized);
    }
    
    public function delete(string $key): bool
    {
        return $this->redis->del($this->prefix . $key) > 0;
    }
    
    public function clear(): bool
    {
        $keys = $this->redis->keys($this->prefix . '*');
        if (!empty($keys)) {
            return $this->redis->del($keys) > 0;
        }
        return true;
    }
    
    public function has(string $key): bool
    {
        return $this->redis->exists($this->prefix . $key) > 0;
    }
    
    public function getMultiple(iterable $keys, mixed $default = null): iterable
    {
        $prefixedKeys = [];
        $keyMap = [];
        foreach ($keys as $key) {
            $prefixedKey = $this->prefix . $key;
            $prefixedKeys[] = $prefixedKey;
            $keyMap[$prefixedKey] = $key;
        }
        
        $values = $this->redis->mget($prefixedKeys);
        
        $result = [];
        foreach ($prefixedKeys as $i => $prefixedKey) {
            $originalKey = $keyMap[$prefixedKey];
            $result[$originalKey] = $values[$i] !== false 
                ? unserialize($values[$i]) 
                : $default;
        }
        
        return $result;
    }
    
    public function setMultiple(iterable $values, null|int|\DateInterval $ttl = null): bool
    {
        $pipe = $this->redis->multi(\Redis::PIPELINE);
        
        $ttlSeconds = null;
        if ($ttl !== null) {
            $ttlSeconds = $ttl instanceof \DateInterval
                ? (new \DateTime())->add($ttl)->getTimestamp() - time()
                : $ttl;
        }
        
        foreach ($values as $key => $value) {
            $serialized = serialize($value);
            if ($ttlSeconds !== null) {
                $pipe->setex($this->prefix . $key, $ttlSeconds, $serialized);
            } else {
                $pipe->set($this->prefix . $key, $serialized);
            }
        }
        
        $results = $pipe->exec();
        return !in_array(false, $results, true);
    }
    
    public function deleteMultiple(iterable $keys): bool
    {
        $prefixedKeys = [];
        foreach ($keys as $key) {
            $prefixedKeys[] = $this->prefix . $key;
        }
        
        return $this->redis->del($prefixedKeys) > 0;
    }
}
```

```php
/**
 * 文件缓存（改造后）
 * 
 * WeAsync 模式下自动切换为内存缓存
 */
class FileCache implements CacheInterface
{
    private CacheInterface $driver;
    
    public function __construct(string $path = 'var/cache')
    {
        $runtime = RuntimeFactory::detectType();
        
        if ($runtime === RuntimeType::WEASYNC) {
            // WeAsync 模式：使用内存缓存
            $this->driver = $this->createMemoryDriver();
            
            // 记录日志（开发调试用）
            if (getenv('WEASYNC_DEBUG')) {
                error_log("FileCache: 自动切换为内存缓存 (WeAsync 模式)");
            }
        } else {
            // FPM 模式：使用文件缓存
            $this->driver = new Driver\FileDriver($path);
        }
    }
    
    private function createMemoryDriver(): CacheInterface
    {
        // 优先使用 APCu（跨 Worker 共享）
        if (extension_loaded('apcu') && apcu_enabled()) {
            return new Driver\ApcuCache();
        }
        
        // 回退到进程内存
        return new Driver\ProcessMemoryCache();
    }
    
    // 代理方法
    public function get(string $key, mixed $default = null): mixed
    {
        return $this->driver->get($key, $default);
    }
    
    public function set(string $key, mixed $value, null|int|\DateInterval $ttl = null): bool
    {
        return $this->driver->set($key, $value, $ttl);
    }
    
    // ... 其他方法类似
}
```

## 6. 淘汰策略详解

### 6.1 支持的淘汰策略

| 策略 | 算法 | 适用场景 | 优缺点 |
|------|------|----------|--------|
| **LRU** | 最近最少使用 | 通用场景 | ✅ 热点数据保留好 / ❌ 需要维护访问顺序 |
| **LFU** | 最不经常使用 | 访问频率差异大 | ✅ 高频数据保留好 / ❌ 新数据容易被淘汰 |
| **TTL** | 最快过期优先 | TTL 差异大 | ✅ 优先保留长期缓存 / ❌ 可能淘汰热点数据 |
| **Random** | 随机淘汰 | 调试/基准测试 | ✅ 实现简单 / ❌ 可能淘汰热点数据 |

### 6.2 配置示例

### 6.4 GC 触发机制

```
┌─────────────────────────────────────────────────────────────────┐
│                        GC 触发流程                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  1. 概率触发（每次 get/set 操作时）                              │
│     └─ 随机数(1, gc_divisor) <= gc_probability 时触发 GC        │
│                                                                  │
│  2. 容量触发（set 操作时）                                       │
│     └─ 当 items >= max_items 或 memory >= max_memory 时         │
│        批量淘汰 eviction_batch 个项                              │
│                                                                  │
│  3. 手动触发                                                     │
│     └─ 调用 $cache->gc() 或 $cache->forceGc()                   │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 6.5 与当前框架的对比

| 特性 | 当前框架 | 改进后 |
|------|---------|--------|
| **过期机制** | 文件 mtime | TTL 字段 |
| **过期清理** | Cron（每天凌晨） | 概率 GC + 访问时检查 |
| **淘汰策略** | 无（只有 TTL） | LRU/LFU/TTL/Random |
| **容量限制** | 无 | max_items + max_memory |
| **WeAsync 支持** | 文件 I/O 阻塞 | 纯内存，无阻塞 |

## 7. 总结

| 运行时 | L1 缓存 | L2 缓存 | 淘汰策略 |
|--------|---------|---------|----------|
| **WeAsync** | ProcessMemory | APCu / Redis | LRU/LFU/TTL/Random |
| **FPM** | ProcessMemory | Redis / File | LRU/LFU/TTL/Random |

### 核心优势

1. **透明切换** - 业务代码无需修改，自动选择最优驱动
2. **多级缓存** - L1 进程内存 + L2 共享存储，命中率高
3. **避免阻塞** - WeAsync 模式下无文件 I/O
4. **可配置淘汰** - 支持 LRU/LFU/TTL/Random 多种策略
5. **防止溢出** - max_items + max_memory 双重限制
6. **概率 GC** - 自动清理过期项，不依赖 Cron
