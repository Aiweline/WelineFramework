## 0. 与 WeAsync 的关系

在 **WeAsync**（照搬 Workerman）模式下，Worker 进程是常驻的，PHP 的静态变量和超全局变量会跨请求保留，这是最常见的 bug 来源：

```
FPM 模式：
  请求1 → 进程1 → Counter::$count = 1 → 进程销毁
  请求2 → 进程2 → Counter::$count = 1 → 进程销毁（互不影响）

WeAsync 模式（同一个 Worker 进程）：
  请求1 → Counter::$count = 1
  请求2 → Counter::$count = 2  ← 污染！
  请求3 → Counter::$count = 3  ← 持续累加！
```

本文档规划如何在 WeAsync 模式下正确隔离和重置状态。

---

## 1. 概述

WeAsync 常驻内存模式下的主要挑战是**状态污染**：
- 静态变量在请求间保留
- 超全局变量需要手动重置
- 闭包捕获的变量可能泄漏

### 1.1 问题示例

```php
class Counter
{
    private static int $count = 0;
    
    public static function increment(): void
    {
        self::$count++; // WeAsync 模式下，值会跨请求累加！
    }
}
```

### 1.2 设计目标

| 目标 | 描述 |
|------|------|
| **显式状态** | 所有可变状态可追踪 |
| **自动重置** | 请求结束时自动清理 |
| **最小侵入** | 不强制修改所有代码 |

## 2. 静态变量管理

### 2.1 StaticStateRegistry

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Runtime;

/**
 * 静态状态注册表
 * 
 * 追踪所有需要在请求间重置的静态变量
 */
final class StaticStateRegistry
{
    /**
     * 注册的重置回调
     * 格式：['ClassName' => callable]
     */
    private static array $resetCallbacks = [];
    
    /**
     * 初始状态快照
     * 格式：['ClassName::$property' => value]
     */
    private static array $initialStates = [];
    
    /**
     * 注册类的静态状态重置回调
     */
    public static function register(string $class, callable $resetCallback): void
    {
        self::$resetCallbacks[$class] = $resetCallback;
    }
    
    /**
     * 注册静态属性的初始值
     */
    public static function registerProperty(
        string $class, 
        string $property, 
        mixed $initialValue
    ): void {
        $key = "{$class}::\${$property}";
        self::$initialStates[$key] = $initialValue;
    }
    
    /**
     * 重置所有注册的静态状态
     */
    public static function resetAll(): void
    {
        // 1. 执行注册的重置回调
        foreach (self::$resetCallbacks as $class => $callback) {
            try {
                $callback();
            } catch (\Throwable $e) {
                // 记录错误但继续重置其他类
                error_log("Failed to reset {$class}: {$e->getMessage()}");
            }
        }
        
        // 2. 重置注册的静态属性
        foreach (self::$initialStates as $key => $initialValue) {
            [$class, $property] = explode('::$', $key);
            
            try {
                $reflection = new \ReflectionProperty($class, $property);
                $reflection->setValue(null, $initialValue);
            } catch (\ReflectionException $e) {
                error_log("Failed to reset {$key}: {$e->getMessage()}");
            }
        }
    }
    
    /**
     * 获取注册的类列表
     */
    public static function getRegisteredClasses(): array
    {
        return array_keys(self::$resetCallbacks);
    }
}
```

### 2.2 Resettable 属性注解

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Runtime\Attribute;

use Attribute;

/**
 * 标记需要在请求间重置的静态属性
 */
#[Attribute(Attribute::TARGET_PROPERTY)]
final class Resettable
{
    public function __construct(
        public readonly mixed $initialValue = null,
    ) {}
}
```

### 2.3 使用示例

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Cache;

use Weline\Framework\Runtime\Attribute\Resettable;
use Weline\Framework\Runtime\ResettableInterface;

class CacheManager implements ResettableInterface
{
    #[Resettable(initialValue: [])]
    private static array $localCache = [];
    
    #[Resettable(initialValue: null)]
    private static ?self $instance = null;
    
    public static function get(string $key): mixed
    {
        return self::$localCache[$key] ?? null;
    }
    
    public static function set(string $key, mixed $value): void
    {
        self::$localCache[$key] = $value;
    }
    
    /**
     * 实现 ResettableInterface
     */
    public function reset(): void
    {
        self::$localCache = [];
        self::$instance = null;
    }
    
    public function getStateInfo(): array
    {
        return [
            'cache_count' => count(self::$localCache),
            'cache_keys' => array_keys(self::$localCache),
        ];
    }
}
```

## 3. 超全局变量隔离

### 3.1 SuperGlobalsManager

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Runtime;

use Psr\Http\Message\ServerRequestInterface;

/**
 * 超全局变量管理器
 */
final class SuperGlobalsManager implements ResettableInterface
{
    /**
     * 原始 $_SERVER 备份
     */
    private static array $originalServer = [];
    
    /**
     * 是否已初始化
     */
    private static bool $initialized = false;
    
    /**
     * 初始化（应用启动时调用一次）
     */
    public static function initialize(): void
    {
        if (self::$initialized) {
            return;
        }
        
        // 备份原始 $_SERVER（包含 PHP 进程级信息）
        self::$originalServer = $_SERVER;
        self::$initialized = true;
    }
    
    /**
     * 从 PSR-7 请求注入超全局变量
     */
    public static function injectFromPsr7(ServerRequestInterface $request): void
    {
        // 保留进程级 SERVER 变量，合并请求级变量
        $_SERVER = array_merge(
            self::getProcessServerVars(),
            $request->getServerParams()
        );
        
        $_GET = $request->getQueryParams();
        $_POST = (array) $request->getParsedBody();
        $_COOKIE = $request->getCookieParams();
        $_FILES = self::convertUploadedFiles($request->getUploadedFiles());
        $_REQUEST = array_merge($_GET, $_POST, $_COOKIE);
    }
    
    /**
     * 重置超全局变量
     */
    public static function reset(): void
    {
        $_GET = [];
        $_POST = [];
        $_COOKIE = [];
        $_FILES = [];
        $_REQUEST = [];
        
        // 恢复进程级 SERVER 变量
        $_SERVER = self::getProcessServerVars();
    }
    
    /**
     * 获取进程级 SERVER 变量
     */
    private static function getProcessServerVars(): array
    {
        return [
            'PHP_SELF' => self::$originalServer['PHP_SELF'] ?? '',
            'argv' => self::$originalServer['argv'] ?? [],
            'argc' => self::$originalServer['argc'] ?? 0,
            'REQUEST_TIME' => time(),
            'REQUEST_TIME_FLOAT' => microtime(true),
        ];
    }
    
    /**
     * 转换 PSR-7 上传文件格式
     */
    private static function convertUploadedFiles(array $uploadedFiles): array
    {
        $files = [];
        
        foreach ($uploadedFiles as $key => $file) {
            if (is_array($file)) {
                $files[$key] = self::convertUploadedFiles($file);
            } else {
                $files[$key] = [
                    'name' => $file->getClientFilename(),
                    'type' => $file->getClientMediaType(),
                    'tmp_name' => $file->getStream()->getMetadata('uri'),
                    'error' => $file->getError(),
                    'size' => $file->getSize(),
                ];
            }
        }
        
        return $files;
    }
    
    public function getStateInfo(): array
    {
        return [
            'get_count' => count($_GET),
            'post_count' => count($_POST),
            'cookie_count' => count($_COOKIE),
            'files_count' => count($_FILES),
        ];
    }
}
```

## 4. 状态管理器实现

### 4.1 StateManager

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Runtime;

/**
 * 状态管理器
 * 
 * 负责协调所有状态的重置
 */
final class StateManager implements StateManagerInterface
{
    /**
     * 注册的可重置对象
     */
    private array $resettables = [];
    
    /**
     * 注册可重置的状态持有者
     */
    public function register(ResettableInterface $resettable): void
    {
        $class = $resettable::class;
        $this->resettables[$class] = $resettable;
    }
    
    /**
     * 重置所有请求级状态
     */
    public function resetRequestState(): void
    {
        // 1. 重置超全局变量
        SuperGlobalsManager::reset();
        
        // 2. 重置注册的可重置对象
        foreach ($this->resettables as $resettable) {
            try {
                $resettable->reset();
            } catch (\Throwable $e) {
                error_log("Failed to reset " . $resettable::class . ": {$e->getMessage()}");
            }
        }
        
        // 3. 重置静态状态注册表
        StaticStateRegistry::resetAll();
        
        // 4. 销毁请求上下文
        RequestContext::destroy();
    }
    
    /**
     * 重置所有状态（包括应用级）
     */
    public function resetAll(): void
    {
        $this->resetRequestState();
        
        // 额外清理应用级状态
        // 通常在进程退出或测试时使用
    }
    
    /**
     * 获取状态快照（用于调试）
     */
    public function getStateSnapshot(): array
    {
        $snapshot = [
            'registered_resettables' => array_keys($this->resettables),
            'static_classes' => StaticStateRegistry::getRegisteredClasses(),
            'resettable_states' => [],
        ];
        
        foreach ($this->resettables as $class => $resettable) {
            $snapshot['resettable_states'][$class] = $resettable->getStateInfo();
        }
        
        return $snapshot;
    }
}
```

## 5. 自动发现与注册

### 5.1 ResettableScanner

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Runtime;

use ReflectionClass;
use ReflectionProperty;
use Weline\Framework\Runtime\Attribute\Resettable;

/**
 * 可重置属性扫描器
 * 
 * 扫描框架类中标记了 #[Resettable] 的静态属性
 */
final class ResettableScanner
{
    /**
     * 扫描指定目录下的类
     */
    public static function scan(string $directory): void
    {
        $files = new \RecursiveIteratorIterator(
            new \RecursiveDirectoryIterator($directory)
        );
        
        foreach ($files as $file) {
            if ($file->isFile() && $file->getExtension() === 'php') {
                self::processFile($file->getPathname());
            }
        }
    }
    
    /**
     * 处理单个文件
     */
    private static function processFile(string $filepath): void
    {
        $content = file_get_contents($filepath);
        
        // 提取命名空间和类名
        if (!preg_match('/namespace\s+([^;]+);/', $content, $nsMatch)) {
            return;
        }
        
        if (!preg_match('/class\s+(\w+)/', $content, $classMatch)) {
            return;
        }
        
        $className = $nsMatch[1] . '\\' . $classMatch[1];
        
        if (!class_exists($className, true)) {
            return;
        }
        
        self::processClass($className);
    }
    
    /**
     * 处理单个类
     */
    private static function processClass(string $className): void
    {
        try {
            $reflection = new ReflectionClass($className);
            
            foreach ($reflection->getProperties(ReflectionProperty::IS_STATIC) as $property) {
                $attributes = $property->getAttributes(Resettable::class);
                
                if (empty($attributes)) {
                    continue;
                }
                
                /** @var Resettable $resettable */
                $resettable = $attributes[0]->newInstance();
                
                StaticStateRegistry::registerProperty(
                    $className,
                    $property->getName(),
                    $resettable->initialValue
                );
            }
        } catch (\ReflectionException) {
            // 忽略反射错误
        }
    }
}
```

## 6. 内存泄漏检测

### 6.1 MemoryLeakDetector

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Runtime\Debug;

/**
 * 内存泄漏检测器
 * 
 * 用于开发环境检测潜在的内存泄漏
 */
final class MemoryLeakDetector
{
    private static int $requestCount = 0;
    private static array $memorySnapshots = [];
    
    /**
     * 请求开始时记录
     */
    public static function onRequestStart(): void
    {
        self::$requestCount++;
        
        if (self::$requestCount % 100 === 0) {
            self::$memorySnapshots[] = [
                'request' => self::$requestCount,
                'memory' => memory_get_usage(true),
                'peak' => memory_get_peak_usage(true),
                'time' => microtime(true),
            ];
            
            // 检测内存增长趋势
            self::analyzeGrowth();
        }
    }
    
    /**
     * 分析内存增长趋势
     */
    private static function analyzeGrowth(): void
    {
        if (count(self::$memorySnapshots) < 3) {
            return;
        }
        
        $recent = array_slice(self::$memorySnapshots, -3);
        $growth = $recent[2]['memory'] - $recent[0]['memory'];
        $avgGrowthPerRequest = $growth / 200; // 200个请求
        
        // 如果平均每个请求增长超过 1KB，发出警告
        if ($avgGrowthPerRequest > 1024) {
            error_log(sprintf(
                'Memory leak warning: %.2f KB growth per request (total: %d requests, memory: %.2f MB)',
                $avgGrowthPerRequest / 1024,
                self::$requestCount,
                $recent[2]['memory'] / 1024 / 1024
            ));
        }
    }
    
    /**
     * 获取内存报告
     */
    public static function getReport(): array
    {
        return [
            'request_count' => self::$requestCount,
            'current_memory' => memory_get_usage(true),
            'peak_memory' => memory_get_peak_usage(true),
            'snapshots' => self::$memorySnapshots,
        ];
    }
}
```

## 7. 最佳实践

### 7.1 避免静态变量

```php
// ❌ 不推荐
class Service
{
    private static array $cache = [];
}

// ✅ 推荐：使用 RequestContext 或 DI
class Service
{
    public function __construct(
        private CacheInterface $cache,
    ) {}
}
```

### 7.2 使用 Resettable 属性

```php
// ✅ 如果必须使用静态变量，标记为可重置
class Registry
{
    #[Resettable(initialValue: [])]
    private static array $data = [];
}
```

### 7.3 实现 ResettableInterface

```php
// ✅ 类实现 ResettableInterface
class MyService implements ResettableInterface
{
    private array $cache = [];
    
    public function reset(): void
    {
        $this->cache = [];
    }
    
    public function getStateInfo(): array
    {
        return ['cache_size' => count($this->cache)];
    }
}
```
