# 16 - 定时任务与常驻内存兼容

> **优先级**: ⭐⭐⭐⭐  
> **依赖**: 01-架构设计, 10-Worker进程管理  
> **预计工作量**: 3-5 天  
> **版本**: v3.0.0

---

## 0. 核心决策

### 0.1 设计目标

在 v3 常驻内存版本中，定时任务系统需要实现**双模式无缝兼容**：

| 模式 | 触发方式 | 执行环境 | 状态管理 |
|------|---------|---------|---------|
| **FPM 模式** | 系统 Crontab → 命令行 | 独立进程，每次完整初始化 | 数据库记录状态 |
| **WeAsync 模式** | 内置定时器 | 常驻进程内执行 | 内存 + 数据库双重管理 |

### 0.2 兼容原则

1. **接口不变** - `CronTaskInterface` 保持不变，现有任务无需修改
2. **命令行保留** - `php bin/w cron:run` 继续可用，支持手动触发
3. **自动切换** - 根据运行时自动选择触发机制
4. **状态同步** - 两种模式的任务状态保持一致

### 0.3 架构概览

```

│                        定时任务系统 v3                            │
```

## 1. 现状分析

### 1.1 当前实现（v2.x）

```php
// 定时任务接口
interface CronTaskInterface
{
    function name(): string;                    // 任务名
    function execute_name(): string;            // 执行名
    function tip(): string;                     // 描述
    function cron_time(): string;               // Cron 表达式
    function execute(): string;                 // 执行逻辑
    function unlock_timeout(int $minute = 30): int;  // 超时解锁
}
```

#### 执行流程

```
1. 系统 Crontab 触发
   */1 * * * * php /path/to/bin/w cron:run

2. 命令行执行
   php bin/w cron:task:run [task_name] [-f] [-p]

3. 进程管理
   - 通过进程名检测是否已运行
   - 创建独立进程执行任务
   - 数据库记录 PID、状态、运行时间
   
4. 任务执行
   - 完整框架初始化
   - 实例化任务类
   - 执行 execute() 方法
   - 保存结果到数据库
   
5. 清理
   - 进程退出
   - 释放所有资源
```

### 1.2 存在的问题

| 问题 | 影响 | v3 需要解决 |
|------|------|------------|
| **冷启动开销** | 每次执行都要完整初始化框架 | ✅ 常驻内存消除冷启动 |
| **进程创建开销** | fork 新进程需要时间和内存 | ✅ 进程内执行，无 fork |
| **资源重复加载** | 配置、类、连接重复创建 | ✅ 复用已加载资源 |
| **状态管理复杂** | 依赖数据库 + 进程管理 | ⚠️ 需要双重状态管理 |
| **并发控制有限** | 只能通过进程锁控制 | ✅ 支持协程并发 |

## 2. 架构设计

### 2.1 调度器抽象层

```php
<?php
declare(strict_types=1);

namespace Weline\Cron\Scheduler;

use Weline\Cron\CronTaskInterface;

/**
 * 定时任务调度器接口
 * 
 * 抽象不同运行时的调度逻辑
 */
interface CronSchedulerInterface
{
    /**
     * 注册定时任务
     * 
     * @param CronTaskInterface $task 任务实例
     * @return void
     */
    public function register(CronTaskInterface $task): void;
    
    /**
     * 启动调度器
     * 
     * - FPM 模式：扫描数据库，执行到期任务
     * - WeAsync 模式：注册所有定时器
     * 
     * @return void
     */
    public function start(): void;
    
    /**
     * 停止调度器
     * 
     * @return void
     */
    public function stop(): void;
    
    /**
     * 手动执行任务
     * 
     * @param string $taskName 任务执行名
     * @param bool $force 是否强制执行（忽略锁）
     * @return void
     */
    public function runTask(string $taskName, bool $force = false): void;
    
    /**
     * 获取所有注册的任务
     * 
     * @return array<string, CronTaskInterface>
     */
    public function getTasks(): array;
    
    /**
     * 获取调度器类型
     * 
     * @return string 'fpm' | 'weasync'
     */
    public function getType(): string;
}
```

### 2.2 FPM 调度器（保持现状）

```php
<?php
declare(strict_types=1);

namespace Weline\Cron\Scheduler;

use Weline\Cron\CronTaskInterface;
use Weline\Cron\Model\CronTask;
use Weline\Cron\Helper\Process;
use Weline\Cron\Helper\CronStatus;
use Cron\CronExpression;

/**
 * FPM 模式调度器
 * 
 * 保持现有逻辑不变，通过系统 Crontab + 命令行触发
 */
class FpmScheduler implements CronSchedulerInterface
{
    /**
     * 注册的任务（从数据库加载）
     */
    private array $tasks = [];
    
    public function __construct(
        private CronTask $cronTaskModel,
    ) {}
    
    /**
     * 注册任务（FPM 模式下不需要显式注册，从数据库读取）
     */
    public function register(CronTaskInterface $task): void
    {
        $this->tasks[$task->execute_name()] = $task;
    }
    
    /**
     * 启动调度器（扫描数据库并执行到期任务）
     * 
     * 这是现有的 cron:task:run 逻辑
     */
    public function start(): void
    {
        // 从数据库加载所有任务
        $tasks = $this->cronTaskModel->select()->fetch()->getItems();
        
        foreach ($tasks as $taskModel) {
            $executeName = $taskModel->getData('execute_name');
            $cronTime = $taskModel->getData('cron_time');
            
            // 检查是否到期
            $cron = new CronExpression($cronTime);
            if (!$cron->isDue()) {
                continue;
            }
            
            // 检查是否已在运行（进程检测）
            $processName = PHP_BINARY . ' ' . BP . 'bin/w cron:task:run -process ' . $executeName;
            $pid = Process::getPidByName($processName);
            
            if ($pid) {
                // 任务已在运行
                $taskModel->setData('status', CronStatus::RUNNING->value)
                    ->setData('pid', $pid)
                    ->save();
                continue;
            }
            
            // 创建新进程执行任务
            $pid = Process::create($processName);
            
            if ($pid) {
                $taskModel->setData('status', CronStatus::BLOCK->value)
                    ->setData('pid', $pid)
                    ->setData('run_time', microtime(true))
                    ->setData('run_date', date('Y-m-d H:i:s'))
                    ->save();
            }
        }
    }
    
    /**
     * 手动执行任务
     */
    public function runTask(string $taskName, bool $force = false): void
    {
        // 现有的 cron:task:run 逻辑
        $task = $this->cronTaskModel->where('execute_name', $taskName)->find()->fetch();
        
        if (!$task->getId()) {
            throw new \RuntimeException("Task not found: {$taskName}");
        }
        
        $class = $task->getData('class');
        $instance = ObjectManager::getInstance($class);
        
        // 执行任务
        $result = $instance->execute();
        
        // 保存结果
        $task->setData('run_times', $task->getData('run_times') + 1)
            ->setData('status', CronStatus::SUCCESS->value)
            ->setData('runtime', microtime(true) - $task->getData('run_time'))
            ->save();
    }
    
    public function stop(): void
    {
        // FPM 模式无需停止
    }
    
    public function getTasks(): array
    {
        return $this->tasks;
    }
    
    public function getType(): string
    {
        return 'fpm';
    }
}
```

### 2.3 WeAsync 调度器（新增）

```php
<?php
declare(strict_types=1);

namespace Weline\Cron\Scheduler;

use Weline\Cron\CronTaskInterface;
use Weline\Cron\Model\CronTask;
use Weline\Cron\Helper\CronStatus;
use Weline\WeAsync\Timer;
use Weline\Framework\Runtime\RequestContext;
use Weline\Framework\Manager\ObjectManager;
use Cron\CronExpression;

/**
 * WeAsync 模式调度器
 * 
 * 使用内置定时器在常驻进程内执行任务
 */
class WeAsyncScheduler implements CronSchedulerInterface
{
    /**
     * 注册的任务
     * @var array<string, CronTaskInterface>
     */
    private array $tasks = [];
    
    /**
     * 定时器 ID 映射
     * @var array<string, int>
     */
    private array $timerIds = [];
    
    /**
     * 正在执行的任务
     * @var array<string, bool>
     */
    private array $runningTasks = [];
    
    public function __construct(
        private CronTask $cronTaskModel,
        private ObjectManager $objectManager,
    ) {}
    
    /**
     * 注册任务
     */
    public function register(CronTaskInterface $task): void
    {
        $executeName = $task->execute_name();
        $this->tasks[$executeName] = $task;
        
        // 同步到数据库（确保数据库有记录）
        $this->syncTaskToDatabase($task);
    }
    
    /**
     * 启动调度器（注册所有定时器）
     */
    public function start(): void
    {
        // 从数据库加载所有任务
        $tasks = $this->cronTaskModel->select()->fetch()->getItems();
        
        foreach ($tasks as $taskModel) {
            $class = $taskModel->getData('class');
            $executeName = $taskModel->getData('execute_name');
            
            // 实例化任务
            try {
                $taskInstance = $this->objectManager->get($class);
                $this->tasks[$executeName] = $taskInstance;
            } catch (\Throwable $e) {
                echo "Failed to load task {$executeName}: {$e->getMessage()}\n";
                continue;
            }
            
            // 注册定时器
            $this->registerTimer($taskInstance, $taskModel);
        }
        
        echo "WeAsync Scheduler started with " . count($this->tasks) . " tasks\n";
    }
    
    /**
     * 注册定时器
     */
    private function registerTimer(CronTaskInterface $task, CronTask $taskModel): void
    {
        $executeName = $task->execute_name();
        $cronTime = $task->cron_time();
        
        // 计算下次执行时间
        $cron = new CronExpression($cronTime);
        $nextRun = $cron->getNextRunDate();
        $delay = $nextRun->getTimestamp() - time();
        
        // 注册一次性定时器
        $timerId = Timer::add($delay, function() use ($task, $taskModel, $executeName) {
            $this->executeTask($task, $taskModel);
            
            // 重新注册下次执行
            $this->registerTimer($task, $taskModel);
        }, [], false);
        
        $this->timerIds[$executeName] = $timerId;
        
        echo "Registered timer for {$executeName}, next run: {$nextRun->format('Y-m-d H:i:s')}\n";
    }
    
    /**
     * 执行任务（在常驻进程内）
     */
    private function executeTask(CronTaskInterface $task, CronTask $taskModel): void
    {
        $executeName = $task->execute_name();
        
        // 检查是否已在执行
        if (isset($this->runningTasks[$executeName])) {
            echo "Task {$executeName} is already running, skip\n";
            
            // 更新数据库状态
            $taskModel->setData('status', CronStatus::RUNNING->value)
                ->setData('runtime_error', 'Previous execution still running')
                ->save();
            
            return;
        }
        
        // 标记为运行中
        $this->runningTasks[$executeName] = true;
        
        // 创建独立的请求上下文
        $context = RequestContext::create();
        
        try {
            echo "[" . date('Y-m-d H:i:s') . "] Executing task: {$executeName}\n";
            
            $startTime = microtime(true);
            
            // 更新数据库：开始执行
            $taskModel->setData('status', CronStatus::BLOCK->value)
                ->setData('run_time', $startTime)
                ->setData('run_date', date('Y-m-d H:i:s'))
                ->setData('runtime_error', '')
                ->save();
            
            // 执行任务
            $result = $task->execute();
            
            $runtime = microtime(true) - $startTime;
            
            // 更新数据库：执行成功
            $cron = new CronExpression($task->cron_time());
            $taskModel->setData('status', CronStatus::SUCCESS->value)
                ->setData('runtime', $runtime)
                ->setData('run_times', $taskModel->getData('run_times') + 1)
                ->setData('next_run_date', $cron->getNextRunDate()->format('Y-m-d H:i:s'))
                ->setData('runtime_error', $result)
                ->save();
            
            echo "Task {$executeName} completed in {$runtime}s\n";
            
        } catch (\Throwable $e) {
            $runtime = microtime(true) - $startTime;
            
            // 记录错误
            $errorMsg = sprintf(
                "Error: %s\nFile: %s:%d\nTrace: %s",
                $e->getMessage(),
                $e->getFile(),
                $e->getLine(),
                $e->getTraceAsString()
            );
            
            $taskModel->setData('status', CronStatus::FAIL->value)
                ->setData('runtime', $runtime)
                ->setData('runtime_error', $errorMsg)
                ->setData('runtime_error_date', date('Y-m-d H:i:s'))
                ->save();
            
            echo "Task {$executeName} failed: {$e->getMessage()}\n";
            
        } finally {
            // 清理运行标记
            unset($this->runningTasks[$executeName]);
            
            // 清理请求上下文（重置所有请求级状态）
            RequestContext::destroy();
            
            // 重置 ObjectManager 的请求级实例
            $this->objectManager->resetRequestScope();
        }
    }
    
    /**
     * 手动执行任务
     */
    public function runTask(string $taskName, bool $force = false): void
    {
        if (!isset($this->tasks[$taskName])) {
            throw new \RuntimeException("Task not found: {$taskName}");
        }
        
        $task = $this->tasks[$taskName];
        $taskModel = $this->cronTaskModel->where('execute_name', $taskName)->find()->fetch();
        
        if (!$force && isset($this->runningTasks[$taskName])) {
            throw new \RuntimeException("Task {$taskName} is already running");
        }
        
        // 立即执行
        $this->executeTask($task, $taskModel);
    }
    
    /**
     * 停止调度器
     */
    public function stop(): void
    {
        foreach ($this->timerIds as $executeName => $timerId) {
            Timer::del($timerId);
            echo "Stopped timer for {$executeName}\n";
        }
        
        $this->timerIds = [];
    }
    
    /**
     * 同步任务到数据库
     */
    private function syncTaskToDatabase(CronTaskInterface $task): void
    {
        $executeName = $task->execute_name();
        
        // 检查是否已存在
        $existing = $this->cronTaskModel->where('execute_name', $executeName)->find()->fetch();
        
        if (!$existing->getId()) {
            // 创建新记录
            $cron = new CronExpression($task->cron_time());
            
            $this->cronTaskModel->setData([
                'name' => $task->name(),
                'execute_name' => $executeName,
                'class' => get_class($task),
                'tip' => $task->tip(),
                'cron_time' => $task->cron_time(),
                'status' => CronStatus::PENDING->value,
                'next_run_date' => $cron->getNextRunDate()->format('Y-m-d H:i:s'),
                'block_unlock_timeout' => $task->unlock_timeout(),
            ])->save();
        }
    }
    
    public function getTasks(): array
    {
        return $this->tasks;
    }
    
    public function getType(): string
    {
        return 'weasync';
    }
}
```

## 3. 运行时集成

### 3.1 自动选择调度器

```php
<?php
declare(strict_types=1);

namespace Weline\Cron\Scheduler;

use Weline\Framework\Runtime\RuntimeType;
use Weline\Framework\Runtime\RuntimeInterface;

/**
 * 调度器工厂
 * 
 * 根据运行时自动选择调度器
 */
class CronSchedulerFactory
{
    public static function create(RuntimeInterface $runtime): CronSchedulerInterface
    {
        return match($runtime->getType()) {
            RuntimeType::WEASYNC => ObjectManager::getInstance(WeAsyncScheduler::class),
            default => ObjectManager::getInstance(FpmScheduler::class),
        };
    }
}
```

### 3.2 WeAsync 启动时集成

```php
<?php
// 在 WeAsync Worker 启动时初始化调度器

use Weline\WeAsync\Worker;
use Weline\Cron\Scheduler\CronSchedulerFactory;

$worker = new Worker('http://0.0.0.0:8080');
$worker->count = 4;

$worker->onWorkerStart = function ($worker) {
    // 初始化框架
    $runtime = RuntimeFactory::create();
    $runtime->bootstrap();
    
    // 初始化定时任务调度器（仅在 Worker 0 中运行）
    if ($worker->id === 0) {
        $scheduler = CronSchedulerFactory::create($runtime);
        $scheduler->start();
        
        // 存储到 Worker 以便后续管理
        $worker->cronScheduler = $scheduler;
    }
    
    $worker->runtime = $runtime;
};

$worker->onWorkerStop = function ($worker) {
    // 停止调度器
    if (isset($worker->cronScheduler)) {
        $worker->cronScheduler->stop();
    }
};

Worker::runAll();
```

### 4.2 WeAsync 模式下的命令行行为

```php
<?php
declare(strict_types=1);

namespace Weline\Cron\Console\Cron\Task;

use Weline\Framework\Runtime\RuntimeType;
use Weline\Framework\App\App;

class Run implements CommandInterface
{
    public function execute(array $args = [], array $data = [])
    {
        $runtime = App::getInstance()->getRuntime();
        
        // 检查运行时类型
        if ($runtime->getType() === RuntimeType::WEASYNC) {
            // WeAsync 模式：通过 RPC 调用 Worker 内的调度器
            $this->executeInWeAsync($args);
        } else {
            // FPM 模式：保持原有逻辑
            $this->executeInFpm($args);
        }
    }
    
    /**
     * FPM 模式执行（现有逻辑）
     */
    private function executeInFpm(array $args): void
    {
        // 现有的进程创建、状态管理逻辑
        // ... 保持不变 ...
    }
    
    /**
     * WeAsync 模式执行（发送请求给 Worker）
     */
    private function executeInWeAsync(array $args): void
    {
        $taskName = $args[0] ?? '';
        $force = in_array('-f', $args) || in_array('--force', $args);
        
        if (!$taskName) {
            throw new \InvalidArgumentException('Task name required');
        }
        
        // 通过 HTTP 请求触发 Worker 内的任务
        $response = $this->sendRequest('http://127.0.0.1:8080/__cron/run', [
            'task' => $taskName,
            'force' => $force,
        ]);
        
        if ($response['status'] === 'success') {
            $this->printing->success("Task {$taskName} triggered successfully");
        } else {
            $this->printing->error("Failed to trigger task: {$response['error']}");
        }
    }
    
    private function sendRequest(string $url, array $data): array
    {
        $ch = curl_init($url);
        curl_setopt($ch, CURLOPT_POST, true);
        curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
        curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
        curl_setopt($ch, CURLOPT_HTTPHEADER, ['Content-Type: application/json']);
        
        $response = curl_exec($ch);
        curl_close($ch);
        
        return json_decode($response, true) ?? ['status' => 'error', 'error' => 'Invalid response'];
    }
}
```

### 4.3 内部 API 端点（WeAsync 模式）

```php
<?php
declare(strict_types=1);

namespace Weline\Cron\Controller\Internal;

/**
 * 定时任务内部 API
 * 
 * 仅在 WeAsync 模式下使用，供命令行调用
 */
class CronApi extends \Weline\Framework\App\Controller\Controller
{
    public function __construct(
        private \Weline\Cron\Scheduler\CronSchedulerInterface $scheduler,
    ) {}
    
    /**
     * 触发任务执行
     * 
     * POST /__cron/run
     * Body: {"task": "task_name", "force": false}
     */
    public function run()
    {
        // 验证请求来源（仅允许本地）
        if ($_SERVER['REMOTE_ADDR'] !== '127.0.0.1') {
            return $this->json(['status' => 'error', 'error' => 'Access denied'], 403);
        }
        
        $data = json_decode(file_get_contents('php://input'), true);
        $taskName = $data['task'] ?? '';
        $force = $data['force'] ?? false;
        
        if (!$taskName) {
            return $this->json(['status' => 'error', 'error' => 'Task name required'], 400);
        }
        
        try {
            $this->scheduler->runTask($taskName, $force);
            return $this->json(['status' => 'success', 'message' => "Task {$taskName} executed"]);
        } catch (\Throwable $e) {
            return $this->json(['status' => 'error', 'error' => $e->getMessage()], 500);
        }
    }
    
    /**
     * 获取任务列表
     * 
     * GET /__cron/tasks
     */
    public function tasks()
    {
        if ($_SERVER['REMOTE_ADDR'] !== '127.0.0.1') {
            return $this->json(['status' => 'error', 'error' => 'Access denied'], 403);
        }
        
        $tasks = $this->scheduler->getTasks();
        
        $taskList = array_map(function($task) {
            return [
                'name' => $task->name(),
                'execute_name' => $task->execute_name(),
                'cron_time' => $task->cron_time(),
                'tip' => $task->tip(),
            ];
        }, $tasks);
        
        return $this->json(['status' => 'success', 'tasks' => $taskList]);
    }
    
    private function json(array $data, int $status = 200): string
    {
        http_response_code($status);
        header('Content-Type: application/json');
        return json_encode($data);
    }
}
```

## 5. 状态管理

### 5.1 双重状态管理

| 状态类型 | FPM 模式 | WeAsync 模式 |
|---------|---------|-------------|
| **运行状态** | 数据库 + 进程检测 | 数据库 + 内存标记 |
| **执行历史** | 数据库 | 数据库 |
| **锁机制** | 进程 PID | 内存布尔值 + 数据库 |
| **超时检测** | 进程存活检测 | 执行时间记录 |

### 5.2 状态同步策略

```php
<?php
// WeAsync 模式下的状态管理

class WeAsyncScheduler
{
    /**
     * 执行任务前：检查并更新状态
     */
    private function checkAndLockTask(CronTask $taskModel): bool
    {
        $executeName = $taskModel->getData('execute_name');
        
        // 1. 检查内存标记（快速检查）
        if (isset($this->runningTasks[$executeName])) {
            return false;
        }
        
        // 2. 检查数据库状态（多 Worker 同步）
        $taskModel->reload();  // 重新加载最新数据
        
        if ($taskModel->getData('status') === CronStatus::RUNNING->value) {
            // 检查超时
            $runTime = $taskModel->getData('run_time');
            $timeout = $taskModel->getData('block_unlock_timeout') * 60;
            
            if (microtime(true) - $runTime > $timeout) {
                // 超时，自动解锁
                echo "Task {$executeName} timeout, auto unlock\n";
            } else {
                return false;
            }
        }
        
        // 3. 上锁
        $this->runningTasks[$executeName] = true;
        
        return true;
    }
    
    /**
     * 执行任务后：释放锁并更新状态
     */
    private function unlockTask(string $executeName): void
    {
        unset($this->runningTasks[$executeName]);
    }
}
```

#### ⚠️ 长时间任务使用协程（可选）

```php
<?php
// 对于耗时较长的任务，可以使用 Fiber 避免阻塞事件循环

class LongRunningTask implements CronTaskInterface
{
    public function execute(): string
    {
        // ✅ 在 WeAsync 模式下使用 Fiber
        if (Runtime::getType() === RuntimeType::WEASYNC) {
            return $this->executeAsync();
        }
        
        return $this->executeSync();
    }
    
    private function executeAsync(): string
    {
        $fiber = new Fiber(function() {
            // 耗时操作
            for ($i = 0; $i < 1000; $i++) {
                $this->processItem($i);
                
                // 定期让出 CPU
                if ($i % 100 === 0) {
                    Fiber::suspend();
                }
            }
            
            return "Processed 1000 items";
        });
        
        $fiber->start();
        
        // 可以通过定时器恢复 Fiber
        Timer::add(0.1, function() use ($fiber) {
            if (!$fiber->isTerminated()) {
                $fiber->resume();
            }
        });
        
        return "Task started";
    }
    
    private function executeSync(): string
    {
        for ($i = 0; $i < 1000; $i++) {
            $this->processItem($i);
        }
        return "Processed 1000 items";
    }
}
```

## 10. 最佳实践

### 10.1 任务设计原则

1. **幂等性** - 任务应该是幂等的，重复执行不会产生副作用
2. **原子性** - 使用事务确保数据一致性
3. **容错性** - 捕获异常，避免任务失败影响其他任务
4. **日志完整** - 记录关键操作，便于排查问题
5. **资源清理** - 及时释放临时资源

### 10.2 任务拆分

对于复杂任务，建议拆分为多个小任务：

### 10.3 错误处理

```php
<?php
class RobustTask implements CronTaskInterface
{
    public function execute(): string
    {
        $errors = [];
        $processed = 0;
        
        try {
            // 分批处理，避免一次失败全部失败
            $items = $this->getItems();
            
            foreach ($items as $item) {
                try {
                    $this->processItem($item);
                    $processed++;
                } catch (\Throwable $e) {
                    // 记录单个项目的错误，继续处理其他项目
                    $errors[] = "Item {$item->id}: {$e->getMessage()}";
                }
            }
            
            $summary = "Processed: {$processed}/" . count($items);
            
            if ($errors) {
                $summary .= "\nErrors:\n" . implode("\n", $errors);
            }
            
            return $summary;
            
        } catch (\Throwable $e) {
            // 致命错误
            return "Fatal error: {$e->getMessage()}\nProcessed: {$processed}";
        }
    }
}
```

## 11. 常见问题

### Q1: WeAsync 模式下任务执行失败会影响 HTTP 请求吗？

**A**: 不会。定时任务在独立的执行上下文中运行，异常不会传播到 HTTP 请求处理流程。

### Q2: 如何保证任务不会并发执行？

**A**: 
- FPM 模式：通过进程名检测和数据库锁
- WeAsync 模式：通过内存标记 + 数据库状态双重检测

### Q3: WeAsync 模式下任务执行时间过长会阻塞 HTTP 请求吗？

**A**: 
- 轻量级任务（< 1s）：不会阻塞，事件循环可以在任务执行间隙处理请求
- 重量级任务（> 1s）：可能阻塞，建议使用 Fiber 协程或拆分为小任务

### Q4: 如何在 WeAsync 模式下手动触发任务？

**A**: 使用命令行：

```bash
php bin/w cron:task:run task_name

# 内部会通过 HTTP 请求调用 Worker 内的 API
```

### Q5: 如何查看任务执行历史？

**A**: 所有任务执行记录都存储在 `cron_task` 表中，包括：
- 执行时间
- 执行结果
- 错误信息
- 运行时长

```php
// 获取未来第 N 次执行时间
$thirdRun = $cron->getNextRunDate('now', 3);
```

**常用 Cron 表达式**：

| 表达式 | 说明 |
|--------|------|
| `* * * * *` | 每分钟 |
| `0 * * * *` | 每小时 |
| `0 0 * * *` | 每天凌晨 |
| `0 0 * * 0` | 每周日凌晨 |
| `0 0 1 * *` | 每月1号凌晨 |
| `*/5 * * * *` | 每5分钟 |
| `0 */2 * * *` | 每2小时 |
| `0 1,13 * * *` | 每天1点和13点 |
| `0 9-17 * * 1-5` | 工作日9-17点 |

## 附录 C: 系统 Crontab 配置

### Linux

```bash
# 编辑 crontab
crontab -e

# 每分钟执行任务调度
*/1 * * * * cd /path/to/project && php bin/w cron:run >> /var/log/cron.log 2>&1

# 查看当前 crontab
crontab -l

# 删除 crontab
crontab -r
```

### Windows

```powershell
# 创建计划任务
schtasks /create /tn "Weline Cron" /tr "php C:\path\to\project\bin\w cron:run" /sc minute /mo 1

# 查看计划任务
schtasks /query /tn "Weline Cron"

# 删除计划任务
schtasks /delete /tn "Weline Cron"
```
