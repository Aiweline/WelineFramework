# 05 - Fiber 协程支持（可选增强）

> **优先级**: ⭐⭐⭐  
> **依赖**: 02-运行时抽象层, 10-WeAsync异步引擎  
> **预计工作量**: 4-5 天

---

## 0. 与 WeAsync 的关系

**WeAsync**（照搬 Workerman）的事件循环已经在 `Event/` 目录中实现（照搬自 Workerman 的 `Events/`）。本文档规划的 Fiber 协程是**可选的增强功能**，用于在单个 Worker 内实现更高的并发。

```
WeAsync 架构：
├── 多进程（Master/Worker）     ← 照搬 Workerman，已实现
├── 事件循环（event/ev/select） ← 照搬 Workerman，已实现
└── Fiber 协程（可选）          ← 本文档规划的增强功能
```

### 优先级说明

| 阶段 | 内容 | 性能 |
|------|------|------|
| **阶段一** | 多进程 + 事件循环（无 Fiber） | 50,000+ QPS |
| **阶段二** | 加入 Fiber 协程支持 | 更高并发，更少 Worker |

建议先完成 WeAsync 核心功能，后续再加入 Fiber 支持。

---

## 1. 概述

本文档规划 PHP 8.1+ Fiber 协程的**可选**集成方案。

### 1.1 为什么需要 Fiber

| 场景 | 无 Fiber | 有 Fiber |
|------|---------|---------|
| 数据库查询（10ms） | Worker 阻塞等待 | 挂起，处理其他请求 |
| 外部 API 调用（100ms） | Worker 阻塞等待 | 挂起，处理其他请求 |
| 1000 并发请求 | 需要 100+ Worker | 4-8 Worker 即可 |

### 1.2 设计目标

| 目标 | 描述 |
|------|------|
| **协程透明** | 业务代码无需感知是否在协程中运行 |
| **自动调度** | 在 I/O 等待时自动让出执行权 |
| **上下文隔离** | 每个协程有独立的请求上下文 |
| **渐进支持** | 可选启用，不影响同步模式 |

## 2. Fiber 基础

### 2.1 Fiber 原理

```php
<?php
// Fiber 是可暂停/恢复的函数
$fiber = new Fiber(function() {
    echo "1. 开始\n";
    $result = Fiber::suspend('等待中');  // 暂停，返回值给调用者
    echo "3. 收到: {$result}\n";
    return "完成";
});

$value = $fiber->start();       // 输出 "1. 开始"，返回 "等待中"
echo "2. 外部: {$value}\n";     // 输出 "2. 外部: 等待中"
$result = $fiber->resume('数据'); // 输出 "3. 收到: 数据"
echo "4. 结果: {$result}\n";    // 输出 "4. 结果: 完成"
```

### 2.2 与事件循环配合

```
┌─────────────────────────────────────────────────────────────┐
│                      Event Loop                              │
│  ┌─────────────────────────────────────────────────────┐    │
│  │                  Fiber Scheduler                      │    │
│  │                                                       │    │
│  │   Fiber1 ──┐    Fiber2 ──┐    Fiber3 ──┐            │    │
│  │   [运行]   │    [挂起]   │    [挂起]   │            │    │
│  │      │     │       ↑     │       ↑     │            │    │
│  │      ↓     │       │     │       │     │            │    │
│  │   [I/O等待]│    [I/O完成]│    [I/O等待]│            │    │
│  │      │     │       │     │       │     │            │    │
│  │      └─────┼───────┘     └───────┼─────┘            │    │
│  │            │                     │                   │    │
│  │            └─────────────────────┘                   │    │
│  │                    调度器轮询                         │    │
│  └─────────────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────────┘
```

## 3. 协程调度器

### 3.1 CoroutineScheduler

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Coroutine;

use Fiber;
use SplQueue;
use Weline\Framework\Server\EventLoop\EventLoopInterface;

/**
 * 协程调度器
 * 
 * 管理多个 Fiber 的执行和调度
 */
final class CoroutineScheduler
{
    /** @var SplQueue<Fiber> 就绪队列 */
    private SplQueue $readyQueue;
    
    /** @var array<int, Fiber> 等待 I/O 的协程 */
    private array $waitingFibers = [];
    
    /** @var array<int, Suspension> 挂起点 */
    private array $suspensions = [];
    
    private int $fiberId = 0;
    private static ?self $instance = null;
    
    public function __construct(
        private readonly EventLoopInterface $eventLoop,
    ) {
        $this->readyQueue = new SplQueue();
        self::$instance = $this;
    }
    
    /**
     * 获取当前调度器实例
     */
    public static function getCurrent(): ?self
    {
        return self::$instance;
    }
    
    /**
     * 创建并启动一个协程
     */
    public function go(callable $callback): int
    {
        $id = ++$this->fiberId;
        
        $fiber = new Fiber(function() use ($callback, $id) {
            // 设置协程上下文
            CoroutineContext::create($id);
            
            try {
                return $callback();
            } finally {
                // 清理协程上下文
                CoroutineContext::destroy($id);
            }
        });
        
        $this->readyQueue->enqueue(['id' => $id, 'fiber' => $fiber, 'value' => null]);
        
        return $id;
    }
    
    /**
     * 挂起当前协程，等待 I/O
     * 
     * @param callable $register 注册 I/O 事件的回调
     * @return mixed I/O 完成后的结果
     */
    public function suspend(callable $register): mixed
    {
        $fiber = Fiber::getCurrent();
        
        if ($fiber === null) {
            throw new \RuntimeException('suspend() must be called within a coroutine');
        }
        
        $suspension = new Suspension();
        $fiberId = $this->getFiberId($fiber);
        $this->suspensions[$fiberId] = $suspension;
        $this->waitingFibers[$fiberId] = $fiber;
        
        // 注册 I/O 事件，完成时恢复协程
        $register(function(mixed $result = null) use ($fiberId, $suspension) {
            $suspension->result = $result;
            $suspension->ready = true;
            
            // 将协程加入就绪队列
            if (isset($this->waitingFibers[$fiberId])) {
                $fiber = $this->waitingFibers[$fiberId];
                unset($this->waitingFibers[$fiberId]);
                $this->readyQueue->enqueue([
                    'id' => $fiberId, 
                    'fiber' => $fiber, 
                    'value' => $result
                ]);
            }
        });
        
        // 挂起当前协程
        return Fiber::suspend();
    }
    
    /**
     * 异步延迟
     */
    public function delay(float $seconds): void
    {
        $this->suspend(function(callable $resume) use ($seconds) {
            $this->eventLoop->addTimer($seconds, function() use ($resume) {
                $resume();
            });
        });
    }
    
    /**
     * 运行调度器（单次调度）
     */
    public function tick(): void
    {
        // 处理就绪队列中的协程
        $processed = 0;
        $maxPerTick = 100; // 每次 tick 最多处理的协程数
        
        while (!$this->readyQueue->isEmpty() && $processed < $maxPerTick) {
            $item = $this->readyQueue->dequeue();
            $fiber = $item['fiber'];
            $value = $item['value'];
            $processed++;
            
            try {
                if (!$fiber->isStarted()) {
                    $fiber->start();
                } elseif ($fiber->isSuspended()) {
                    $fiber->resume($value);
                }
            } catch (\Throwable $e) {
                // 协程内部异常
                error_log("Coroutine error: {$e->getMessage()}");
            }
        }
    }
    
    /**
     * 是否有等待中的协程
     */
    public function hasPending(): bool
    {
        return !$this->readyQueue->isEmpty() || !empty($this->waitingFibers);
    }
    
    /**
     * 获取协程 ID
     */
    private function getFiberId(Fiber $fiber): int
    {
        return spl_object_id($fiber);
    }
    
    /**
     * 获取统计信息
     */
    public function getStats(): array
    {
        return [
            'ready' => $this->readyQueue->count(),
            'waiting' => count($this->waitingFibers),
            'total_created' => $this->fiberId,
        ];
    }
}

/**
 * 挂起点
 */
final class Suspension
{
    public bool $ready = false;
    public mixed $result = null;
}
```

### 3.2 CoroutineContext（协程上下文）

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Coroutine;

use Fiber;

/**
 * 协程上下文
 * 
 * 每个协程有独立的上下文，存储请求级数据
 */
final class CoroutineContext
{
    /** @var array<int, self> 所有协程的上下文 */
    private static array $contexts = [];
    
    /** 当前协程的数据 */
    private array $data = [];
    
    /** 当前协程的实例存储 */
    private array $instances = [];
    
    /** 协程 ID */
    private int $id;
    
    /** 创建时间 */
    private float $startTime;
    
    private function __construct(int $id)
    {
        $this->id = $id;
        $this->startTime = microtime(true);
    }
    
    /**
     * 创建协程上下文
     */
    public static function create(int $coroutineId): self
    {
        $context = new self($coroutineId);
        self::$contexts[$coroutineId] = $context;
        return $context;
    }
    
    /**
     * 销毁协程上下文
     */
    public static function destroy(int $coroutineId): void
    {
        unset(self::$contexts[$coroutineId]);
    }
    
    /**
     * 获取当前协程的上下文
     */
    public static function current(): ?self
    {
        $fiber = Fiber::getCurrent();
        
        if ($fiber === null) {
            return null;
        }
        
        $id = spl_object_id($fiber);
        return self::$contexts[$id] ?? null;
    }
    
    /**
     * 获取或创建当前上下文
     */
    public static function getOrCreate(): self
    {
        $context = self::current();
        
        if ($context === null) {
            // 非协程环境，创建临时上下文
            $id = 0;
            if (!isset(self::$contexts[$id])) {
                self::$contexts[$id] = new self($id);
            }
            return self::$contexts[$id];
        }
        
        return $context;
    }
    
    /**
     * 设置数据
     */
    public function set(string $key, mixed $value): void
    {
        $this->data[$key] = $value;
    }
    
    /**
     * 获取数据
     */
    public function get(string $key, mixed $default = null): mixed
    {
        return $this->data[$key] ?? $default;
    }
    
    /**
     * 检查数据是否存在
     */
    public function has(string $key): bool
    {
        return isset($this->data[$key]);
    }
    
    /**
     * 删除数据
     */
    public function delete(string $key): void
    {
        unset($this->data[$key]);
    }
    
    /**
     * 存储实例
     */
    public function setInstance(string $class, object $instance): void
    {
        $this->instances[$class] = $instance;
    }
    
    /**
     * 获取实例
     */
    public function getInstance(string $class): ?object
    {
        return $this->instances[$class] ?? null;
    }
    
    /**
     * 检查实例是否存在
     */
    public function hasInstance(string $class): bool
    {
        return isset($this->instances[$class]);
    }
    
    /**
     * 获取协程 ID
     */
    public function getId(): int
    {
        return $this->id;
    }
    
    /**
     * 获取执行时间（毫秒）
     */
    public function getElapsedMs(): float
    {
        return (microtime(true) - $this->startTime) * 1000;
    }
    
    /**
     * 清理所有数据
     */
    public function clear(): void
    {
        $this->data = [];
        $this->instances = [];
    }
}
```

## 4. 异步 I/O 包装

### 4.1 Async 工具类

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Coroutine;

/**
 * 异步操作工具类
 */
final class Async
{
    /**
     * 创建协程
     */
    public static function go(callable $callback): int
    {
        $scheduler = CoroutineScheduler::getCurrent();
        
        if ($scheduler === null) {
            // 非异步环境，直接执行
            $callback();
            return 0;
        }
        
        return $scheduler->go($callback);
    }
    
    /**
     * 异步延迟
     */
    public static function sleep(float $seconds): void
    {
        $scheduler = CoroutineScheduler::getCurrent();
        
        if ($scheduler === null) {
            // 非异步环境，同步睡眠
            usleep((int)($seconds * 1000000));
            return;
        }
        
        $scheduler->delay($seconds);
    }
    
    /**
     * 等待多个协程完成
     */
    public static function all(array $callbacks): array
    {
        $scheduler = CoroutineScheduler::getCurrent();
        
        if ($scheduler === null) {
            // 非异步环境，顺序执行
            $results = [];
            foreach ($callbacks as $key => $callback) {
                $results[$key] = $callback();
            }
            return $results;
        }
        
        $results = [];
        $completed = 0;
        $total = count($callbacks);
        $waitingFiber = Fiber::getCurrent();
        
        foreach ($callbacks as $key => $callback) {
            $scheduler->go(function() use ($key, $callback, &$results, &$completed, $total, $waitingFiber, $scheduler) {
                $results[$key] = $callback();
                $completed++;
                
                // 所有完成后恢复等待的协程
                if ($completed >= $total && $waitingFiber !== null) {
                    // 通过调度器恢复
                }
            });
        }
        
        // 挂起等待所有完成
        while ($completed < $total) {
            Fiber::suspend();
        }
        
        return $results;
    }
    
    /**
     * 等待任意一个完成
     */
    public static function any(array $callbacks): mixed
    {
        $scheduler = CoroutineScheduler::getCurrent();
        
        if ($scheduler === null) {
            // 非异步环境，返回第一个
            foreach ($callbacks as $callback) {
                return $callback();
            }
            return null;
        }
        
        $result = null;
        $done = false;
        
        foreach ($callbacks as $callback) {
            $scheduler->go(function() use ($callback, &$result, &$done) {
                if ($done) return;
                $result = $callback();
                $done = true;
            });
        }
        
        while (!$done) {
            Fiber::suspend();
        }
        
        return $result;
    }
    
    /**
     * 带超时的操作
     */
    public static function timeout(float $seconds, callable $callback): mixed
    {
        $scheduler = CoroutineScheduler::getCurrent();
        
        if ($scheduler === null) {
            return $callback();
        }
        
        $result = null;
        $done = false;
        $timedOut = false;
        
        // 启动任务协程
        $scheduler->go(function() use ($callback, &$result, &$done) {
            $result = $callback();
            $done = true;
        });
        
        // 启动超时协程
        $scheduler->go(function() use ($seconds, &$done, &$timedOut, $scheduler) {
            $scheduler->delay($seconds);
            if (!$done) {
                $timedOut = true;
                $done = true;
            }
        });
        
        while (!$done) {
            Fiber::suspend();
        }
        
        if ($timedOut) {
            throw new TimeoutException("Operation timed out after {$seconds} seconds");
        }
        
        return $result;
    }
}

/**
 * 超时异常
 */
class TimeoutException extends \RuntimeException {}
```

### 4.2 异步 Socket 包装

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Coroutine\IO;

use Weline\Framework\Coroutine\CoroutineScheduler;
use Fiber;

/**
 * 异步 Socket 操作
 * 
 * 将阻塞的 socket 操作转为协程友好的异步操作
 */
final class AsyncSocket
{
    /**
     * 异步读取
     */
    public static function read($socket, int $length): string
    {
        $scheduler = CoroutineScheduler::getCurrent();
        
        if ($scheduler === null) {
            // 非协程环境，同步读取
            return fread($socket, $length) ?: '';
        }
        
        // 设置为非阻塞
        stream_set_blocking($socket, false);
        
        // 先尝试读取
        $data = @fread($socket, $length);
        if ($data !== false && $data !== '') {
            return $data;
        }
        
        // 没有数据，挂起等待
        return $scheduler->suspend(function(callable $resume) use ($socket, $length) {
            $scheduler = CoroutineScheduler::getCurrent();
            $eventLoop = $scheduler->eventLoop;
            
            $eventId = $eventLoop->addRead($socket, function($socket) use ($resume, $length, &$eventId, $eventLoop) {
                $data = @fread($socket, $length);
                $eventLoop->remove($eventId);
                $resume($data ?: '');
            });
        });
    }
    
    /**
     * 异步写入
     */
    public static function write($socket, string $data): int
    {
        $scheduler = CoroutineScheduler::getCurrent();
        
        if ($scheduler === null) {
            return fwrite($socket, $data) ?: 0;
        }
        
        stream_set_blocking($socket, false);
        
        $written = @fwrite($socket, $data);
        if ($written !== false && $written > 0) {
            return $written;
        }
        
        return $scheduler->suspend(function(callable $resume) use ($socket, $data) {
            $scheduler = CoroutineScheduler::getCurrent();
            $eventLoop = $scheduler->eventLoop;
            
            $eventId = $eventLoop->addWrite($socket, function($socket) use ($resume, $data, &$eventId, $eventLoop) {
                $written = @fwrite($socket, $data);
                $eventLoop->remove($eventId);
                $resume($written ?: 0);
            });
        });
    }
    
    /**
     * 异步连接
     */
    public static function connect(string $address, float $timeout = 30.0): mixed
    {
        $scheduler = CoroutineScheduler::getCurrent();
        
        $socket = @stream_socket_client(
            $address,
            $errno,
            $errstr,
            0, // 超时为 0，非阻塞
            STREAM_CLIENT_CONNECT | STREAM_CLIENT_ASYNC_CONNECT
        );
        
        if ($socket === false) {
            throw new \RuntimeException("Failed to connect: [{$errno}] {$errstr}");
        }
        
        if ($scheduler === null) {
            // 非协程环境，等待连接完成
            stream_set_blocking($socket, true);
            return $socket;
        }
        
        // 协程环境，异步等待连接完成
        return $scheduler->suspend(function(callable $resume) use ($socket, $timeout) {
            $scheduler = CoroutineScheduler::getCurrent();
            $eventLoop = $scheduler->eventLoop;
            
            $eventId = $eventLoop->addWrite($socket, function($socket) use ($resume, &$eventId, $eventLoop) {
                $eventLoop->remove($eventId);
                $resume($socket);
            });
            
            // 超时处理
            $eventLoop->addTimer($timeout, function() use ($resume, &$eventId, $eventLoop, $socket) {
                $eventLoop->remove($eventId);
                @fclose($socket);
                $resume(null);
            });
        });
    }
}
```

### 4.3 异步数据库（示例）

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Coroutine\IO;

use Weline\Framework\Coroutine\CoroutineScheduler;
use Weline\Framework\Database\Pool\ConnectionPoolInterface;

/**
 * 协程友好的数据库操作
 * 
 * 注意：PDO 本身是阻塞的，这里通过连接池减少阻塞时间
 * 真正的异步数据库需要使用 mysqli 的非阻塞模式或 amphp/mysql
 */
final class AsyncDatabase
{
    public function __construct(
        private readonly ConnectionPoolInterface $pool,
    ) {}
    
    /**
     * 执行查询（协程友好）
     */
    public function query(string $sql, array $params = []): array
    {
        // 从连接池获取连接
        $pdo = $this->pool->acquire();
        
        try {
            $stmt = $pdo->prepare($sql);
            $stmt->execute($params);
            return $stmt->fetchAll(\PDO::FETCH_ASSOC);
        } finally {
            // 归还连接
            $this->pool->release($pdo);
            
            // 在协程环境中，让出执行权给其他协程
            $scheduler = CoroutineScheduler::getCurrent();
            if ($scheduler !== null) {
                // 短暂让出，让其他协程有机会执行
                Async::sleep(0.001);
            }
        }
    }
    
    /**
     * 执行非查询操作
     */
    public function execute(string $sql, array $params = []): int
    {
        $pdo = $this->pool->acquire();
        
        try {
            $stmt = $pdo->prepare($sql);
            $stmt->execute($params);
            return $stmt->rowCount();
        } finally {
            $this->pool->release($pdo);
        }
    }
}
```

## 5. 与 HttpWorker 集成

### 5.1 更新 HttpWorker 支持协程

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Server;

use Weline\Framework\Coroutine\CoroutineScheduler;
use Weline\Framework\Coroutine\CoroutineContext;
use Weline\Framework\Server\EventLoop\EventLoopInterface;

/**
 * 支持协程的 HTTP Worker
 */
final class CoroutineHttpWorker implements WorkerInterface
{
    private $socket;
    private EventLoopInterface $eventLoop;
    private CoroutineScheduler $scheduler;
    private bool $running = true;
    
    public function __construct(
        private readonly RuntimeInterface $runtime,
        private readonly string $host = '0.0.0.0',
        private readonly int $port = 8080,
        private readonly int $maxCoroutines = 10000, // 最大协程数
    ) {}
    
    public function run(int $workerId): void
    {
        // 创建事件循环
        $this->eventLoop = EventLoopFactory::create();
        
        // 创建协程调度器
        $this->scheduler = new CoroutineScheduler($this->eventLoop);
        
        // 启动框架
        $this->runtime->bootstrap();
        
        // 创建监听 Socket
        $this->createSocket();
        
        // 注册 accept 事件
        $this->eventLoop->addRead($this->socket, function($socket) {
            $this->onAccept($socket);
        });
        
        // 添加调度器 tick
        $this->eventLoop->addTimer(0.001, function() {
            $this->scheduler->tick();
        }, persistent: true);
        
        echo "Coroutine Worker {$workerId} started\n";
        echo "Max coroutines: {$this->maxCoroutines}\n";
        
        // 启动事件循环
        $this->eventLoop->run();
    }
    
    /**
     * 处理新连接
     */
    private function onAccept($socket): void
    {
        $client = @stream_socket_accept($socket, 0, $peername);
        
        if ($client === false) {
            return;
        }
        
        stream_set_blocking($client, false);
        
        // 为每个连接创建一个协程处理
        $this->scheduler->go(function() use ($client, $peername) {
            $this->handleConnection($client, $peername);
        });
    }
    
    /**
     * 在协程中处理连接
     */
    private function handleConnection($client, string $peername): void
    {
        $buffer = '';
        
        try {
            while (true) {
                // 异步读取数据
                $data = $this->asyncRead($client);
                
                if ($data === '') {
                    break; // 连接关闭
                }
                
                $buffer .= $data;
                
                // 尝试解析请求
                $request = HttpParser::parse($buffer);
                
                if ($request === null) {
                    continue; // 请求不完整
                }
                
                $buffer = substr($buffer, $request->rawLength);
                
                // 处理请求
                $this->handleRequest($client, $request, $peername);
                
                if (!$request->keepAlive) {
                    break;
                }
            }
        } finally {
            @fclose($client);
        }
    }
    
    /**
     * 处理单个请求
     */
    private function handleRequest($client, ParsedRequest $request, string $peername): void
    {
        try {
            // 创建 PSR-7 请求
            $psrRequest = $this->createPsrRequest($request, $peername);
            
            // 处理请求（在当前协程中执行）
            $psrResponse = $this->runtime->handle($psrRequest);
            
            // 发送响应
            $this->sendResponse($client, $psrResponse, $request->keepAlive);
            
        } catch (\Throwable $e) {
            $this->sendErrorResponse($client, 500, $e->getMessage());
        } finally {
            $this->runtime->reset();
        }
    }
    
    /**
     * 异步读取
     */
    private function asyncRead($socket): string
    {
        // 先尝试直接读取
        $data = @fread($socket, 65535);
        
        if ($data !== false && $data !== '') {
            return $data;
        }
        
        // 没有数据，挂起等待
        return $this->scheduler->suspend(function(callable $resume) use ($socket) {
            $eventId = $this->eventLoop->addRead($socket, function($socket) use ($resume, &$eventId) {
                $this->eventLoop->remove($eventId);
                $data = @fread($socket, 65535);
                $resume($data ?: '');
            });
        });
    }
    
    // ... 其他方法与非协程版本类似
}
```

## 6. 使用示例

### 6.1 在控制器中使用协程

```php
<?php
use Weline\Framework\Coroutine\Async;

class UserController
{
    public function show(int $id): array
    {
        // 并发获取多个数据
        $results = Async::all([
            'user' => fn() => $this->userService->find($id),
            'orders' => fn() => $this->orderService->findByUser($id),
            'address' => fn() => $this->addressService->findByUser($id),
        ]);
        
        // 三个查询并发执行，总时间 ≈ max(单个查询时间)
        
        return [
            'user' => $results['user'],
            'orders' => $results['orders'],
            'address' => $results['address'],
        ];
    }
    
    public function batchProcess(array $ids): array
    {
        $results = [];
        
        // 并发处理，但限制同时执行数
        foreach (array_chunk($ids, 10) as $chunk) {
            $batchResults = Async::all(
                array_map(fn($id) => fn() => $this->processOne($id), $chunk)
            );
            $results = array_merge($results, $batchResults);
        }
        
        return $results;
    }
}
```

### 6.2 带超时的操作

```php
<?php
use Weline\Framework\Coroutine\Async;
use Weline\Framework\Coroutine\TimeoutException;

try {
    $result = Async::timeout(5.0, function() {
        // 如果这个操作超过 5 秒，会抛出 TimeoutException
        return $this->slowApiCall();
    });
} catch (TimeoutException $e) {
    // 超时处理
    return $this->fallbackResult();
}
```
