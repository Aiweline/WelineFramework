# 06 - 数据库连接池

> **优先级**: ⭐⭐⭐⭐  
> **依赖**: 02-运行时抽象层, 10-WeAsync异步引擎  
> **预计工作量**: 3-4 天

---

## 0. 与 WeAsync 的关系

在 **WeAsync**（照搬 Workerman）模式下，Worker 进程是常驻的，数据库连接需要池化管理：

```
FPM 模式：
  请求 → 创建 PDO → 使用 → 请求结束 → PDO 销毁

WeAsync 模式：
  Worker 启动 → 创建连接池
  请求1 → 从池获取 PDO → 使用 → 归还池
  请求2 → 从池获取 PDO → 使用 → 归还池
  ... 连接复用
  Worker 停止 → 关闭所有连接
```

---

## 1. 概述

在 WeAsync 常驻内存模式下，数据库连接需要池化管理，避免：
- 连接过多导致数据库压力
- 连接泄漏
- 长连接超时断开

### 1.1 设计目标

| 目标 | 描述 |
|------|------|
| **连接复用** | 减少连接建立开销 |
| **自动管理** | 自动处理超时、断开重连 |
| **并发安全** | 支持多请求并发 |
| **可配置** | 池大小、超时时间可配置 |

---

## 1.2 FPM 与 WeAsync 兼容策略

数据库连接在 **FPM** 与 **WeAsync** 下的行为不同，通过**统一抽象、按运行时切换实现**兼容：

### 策略概览

| 运行时 | 是否用连接池 | 连接生命周期 | 获取方式 | 释放时机 |
|--------|--------------|--------------|----------|----------|
| **FPM** | 否 | 单请求内有效 | 每请求新建一条连接 | 请求结束 `terminate()` 时关闭 |
| **WeAsync** | 是 | 多请求复用 | 从池 `acquire()` | 请求结束 `reset()` 时 `release()` 归还池 |

### 统一入口：ConnectionProviderInterface

所有需要 PDO 的地方（如 DbManager、Adapter）只依赖「连接提供者」接口，由运行时在 bootstrap 时绑定不同实现：

```php
<?php
namespace Weline\Framework\Database\Pool;

/**
 * 连接提供者接口（统一入口，屏蔽 FPM / 池 差异）
 */
interface ConnectionProviderInterface
{
    /**
     * 获取当前请求可用的数据库连接
     * - FPM：返回本请求新建的连接（请求内单例）
     * - 常驻：从连接池 acquire，请求结束时由运行时 release
     */
    public function getConnection(string $name = 'default'): \PDO;

    /**
     * 归还/关闭连接（由运行时在 reset/terminate 时调用，业务一般不直接调）
     */
    public function releaseConnection(\PDO $connection, string $name = 'default'): void;
}
```

- **FPM**：实现类内部「按请求」创建新 PDO，存于 RequestContext；`releaseConnection` 在 `terminate()` 时关闭该连接。
- **常驻**：实现类内部持有一个 `ConnectionPoolInterface`，`getConnection` = `pool->acquire()`，`releaseConnection` = `pool->release()`，在 `reset()` 时对当前请求拿到的连接做 `release`。

### FPM 实现（每请求新建，不用池）

```php
<?php
namespace Weline\Framework\Database\Pool;

use Weline\Framework\Runtime\RequestContext;

/**
 * FPM 连接提供者：每请求新建连接，请求结束关闭，不使用连接池
 */
final class FpmConnectionProvider implements ConnectionProviderInterface
{
    private const CONTEXT_KEY = '__db_connections';

    public function getConnection(string $name = 'default'): \PDO
    {
        $ctx = RequestContext::current();
        $key = self::CONTEXT_KEY . '.' . $name;

        if ($ctx->has($key)) {
            return $ctx->get($key);
        }

        $factory = new PdoConnectionFactory(/* 从配置读取 */);
        $pdo = $factory->create();
        $ctx->set($key, $pdo);
        return $pdo;
    }

    public function releaseConnection(\PDO $connection, string $name = 'default'): void
    {
        $factory = new PdoConnectionFactory(/* ... */);
        $factory->close($connection);
        RequestContext::current()->remove(self::CONTEXT_KEY . '.' . $name);
    }
}
```

- FPM 下 `terminate()` 中对该请求创建过的连接调用 `releaseConnection`，即关闭连接，不建池、不复用。

### 常驻实现（用连接池）

```php
<?php
namespace Weline\Framework\Database\Pool;

use Weline\Framework\Runtime\RequestContext;

/**
 * 池化连接提供者：从 ConnectionPool acquire，请求结束 release 归还池
 */
final class PooledConnectionProvider implements ConnectionProviderInterface
{
    private const CONTEXT_KEY = '__db_pooled_connections'; // 记录本请求 acquire 过的，便于 reset 时 release

    public function __construct(
        private readonly ConnectionPoolManager $poolManager,
    ) {}

    public function getConnection(string $name = 'default'): \PDO
    {
        $ctx = RequestContext::current();
        $key = self::CONTEXT_KEY . '.' . $name;

        if ($ctx->has($key)) {
            return $ctx->get($key)['pdo'];
        }

        $pool = $this->poolManager->getPool($name);
        $pdo = $pool->acquire();
        $ctx->set($key, ['pdo' => $pdo, 'pool' => $pool]);
        return $pdo;
    }

    public function releaseConnection(\PDO $connection, string $name = 'default'): void
    {
        $ctx = RequestContext::current();
        $key = self::CONTEXT_KEY . '.' . $name;
        if (!$ctx->has($key)) {
            return;
        }
        $entry = $ctx->get($key);
        $entry['pool']->release($entry['pdo']);
        $ctx->remove($key);
    }
}
```

- 常驻下在 `reset()` 中对当前请求记录过的连接调用 `releaseConnection`，即归还池；进程退出时 `terminate()` 里可再调 `ConnectionPoolManager::closeAll()` 关闭所有池。

### 运行时绑定（bootstrap 时注入）

在运行时抽象层（如 02-运行时抽象层）的 bootstrap / 容器配置中按类型选择提供者：

```php
// 伪代码：根据运行时类型绑定 ConnectionProviderInterface
if (Runtime::current()->isPersistent()) {
    $container->bind(ConnectionProviderInterface::class, PooledConnectionProvider::class);
} else {
    $container->bind(ConnectionProviderInterface::class, FpmConnectionProvider::class);
}
```

- **FPM**：`isPersistent() === false` → 使用 `FpmConnectionProvider`，每请求新建 + 关闭，兼容现有 FPM 行为。
- **常驻**：`isPersistent() === true` → 使用 `PooledConnectionProvider`，连接池 acquire/release。

### 小结

- **连接池**：只在常驻内存模式下启用；FPM 下不建池，每请求一连接一关闭。
- **兼容 FPM**：通过 `ConnectionProviderInterface` 统一入口，FPM 与常驻各一套实现，业务与 DbManager 只调 `getConnection` / 由运行时在 `reset`/`terminate` 做 `releaseConnection`，无需关心底层是否用池。

## 2. 连接池接口

### 2.1 ConnectionPoolInterface

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Database\Pool;

/**
 * 连接池接口
 */
interface ConnectionPoolInterface
{
    /**
     * 从池中获取连接
     * 
     * @param float $timeout 超时时间（秒），0 表示不等待
     * @return mixed 连接对象
     * @throws PoolExhaustedException 池已耗尽且超时
     */
    public function acquire(float $timeout = 0): mixed;
    
    /**
     * 归还连接到池
     */
    public function release(mixed $connection): void;
    
    /**
     * 销毁连接（出错时调用）
     */
    public function destroy(mixed $connection): void;
    
    /**
     * 获取池状态
     */
    public function getStats(): PoolStats;
    
    /**
     * 关闭池中所有连接
     */
    public function close(): void;
}

/**
 * 连接池统计
 */
final class PoolStats
{
    public function __construct(
        public readonly int $total,      // 总连接数
        public readonly int $idle,       // 空闲连接数
        public readonly int $inUse,      // 使用中连接数
        public readonly int $waiting,    // 等待获取的请求数
    ) {}
}
```

## 3. 通用连接池实现

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Database\Pool;

use SplQueue;
use Weline\Framework\Runtime\ResettableInterface;

/**
 * 通用连接池
 */
final class ConnectionPool implements ConnectionPoolInterface, ResettableInterface
{
    private SplQueue $idle;
    private array $inUse = [];
    private int $total = 0;
    private int $waiting = 0;
    
    public function __construct(
        private readonly ConnectionFactoryInterface $factory,
        private readonly int $minSize = 1,
        private readonly int $maxSize = 10,
        private readonly float $maxIdleTime = 60.0,
        private readonly float $maxLifetime = 3600.0,
        private readonly float $validationInterval = 30.0,
    ) {
        $this->idle = new SplQueue();
        $this->warmup();
    }
    
    /**
     * 预热连接池
     */
    private function warmup(): void
    {
        for ($i = 0; $i < $this->minSize; $i++) {
            $this->createConnection();
        }
    }
    
    /**
     * 创建新连接
     */
    private function createConnection(): PooledConnection
    {
        $raw = $this->factory->create();
        $pooled = new PooledConnection($raw, time());
        $this->total++;
        $this->idle->enqueue($pooled);
        return $pooled;
    }
    
    public function acquire(float $timeout = 0): mixed
    {
        $deadline = $timeout > 0 ? microtime(true) + $timeout : null;
        
        while (true) {
            // 1. 尝试从空闲池获取
            if (!$this->idle->isEmpty()) {
                $pooled = $this->idle->dequeue();
                
                // 验证连接
                if (!$this->isValid($pooled)) {
                    $this->destroyPooled($pooled);
                    continue;
                }
                
                $this->inUse[spl_object_id($pooled)] = $pooled;
                return $pooled->connection;
            }
            
            // 2. 尝试创建新连接
            if ($this->total < $this->maxSize) {
                $pooled = $this->createConnection();
                $this->idle->dequeue(); // 从 idle 取出
                $this->inUse[spl_object_id($pooled)] = $pooled;
                return $pooled->connection;
            }
            
            // 3. 池已满，等待或超时
            if ($deadline !== null && microtime(true) >= $deadline) {
                throw new PoolExhaustedException(
                    "Connection pool exhausted (max: {$this->maxSize})"
                );
            }
            
            $this->waiting++;
            usleep(10000); // 等待 10ms
            $this->waiting--;
        }
    }
    
    public function release(mixed $connection): void
    {
        $pooled = $this->findPooled($connection);
        if ($pooled === null) {
            return;
        }
        
        unset($this->inUse[spl_object_id($pooled)]);
        
        // 重置连接状态
        try {
            $this->factory->reset($pooled->connection);
            $pooled->lastUsed = time();
            $this->idle->enqueue($pooled);
        } catch (\Throwable) {
            $this->destroyPooled($pooled);
        }
    }
    
    public function destroy(mixed $connection): void
    {
        $pooled = $this->findPooled($connection);
        if ($pooled !== null) {
            unset($this->inUse[spl_object_id($pooled)]);
            $this->destroyPooled($pooled);
        }
    }
    
    private function findPooled(mixed $connection): ?PooledConnection
    {
        foreach ($this->inUse as $pooled) {
            if ($pooled->connection === $connection) {
                return $pooled;
            }
        }
        return null;
    }
    
    private function isValid(PooledConnection $pooled): bool
    {
        $now = time();
        
        // 检查最大生命周期
        if (($now - $pooled->createdAt) > $this->maxLifetime) {
            return false;
        }
        
        // 检查空闲超时
        if (($now - $pooled->lastUsed) > $this->maxIdleTime) {
            return false;
        }
        
        // 验证连接
        return $this->factory->validate($pooled->connection);
    }
    
    private function destroyPooled(PooledConnection $pooled): void
    {
        try {
            $this->factory->close($pooled->connection);
        } catch (\Throwable) {
            // 忽略关闭错误
        }
        $this->total--;
    }
    
    public function getStats(): PoolStats
    {
        return new PoolStats(
            total: $this->total,
            idle: $this->idle->count(),
            inUse: count($this->inUse),
            waiting: $this->waiting,
        );
    }
    
    public function close(): void
    {
        // 关闭所有空闲连接
        while (!$this->idle->isEmpty()) {
            $pooled = $this->idle->dequeue();
            $this->destroyPooled($pooled);
        }
        
        // 关闭使用中连接
        foreach ($this->inUse as $pooled) {
            $this->destroyPooled($pooled);
        }
        
        $this->inUse = [];
    }
    
    public function reset(): void
    {
        // 连接池不需要在请求间重置
        // 连接会被复用
    }
    
    public function getStateInfo(): array
    {
        $stats = $this->getStats();
        return [
            'total' => $stats->total,
            'idle' => $stats->idle,
            'in_use' => $stats->inUse,
            'waiting' => $stats->waiting,
        ];
    }
}

/**
 * 池化连接包装
 */
final class PooledConnection
{
    public int $lastUsed;
    
    public function __construct(
        public readonly mixed $connection,
        public readonly int $createdAt,
    ) {
        $this->lastUsed = $createdAt;
    }
}
```

## 4. PDO 连接工厂

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Database\Pool;

/**
 * PDO 连接工厂
 */
final class PdoConnectionFactory implements ConnectionFactoryInterface
{
    public function __construct(
        private readonly string $dsn,
        private readonly string $username,
        private readonly string $password,
        private readonly array $options = [],
    ) {}
    
    public function create(): \PDO
    {
        $defaultOptions = [
            \PDO::ATTR_ERRMODE => \PDO::ERRMODE_EXCEPTION,
            \PDO::ATTR_DEFAULT_FETCH_MODE => \PDO::FETCH_ASSOC,
            \PDO::ATTR_EMULATE_PREPARES => false,
        ];
        
        $options = array_replace($defaultOptions, $this->options);
        
        return new \PDO($this->dsn, $this->username, $this->password, $options);
    }
    
    public function validate(mixed $connection): bool
    {
        try {
            $connection->query('SELECT 1');
            return true;
        } catch (\PDOException) {
            return false;
        }
    }
    
    public function reset(mixed $connection): void
    {
        // 回滚未提交的事务
        if ($connection->inTransaction()) {
            $connection->rollBack();
        }
    }
    
    public function close(mixed $connection): void
    {
        // PDO 没有显式关闭方法，置空即可
    }
}
```

## 5. 连接池管理器

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Database\Pool;

use Weline\Framework\App\Env;

/**
 * 连接池管理器
 */
final class ConnectionPoolManager
{
    private static array $pools = [];
    
    /**
     * 获取默认数据库连接池
     */
    public static function getDefaultPool(): ConnectionPoolInterface
    {
        return self::getPool('default');
    }
    
    /**
     * 获取指定名称的连接池
     */
    public static function getPool(string $name): ConnectionPoolInterface
    {
        if (!isset(self::$pools[$name])) {
            self::$pools[$name] = self::createPool($name);
        }
        
        return self::$pools[$name];
    }
    
    /**
     * 创建连接池
     */
    private static function createPool(string $name): ConnectionPoolInterface
    {
        $config = self::getPoolConfig($name);
        
        $factory = new PdoConnectionFactory(
            dsn: $config['dsn'],
            username: $config['username'],
            password: $config['password'],
            options: $config['options'] ?? [],
        );
        
        return new ConnectionPool(
            factory: $factory,
            minSize: $config['pool']['min'] ?? 1,
            maxSize: $config['pool']['max'] ?? 10,
            maxIdleTime: $config['pool']['idle_timeout'] ?? 60.0,
            maxLifetime: $config['pool']['max_lifetime'] ?? 3600.0,
        );
    }
    
    /**
     * 获取连接池配置
     */
    private static function getPoolConfig(string $name): array
    {
        $env = Env::getInstance();
        $dbConfig = $env->getDbConfig();
        
        if ($name === 'default') {
            $config = $dbConfig;
        } else {
            $config = $dbConfig['connections'][$name] ?? $dbConfig;
        }
        
        // 构建 DSN
        $driver = $config['adapter'] ?? 'mysql';
        $host = $config['hostname'] ?? 'localhost';
        $port = $config['hostport'] ?? 3306;
        $database = $config['database'] ?? '';
        $charset = $config['charset'] ?? 'utf8mb4';
        
        $dsn = match($driver) {
            'mysql' => "mysql:host={$host};port={$port};dbname={$database};charset={$charset}",
            'pgsql' => "pgsql:host={$host};port={$port};dbname={$database}",
            'sqlite' => "sqlite:{$database}",
            default => throw new \RuntimeException("Unsupported database driver: {$driver}"),
        };
        
        return [
            'dsn' => $dsn,
            'username' => $config['username'] ?? '',
            'password' => $config['password'] ?? '',
            'options' => $config['options'] ?? [],
            'pool' => $config['pool'] ?? [],
        ];
    }
    
    /**
     * 关闭所有连接池
     */
    public static function closeAll(): void
    {
        foreach (self::$pools as $pool) {
            $pool->close();
        }
        self::$pools = [];
    }
    
    /**
     * 获取所有连接池状态
     */
    public static function getAllStats(): array
    {
        $stats = [];
        foreach (self::$pools as $name => $pool) {
            $stats[$name] = $pool->getStats();
        }
        return $stats;
    }
}
```

## 6. 使用示例

### 6.1 配置文件

```php
<?php
// app/etc/env.php
return [
    'db' => [
        'adapter' => 'mysql',
        'hostname' => 'localhost',
        'hostport' => 3306,
        'database' => 'weline',
        'username' => 'root',
        'password' => '',
        'charset' => 'utf8mb4',
        
        // 连接池配置
        'pool' => [
            'min' => 2,           // 最小连接数
            'max' => 20,          // 最大连接数
            'idle_timeout' => 60, // 空闲超时（秒）
            'max_lifetime' => 3600, // 最大生命周期（秒）
        ],
    ],
];
```

### 6.2 使用连接池

```php
<?php
use Weline\Framework\Database\Pool\ConnectionPoolManager;

// 获取连接
$pool = ConnectionPoolManager::getDefaultPool();
$pdo = $pool->acquire();

try {
    $stmt = $pdo->prepare('SELECT * FROM users WHERE id = ?');
    $stmt->execute([1]);
    $user = $stmt->fetch();
} finally {
    // 归还连接
    $pool->release($pdo);
}
```
