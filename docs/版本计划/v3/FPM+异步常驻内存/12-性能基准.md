## 0. WeAsync 预期性能

**WeAsync**（照搬 Workerman）的性能与 Workerman 一致：

| 场景 | FPM | WeAsync (event) | 提升倍数 |
|------|-----|-----------------|---------|
| Hello World | 3,000-5,000 | **50,000+** | 10-15x |
| JSON API | 2,000-3,000 | **30,000+** | 10-15x |
| 数据库查询 | 500-1,000 | **10,000-15,000** | 10-20x |
| 模板渲染 | 300-500 | **5,000-8,000** | 10-15x |

> 性能提升主要来自：避免每次请求都加载框架、连接池复用、事件驱动 I/O

## 5. 基准测试类

```php
<?php
declare(strict_types=1);

namespace Weline\Framework\Benchmark;

/**
 * 性能基准测试
 */
final class Benchmark
{
    private array $results = [];
    
    /**
     * 测量单次执行时间
     */
    public function measure(string $name, callable $callback): float
    {
        $start = hrtime(true);
        $callback();
        $elapsed = (hrtime(true) - $start) / 1e6; // 毫秒
        
        $this->results[$name] = $elapsed;
        return $elapsed;
    }
    
    /**
     * 测量多次执行的平均时间
     */
    public function measureAverage(string $name, callable $callback, int $iterations = 100): array
    {
        $times = [];
        
        for ($i = 0; $i < $iterations; $i++) {
            $start = hrtime(true);
            $callback();
            $times[] = (hrtime(true) - $start) / 1e6;
        }
        
        sort($times);
        
        $result = [
            'min' => $times[0],
            'max' => $times[$iterations - 1],
            'avg' => array_sum($times) / $iterations,
            'p50' => $times[(int)($iterations * 0.5)],
            'p95' => $times[(int)($iterations * 0.95)],
            'p99' => $times[(int)($iterations * 0.99)],
        ];
        
        $this->results[$name] = $result;
        return $result;
    }
    
    /**
     * 获取内存使用
     */
    public static function getMemoryUsage(): array
    {
        return [
            'current' => memory_get_usage(true) / 1024 / 1024, // MB
            'peak' => memory_get_peak_usage(true) / 1024 / 1024, // MB
        ];
    }
    
    /**
     * 获取结果报告
     */
    public function getReport(): string
    {
        $report = "Performance Benchmark Report\n";
        $report .= str_repeat("=", 60) . "\n\n";
        
        foreach ($this->results as $name => $result) {
            $report .= "{$name}:\n";
            
            if (is_array($result)) {
                $report .= sprintf("  Min: %.2fms\n", $result['min']);
                $report .= sprintf("  Max: %.2fms\n", $result['max']);
                $report .= sprintf("  Avg: %.2fms\n", $result['avg']);
                $report .= sprintf("  P50: %.2fms\n", $result['p50']);
                $report .= sprintf("  P95: %.2fms\n", $result['p95']);
                $report .= sprintf("  P99: %.2fms\n", $result['p99']);
            } else {
                $report .= sprintf("  Time: %.2fms\n", $result);
            }
            
            $report .= "\n";
        }
        
        $memory = self::getMemoryUsage();
        $report .= "Memory Usage:\n";
        $report .= sprintf("  Current: %.2f MB\n", $memory['current']);
        $report .= sprintf("  Peak: %.2f MB\n", $memory['peak']);
        
        return $report;
    }
}
```

## 6. 内置基准命令

```php
<?php
// bin/w benchmark:run

use Weline\Framework\Benchmark\Benchmark;
use Weline\Framework\Runtime\RuntimeFactory;
use Weline\Framework\Manager\ObjectManager;

$benchmark = new Benchmark();

echo "Running benchmarks...\n\n";

// 1. 框架启动时间
$benchmark->measure('Framework Bootstrap', function() {
    \Weline\Framework\App::init();
});

// 2. ObjectManager 获取实例
$benchmark->measureAverage('ObjectManager::getInstance', function() {
    ObjectManager::getInstance(\Weline\Framework\Http\Request::class);
}, 1000);

// 3. 路由匹配
$router = ObjectManager::getInstance(\Weline\Framework\Router\Core::class);
$benchmark->measureAverage('Router Match', function() use ($router) {
    // 模拟路由匹配
}, 1000);

// 4. 模板渲染
$template = ObjectManager::getInstance(\Weline\Framework\View\Template::class);
$benchmark->measureAverage('Template Render', function() use ($template) {
    // 模拟模板渲染
}, 100);

echo $benchmark->getReport();
```

## 7. 持续基准测试

### 7.1 GitHub Actions 配置

```yaml
name: Benchmark

on:
  push:
    branches: [main, dev]
  pull_request:
    branches: [main]

jobs:
  benchmark:
    runs-on: ubuntu-latest
    
    steps:
      - uses: actions/checkout@v3
      
      - name: Setup PHP
        uses: shivammathur/setup-php@v2
        with:
          php-version: '8.4'
          extensions: pcntl, posix, sockets
          
      - name: Install dependencies
        run: composer install
        
      - name: Run benchmarks
        run: php bin/w benchmark:run
        
      - name: Store results
        uses: benchmark-action/github-action-benchmark@v1
        with:
          tool: 'customSmallerIsBetter'
          output-file-path: benchmark-results.json
```
