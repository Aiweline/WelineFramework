# FontSubLetter 模块文档索引

## 概述

FontSubLetter 是一个专业的字体子集生成工具，支持多种字体格式的子集生成，提供完整的后台管理功能。

## 技术栈

- **后端框架**: Weline Framework
- **字体处理**: php-font-lib, 纯PHP实现
- **前端技术**: HTML5, CSS3, JavaScript, Bootstrap 5
- **数据库**: SQLite
- **图表库**: Chart.js
- **日期选择器**: DateRangePicker

## 支持的字体格式

- **TTF** (TrueType Font)
- **OTF** (OpenType Font)
- **WOFF** (Web Open Font Format)
- **WOFF2** (Web Open Font Format 2.0)

## 主要功能

1. **字体上传**: 支持多种格式的字体文件上传
2. **字符提取**: 自动提取字体支持的字符
3. **子集生成**: 根据选择的字符生成字体子集
4. **后台管理**: 完整的用户上传记录和统计管理
5. **本地存储**: 浏览器本地存储用户选择
6. **错误处理**: 完善的错误处理和用户提示

## 使用指南

### 1. 前端使用

1. 访问字体子集生成页面
2. 上传字体文件
3. 选择需要的字符
4. 生成并下载字体子集

### 2. 后台管理

1. 访问后台管理界面
2. 查看用户上传记录
3. 查看字符提取统计
4. 导出数据报告

### 3. API接口

- `POST /fontsubletter/pc/index/upload` - 字体上传
- `POST /fontsubletter/pc/index/extract` - 字符提取
- `POST /fontsubletter/pc/index/generate` - 子集生成
- `GET /fontsubletter/backend/admin/index` - 后台管理
- `GET /fontsubletter/backend/record/index` - 记录管理
