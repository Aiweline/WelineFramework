# WelineTools_FontSubLetter - 字体字符提取与压缩工具模块

## 模块简介

`WelineTools_FontSubLetter` 是一个专业的在线字体压缩工具模块，支持多种字体格式，提供智能字符提取、可视化字符选择和高效字体压缩功能。本模块采用纯PHP实现，无需外部依赖，环境简单，部署容易。

## 功能特性

### 🎯 核心功能
- **字体上传**: 支持 TTF、OTF、WOFF、WOFF2 等多种字体格式
- **字符提取**: 智能提取字体中包含的所有字符
- **字符选择**: 可视化界面选择需要的字符
- **字体压缩**: 生成只包含选定字符的子集字体
- **文件下载**: 一键下载压缩后的字体文件

### 🚀 技术优势
- **纯PHP实现**: 无需外部依赖，环境简单
- **高压缩率**: 压缩率可达80%以上
- **处理速度快**: 优化的算法确保快速处理
- **安全可靠**: 在线处理，数据安全
- **用户友好**: 直观的界面设计
- **部署简单**: 无需Python环境，降低部署复杂度

## 依赖库

### Composer 依赖
```json
{
    "require": {
        "php": ">=7.4",
        "phenx/php-font-lib": "^0.5.4"
    }
}
```

### 主要使用的库
- **phenx/php-font-lib**: 用于字体文件解析和处理
- **GD库**: PHP内置图形处理库，用于字符提取

## 安装说明

### 1. 模块安装
```bash
# 将模块文件放置到指定目录
cp -r WelineTools_FontSubLetter /path/to/your/project/modules/

# 安装依赖
composer install

# 运行模块安装命令
php install.php
```

### 2. 数据库表创建
模块会自动创建以下数据表：
- `weline_font_sub_letter_records`: 字体处理记录表
- `weline_font_sub_letter_char_maps`: 字体字符映射表

### 3. 权限配置
确保 `media/fonts/` 目录具有写入权限：
```bash
chmod -R 755 media/fonts/
```

## 使用方法

### 后端管理界面
访问路径：`/admin/font-sub-letter/index`

### 前端用户界面
访问路径：`/font-sub-letter`

### 使用步骤
1. **上传字体文件**: 拖拽或选择字体文件上传
2. **提取字符**: 点击"提取字符"按钮分析字体
3. **选择字符**: 在字符网格中选择需要的字符
4. **生成字体**: 点击"生成字体"创建子集字体
5. **下载文件**: 下载压缩后的字体文件

## 技术实现

### 支持的字体格式
- **TTF** (TrueType Font)
- **OTF** (OpenType Font)
- **WOFF** (Web Open Font Format)
- **WOFF2** (Web Open Font Format 2.0)

### 文件大小限制
- 最大文件大小：100MB
- 支持的文件扩展名：`.ttf`, `.otf`, `.woff`, `.woff2`

### 字符提取算法
模块使用以下算法提取字体字符：
1. 使用GD库解析字体文件
2. 测试常用字符的可用性
3. 提取可用字符编码
4. 生成字符列表

### 字体压缩原理
1. 分析原始字体文件结构
2. 提取选定字符的字形数据
3. 重新构建字体文件
4. 生成优化的子集字体
