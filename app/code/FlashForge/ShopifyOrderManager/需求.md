# Shopify订单管理系统需求

## 核心功能

### 1. 多店铺管理
- 支持多个Shopify店铺配置
- 每个店铺独立的API配置（shop_url, api_key, access_token）
- 店铺状态管理（启用/禁用）

### 2. 订单自动同步
- **定时同步**：每10分钟自动从所有活跃店铺同步订单
- **手动同步**：支持命令行手动触发同步
- **增量同步**：基于最后同步时间，只同步新增和更新的订单
- **事务处理**：确保数据一致性，失败时自动回滚

### 3. 订单信息同步
#### 订单主表 (shopify_orders)
- 订单号 (order_number)
- 店铺ID (shop_id)
- Shopify订单ID (shopify_order_id)
- 客户信息 (customer_email, customer_name)
- 价格信息：
  - 订单总价 (total_price)
  - 商品小计 (subtotal_price)
  - 税费 (total_tax)
  - 运费 (total_shipping_price)
  - 折扣总额 (total_discounts)
- 状态信息：
  - 支付状态 (financial_status)
  - 发货状态 (fulfillment_status)
  - 订单状态 (order_status)
- 地址信息：
  - 收货地址 (shipping_address) - JSON格式
  - 账单地址 (billing_address) - JSON格式
- 其他信息：
  - 货币 (currency)
  - 支付方式 (gateway)
  - 订单标签 (tags)
  - 订单备注 (note)
  - Shopify创建时间 (shopify_created_at)
  - Shopify更新时间 (shopify_updated_at)

#### 订单项目表 (shopify_order_items)
- 订单ID (order_id)
- 产品信息：
  - Shopify产品ID (shopify_product_id)
  - Shopify变体ID (shopify_variant_id)
  - 产品标题 (product_title)
  - 变体标题 (variant_title)
  - SKU (sku)
  - 供应商 (vendor)
  - 产品类型 (product_type)
- **真实价格信息**：
  - 单价 (price) - 订单中的真实单价
  - 数量 (quantity)
  - 总折扣 (total_discount) - 该商品的总折扣金额
  - 单件税费 (unit_tax) - 计算得出的单件税费
  - 总税费 (total_tax) - 该商品的总税费
- 发货信息：
  - 可发货数量 (fulfillable_quantity)
  - 发货状态 (fulfillment_status)
  - 是否需要配送 (requires_shipping)
  - 配送服务 (fulfillment_service)
- 其他属性：
  - 是否征税 (taxable)
  - 是否礼品卡 (gift_card)
  - 产品属性 (properties) - JSON格式

### 4. 价格计算逻辑
#### 订单项目真实价格计算
1. **基础价格**：使用Shopify API返回的 `price` 字段（已包含折扣的最终价格）
2. **折扣处理**：记录 `total_discount` 字段（该商品的总折扣金额）
3. **税费计算**：
   - 从 `tax_lines` 中提取税费信息
   - 计算单件税费：`unit_tax = total_tax / quantity`
4. **最终价格**：`final_price = (price * quantity) - total_discount + total_tax`

1. **数据筛选**：支持按店铺、时间范围筛选
2. **数据组织**：每个订单项目占一行，确保数据完整性

5. **状态转换**：将英文状态转换为中文显示

#### 5.7 导出功能实现建议

**数据查询优化：**
```php
// 使用JOIN查询减少数据库访问次数
$query = $this->orderModel->select()
    ->join('shopify_order_items', 'shopify_orders.order_id = shopify_order_items.order_id')
    ->join('shops', 'shopify_orders.shop_id = shops.shop_id')
    ->where('shopify_orders.shopify_created_at', $startDate, '>=')
    ->where('shopify_orders.shopify_created_at', $endDate, '<=');
```

**地址信息解析：**
```php
// 解析shipping_address JSON字段
private function parseShippingAddress($shippingAddressJson): array
{
    $address = json_decode($shippingAddressJson, true) ?: [];
    return [
        'province' => $address['province'] ?? '',
        'country' => $address['country'] ?? '',
        'city' => $address['city'] ?? '',
        'zip' => $address['zip'] ?? ''
    ];
}
```

**价格计算逻辑：**
```php
// 计算商品小计和总价
private function calculateItemPrices($item): array
{
    $quantity = intval($item['quantity']);
    $price = floatval($item['price']);
    $totalDiscount = floatval($item['total_discount']);
    $totalTax = floatval($item['total_tax']);
    
    $subtotal = ($quantity * $price) - $totalDiscount;
    $total = $subtotal + $totalTax;
    
    return [
        'subtotal' => $subtotal,
        'total' => $total
    ];
}
```

#### 5.8 导出功能限制和注意事项

**导出限制：**

## 数据同步流程

1. **获取活跃店铺列表**
2. **遍历每个店铺**：
   - 初始化Shopify API连接
   - 测试API连接
   - 获取最后同步时间
   - 调用Shopify API获取更新订单
3. **处理每个订单**：
   - 检查订单是否已存在
   - 更新或创建订单记录
   - 处理订单项目（删除旧项目，插入新项目）
   - 计算真实价格和税费
4. **更新同步时间**
5. **发送通知**（成功/失败）

## 注意事项

1. **价格准确性**：确保同步的价格是订单中的真实价格，包含所有折扣和优惠
2. **数据一致性**：使用事务确保订单和订单项目的数据一致性
3. **错误恢复**：API调用失败时能够重试和恢复
4. **性能优化**：批量处理订单项目，避免逐条插入
5. **监控告警**：及时通知同步状态和错误信息
