### 3. 统计API接口 ✅

#### 3.1 新增文件
- **文件**: `app/code/Weline/Visitor/Api/Rest/V1/Statistics.php`

#### 3.2 API接口列表

##### `getWebsite()` - 获取站点统计信息
- **路径**: `/visitor/rest/v1/statistics/website`
- **方法**: GET
- **功能**: 获取指定站点的完整统计信息
- **参数**:
  - `websiteId` (可选): 站点ID，默认使用当前请求的站点ID
  - `startDate` (可选): 开始日期（Y-m-d H:i:s格式）
  - `endDate` (可选): 结束日期（Y-m-d H:i:s格式）
- **返回**: 包含总记录数、事件统计、未处理记录数等

##### `getEvent()` - 获取事件统计信息
- **路径**: `/visitor/rest/v1/statistics/event`
- **方法**: GET
- **功能**: 获取指定站点和事件的统计信息
- **参数**:
  - `websiteId` (可选): 站点ID
  - `event` (必填): 事件名称
- **返回**: 事件数量统计

##### `getWebsites()` - 获取所有站点列表
- **路径**: `/visitor/rest/v1/statistics/websites`
- **方法**: GET
- **功能**: 获取系统中所有有像素记录的站点ID列表
- **返回**: 站点ID数组和总数

## 使用示例

### 1. 前端发送像素数据
```javascript
// 站点ID会自动从Cookie或window.site获取
WelinePixel.send({
    eventName: 'click',
    // websiteId 和 siteId 会自动填充
});
```

### 2. 后端查询特定站点的记录
```php
// 获取站点ID为1的所有像素记录
$pixels = \Weline\Visitor\Model\Pixel::getPixelsByWebsiteId(1);

// 获取站点统计摘要
$summary = \Weline\Visitor\Model\Pixel::getWebsiteSummary(1);

// 获取时间范围内的统计
$stats = \Weline\Visitor\Model\Pixel::getWebsiteStatsByDateRange(
    1, 
    '2025-01-01 00:00:00', 
    '2025-01-31 23:59:59'
);
```

### 3. 使用统计API
```bash
# 获取站点统计信息
GET /visitor/rest/v1/statistics/website?websiteId=1

# 获取时间范围内的统计
GET /visitor/rest/v1/statistics/website?websiteId=1&startDate=2025-01-01&endDate=2025-01-31

# 获取事件统计
GET /visitor/rest/v1/statistics/event?websiteId=1&event=login

# 获取所有站点列表
GET /visitor/rest/v1/statistics/websites
```

### 5.2 Statistics API接口扩展

#### 5.2.1 新增接口（8个）
1. `getBusinessValue()` - 获取商业价值分析数据
   - 路径: `/visitor/rest/v1/statistics/business-value`
   - 参数: websiteId, period (daily|weekly|monthly|quarterly|yearly), startDate, endDate

2. `getDashboard()` - 获取实时大屏数据
   - 路径: `/visitor/rest/v1/statistics/dashboard`
   - 参数: websiteId, interval (10|30), hours

3. `getDailyComparison()` - 获取每日对比数据
   - 路径: `/visitor/rest/v1/statistics/daily-comparison`
   - 参数: websiteId, days

4. `getChangePercentage()` - 获取变化百分比数据
   - 路径: `/visitor/rest/v1/statistics/change-percentage`
   - 参数: websiteId, interval, hours

5. `getTopEvents()` - 获取热门事件Top N
   - 路径: `/visitor/rest/v1/statistics/top-events`
   - 参数: websiteId, limit, startDate, endDate

6. `getEventDistribution()` - 获取事件分布统计
   - 路径: `/visitor/rest/v1/statistics/event-distribution`
   - 参数: websiteId, startDate, endDate

7. `getTimeRangeStats()` - 获取时间范围统计（增强版）
   - 路径: `/visitor/rest/v1/statistics/time-range`
   - 参数: websiteId, startDate, endDate

8. `getRealtimeStats()` - 获取实时统计数据
   - 路径: `/visitor/rest/v1/statistics/realtime`
   - 参数: websiteId, minutes
