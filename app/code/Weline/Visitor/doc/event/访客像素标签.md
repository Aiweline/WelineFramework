# Weline Visitor 模块 - 事件文档

## 概述

本文档详细说明了 Weline Visitor 模块提供的 `Weline_Visitor::taglib_pixel` 事件及其使用方法。该事件在渲染访客像素标签时触发。

## 事件列表

### 1. Weline_Visitor::taglib_pixel - 访客像素标签事件

#### 基本信息

- **事件名称**：`Weline_Visitor::taglib_pixel`
- **事件类型**：视图渲染事件
- **触发时机**：在渲染访客像素标签时
- **触发位置**：`app/code/Weline/Visitor/Taglib/Pixel.php` 第 65 行
- **配置文件**：`app/code/Weline/Visitor/etc/event.xml`

#### 功能说明

`Weline_Visitor::taglib_pixel` 事件在渲染访客像素标签时触发，允许其他模块自定义像素代码。可以通过修改事件数据中的 `pixel_code` 字段来设置像素代码，通过 `enable` 字段来控制是否启用。

该事件主要用于：
- 自定义像素代码
- 像素标签启用控制
- 像素标签数据收集
- 像素标签日志记录

#### 触发时机

```php
// app/code/Weline/Visitor/Taglib/Pixel.php
$data = new DataObject(['pixel_code' => '', 'name' => $name, 'enable' => 1]);
$event = w_obj(EventsManager::class);
$event->dispatch('Weline_Visitor::taglib_pixel', $data);
if (empty($data->getData('enable'))) {
    return '';
}
$tp->assign('pixel_code', $data->getData('pixel_code'));
```

#### 使用场景

- **像素代码注入**：注入自定义的像素跟踪代码
- **启用控制**：根据条件控制像素标签是否启用
- **数据收集**：收集像素标签相关的数据
- **日志记录**：记录像素标签的渲染情况

#### 使用方法

##### 基本用法

在模块的 `etc/event.xml` 文件中注册观察者：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="Weline_Visitor::taglib_pixel">
        <observer name="Your_Module::taglib_pixel"
                  instance="Your\Module\Observer\TaglibPixelObserver"
                  disabled="false"
                  shared="true"
                  sort="100"/>
    </event>
</config>
```

创建观察者类：

```php
<?php

namespace Your\Module\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class TaglibPixelObserver implements ObserverInterface
{
    /**
     * 执行观察者逻辑
     *
     * @param Event $event
     * @return void
     */
    public function execute(Event &$event): void
    {
        $data = $event->getData();
        $name = $data->getData('name');
        
        // 根据名称设置不同的像素代码
        if ($name === 'facebook') {
            $data->setData('pixel_code', '<script>/* Facebook Pixel Code */</script>');
        } elseif ($name === 'google') {
            $data->setData('pixel_code', '<script>/* Google Analytics Code */</script>');
        }
        
        // 控制是否启用
        // $data->setData('enable', 0); // 禁用
    }
}
```

#### 事件数据

`Weline_Visitor::taglib_pixel` 事件传递的数据：

```php
[
    'pixel_code' => string,  // 像素代码（可以修改）
    'name' => string,        // 像素标签名称
    'enable' => int,         // 是否启用（可以修改，0=禁用，1=启用）
]
```

**数据说明**：
- `pixel_code`：像素跟踪代码，观察者可以设置此值来自定义像素代码
- `name`：像素标签的名称，用于区分不同的像素标签
- `enable`：是否启用像素标签，设置为 0 可以禁用像素标签

#### 注意事项

1. **执行顺序**：观察者按照 `sort` 属性排序执行，数值越小越先执行
2. **代码安全**：确保注入的像素代码是安全的，避免XSS攻击
3. **性能考虑**：该事件在每次渲染像素标签时都会触发，应避免执行耗时操作
4. **启用控制**：如果 `enable` 设置为 0，像素标签将不会渲染
