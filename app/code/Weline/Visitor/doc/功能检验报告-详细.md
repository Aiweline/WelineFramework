## 发现并修复的问题

### ✅ 问题1: 加密服务版本号null处理
- **问题**: `encrypt()` 方法在版本号为null时调用 `getTokenByVersion(null)` 会失败
- **修复**: 添加null检查，使用 `getCurrentVersionToken()` 替代
- **状态**: ✅ 已修复

### ✅ 问题2: Observer URL获取方式
- **问题**: 使用 `$_SERVER['WELINE_BASE_URL']` 可能不存在
- **修复**: 使用 `Env::getInstance()->getBaseUrl()` 并添加fallback逻辑
- **状态**: ✅ 已修复

### ✅ 问题3: PixelEncryptionToken表名定义
- **问题**: 模型缺少 `$table` 属性定义
- **修复**: 添加 `public string $table = 'w_pixel_encryption_token';`
- **状态**: ✅ 已修复

### ✅ 问题4: createTable表名参数
- **问题**: `createTable()` 第一个参数应该是表名，不是注释
- **修复**: 修改为 `createTable('pixel_encryption_token', 'weline 像素加密Token')`
- **状态**: ✅ 已修复
