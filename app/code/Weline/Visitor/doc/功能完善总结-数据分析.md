# 像素数据分析功能完善总结

## 完成时间
2025-01-XX

## 功能概述

本次完善为像素系统添加了完整的数据分析功能，包括商业价值分析、实时大屏展示、数据对比分析、A/B测试管理和综合统计报告。

## 已完成功能

### 1. 商业价值分析 ✅

#### 功能特性
- **多时间维度支持**：每天、每周、每月、每季度、每年
- **价值趋势图**：可视化展示价值变化趋势
- **事件趋势图**：可视化展示事件数量变化
- **详细数据表格**：包含转化率、平均价值等指标
- **总价值统计**：汇总所有时间段的总体价值
- **总事件数统计**：汇总所有时间段的事件总数

#### API接口
```
GET /visitor/rest/v1/analytics/business-value
```

#### 参数
- `websiteId` (可选): 站点ID
- `period` (必填): 时间维度 (daily|weekly|monthly|quarterly|yearly)
- `startDate` (可选): 开始日期
- `endDate` (可选): 结束日期

### 2. 实时大屏展示 ✅

#### 功能特性
- **实时数据展示**：10分钟/30分钟粒度
- **自动刷新功能**：可开启/关闭，默认每30秒刷新
- **当前数据卡片**：当前价值、事件数、变化百分比
- **实时趋势图**：价值趋势和事件趋势
- **变化百分比显示**：相比上一时段的变化
- **深色主题**：适合大屏展示

#### API接口
```
GET /visitor/rest/v1/analytics/dashboard
```

#### 参数
- `websiteId` (可选): 站点ID
- `interval` (可选): 时间间隔，10或30分钟
- `hours` (可选): 获取最近N小时的数据

#### 访问页面
```
/visitor/analytics/dashboard
```

### 3. 数据对比分析 ✅

#### 功能特性
- **每日对比**：每天相比昨天的数据分析
- **对比数据表格**：显示今天和昨天的详细数据
- **变化值显示**：数值变化和百分比变化
- **对比趋势图**：今天和昨天的趋势对比
- **多天对比**：支持查看最近N天的对比数据

#### API接口
```
GET /visitor/rest/v1/analytics/daily-comparison
```

#### 参数
- `websiteId` (可选): 站点ID
- `days` (可选): 获取最近N天的对比数据

### 4. 数据变化百分比 ✅

#### 功能特性
- **时刻对比**：每个时刻相比前一时刻的变化百分比
- **支持多粒度**：10分钟和30分钟
- **变化趋势**：可视化展示数据变化趋势

#### API接口
```
GET /visitor/rest/v1/analytics/change-percentage
```

### 5. A/B测试管理 ✅

#### 功能特性
- **测试创建**：通过表单创建A/B测试配置
- **测试列表**：查看所有测试，支持状态筛选
- **测试详情**：查看测试数据和对比结果
- **自动判断获胜者**：根据转化率自动判断获胜变体
- **改进百分比**：显示变体B相比变体A的改进情况
- **状态管理**：支持active、paused、completed、draft状态

#### 数据库模型
- **表名**：`w_ab_test`
- **字段**：测试ID、站点ID、名称、描述、状态、开始/结束时间、变体配置、流量分配

#### API接口
```
POST /visitor/rest/v1/analytics/ab-test/create  # 创建测试
GET  /visitor/rest/v1/analytics/ab-test/list    # 获取测试列表
GET  /visitor/rest/v1/analytics/ab-test         # 获取测试数据
```

### 6. 综合统计报告 ✅

#### 功能特性
- **多维度统计**：汇总多个维度的统计数据
- **统计卡片**：总记录数、总价值、事件类型数、未处理记录
- **热门事件Top 10**：按事件数量排序的热门事件
- **事件分布图**：饼图展示事件分布
- **每日趋势图**：双Y轴图表展示价值和事件趋势
- **数据导出**：支持CSV格式导出

#### API接口
```
GET /visitor/rest/v1/analytics/report  # 获取综合报告
GET /visitor/rest/v1/analytics/export  # 导出数据
```

### 7. 数据导出功能 ✅

#### 功能特性
- **CSV格式导出**：支持Excel打开，包含BOM以支持中文
- **JSON格式导出**：支持API调用获取JSON数据
- **时间范围筛选**：支持按时间范围导出
- **站点筛选**：支持按站点ID筛选

#### API接口
```
GET /visitor/rest/v1/analytics/export
```

#### 参数
- `websiteId` (可选): 站点ID
- `startDate` (可选): 开始日期
- `endDate` (可选): 结束日期
- `format` (可选): csv|json，默认csv

## 使用示例

### 示例1: 获取每日商业价值数据
```javascript
fetch('/visitor/rest/v1/analytics/business-value?period=daily&websiteId=1')
  .then(res => res.json())
  .then(data => console.log(data));
```

### 示例2: 创建A/B测试
```javascript
fetch('/visitor/rest/v1/analytics/ab-test/create', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    testId: 'button_test',
    name: '按钮测试',
    websiteId: 1,
    variantA: { color: 'blue' },
    variantB: { color: 'red' },
    status: 'active'
  })
});
```

window.open('/visitor/rest/v1/analytics/export?websiteId=1&format=csv');
