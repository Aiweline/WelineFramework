## 模块概述

Weline_Meta 模块用于统一管理系统中各种文件的元数据，支持多语言翻译，方便主题切换和组件管理。

### 核心特性

- ✅ 统一的元数据格式规范
- ✅ 支持多语言翻译（集成I18n模块）
- ✅ 自动扫描和收集元数据
- ✅ 编译时翻译（Meta Taglib）
- ✅ 后端管理界面
- ✅ 事件机制，支持其他模块注册收集器

## 数据库设计

### 表结构：w_meta

```sql
CREATE TABLE `w_meta` (
  `meta_id` int(11) NOT NULL AUTO_INCREMENT COMMENT '元数据ID',
  `namespace` varchar(100) NOT NULL COMMENT '命名空间：theme, module, api等',
  `meta_type` varchar(50) NOT NULL COMMENT '元数据类型：layout, component, partial, api等',
  `meta_identify` varchar(255) NOT NULL COMMENT '元数据标识：可能是id、文件名、路径等，根据类型而定',
  `file_path` varchar(500) DEFAULT NULL COMMENT '文件相对路径（相对于模块或主题根目录）',
  `file_full_path` varchar(1000) DEFAULT NULL COMMENT '文件完整路径',
  `area` varchar(50) DEFAULT NULL COMMENT '区域：frontend, backend（可选）',
  `category` varchar(100) DEFAULT NULL COMMENT '分类：account, product, cart等（可选）',
  `meta_data` text DEFAULT NULL COMMENT '元数据JSON（存储所有字段的原始值）',
  `created_at` datetime DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  `updated_at` datetime DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  PRIMARY KEY (`meta_id`),
  UNIQUE KEY `uk_namespace_type_identify` (`namespace`, `meta_type`, `meta_identify`),
  KEY `idx_namespace` (`namespace`),
  KEY `idx_meta_type` (`meta_type`),
  KEY `idx_file_path` (`file_path`),
  KEY `idx_namespace_type` (`namespace`, `meta_type`),
  KEY `idx_namespace_area` (`namespace`, `area`),
  KEY `idx_type_area` (`meta_type`, `area`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='元数据表';
```

### 索引优化说明

为了提高查询效率，wl_meta表设计了以下索引：

#### 1. 主键索引
- **PRIMARY KEY (`meta_id`)**: 主键索引，用于快速定位单条记录

#### 2. 唯一索引
- **UNIQUE KEY `uk_namespace_type_identify` (`namespace`, `meta_type`, `meta_identify`)**: 
  - 防止重复数据
  - 覆盖最常见的查询组合：根据命名空间、类型和标识查找元数据
  - 提升 `WHERE namespace = ? AND meta_type = ? AND meta_identify = ?` 查询效率

#### 3. 单列索引
- **KEY `idx_namespace` (`namespace`)**: 
  - 提升按命名空间查询的效率
  - 适用于：`WHERE namespace = ?`
  
- **KEY `idx_meta_type` (`meta_type`)**: 
  - 提升按类型查询的效率
  - 适用于：`WHERE meta_type = ?`
  
- **KEY `idx_file_path` (`file_path`)**: 
  - 提升按文件路径查询的效率
  - 适用于：`WHERE file_path = ?` 或 `WHERE file_path LIKE ?`

#### 4. 复合索引（提升多条件查询效率）
- **KEY `idx_namespace_type` (`namespace`, `meta_type`)**: 
  - 提升按命名空间和类型组合查询的效率
  - 适用于：`WHERE namespace = ? AND meta_type = ?`
  - 可以覆盖只查询namespace的情况（最左前缀原则）
  
- **KEY `idx_namespace_area` (`namespace`, `area`)**: 
  - 提升按命名空间和区域组合查询的效率
  - 适用于：`WHERE namespace = ? AND area = ?`
  - 可以覆盖只查询namespace的情况
  
- **KEY `idx_type_area` (`meta_type`, `area`)**: 
  - 提升按类型和区域组合查询的效率
  - 适用于：`WHERE meta_type = ? AND area = ?`
  - 可以覆盖只查询meta_type的情况

#### 索引使用建议

1. **查询优化**：
   - 优先使用唯一索引 `uk_namespace_type_identify` 进行精确查询
   - 使用复合索引时，遵循最左前缀原则，确保查询条件顺序与索引字段顺序一致

2. **常见查询场景**：
   - 按命名空间查询：使用 `idx_namespace` 或 `idx_namespace_type`
   - 按类型查询：使用 `idx_meta_type` 或 `idx_type_area`
   - 按文件路径查询：使用 `idx_file_path`
   - 组合查询：优先使用对应的复合索引

3. **索引维护**：
   - 索引会占用额外的存储空间
   - 插入和更新操作会更新相关索引，略微影响性能
   - 对于元数据这种读多写少的场景，索引带来的查询性能提升远大于维护成本

### 翻译存储

翻译使用I18n模块的 `i18n_dictionary` 和 `i18n_locale_dictionary` 表存储：

- **翻译键格式**：`@meta::{namespace}.{type}.{identify}.{group}.{field}`
- **示例**：`@meta::theme.layout.account_auth.info.name`

## 元数据格式规范

### 标准格式

```php
<?php
/**
 * ============================================
 * 元数据区域 (Metadata Block)
 * ============================================
 * 
 * @meta.namespace theme                    # 命名空间：theme, module, api等
 * @meta.theme.type layout                 # 类型：layout, component, partial等
 * @meta.theme.area frontend               # 区域：frontend, backend（可选）
 * @meta.theme.category account            # 分类：account, product, cart等（可选）
 * @meta.theme.identify auth               # 标识：auth, default, 1280等（文件名或ID）
 * 
 * @meta.theme.info.name 名称：个人中心认证页面布局
 * @meta.theme.info.description 描述：登录/注册等认证页面的专用布局
 * @meta.theme.info.icon 图标：mdi mdi-account-lock
 * @meta.theme.info.version 版本：1.0.0
 * @meta.theme.info.author 作者：Weline Team
 * 
 * @param title 页面标题
 * @param content 认证表单内容（HTML字符串）
 * @param class 额外CSS类
 */
```

### 翻译键生成规则

```
@meta::{namespace}.{type}.{identify}.{group}.{field}
```

**示例**：
- `@meta::theme.layout.account_auth.info.name`
- `@meta::theme.layout.account_auth.info.description`
- `@meta::theme.component.frontend.button.info.name`
- `@meta::theme.partial.backend.header.info.name`

**说明**：
- `namespace`: 命名空间（如：theme）
- `type`: 类型（如：layout, component, partial）
- `identify`: 标识（文件名或自定义ID）
- `group`: 分组（如：info, config, custom）
- `field`: 字段名（如：name, description, icon等）

### 1. 元数据模型 (Model/Meta.php)

```php
<?php
declare(strict_types=1);

namespace Weline\Meta\Model;

use Weline\Framework\Database\AbstractModel;
use Weline\Framework\Database\Api\Db\Ddl\TableInterface;
use Weline\Framework\Setup\Data\Context;
use Weline\Framework\Setup\Db\ModelSetup;

class Meta extends AbstractModel
{
    public const table = 'wl_meta';
    
    const fields_ID = 'meta_id';
    const fields_NAMESPACE = 'namespace';
    const fields_META_TYPE = 'meta_type';
    const fields_META_IDENTIFY = 'meta_identify';
    const fields_FILE_PATH = 'file_path';
    const fields_FILE_FULL_PATH = 'file_full_path';
    const fields_AREA = 'area';
    const fields_CATEGORY = 'category';
    const fields_META_DATA = 'meta_data';

    /**
     * 安装表结构
     */
    public function install(ModelSetup $setup, Context $context): void
    {
        if (!$setup->tableExist()) {
            $setup->createTable()
                ->addColumn(self::fields_ID, TableInterface::column_type_INTEGER, null, 'not null auto_increment', '元数据ID')
                ->addColumn(self::fields_NAMESPACE, TableInterface::column_type_VARCHAR, 100, 'not null', '命名空间')
                ->addColumn(self::fields_META_TYPE, TableInterface::column_type_VARCHAR, 50, 'not null', '元数据类型')
                ->addColumn(self::fields_META_IDENTIFY, TableInterface::column_type_VARCHAR, 255, 'not null', '元数据标识')
                ->addColumn(self::fields_FILE_PATH, TableInterface::column_type_VARCHAR, 500, 'default null', '文件相对路径')
                ->addColumn(self::fields_FILE_FULL_PATH, TableInterface::column_type_VARCHAR, 1000, 'default null', '文件完整路径')
                ->addColumn(self::fields_AREA, TableInterface::column_type_VARCHAR, 50, 'default null', '区域')
                ->addColumn(self::fields_CATEGORY, TableInterface::column_type_VARCHAR, 100, 'default null', '分类')
                ->addColumn(self::fields_META_DATA, TableInterface::column_type_TEXT, null, 'default null', '元数据JSON')
                ->addColumn('created_at', TableInterface::column_type_DATETIME, null, 'default current_timestamp', '创建时间')
                ->addColumn('updated_at', TableInterface::column_type_DATETIME, null, 'default current_timestamp on update current_timestamp', '更新时间')
                ->addIndex(TableInterface::index_type_PRIMARY, 'PRIMARY', self::fields_ID, '主键')
                ->addIndex(TableInterface::index_type_UNIQUE, 'uk_namespace_type_identify', 
                    [self::fields_NAMESPACE, self::fields_META_TYPE, self::fields_META_IDENTIFY], '唯一索引')
                ->addIndex(TableInterface::index_type_KEY, 'idx_namespace', self::fields_NAMESPACE, '命名空间索引')
                ->addIndex(TableInterface::index_type_KEY, 'idx_meta_type', self::fields_META_TYPE, '类型索引')
                ->addIndex(TableInterface::index_type_KEY, 'idx_file_path', self::fields_FILE_PATH, '文件路径索引')
                ->addAdditional('ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT=\'元数据表\'')
                ->create();
        }
    }

    /**
     * 生成翻译键
     */
    public function generateTranslationKey(string $group = 'info', string $field = 'name'): string
    {
        $namespace = $this->getData(self::fields_NAMESPACE);
        $type = $this->getData(self::fields_META_TYPE);
        $identify = $this->getData(self::fields_META_IDENTIFY);
        
        return "@meta::{$namespace}.{$type}.{$identify}.{$group}.{$field}";
    }

    /**
     * 保存元数据并收集翻译
     */
    public function saveMeta(array $metaData): void
    {
        // 保存元数据
        $this->save();
        
        // 触发翻译收集事件（由I18n模块监听并处理）
        $this->getEventManager()->dispatch('Weline_I18n::collect_translations', [
            'translations' => $translations,
            'module' => 'Weline_Meta' // 标识翻译词来源模块
        ]);
    }
}
```

### 2. 扫描服务 (Service/Scanner.php)

**重要说明**：Meta模块不提供自动扫描功能，只提供扫描器工具。其他模块通过事件调用扫描器，指定路径进行扫描。扫描时会严格验证格式，不符合格式的文件会报错。

```php
<?php
declare(strict_types=1);

namespace Weline\Meta\Service;

use Weline\Framework\App\Exception;
use Weline\Framework\Manager\ObjectManager;
use Weline\Meta\Model\Meta;

class Scanner
{
    /**
     * 扫描指定路径的文件并提取元数据
     * 
     * @param string $scanPath 扫描路径，格式：ModuleName::path，如 Weline_Theme::theme
     * @param string $namespace 命名空间（如果文件中有@meta.namespace，会覆盖此值）
     * @param callable|null $identifyGenerator 标识生成器回调函数
     * @param bool $strictMode 严格模式：如果文件不符合格式要求，会抛出异常
     * @return array 扫描结果数组，包含成功和失败的文件列表
     * @throws Exception 如果strictMode=true且文件不符合格式要求
     */
    public function scanPath(string $scanPath, string $namespace = '', bool $strictMode = true): array
    {
        // 解析路径格式：ModuleName::path
        if (strpos($scanPath, '::') === false) {
            throw new Exception(__('扫描路径格式错误，应为：ModuleName::path，如：Weline_Theme::theme'));
        }
        
        [$moduleName, $relativePath] = explode('::', $scanPath, 2);
        
        // 获取模块路径
        $modulePath = $this->getModulePath($moduleName);
        if (!$modulePath) {
            throw new Exception(__('模块不存在：%1', $moduleName));
        }
        
        $fullPath = rtrim($modulePath, '/') . '/' . ltrim($relativePath, '/');
        if (!is_dir($fullPath)) {
            throw new Exception(__('扫描路径不存在：%1', $fullPath));
        }
        
        $results = [
            'success' => [],
            'failed' => [],
            'skipped' => []
        ];
        
        // 递归扫描目录
        $iterator = new \RecursiveIteratorIterator(
            new \RecursiveDirectoryIterator($fullPath, \RecursiveDirectoryIterator::SKIP_DOTS)
        );
        
        foreach ($iterator as $file) {
            if ($file->isFile() && $file->getExtension() === 'phtml') {
                try {
                    $meta = $this->scanFile($file->getPathname(), $namespace, null, $strictMode);
                    if ($meta) {
                        $results['success'][] = $file->getPathname();
                    } else {
                        $results['skipped'][] = $file->getPathname();
                    }
                } catch (\Exception $e) {
                    $results['failed'][] = [
                        'file' => $file->getPathname(),
                        'error' => $e->getMessage()
                    ];
                    if ($strictMode) {
                        throw $e;
                    }
                }
            }
        }
        
        return $results;
    }

    /**
     * 扫描单个文件并提取元数据
     * 
     * @param string $filePath 文件路径
     * @param string $namespace 命名空间（如果文件中有@meta.namespace，会覆盖此值）
     * @param callable|null $identifyGenerator 标识生成器回调函数
     * @param bool $strictMode 严格模式：如果文件不符合格式要求，会抛出异常
     * @return Meta|null 返回Meta对象，如果文件没有元数据则返回null
     * @throws Exception 如果strictMode=true且文件不符合格式要求
     */
    public function scanFile(string $filePath, string $namespace = '', ?callable $identifyGenerator = null, bool $strictMode = true): ?Meta
    {
        if (!file_exists($filePath)) {
            throw new Exception(__('文件不存在：%1', $filePath));
        }

        $content = file_get_contents($filePath);
        $metaData = $this->extractMetadata($content, $filePath, $strictMode, $namespace);
        
        // 如果没有元数据，返回null（不报错）
        if (empty($metaData)) {
            return null;
        }
        
        // 验证必需字段
        $this->validateMetadata($metaData, $filePath, $strictMode);
        
        // 使用文件中的namespace或传入的namespace
        $finalNamespace = $metaData['namespace'] ?? $namespace;
        if (empty($finalNamespace)) {
            if ($strictMode) {
                throw new Exception(__('文件缺少命名空间定义：%1。请添加 @meta.namespace 或通过参数传入', $filePath));
            }
            return null;
        }

        // 生成标识
        $identify = $identifyGenerator ? $identifyGenerator($filePath, $metaData) : ($metaData['identify'] ?? $this->generateIdentify($filePath));
        
        // 创建或更新元数据
        $meta = ObjectManager::getInstance()->get(Meta::class);
        $meta->load([
            Meta::fields_NAMESPACE => $finalNamespace,
            Meta::fields_META_TYPE => $metaData['type'],
            Meta::fields_META_IDENTIFY => $identify
        ]);

        $meta->setData(Meta::fields_NAMESPACE, $finalNamespace);
        $meta->setData(Meta::fields_META_TYPE, $metaData['type']);
        $meta->setData(Meta::fields_META_IDENTIFY, $identify);
        $meta->setData(Meta::fields_FILE_PATH, $this->getRelativePath($filePath));
        $meta->setData(Meta::fields_FILE_FULL_PATH, $filePath);
        $meta->setData(Meta::fields_AREA, $metaData['area'] ?? null);
        $meta->setData(Meta::fields_CATEGORY, $metaData['category'] ?? null);
        $meta->setData(Meta::fields_META_DATA, json_encode($metaData, JSON_UNESCAPED_UNICODE));

        $meta->saveMeta($metaData);
        
        return $meta;
    }

    /**
     * 提取元数据（严格验证格式）
     */
    protected function extractMetadata(string $content, string $filePath, bool $strictMode, string $defaultNamespace = ''): array
    {
        $metaData = [];
        
        // 检查是否有元数据区域标识
        $hasMetaBlock = preg_match('/元数据区域|Metadata Block/', $content);
        
        // 第一步：提取 @meta.namespace（必须先提取，因为后续字段依赖命名空间）
        $namespace = $defaultNamespace;
        if (preg_match('/@meta\.namespace\s+(\S+)/', $content, $matches)) {
            $namespace = trim($matches[1]);
            $metaData['namespace'] = $namespace;
        } elseif (empty($namespace)) {
            // 如果没有命名空间且严格模式，报错
            if ($strictMode && $hasMetaBlock) {
                throw new Exception(__('文件缺少必需字段 @meta.namespace：%1', $filePath));
            }
            return []; // 没有命名空间，无法继续提取
        }
        
        // 第二步：使用命名空间提取其他字段
        // 提取 @meta.{namespace}.type（必需）
        if (preg_match('/@meta\.' . preg_quote($namespace, '/') . '\.type\s+(\S+)/', $content, $matches)) {
            $metaData['type'] = trim($matches[1]);
        } elseif ($strictMode && $hasMetaBlock) {
            throw new Exception(__('文件缺少必需字段 @meta.%1.type：%2', $namespace, $filePath));
        }
        
        // 提取 @meta.{namespace}.area
        if (preg_match('/@meta\.' . preg_quote($namespace, '/') . '\.area\s+(\S+)/', $content, $matches)) {
            $metaData['area'] = trim($matches[1]);
        }
        
        // 提取 @meta.{namespace}.category
        if (preg_match('/@meta\.' . preg_quote($namespace, '/') . '\.category\s+(\S+)/', $content, $matches)) {
            $metaData['category'] = trim($matches[1]);
        }
        
        // 提取 @meta.{namespace}.identify
        if (preg_match('/@meta\.' . preg_quote($namespace, '/') . '\.identify\s+(\S+)/', $content, $matches)) {
            $metaData['identify'] = trim($matches[1]);
        }
        
        // 提取 @meta.{namespace}.info.* 字段（至少需要name和description）
        if (preg_match_all('/@meta\.' . preg_quote($namespace, '/') . '\.info\.(\w+)\s+(.+)/', $content, $matches, PREG_SET_ORDER)) {
            foreach ($matches as $match) {
                $field = trim($match[1]);
                $value = trim($match[2]);
                // 提取值（格式：名称：值 或 描述：值）
                if (preg_match('/^[^：:]+[：:]\s*(.+)$/u', $value, $valueMatch)) {
                    $metaData['info'][$field] = trim($valueMatch[1]);
                } else {
                    $metaData['info'][$field] = $value;
                }
            }
        }
        
        // 严格模式：如果有元数据区域标识，必须至少有一个info字段
        if ($strictMode && $hasMetaBlock && empty($metaData['info'])) {
            throw new Exception(__('文件定义了元数据区域但缺少 @meta.%1.info.* 字段：%2', $namespace, $filePath));
        }
        
        return $metaData;
    }

    /**
     * 验证元数据格式
     */
    protected function validateMetadata(array $metaData, string $filePath, bool $strictMode): void
    {
        if ($strictMode) {
            // 必需字段验证
            if (empty($metaData['type'])) {
                throw new Exception(__('元数据缺少必需字段 type：%1', $filePath));
            }
            
            // info字段至少需要name
            if (empty($metaData['info']['name'])) {
                $namespace = $metaData['namespace'] ?? 'unknown';
                throw new Exception(__('元数据缺少必需字段 @meta.%1.info.name：%2', $namespace, $filePath));
            }
        }
    }

    /**
     * 获取模块路径
     */
    protected function getModulePath(string $moduleName): ?string
    {
        $moduleManager = ObjectManager::getInstance()->get(\Weline\Framework\Module\Manager::class);
        $module = $moduleManager->getModule($moduleName);
        return $module ? $module->getPath() : null;
    }

    /**
     * 生成标识（从文件路径）
     */
    protected function generateIdentify(string $filePath): string
    {
        $basename = basename($filePath, '.phtml');
        return str_replace(['/', '\\'], '_', $basename);
    }

    /**
     * 获取相对路径
     */
    protected function getRelativePath(string $filePath): string
    {
        $rootPath = BP . '/';
        if (strpos($filePath, $rootPath) === 0) {
            return substr($filePath, strlen($rootPath));
        }
        return $filePath;
    }
}
```

### 3. Meta Taglib (Taglib/Meta.php)

```php
<?php
declare(strict_types=1);

namespace Weline\Meta\Taglib;

use Weline\Framework\Manager\ObjectManager;
use Weline\I18n\Model\Locale\Dictionary as LocaleDictionary;
use Weline\Taglib\TaglibInterface;

class Meta implements TaglibInterface
{
    static public function name(): string
    {
        return 'meta';
    }

    static function tag(): bool
    {
        return false; // 不支持成对标签，只支持@meta{}格式
    }

    static function attr(): array
    {
        return []; // 不需要属性，通过@meta{}格式解析
    }

    static function callback(): callable
    {
        return function ($tag_key, $config, $tag_data, $attributes) {
            // 只支持 @meta{} 格式
            if ($tag_key !== '@tag{}') {
                return '';
            }

            $content = trim($tag_data[1] ?? '');
            if (empty($content)) {
                return '';
            }

            // 解析格式：@meta{key|"默认值"}
            $translationKey = '';
            $defaultValue = '';

            // 检查是否有默认值（用|分隔）
            if (strpos($content, '|') !== false) {
                [$keyPart, $defaultPart] = explode('|', $content, 2);
                $translationKey = trim($keyPart);
                $defaultValue = trim($defaultPart);
                // 移除引号（支持单引号和双引号）
                $defaultValue = trim($defaultValue, '\'"');
            } else {
                $translationKey = trim($content);
            }

            if (empty($translationKey)) {
                return $defaultValue;
            }

            // 如果翻译键不是以@meta::开头，自动添加
            if (!str_starts_with($translationKey, '@meta::')) {
                $translationKey = '@meta::' . $translationKey;
            }

            // 验证翻译键格式
            if (!preg_match('/^@meta::(.+)$/', $translationKey, $matches)) {
                return $defaultValue;
            }

            $keyParts = explode('.', $matches[1]);
            if (count($keyParts) < 5) {
                return $defaultValue;
            }

            // 获取当前语言
            $locale = \Weline\Framework\Http\Cookie::getLangLocal() ?? 'zh_Hans_CN';

            // 从I18n Dictionary获取翻译
            /** @var LocaleDictionary $localeDict */
            $localeDict = ObjectManager::getInstance()->get(LocaleDictionary::class);
            $md5 = LocaleDictionary::generateMd5($translationKey, $locale);
            $localeDict->load($md5, LocaleDictionary::fields_MD5);
            
            if ($localeDict->getId()) {
                $translation = $localeDict->getData(LocaleDictionary::fields_TRANSLATE);
                if (!empty($translation)) {
                    return $translation;
                }
            }
            
            // 优先级：翻译值 > 默认值
            return $defaultValue;
        };
    }
}
```

### 4. 翻译收集观察者 (I18n模块)

Meta模块通过触发 `Weline_I18n::collect_translations` 事件来提交翻译词，I18n模块的观察者会监听该事件并将翻译词存储到字典表中。

## Theme模块集成

### Theme模块特殊处理

Theme模块下的文件默认命名空间都是`theme`，类型根据目录自动识别。

### 使用方式

Theme模块有两种方式使用Meta模块：

#### 方式1：通过事件调用Meta的扫描器（推荐，规范化）

Theme模块通过事件触发Meta模块的扫描器，指定路径进行扫描。这种方式会严格按照Meta的格式要求验证文件，不符合格式的文件会报错。

**Theme观察者 (Theme/Observer/ScanThemeFiles.php)**

```php
<?php
declare(strict_types=1);

namespace Weline\Theme\Observer;

use Weline\Framework\Event\EventManager;
use Weline\Framework\Manager\ObjectManager;
use Weline\Meta\Service\Scanner;

class ScanThemeFiles
{
    /**
     * 扫描Theme模块的模板文件
     */
    public function execute(array $data): void
    {
        // 方式1：直接调用Meta的扫描器
        /** @var Scanner $scanner */
        $scanner = ObjectManager::getInstance()->get(Scanner::class);
        
        // 扫描Theme模块的theme目录
        // 路径格式：ModuleName::path
        $scanPath = 'Weline_Theme::view/theme';
        $namespace = 'theme'; // Theme模块默认命名空间
        
        try {
            $results = $scanner->scanPath($scanPath, $namespace, true); // strictMode=true，严格验证格式
            
            // 处理扫描结果
            if (!empty($results['failed'])) {
                foreach ($results['failed'] as $failed) {
                    // 记录错误日志
                    \Weline\Framework\App\State::getInstance()->getLogger()->error(
                        __('元数据扫描失败：%1 - %2', $failed['file'], $failed['error'])
                    );
                }
            }
        } catch (\Exception $e) {
            // 严格模式下，任何错误都会抛出异常
            \Weline\Framework\App\State::getInstance()->getLogger()->error(
                __('Theme元数据扫描失败：%1', $e->getMessage())
            );
        }
    }
}
```

#### 方式2：Theme模块自己实现扫描逻辑（更高灵活性）

Theme模块可以自己实现扫描逻辑，然后直接调用Meta的事件写入数据。这种方式更灵活，可以自定义扫描规则。

**Theme自定义扫描器 (Theme/Service/MetaScanner.php)**

```php
<?php
declare(strict_types=1);

namespace Weline\Theme\Service;

use Weline\Framework\Event\EventManager;
use Weline\Framework\Manager\ObjectManager;
use Weline\Meta\Model\Meta;

class MetaScanner
{
    /**
     * 扫描Theme模块的所有模板文件（自定义逻辑）
     */
    public function scanThemeFiles(): void
    {
        $themePath = BP . '/app/code/Weline/Theme/view/theme';
        $this->scanDirectory($themePath, 'theme');
    }

    /**
     * 递归扫描目录
     */
    protected function scanDirectory(string $dir, string $namespace): void
    {
        $iterator = new \RecursiveIteratorIterator(
            new \RecursiveDirectoryIterator($dir, \RecursiveDirectoryIterator::SKIP_DOTS)
        );

        foreach ($iterator as $file) {
            if ($file->isFile() && $file->getExtension() === 'phtml') {
                $filePath = $file->getPathname();
                
                // 根据目录结构确定类型
                $type = $this->determineType($filePath);
                
                // 生成标识
                $identify = $this->generateIdentify($filePath, $type);
                
                // 提取元数据（自定义逻辑）
                $metaData = $this->extractMetadata($filePath, $type, $identify);
                
                if (!empty($metaData)) {
                    // 直接调用Meta的事件写入数据
                    $this->saveMetaData($namespace, $type, $identify, $filePath, $metaData);
                }
            }
        }
    }

    /**
     * 提取元数据（自定义逻辑）
     */
    protected function extractMetadata(string $filePath, string $type, string $identify): array
    {
        $content = file_get_contents($filePath);
        $metaData = [];
        
        // 自定义提取逻辑
        // 例如：从文件名推断名称，从目录结构推断分类等
        
        return $metaData;
    }

    /**
     * 保存元数据（调用Meta的事件）
     */
    protected function saveMetaData(string $namespace, string $type, string $identify, string $filePath, array $metaData): void
    {
        /** @var Meta $meta */
        $meta = ObjectManager::getInstance()->get(Meta::class);
        $meta->load([
            Meta::fields_NAMESPACE => $namespace,
            Meta::fields_META_TYPE => $type,
            Meta::fields_META_IDENTIFY => $identify
        ]);

        $meta->setData(Meta::fields_NAMESPACE, $namespace);
        $meta->setData(Meta::fields_META_TYPE, $type);
        $meta->setData(Meta::fields_META_IDENTIFY, $identify);
        $meta->setData(Meta::fields_FILE_PATH, str_replace(BP . '/', '', $filePath));
        $meta->setData(Meta::fields_FILE_FULL_PATH, $filePath);
        $meta->setData(Meta::fields_AREA, $metaData['area'] ?? null);
        $meta->setData(Meta::fields_CATEGORY, $metaData['category'] ?? null);
        $meta->setData(Meta::fields_META_DATA, json_encode($metaData, JSON_UNESCAPED_UNICODE));

        // 调用Meta的事件保存并收集翻译
        $meta->saveMeta($metaData);
    }

    /**
     * 根据路径确定类型
     */
    protected function determineType(string $filePath): string
    {
        if (strpos($filePath, '/layouts/') !== false) {
            return 'layout';
        } elseif (strpos($filePath, '/components/') !== false) {
            return 'component';
        } elseif (strpos($filePath, '/partials/') !== false) {
            return 'partial';
        }
        return 'unknown';
    }

    /**
     * 生成标识
     */
    protected function generateIdentify(string $filePath, string $type): string
    {
        $relativePath = str_replace(BP . '/app/code/Weline/Theme/view/theme/', '', $filePath);
        $relativePath = str_replace('.phtml', '', $relativePath);
        
        // 移除类型目录
        $relativePath = str_replace(['layouts/', 'components/', 'partials/'], '', $relativePath);
        
        // 转换为标识格式
        return str_replace(['/', '\\'], '_', $relativePath);
    }
}
```

### 扫描控制器 (Controller/Backend/Scanner.php)

```php
<?php
declare(strict_types=1);

namespace Weline\Meta\Controller\Backend;

use Weline\Backend\Controller\BackendController;
use Weline\Framework\Manager\ObjectManager;
use Weline\Meta\Service\Scanner;

class Scanner extends BackendController
{
    /**
     * 扫描页面
     */
    public function index()
    {
        return $this->fetch();
    }

    /**
     * 执行扫描
     */
    public function scan()
    {
        $scanPath = $this->request->getPost('scan_path'); // 例如：Weline_Theme::view/theme
        $namespace = $this->request->getPost('namespace', 'theme');
        $strictMode = (bool)$this->request->getPost('strict_mode', true);
        
        if (empty($scanPath)) {
            $this->getMessageManager()->addError(__('请指定扫描路径'));
            $this->redirect('*/index');
            return;
        }
        
        try {
            // 触发扫描事件，让其他模块的观察者处理
            $this->getEventManager()->dispatch('meta_scan_path', [
                'scan_path' => $scanPath,
                'namespace' => $namespace,
                'strict_mode' => $strictMode
            ]);
            
            $this->getMessageManager()->addSuccess(__('扫描任务已提交，请查看扫描结果'));
        } catch (\Exception $e) {
            $this->getMessageManager()->addError(__('扫描失败：%1', $e->getMessage()));
        }
        
        $this->redirect('*/index');
    }

    /**
     * 直接扫描（不通过事件）
     */
    public function scanDirect()
    {
        $scanPath = $this->request->getPost('scan_path');
        $namespace = $this->request->getPost('namespace', 'theme');
        $strictMode = (bool)$this->request->getPost('strict_mode', true);
        
        if (empty($scanPath)) {
            $this->getMessageManager()->addError(__('请指定扫描路径'));
            $this->redirect('*/index');
            return;
        }
        
        try {
            /** @var Scanner $scanner */
            $scanner = ObjectManager::getInstance()->get(Scanner::class);
            $results = $scanner->scanPath($scanPath, $namespace, $strictMode);
            
            $successCount = count($results['success']);
            $failedCount = count($results['failed']);
            $skippedCount = count($results['skipped']);
            
            $this->getMessageManager()->addSuccess(__(
                '扫描完成：成功 %1 个，失败 %2 个，跳过 %3 个',
                $successCount,
                $failedCount,
                $skippedCount
            ));
            
            if ($failedCount > 0) {
                foreach ($results['failed'] as $failed) {
                    $this->getMessageManager()->addWarning(__(
                        '文件扫描失败：%1 - %2',
                        basename($failed['file']),
                        $failed['error']
                    ));
                }
            }
            
            $this->assign('results', $results);
        } catch (\Exception $e) {
            $this->getMessageManager()->addError(__('扫描失败：%1', $e->getMessage()));
        }
        
        return $this->fetch('templates/Backend/Scanner/results.phtml');
    }
}
```

## I18n集成

### 翻译键格式

```
@meta::{namespace}.{type}.{identify}.{group}.{field}
```

### 翻译存储

翻译存储在I18n模块的表中：
- `i18n_dictionary`: 存储词汇（word）
- `i18n_locale_dictionary`: 存储翻译（word, locale_code, translate）

### 使用示例

在模板中使用Meta标签：

```php
<!-- 使用完整的翻译键 -->
@meta{@meta::theme.layout.account_auth.info.name|个人中心认证页面布局}

@meta{theme.layout.account_auth.info.name|个人中心认证页面布局}

<?php $metaKey = 'theme.layout.account_auth.info.name'; ?>
@meta{<?= $metaKey ?>|个人中心认证页面布局}
```

**注意**：默认值不需要引号包裹，直接使用 `|` 后面的内容即可。
