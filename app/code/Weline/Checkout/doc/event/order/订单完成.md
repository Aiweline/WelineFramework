# Weline_Checkout::order::completed - 订单完成

## 事件说明

当订单状态变更为已完成时触发，允许其他模块执行完成后的操作，如增加积分、发送通知等。

## 触发时机

在 `OrderService::updateOrderStatus()` 方法中，当状态变更为 `completed` 时。

## 数据格式

```php
[
    'order_id' => 1,
    'order' => Order对象
]
```

## 可用数据

- `order_id` (int) - 订单ID
- `order` (Order) - 订单对象

## 使用场景

- 增加用户积分
- 发送订单完成通知
- 触发评价提醒
- 更新统计数据

## 使用示例

```php
namespace Weline\YourModule\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class OrderCompletedObserver implements ObserverInterface
{
    public function execute(Event &$event): void
    {
        $data = $event->getData();
        $order = $data['order'];
        
        // 增加用户积分
        $points = (int)($order->getTotalAmount() * 0.01);
        $this->addUserPoints($order->getCustomerId(), $points);
        
        // 发送订单完成通知
        $this->sendOrderCompleteNotification($order);
        
        // 触发评价提醒
        $this->triggerReviewReminder($order);
    }
}
```

## 注意事项

- 订单状态已更新为已完成
- 建议在观察者中进行异步操作，避免阻塞主流程
