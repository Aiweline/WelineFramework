# Weline_Checkout::order::refunded - 订单退款

## 事件说明

当订单被退款时触发，允许其他模块执行退款后的操作，如更新账户余额、发送通知等。

## 触发时机

在 `OrderService::updateOrderStatus()` 方法中，当状态变更为 `refunded` 时。

## 数据格式

```php
[
    'order_id' => 1,
    'order' => Order对象
]
```

## 可用数据

- `order_id` (int) - 订单ID
- `order` (Order) - 订单对象

## 使用场景

- 更新账户余额
- 发送退款通知
- 记录退款日志
- 更新统计数据

## 使用示例

```php
namespace Weline\YourModule\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class OrderRefundedObserver implements ObserverInterface
{
    public function execute(Event &$event): void
    {
        $data = $event->getData();
        $order = $data['order'];
        
        // 更新账户余额
        $this->updateAccountBalance($order->getCustomerId(), $order->getTotalAmount());
        
        // 发送退款通知
        $this->sendRefundNotification($order);
        
        // 记录退款日志
        error_log("订单退款: {$order->getOrderNumber()}, 金额: {$order->getTotalAmount()}");
    }
}
```

## 注意事项

- 订单状态已更新为已退款
- 建议在观察者中进行异步操作，避免阻塞主流程
