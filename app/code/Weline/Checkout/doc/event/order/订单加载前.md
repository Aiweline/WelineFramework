# Weline_Checkout::order::load::before - 订单加载前

## 事件说明

在加载订单数据之前触发，允许其他模块执行预处理操作。

## 触发时机

在 `OrderService::getOrder()` 方法中，执行订单加载之前。

## 数据格式

```php
[
    'order_id' => 1
]
```

## 可用数据

- `order_id` (int) - 订单ID

## 使用场景

- 记录订单访问日志
- 验证订单访问权限
- 执行预处理操作

## 使用示例

```php
namespace Weline\YourModule\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class OrderLoadBeforeObserver implements ObserverInterface
{
    public function execute(Event &$event): void
    {
        $data = $event->getData();
        $orderId = $data['order_id'];
        
        // 记录订单访问日志
        error_log("加载订单: {$orderId}");
    }
}
```

## 注意事项

- 此时订单尚未加载，无法访问订单数据
- 主要用于记录日志或权限验证
