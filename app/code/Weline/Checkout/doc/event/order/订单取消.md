# Weline_Checkout::order::cancelled - 订单取消

## 事件说明

当订单被取消时触发，允许其他模块执行取消后的操作，如恢复库存、处理退款等。

## 触发时机

在 `OrderService::updateOrderStatus()` 方法中，当状态变更为 `cancelled` 时。

## 数据格式

```php
[
    'order_id' => 1,
    'order' => Order对象
]
```

## 可用数据

- `order_id` (int) - 订单ID
- `order` (Order) - 订单对象

## 使用场景

- 恢复库存
- 处理退款
- 发送取消通知
- 更新统计数据

## 使用示例

```php
namespace Weline\YourModule\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class OrderCancelledObserver implements ObserverInterface
{
    public function execute(Event &$event): void
    {
        $data = $event->getData();
        $order = $data['order'];
        
        // 恢复库存
        $items = $order->getItems();
        foreach ($items as $item) {
            $this->restoreInventory($item['product_id'], $item['quantity']);
        }
        
        // 如果已支付，处理退款
        if ($order->isPaid()) {
            $this->processRefund($order);
        }
        
        // 发送取消通知
        $this->sendOrderCancelledNotification($order);
    }
}
```

## 注意事项

- 订单状态已更新为已取消
- 建议在观察者中进行异步操作，避免阻塞主流程
