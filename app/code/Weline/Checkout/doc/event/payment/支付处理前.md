# Weline_Checkout::payment::process::before - 支付处理前

## 事件说明

在支付处理之前触发，允许其他模块准备支付参数、选择支付网关、验证支付限额等。

## 触发时机

在 `PaymentService::processPayment()` 方法中，数据验证通过后、创建支付交易记录之前。

## 数据格式

```php
[
    'order_id' => 1,
    'order' => Order对象,
    'payment_method' => 'alipay',
    'payment_data' => [...]
]
```

## 可用数据

- `order_id` (int) - 订单ID
- `order` (Order) - 订单对象
- `payment_method` (string) - 支付方式
- `payment_data` (array, 引用传递) - 支付数据，可以在观察者中修改

## 使用场景

- 准备支付参数
- 选择支付网关
- 验证支付限额
- 添加支付配置

## 使用示例

```php
namespace Weline\YourModule\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class PaymentProcessBeforeObserver implements ObserverInterface
{
    public function execute(Event &$event): void
    {
        $data = $event->getData();
        $order = $data['order'];
        $paymentData = &$data['payment_data'];
        
        // 验证支付限额
        $amount = $order->getTotalAmount();
        if ($amount > 50000) {
            throw new \Exception('支付金额超过限额');
        }
        
        // 准备支付参数
        $paymentData['payment_params'] = [
            'out_trade_no' => $order->getOrderNumber(),
            'total_amount' => $amount,
            'subject' => '订单支付',
            'body' => '订单号：' . $order->getOrderNumber()
        ];
    }
}
```

## 注意事项

- 支付数据通过引用传递，可以直接修改
- 可以抛出异常阻止支付处理
