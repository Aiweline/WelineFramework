# Weline_Checkout::checkout::calculate_totals::before - 计算订单总额前

## 事件说明

在计算订单总额之前触发，允许其他模块修改计算参数（运费、税费、折扣等）。

## 触发时机

在 `CheckoutService::calculateTotals()` 方法中，执行总额计算之前。

## 数据格式

```php
[
    'items' => [
        ['quantity' => 2, 'price' => 100.00],
        // ... 其他订单项
    ],
    'shipping_amount' => 10.00,
    'tax_amount' => 0.00,
    'discount_amount' => 5.00
]
```

## 可用数据

- `items` (array, 引用传递) - 订单项数组，可以在观察者中修改
- `shipping_amount` (float, 引用传递) - 运费，可以在观察者中修改
- `tax_amount` (float, 引用传递) - 税费，可以在观察者中修改
- `discount_amount` (float, 引用传递) - 折扣金额，可以在观察者中修改

## 使用场景

- 根据订单项计算运费
- 根据地区计算税费
- 应用折扣或优惠
- 修改订单项价格

## 使用示例

```php
namespace Weline\YourModule\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class CalculateTotalsBeforeObserver implements ObserverInterface
{
    public function execute(Event &$event): void
    {
        $data = $event->getData();
        $items = &$data['items'];
        $shippingAmount = &$data['shipping_amount'];
        $discountAmount = &$data['discount_amount'];
        
        // 根据订单项计算运费
        $totalWeight = 0;
        foreach ($items as $item) {
            $totalWeight += ($item['weight'] ?? 0) * ($item['quantity'] ?? 1);
        }
        $shippingAmount = $this->calculateShipping($totalWeight);
        
        // 应用会员折扣
        $discountAmount += $this->calculateMemberDiscount($items);
    }
}
```

## 注意事项

- 所有参数通过引用传递，可以直接修改
- 修改后的参数会影响总额计算
- 建议保持数据一致性，避免计算错误
