# 布局切换后事件 (layout_switch_after)

## 事件名称
`Weline_Layout::layout_switch_after`

## 触发时机
在布局切换后触发，可用于清除缓存、更新索引等。

## 事件参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| module_code | string | 模块代码 |
| layout_type | string | 布局类型 |
| layout_code | string | 布局代码 |

## 使用场景

- 清除旧布局相关缓存
- 更新布局索引
- 记录布局切换日志
- 同步布局信息到其他系统

## 注册观察者

在模块的 `etc/event.xml` 中注册：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="Weline_Layout::layout_switch_after">
        <observer name="Your_Module::layout_switch_after"
                  instance="Your\Module\Observer\LayoutSwitchAfterObserver"
                  disabled="false"
                  shared="true"
                  sort="0"/>
    </event>
</config>
```
