# 布局计划触发事件 (layout_schedule_trigger)

## 事件名称
`Weline_Layout::layout_schedule_trigger`

## 触发时机
布局定时计划触发时触发。

## 事件参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| schedule_id | int | 计划ID |
| layout_id | int | 布局ID |
| module_code | string | 模块代码 |
| layout_type | string | 布局类型 |
| layout_code | string | 布局代码 |

## 使用场景

- 记录计划触发日志
- 执行计划相关的后续操作
- 通知相关人员
- 同步计划状态到其他系统

## 观察者示例

```php
<?php
namespace YourModule\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class LayoutScheduleTriggerObserver implements ObserverInterface
{
    public function execute(Event &$event): void
    {
        $data = $event->getData();
        $scheduleId = $data['schedule_id'] ?? 0;
        $layoutId = $data['layout_id'] ?? 0;
        $moduleCode = $data['module_code'] ?? '';
        $layoutType = $data['layout_type'] ?? '';
        $layoutCode = $data['layout_code'] ?? '';
        
        // 记录计划触发日志
        $this->logScheduleTrigger($scheduleId, $layoutId, $moduleCode, $layoutType, $layoutCode);
        
        // 执行后续操作
        $this->handleScheduleTrigger($scheduleId, $layoutId, $moduleCode, $layoutType, $layoutCode);
    }
    
    private function logScheduleTrigger(int $scheduleId, int $layoutId, string $moduleCode, string $layoutType, string $layoutCode): void
    {
        // 记录日志逻辑
    }
    
    private function handleScheduleTrigger(int $scheduleId, int $layoutId, string $moduleCode, string $layoutType, string $layoutCode): void
    {
        // 处理计划触发逻辑
    }
}
```

## 注册观察者

在模块的 `etc/event.xml` 中注册：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="Weline_Layout::layout_schedule_trigger">
        <observer name="Your_Module::layout_schedule_trigger"
                  instance="Your\Module\Observer\LayoutScheduleTriggerObserver"
                  disabled="false"
                  shared="true"
                  sort="0"/>
    </event>
</config>
```
