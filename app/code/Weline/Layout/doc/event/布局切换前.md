# 布局切换前事件 (layout_switch_before)

## 事件名称
`Weline_Layout::layout_switch_before`

## 触发时机
在布局切换之前触发，可用于验证或准备工作。

## 事件参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| module_code | string | 模块代码 |
| layout_type | string | 布局类型 |
| old_layout | string | 旧布局代码 |
| new_layout | string | 新布局代码 |

## 使用场景

- 验证布局切换是否允许
- 执行布局切换前的准备工作
- 记录布局切换日志
- 清理旧布局相关缓存

## 观察者示例

```php
use Weline\Framework\Event\ObserverInterface;

        // 验证布局切换是否允许

            throw new \Exception('不允许切换到此布局');
        }
        
        // 执行切换前的准备工作
        $this->prepareLayoutSwitch($moduleCode, $layoutType, $oldLayout, $newLayout);
    }
    
    private function canSwitchLayout(string $moduleCode, string $layoutType, string $oldLayout, string $newLayout): bool
    {
        // 验证逻辑
        return true;
    }
    
    private function prepareLayoutSwitch(string $moduleCode, string $layoutType, string $oldLayout, string $newLayout): void
    {
        // 准备工作逻辑
    }
}
```

## 注册观察者

在模块的 `etc/event.xml` 中注册：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="Weline_Layout::layout_switch_before">
        <observer name="Your_Module::layout_switch_before"
                  instance="Your\Module\Observer\LayoutSwitchBeforeObserver"
                  disabled="false"
                  shared="true"
                  sort="0"/>
    </event>
</config>
```
