# Currency 模块需求文档

## 一、功能需求

### 1. 货币管理
- 支持货币的增删改查
- 货币基本信息：代码、名称、汇率、符号、位置、格式、状态
- 货币格式化配置：图标、千分位分隔符、小数分隔符、基准货币

### 2. 货币导入
- 支持从第三方API导入汇率
- 支持自动导入（Cron定时任务）
- 支持手动导入
- 导入时自动更新货币格式化信息

### 3. 货币格式化
- 支持货币符号位置调整（前面/后面）
- 支持千分位分隔符配置（逗号、点、空格等）
- 支持小数分隔符配置
- 支持货币图标显示

### 4. 配置管理
- 导入开关配置
- API渠道选择
- API密钥配置
- Cron执行时间配置
- 基准货币设置

## 二、技术实现

### 1. 数据库扩展
- 新增字段：icon、thousand_separator、decimal_separator、base_currency
- 支持数据库升级和迁移

### 2. 第三方API集成
- 实现exchangerate-api.com免费API
- 支持错误处理和重试机制
- 支持API限流

### 3. Cron定时任务
- 实现定期自动导入汇率
- 支持配置开关控制
- 记录执行日志

### 4. 缓存机制
- 货币数据缓存
- 格式化结果缓存
- 数据变更时自动清除缓存

## 三、API接口说明

### ExchangeRateApiInterface
- `getExchangeRates()`: 获取实时汇率
- `getExchangeRate()`: 获取单个货币汇率
- `getSupportedCurrencies()`: 获取支持的货币列表
- `getCurrencyInfo()`: 获取货币详细信息
- `testConnection()`: 测试API连接

## 四、使用示例

### 格式化货币金额
```php
use Weline\Currency\Helper\CurrencyFormatter;

// 格式化金额
$formatted = CurrencyFormatter::format(1234.56, 'USD');
// 输出: $1,234.56
```

### 获取货币信息
```php
use Weline\Currency\Data\CurrencyData;

// 获取货币信息
$currency = CurrencyData::getCurrency('USD');
// 返回: ['code' => 'USD', 'name' => '美元', 'rate' => 7.2, ...]
```

### 手动导入汇率
```php
use Weline\Currency\Service\CurrencyImportService;
use Weline\Currency\Api\ExchangeRateApi;

$api = new ExchangeRateApi();
$importService = new CurrencyImportService($api);
$result = $importService->importAll('CNY');
```

## 五、配置说明

### 配置项
- `import_enabled`: 是否启用自动导入（true/false）
- `import_provider`: 导入渠道（exchangerate-api）
- `import_api_key`: API密钥（可选）
- `import_cron_time`: Cron执行时间表达式（默认：0 2 * * *）
- `base_currency`: 基准货币代码（默认：CNY）
