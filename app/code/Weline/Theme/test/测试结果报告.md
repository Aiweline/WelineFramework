## 🔧 修复的问题

### 1. CLI环境兼容性问题

**问题**：`$_SERVER['REQUEST_SCHEME']` 在CLI环境下不存在

**修复**：
```php
// 修复前
$baseUrl = rtrim($_SERVER['REQUEST_SCHEME'] . '://' . $_SERVER['HTTP_HOST'] . dirname($_SERVER['SCRIPT_NAME']), '/');

// 修复后
$scheme = $_SERVER['REQUEST_SCHEME'] ?? ($_SERVER['HTTPS'] ?? '' === 'on' ? 'https' : 'http');
$host = $_SERVER['HTTP_HOST'] ?? 'localhost';
$scriptName = $_SERVER['SCRIPT_NAME'] ?? '/';
$baseUrl = rtrim($scheme . '://' . $host . dirname($scriptName), '/');
```

### 2. Windows路径格式兼容性问题

**问题**：测试断言期望路径包含 "pub/static"，但Windows使用反斜杠

**修复**：
```php
// 修复前
$this->assertStringContainsString('pub/static', $cssPath);

// 修复后
$this->assertTrue(
    strpos($cssPath, 'pub' . DS . 'static') !== false || 
    strpos($cssPath, 'pub/static') !== false,
    '路径应包含 pub/static 或 pub' . DS . 'static'
);
```

### 3. Risky测试问题

**问题**：`testVariableGrouping` 没有执行任何断言

**修复**：添加了适当的断言来验证变量分组功能
