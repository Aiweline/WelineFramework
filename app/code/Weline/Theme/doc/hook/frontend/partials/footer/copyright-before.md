# Weline Theme 模块 - Hook 文档

## 概述

本文档详细说明了 Weline Theme 模块提供的 `Weline_Theme::frontend::partials::footer::copyright-before` Hook 及其使用方法。

## Hook 信息

- **Hook 名称**：`Weline_Theme::frontend::partials::footer::copyright-before`
- **显示名称**：页脚版权信息之前
- **功能说明**：在渲染页脚版权信息之前触发，允许其他模块在版权信息开始处注入内容。

## 使用方法

在模块的 `view/hooks/` 目录下创建文件：`view/hooks/Weline_Theme--frontend--partials--footer--copyright-before.phtml`
