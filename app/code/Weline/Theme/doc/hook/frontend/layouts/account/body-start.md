# Weline Theme 模块 - Hook 文档

## Hook 信息

- **Hook 名称**：`Weline_Theme::frontend::layouts::account::body-start`
- **显示名称**：账户布局 Body 开始
- **功能说明**：在渲染账户布局的 <body> 标签开始处触发，允许其他模块在 body 开始处注入内容。

## 使用方法

在模块的 `view/hooks/` 目录下创建文件：`view/hooks/Weline_Theme--frontend--layouts--account--body-start.phtml`
