# 分类筛选侧栏部件

## 概述

分类筛选侧栏部件用于分类页左侧，提供属性筛选功能，支持属性分组、数量展示与已选条件展示（亚马逊风格）。

## 基本信息

- **部件类型**：`sidebar`
- **部件代码**：`category-filters`
- **适用区域**：`frontend`（前端）
- **版本**：`1.0.0`
- **模块**：`Weline_Theme`

## 功能特性

- ✅ 支持属性分组展示
- ✅ 支持数量展示（可配置）
- ✅ 支持已选条件展示
- ✅ 亚马逊风格 UI 设计
- ✅ 支持自定义筛选 HTML
- ✅ 响应式设计，移动端自适应

## 使用方法

### 在模板中使用

```phtml
<w:widget type="sidebar" name="category-filters"/>
```

### 传入自定义筛选 HTML

```phtml
<w:widget type="sidebar" name="category-filters" params='{"html":"<div>自定义筛选内容</div>"}'/>
```

## 参数说明

### html（可选）

- **类型**：`string`
- **默认值**：`''`
- **说明**：业务模块渲染好的筛选HTML。如果为空，则使用默认亚马逊风格示例内容。

## 样式说明

部件使用的 CSS 类名（由主题 layouts 提供样式）：

- `.category-filter-mock` - 筛选容器
- `.category-filter-header` - 筛选头部
- `.category-filter-title` - 筛选标题
- `.category-filter-clear` - 清除全部链接
- `.category-filter-applied` - 已选条件容器
- `.filter-chip` - 筛选标签
- `.chip-remove` - 移除按钮
- `.category-filter-group` - 筛选分组
- `.category-filter-group-title` - 分组标题
- `.category-filter-list` - 筛选列表
- `.category-filter-item` - 筛选项
- `.label` - 标签文本
- `.count` - 数量显示
- `.stars` - 评分星星

## 模板位置

- **模板文件**：`app/code/Weline/Theme/view/theme/frontend/widgets/category-filters/default.phtml`

## 示例

### 基本使用

```phtml
<aside class="category-filter-sidebar">
    <w:widget type="sidebar" name="category-filters"/>
</aside>
```

### 传入自定义内容

```phtml
<w:widget type="sidebar" name="category-filters" params='{"html":"<div class=\"custom-filters\">自定义筛选内容</div>"}'/>
```

## 开发说明

### 扩展筛选功能

如果需要扩展筛选功能，可以通过以下方式：

1. **通过参数传入 HTML**：使用 `html` 参数传入业务模块渲染的筛选 HTML
2. **修改模板**：直接修改 `default.phtml` 模板文件
3. **创建 Block 类**：在 widget.php 中指定 `block_class`，使用 Block 类处理复杂逻辑

### 样式定制

样式由主题 layouts 提供，如需定制样式，可以在布局文件的 `<style>` 标签中覆盖相关 CSS 类。
