# 翻译包创建指南

本文档介绍如何创建 Weline Framework 的翻译包（语言包）。

## 目录结构

翻译包统一存放在 `app/i18n/` 目录下，结构如下：

```
app/i18n/
├── {厂商名}/
│   ├── {语言代码}/
│   │   └── register.php
│   └── {语言代码}/
│       └── register.php
```

### 示例

```
app/i18n/
├── Weline/
│   ├── en_US/
│   │   └── register.php
│   └── zh_Hans_CN/
│       └── register.php
├── MyCompany/
│   ├── ja_JP/
│   │   └── register.php
│   └── ko_KR/
│       └── register.php
```

## 语言代码格式

语言代码遵循 **IETF BCP 47** 标准，格式为：`{语言}_{文字}_{地区}`

### 常用语言代码

| 语言代码 | 语言 | 国家/地区 |
|---------|------|----------|
| `zh_Hans_CN` | 简体中文 | 中国 |
| `zh_Hant_TW` | 繁体中文 | 台湾 |
| `zh_Hant_HK` | 繁体中文 | 香港 |
| `en_US` | 英语 | 美国 |
| `en_GB` | 英语 | 英国 |
| `ja_JP` | 日语 | 日本 |
| `ko_KR` | 韩语 | 韩国 |
| `fr_FR` | 法语 | 法国 |
| `de_DE` | 德语 | 德国 |
| `es_ES` | 西班牙语 | 西班牙 |
| `pt_BR` | 葡萄牙语 | 巴西 |
| `ru_RU` | 俄语 | 俄罗斯 |
| `ar_SA` | 阿拉伯语 | 沙特阿拉伯 |
| `th_TH` | 泰语 | 泰国 |
| `vi_VN` | 越南语 | 越南 |

## 创建翻译包

### 步骤 1：创建目录

```bash
mkdir -p app/i18n/{厂商名}/{语言代码}
```

例如，创建日语翻译包：

```bash
mkdir -p app/i18n/Weline/ja_JP
```

### 步骤 2：创建 register.php

在翻译包目录下创建 `register.php` 文件：

```php
<?php

use Weline\Framework\Register\Register;

Register::register(
    Register::I18N,      // 注册类型：翻译包
    __DIR__,             // 翻译包目录
    '1.0.0',             // 版本号
    '日语翻译包'          // 描述
);
```

### 步骤 3：执行系统升级

创建完成后，执行系统升级命令使翻译包生效：

```bash
php bin/m system:upgrade
```

## 完整示例

### 创建日语翻译包

1. 创建目录：
```bash
mkdir -p app/i18n/Weline/ja_JP
```

2. 创建 `app/i18n/Weline/ja_JP/register.php`：
```php
<?php

/*
 * 日语翻译包
 * 作者：Your Name
 * 邮箱：your@email.com
 */

use Weline\Framework\Register\Register;

Register::register(
    Register::I18N,
    __DIR__,
    '1.0.0',
    '日语翻译包'
);
```

3. 执行升级：
```bash
php bin/m system:upgrade
```

## 翻译文件

翻译包注册后，系统会在后台【国际化 > 区域管理】中显示对应的语言选项。

翻译词条存储在数据库中，可通过以下方式管理：

1. **后台管理**：国际化 > 翻译词条管理
2. **CSV 导入**：使用 AI 翻译功能批量导入
3. **命令行收集**：`php bin/m i18n:collect` 收集系统中的待翻译词条

## 注意事项

1. **厂商名**：建议使用公司或项目名称，避免与其他翻译包冲突
2. **语言代码**：必须使用标准的语言代码格式
3. **版本号**：遵循语义化版本规范（Semantic Versioning）
4. **默认语言**：系统默认安装 `en_US` 和 `zh_Hans_CN` 两个翻译包

## 相关命令

```bash
# 系统升级（安装翻译包）
php bin/m system:upgrade

# 收集翻译词条
php bin/m i18n:collect

# 查看已安装的区域
php bin/m i18n:locals
```
