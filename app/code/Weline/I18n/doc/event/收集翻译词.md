# Weline I18n 模块 - 事件文档

## 概述

本文档详细说明了 Weline I18n 模块提供的 `Weline_I18n::collect_translations` 事件及其使用方法。该事件用于收集翻译词，允许其他模块提交翻译词到I18n字典表中。

## 事件列表

### 1. Weline_I18n::collect_translations - 收集翻译词事件

#### 基本信息

- **事件名称**：`Weline_I18n::collect_translations`
- **事件类型**：国际化事件
- **触发时机**：当其他模块需要收集翻译词时
- **观察者类**：`Weline\I18n\Observer\CollectTranslations`
- **配置文件**：`app/code/Weline/I18n/etc/event.xml`

#### 功能说明

`Weline_I18n::collect_translations` 事件用于收集翻译词，当其他模块需要将翻译词提交到I18n字典表时触发。I18n模块的观察者会监听该事件并将翻译词存储到字典表中。

该事件主要用于：
- 收集模块中的翻译词
- 将翻译词存储到I18n字典表
- 支持多语言翻译管理
- 自动提取和收集翻译内容

#### 触发时机

该事件由其他模块主动触发，通常在以下场景中：
- 保存元数据时收集翻译词
- 模块安装时收集翻译词
- 资源编译后收集翻译词
- 模板编译后收集翻译词

#### 使用场景

- **元数据翻译**：在保存元数据时收集翻译词
- **模块翻译**：在模块安装或升级时收集翻译词
- **资源翻译**：在资源编译后收集翻译词
- **模板翻译**：在模板编译后收集翻译词
- **动态翻译**：动态生成的内容需要翻译时

#### 使用方法

##### 基本用法

在其他模块中触发事件：

```php
use Weline\Framework\Event\EventsManager;
use Weline\Framework\Manager\ObjectManager;

// 获取事件管理器实例
/** @var EventsManager $eventsManager */
$eventsManager = ObjectManager::getInstance(EventsManager::class);

// 触发翻译收集事件
$eventsManager->dispatch('Weline_I18n::collect_translations', [
    'translations' => [
        [
            'word' => '@meta::namespace.type.identify.info.name',
            'translate' => '翻译值',
            'module' => 'Weline_Meta'  // 可选：标识翻译词来源模块
        ],
        [
            'word' => 'another.translation.key',
            'translate' => '另一个翻译值'
        ]
    ],
    'module' => 'Weline_YourModule'  // 可选：全局模块标识
]);
```

##### 在模型中使用

```php
<?php

namespace Your\Module\Model;

use Weline\Framework\Database\AbstractModel;

class YourModel extends AbstractModel
{
    public function saveMeta(array $metaData): static
    {
        // 保存数据
        $this->save();
        
        // 收集翻译词
        $translations = [];
        if (!empty($metaData['info']) && is_array($metaData['info'])) {
            foreach ($metaData['info'] as $field => $value) {
                if (!empty($value)) {
                    $translations[] = [
                        'word' => $this->generateTranslationKey('info', $field),
                        'translate' => $value,
                        'module' => 'Weline_YourModule'
                    ];
                }
            }
        }
        
        // 触发翻译收集事件
        if (!empty($translations)) {
            $this->getEventManager()->dispatch('Weline_I18n::collect_translations', [
                'translations' => $translations,
                'module' => 'Weline_YourModule'
            ]);
        }
        
        return $this;
    }
}
```

#### 事件数据格式

`Weline_I18n::collect_translations` 事件传递的数据：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| translations | array | 是 | 翻译词数组，每个元素包含 `word` 和 `translate` |
| module | string | 否 | 翻译词来源模块，用于标识翻译词的来源 |

**翻译词数组格式**：

```php
[
    'translations' => [
        [
            'word' => '翻译键',        // 必填：翻译键，用于标识翻译词
            'translate' => '翻译值',   // 必填：翻译值，实际的翻译内容
            'module' => 'Weline_Meta'  // 可选：翻译词来源模块
        ],
        // ... 更多翻译词
    ],
    'module' => 'Weline_YourModule'  // 可选：全局模块标识
]
```

#### 观察者处理

I18n模块的观察者会自动处理该事件：

1. **数据验证**：检查翻译词数据格式是否正确
2. **去重处理**：检查翻译词是否已存在于字典表中
3. **存储翻译**：将翻译词存储到 `i18n_dictionary` 表
4. **模块关联**：关联翻译词与来源模块

#### 注意事项

1. **数据格式**：翻译词数组必须包含 `word` 和 `translate` 字段
2. **翻译键格式**：建议使用有意义的翻译键格式，如 `@module::namespace.type.field`
3. **模块标识**：建议提供 `module` 字段，便于管理和追踪翻译词来源
4. **性能考虑**：批量提交翻译词时，建议一次性提交多个翻译词
5. **错误处理**：观察者会自动处理错误，不会影响主流程
6. **去重机制**：如果翻译词已存在，观察者会自动跳过
