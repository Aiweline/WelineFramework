# Geo 模块快速入门指南

## 🎯 5分钟快速上手

本指南将帮助您在5分钟内完成第一个Feed的创建和推送。

---

## 步骤1：安装模块（1分钟）

```bash
php bin/m module:upgrade Weline_Geo
```

---

## 步骤2：配置平台和账户（2分钟）

### 2.1 添加平台

1. 访问后台：**生成式搜索引擎优化 > 平台管理**
2. 点击 **"添加平台"**
3. 填写：
   - 平台代码：选择 `perplexity`（示例）
   - 平台名称：`Perplexity AI`
   - 是否启用：✓
4. 保存

### 2.2 添加账户

1. 在平台列表中，点击 **"账户"** 按钮
2. 点击 **"添加账户"**
3. 填写：
   - 账户名称：`我的Perplexity账户`
   - API密钥：输入您的API密钥
   - 设为默认账户：✓
   - 激活：✓
4. 保存
5. 点击 **"测试连接"** 验证

---

## 步骤3：创建Feed（1分钟）

1. 访问：**生成式搜索引擎优化 > Feed管理**
2. 点击 **"添加Feed"**
3. 填写：
   - Feed名称：`我的内容Feed`
   - Feed类型：`内容`
   - 数据源类型：`数据库`
   - 数据源配置：
   ```json
   {
     "table": "cms_page",
     "fields": {
       "title": "title",
       "content": "content",
       "url": "url"
     },
     "where": {
       "status": 1
     }
   }
   ```
   - 更新频率：`每天`
   - 是否启用：✓
4. 保存

---

## 步骤4：推送Feed（1分钟）

### 方式1：一键推送（推荐）

1. 访问：**生成式搜索引擎优化 > 推送管理 > 一键推送**
2. 选择Feed：`我的内容Feed`
3. 选择平台：`Perplexity AI` ✓
4. 点击 **"开始推送"**

### 方式2：命令行推送

```bash
php bin/m geo:feed:push 1
```

---

## ✅ 完成！

恭喜！您已经成功完成了第一个Feed的推送。
