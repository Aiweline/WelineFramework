# Sitemap 管理前端使用指南

## 快速开始

访问：`后台 → SEO管理 → Sitemap管理`

---

## 🎨 界面功能

### 顶部工具栏

```
[搜索站点...] [展开所有 ↓] [折叠所有 ↑]
```

**功能说明：**
- **搜索框**：实时搜索站点（支持名称、代码、域名）
- **展开所有**：一键展开当前页所有站点
- **折叠所有**：一键折叠当前页所有站点

---

## 📋 使用场景

### 场景 1：查找特定站点

1. 在搜索框输入站点名称或域名（如：`default`）
2. 系统自动过滤匹配的站点
3. 点击站点卡片头部展开查看详情

### 场景 2：浏览所有站点

1. 使用页面底部的分页控件切换页码
2. 每页显示 5 个站点
3. 点击站点卡片查看平台分组和文件

### 场景 3：批量查看站点

1. 点击顶部"展开所有"按钮
2. 当前页所有站点同时展开
3. 查看完毕后点击"折叠所有"恢复

### 场景 4：复制 sitemap URL

1. 展开站点查看平台
2. 找到目标文件（如平台索引）
3. 点击"复制"按钮
4. URL 自动复制到剪贴板

## 🎯 操作指南

### 生成 Sitemap

**方法 1：调用所有生成器**
1. 点击左侧"调用所有生成器"按钮
2. 系统自动同步 URL 并生成所有站点的 sitemap
3. 成功后页面自动刷新

**方法 2：命令行生成**
```bash
php bin/w sitemap:test
```

### 提交到搜索引擎

**手动提交：**
1. 展开目标站点
2. 找到对应平台（如 Google）
3. 点击"复制提交URL"按钮
4. 前往搜索引擎控制台粘贴提交

**自动提交：**
- 配置 SEO 账户和站点绑定
- Cron 定时任务自动提交
- 查看提交日志验证

### 查看 Sitemap 内容

**方式 1：点击查看按钮**
- 直接在浏览器新标签页打开
- 查看 XML 原始内容

**方式 2：访问 URL**
- 跨站总索引：`/sitemaps/sitemap.xml`
- 平台索引：`/sitemaps/{website}/{platform}/sitemap.xml`
- 模块文件：`/sitemaps/{website}/{platform}/sitemap_{module}_{n}.xml`

## ⚙️ 高级功能

### 搜索技巧

**精确搜索：**
- 输入完整站点代码：`default`
- 输入域名：`example.com`

**模糊搜索：**
- 输入部分关键词：`shop`
- 匹配所有包含该词的站点

### 分页配置

**默认配置：**
- 每页 5 个站点
- 最多显示 7 个页码按钮

**修改配置：**
在模板中调整 `ITEMS_PER_PAGE` 常量

### 键盘快捷键（建议）

- `Ctrl + F`：快速聚焦搜索框
- `Esc`：清空搜索
- `↑/↓`：切换页码

## 🐛 故障排查

### 问题：没有显示站点

**原因：** 未生成 sitemap 或数据库无 URL

**解决：**
1. 点击"调用所有生成器"
2. 或运行 `php bin/w sitemap:test`
3. 刷新页面

### 问题：搜索无结果

**原因：** 输入的关键词不匹配

**解决：**
1. 检查关键词拼写
2. 尝试搜索站点代码而非名称
3. 清空搜索查看所有站点

### 问题：分页不显示

**原因：** 站点数量 ≤ 5 个

**解决：** 这是正常的，少量站点不需要分页

## 🚀 性能建议

**建议操作：**

2. 避免频繁"展开所有"

- 分页切换：< 50ms
- 展开折叠：< 200ms
