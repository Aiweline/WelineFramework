# SEO 平台绑定功能说明

## 功能概述

已实现 SEO 账户与平台适配器的关联逻辑，**只为站点绑定的平台生成 Sitemap**，未绑定的平台不会生成任何文件。

---

## ✅ 已实现的功能

### 1. **核心逻辑修复**

**问题：** 之前的实现中，如果站点没有绑定 SEO 账户，系统会默认为所有平台（Google、Bing、百度）生成 Sitemap。

```php
// 旧逻辑（错误）
if (empty($adapters)) {
    $adapters = $this->adapterRegistry->getAdapters(); // ❌ 返回所有平台
}
```

**修复：** 移除默认返回所有适配器的逻辑，未绑定账户的站点返回空数组。

```php
// 新逻辑（正确）
if (empty($bindings)) {
    return []; // ✅ 返回空数组，不生成任何 sitemap
}
```

### 2. **数据库schema更新**

#### `weline_seo_account` 表

**新增字段：**
- `platform` (VARCHAR(50), NOT NULL) - Sitemap 平台代码（google/bing/baidu）
- **索引：** `idx_platform`

**升级逻辑：**
```php
public function upgrade(ModelSetup $setup, Context $context): void
{
    // 添加 platform 字段（v1.0.1）
    if (!$setup->columnExist(self::fields_PLATFORM)) {
        $setup->addColumn(
            self::fields_PLATFORM,
            TableInterface::column_type_VARCHAR,
            50,
            'not null default ""',
            'Sitemap平台代码：google/bing/baidu等',
            self::fields_MODULE
        );
    }
}
```

#### `weline_seo_website_account` 表

**索引修改：**
- **删除：** `unique_website` （单一站点唯一索引）
- **新增：** `unique_website_account` （站点-账户组合唯一索引）
- **新增：** `idx_website` （站点索引）

**支持一个站点绑定多个 SEO 账户（多平台）。**

### 3. **模型方法更新**

#### `SeoAccount` 模型

```php
// 获取平台代码
public function getPlatform(): string

// 设置平台代码
public function setPlatform(string $platform): self
```

#### `SeoWebsiteAccount` 模型

```php
// 根据站点ID获取所有绑定（支持多个）
public function getByWebsiteId(int $websiteId): array

// 根据站点ID和账户ID获取唯一绑定
public function getByWebsiteAndAccount(int $websiteId, int $accountId): ?self

// 绑定站点与账户
public function bindWebsiteAccount(int $websiteId, int $accountId, bool $isAutoSubmit = true): self

// 解绑站点与账户
public function unbindWebsiteAccount(int $websiteId, int $accountId): bool

// 获取站点绑定的所有平台代码
public function getWebsitePlatforms(int $websiteId): array
```

### 4. **服务层更新**

#### `WebSitemapData` 服务

```php
/**
 * 获取站点绑定的平台适配器列表
 *
 * @param int $websiteId
 * @return SitemapPlatformAdapterInterface[] [platform_code => adapter, ...]
 */
public function getWebsiteAdapters(int $websiteId): array
{
    $bindings = $this->seoWebsiteAccountModel->reset()
        ->where(SeoWebsiteAccount::fields_WEBSITE_ID, $websiteId)
        ->select()
        ->fetchArray();
    
    if (empty($bindings)) {
        // 🔥 重要：如果站点没有绑定 SEO 账户，返回空数组
        // 不生成任何平台的 sitemap
        return [];
    }
    
    // ... 遍历绑定，获取对应的平台适配器
    // 优先使用账户的 platform 字段
    // 如果为空，尝试从 provider 推断（向后兼容）
    
    return $adapters;
}
```

## 📊 使用场景

### 场景 1：站点绑定单一平台

```
站点: default
绑定: Google 账户

结果:
  pub/sitemaps/default/
  └── google/
      ├── sitemap.xml
      └── sitemap_builder_1.xml

```

```
绑定: Google 账户 + Bing 账户

结果:
  pub/sitemaps/my/
  ├── google/
  │   ├── sitemap.xml
  │   └── sitemap_builder_1.xml
  └── bing/
      ├── sitemap.xml
      └── sitemap_builder_1.xml

不生成: baidu/
```

### 场景 3：站点未绑定任何平台

```
站点: shop_us
绑定: 无

结果:
  不生成任何 sitemap 文件
```

## ⚙️ 配置步骤

### 1. 运行数据库升级

```bash
php bin/w setup:upgrade
```

**这将：**
- 给 `weline_seo_account` 表添加 `platform` 字段
- 修改 `weline_seo_website_account` 表的索引（支持多平台绑定）

### 2. 更新现有账户（可选）

如果系统中已有 SEO 账户，需要手动设置 `platform` 字段：

```sql
UPDATE weline_seo_account 
SET platform = 'google' 
WHERE provider LIKE '%google%';

UPDATE weline_seo_account 
SET platform = 'bing' 
WHERE provider LIKE '%bing%';

UPDATE weline_seo_account 
SET platform = 'baidu' 
WHERE provider LIKE '%baidu%';
```

或者使用框架代码：

```php
$accounts = $seoAccountModel->reset()->select()->fetchArray();
foreach ($accounts as $row) {
    $account = $seoAccountModel->reset()->load($row['account_id']);
    
    // 从 provider 推断平台
    $provider = strtolower($account->getData(SeoAccount::fields_PROVIDER));
    if (strpos($provider, 'google') !== false) {
        $account->setPlatform('google')->save();
    } elseif (strpos($provider, 'bing') !== false) {
        $account->setPlatform('bing')->save();
    } elseif (strpos($provider, 'baidu') !== false) {
        $account->setPlatform('baidu')->save();
    }
}
```

### 3. 绑定站点与账户

在后台 SEO 配置界面或使用代码：

```php
use Weline\Seo\Model\SeoWebsiteAccount;
use Weline\Framework\Manager\ObjectManager;

$binding = ObjectManager::getInstance(SeoWebsiteAccount::class);

// 绑定站点 1 到 Google 账户 ID 1
$binding->bindWebsiteAccount(
    websiteId: 1,
    accountId: 1,
    isAutoSubmit: true
);

// 绑定站点 1 到 Bing 账户 ID 2（支持多平台）
$binding->bindWebsiteAccount(
    websiteId: 1,
    accountId: 2,
    isAutoSubmit: true
);
```
