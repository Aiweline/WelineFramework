## 国际化规范遵循

✅ **基础语言是中文**: 所有代码中的文本都是中文  
✅ **使用 `__()` 函数**: 所有用户可见文本都使用 `__()` 函数包装  
✅ **占位符格式**: 使用 `%{1}`、`%{2}` 等命名占位符格式  
✅ **翻译文件格式**: CSV格式，中文列保持原样，英文列提供翻译  
✅ **文件位置正确**: 翻译文件放在模块的 `i18n/` 目录下  
✅ **不需要 register.php**: 模块翻译文件不需要注册文件  

## 注意事项

1. **数据库注释**: 数据库表的注释（`createTable()` 和 `addColumn()` 中的注释）不需要翻译，这些是给数据库管理员看的
2. **菜单配置**: 菜单配置文件（`menu.xml`）中的 title 属性由框架自动处理国际化
3. **代码注释**: PHP代码中的注释（`/** */` 和 `//`）不需要翻译
4. **变量名和常量**: 变量名、常量名、类名等不需要翻译
