# i18n 批量更新脚本说明

由于条件类文件较多，可以使用以下模式批量更新剩余文件。

## 更新模式

### 1. getName() 方法
```php
// 查找
public function getName(): string
{
    return '文本';
}

// 替换为
public function getName(): string
{
    return __('文本');
}
```

### 2. getDescription() 方法
```php
// 查找
public function getDescription(): string
{
    return '描述文本';
}

// 替换为
public function getDescription(): string
{
    return __('描述文本');
}
```

### 3. getFormFields() 中的 label
```php
// 查找
'label' => '标签文本',

// 替换为
'label' => __('标签文本'),
```

### 4. getFormFields() 中的 options
```php
// 查找
'options' => [
    'key' => '选项文本',
],

// 替换为
'options' => [
    'key' => __('选项文本'),
],
```

## 快速更新方法

可以使用 IDE 的查找替换功能（支持正则表达式）：

1. 查找：`return '([^']+)';`（在 getName 或 getDescription 方法中）
2. 替换：`return __('$1');`

注意：需要逐个文件检查，确保替换正确。
