    'description' => '测试描述'

**英文翻译**:
```php
[
    'id' => $ruleId,
    'local_code' => 'en_US',

    'description' => 'Test Description'
]
```

## 边界情况和异常处理

### 边界情况

1. **空翻译**: 当翻译记录不存在时，应回退到原始字段值
2. **多语言**: 支持同时存在多个语言的翻译记录
3. **大量数据**: 测试在大量规则和翻译数据下的性能
4. **特殊字符**: 测试包含特殊字符的翻译内容

### 异常处理

1. **规则不存在**: 查询不存在的规则ID时，应返回空结果
2. **翻译不存在**: 加载不存在的翻译时，应回退到原始字段
3. **数据库错误**: 数据库操作失败时，应正确处理异常

## 测试覆盖率要求

- **模型层**: ≥ 90%
- **控制器层**: ≥ 80%
- **视图层**: ≥ 70%
- **整体覆盖率**: ≥ 85%
