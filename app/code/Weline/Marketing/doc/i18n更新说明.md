## 更新模式

### getName() 方法
```php
// 更新前
public function getName(): string
{
    return '客户组';
}

// 更新后
public function getName(): string
{
    return __('客户组');
}
```

### getDescription() 方法
```php
// 更新前
public function getDescription(): string
{
    return '根据客户所属的客户组进行条件判断';
}

// 更新后
public function getDescription(): string
{
    return __('根据客户所属的客户组进行条件判断');
}
```

### getFormFields() 方法
```php
// 更新前
'label' => '操作符',
'options' => [
    'in' => '属于',
    'not_in' => '不属于',
],

// 更新后
'label' => __('操作符'),
'options' => [
    'in' => __('属于'),
    'not_in' => __('不属于'),
],
```

### 动作执行返回消息
```php
// 更新前
'messages' => [sprintf('享受 %.2f%% 折扣，优惠 %.2f 元', $discountValue, $discountAmount)],

// 更新后
'messages' => [sprintf(__('享受 %.2f%% 折扣，优惠 %.2f 元'), $discountValue, $discountAmount)],
```

## 注意事项

1. 数据库表的注释（`createTable()` 和 `addColumn()` 中的注释）不需要翻译，这些是给数据库管理员看的
2. 所有用户可见的文本必须使用 `__()` 函数
3. 翻译文件中的中文列保持原样，英文列提供英文翻译
4. 占位符使用 `%{1}`、`%{2}` 格式，不要使用 `%1`、`%2`
