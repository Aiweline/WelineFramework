## 🎯 模块简介

`Weline_AutoLeadAgent` 是一个基于零信任架构的智能寻客系统，提供：

### 核心功能
1. **自动寻客** - 结合店铺描述自动查找潜在客户
2. **安全防护** - 三层动态防御架构
3. **WASM核心** - 高性能算法核心
4. **LLM集成** - WebLLM (Phi-3 Mini) 智能分析
5. **多端支持** - Chrome插件 + Web兼容模式

### 技术栈
- **后端**: PHP 8.1+, WelineFramework
- **前端**: 原生 JavaScript, WebAssembly, WebLLM
- **WASM**: C/C++, Emscripten
- **Chrome插件**: Manifest V3
- **测试**: PHPUnit
