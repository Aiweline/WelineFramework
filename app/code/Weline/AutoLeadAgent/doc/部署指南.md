## 🔧 环境要求

### 服务器要求

- **PHP**: >= 8.1
- **数据库**: MySQL 5.8+ / MariaDB 10.6+ / SQLite 3
- **Web服务器**: Nginx / Apache
- **内存**: >= 512MB（推荐1GB+）
- **磁盘空间**: >= 100MB

### 依赖模块

- `Weline_Framework` - 核心框架
- `Weline_Store` - 店铺数据
- `Weline_I18n` - 国际化支持
- `Weline_Backend` - 后台管理

### 运行环境

- **Chrome 浏览器** - 任务执行需在 Chrome 内
- **Browser MCP 扩展** - 提供 browser_navigate、browser_snapshot、browser_extract 等工具

### 可选依赖

- **Emscripten SDK** - 用于编译 agent_brain WASM（Emscripten 路径）

## 📦 安装步骤

### 1. 安装模块

```bash
# 进入项目根目录
cd /path/to/weline-framework

# 运行模块安装
php bin/w setup:upgrade
```

### 2. 验证安装

```bash
# 检查模块是否已安装
php bin/w module:list | grep AutoLeadAgent

# 检查数据库表
php bin/w db:show-tables | grep auto_lead_agent
```

### 3. 配置环境变量

编辑 `app/etc/env.php`:

```php
return [
    'auto_lead_agent' => [
        'secret_key' => 'your-strong-secret-key-here',
        'allowed_domains' => [
            'your-domain.com',
            'www.your-domain.com',
        ],
        'default_token_ttl' => 3600,
    ],
];
```

## ⚙️ 配置说明

### 基础配置

#### 1. 密钥配置

**位置**: `app/etc/env.php`

```php
'auto_lead_agent' => [
    'secret_key' => 'your-strong-secret-key-here',
],
```

**安全建议**:
- 使用强随机密钥（至少32字符）
- 生产环境必须更改默认密钥
- 密钥应保密，不要提交到版本控制

#### 2. 域名白名单

```php
'allowed_domains' => [
    'your-domain.com',
    'www.your-domain.com',
],
```

**说明**:
- 只有白名单中的域名可以使用Agent
- 支持子域名
- 支持通配符（开发中）

#### 3. Token配置

```php
'default_token_ttl' => 3600, // 秒，默认1小时
```

**建议**:
- 生产环境: 3600秒（1小时）
- 开发环境: 86400秒（24小时）
- 测试环境: 7200秒（2小时）

### 数据库配置

模块会自动创建以下表：

- `weline_auto_lead_agent_token` - Token存储
- `weline_auto_lead_agent_wasm_hash` - WASM哈希记录
- `weline_auto_lead_agent_lead_candidate` - 潜在客户
- `weline_auto_lead_agent_search_task` - 搜索任务

**索引优化**:
- 所有表已创建必要索引
- 建议定期清理过期数据

### Web服务器配置

#### Nginx配置

```nginx
location /Weline/AutoLeadAgent/ {
    alias /path/to/weline-framework/app/code/Weline/AutoLeadAgent/view/statics/;
    
    # WASM文件
    location ~* \.wasm$ {
        add_header Content-Type application/wasm;
        add_header Cache-Control "public, max-age=31536000";
    }
    
    # JavaScript文件
    location ~* \.js$ {
        add_header Content-Type application/javascript;
        add_header Cache-Control "public, max-age=3600";
    }
}
```

#### Apache配置

```apache
<Directory "/path/to/weline-framework/app/code/Weline/AutoLeadAgent/view/statics/">
    Options Indexes FollowSymLinks
    AllowOverride All
    Require all granted
    
    # WASM文件
    <FilesMatch "\.wasm$">
        Header set Content-Type "application/wasm"
        Header set Cache-Control "public, max-age=31536000"
    </FilesMatch>
</Directory>
```

## 🔨 WASM编译

### 安装Emscripten

```bash
# 下载并安装Emscripten SDK
git clone https://github.com/emscripten-core/emsdk.git
cd emsdk
./emsdk install latest
./emsdk activate latest
source ./emsdk_env.sh
```

### 编译WASM

```bash
# 进入WASM源码目录
cd app/code/Weline/AutoLeadAgent/wasm-src

# 创建构建目录
mkdir build
cd build

# 配置CMake
emcmake cmake ..

# 编译
emmake make

# 复制WASM文件
cp agent_core.wasm ../../view/statics/wasm/agent-core.wasm
```

### 注册WASM哈希

编译完成后，需要注册WASM哈希：

```bash
# 自动注册（在安装脚本中）
php bin/w setup:upgrade

# 或手动注册
php -r "
require 'app/bootstrap.php';
\$wasmService = \Weline\Framework\Manager\ObjectManager::getInstance(\Weline\AutoLeadAgent\Service\WasmService::class);
\$hash = \$wasmService->registerHash();
echo 'WASM哈希已注册: ' . \$hash . PHP_EOL;
"
```

## 🔒 安全加固

### 1. HTTPS强制

生产环境必须使用HTTPS：

```nginx
# Nginx配置
if ($scheme != "https") {
    return 301 https://$host$request_uri;
}
```

### 2. Token安全

- 使用强随机密钥
- 定期轮换密钥
- 限制Token有效期
- 记录Token使用日志

### 3. 域名白名单

严格限制授权域名：

```php
'allowed_domains' => [
    'your-domain.com', // 只允许主域名
],
```

### 4. WASM验证

始终验证WASM文件哈希：

```php
// 在加载WASM前验证哈希
$expectedHash = $wasmService->getLatestHash();
$actualHash = hash_file('sha256', $wasmPath);
if ($expectedHash !== $actualHash) {
    throw new Exception('WASM文件已被篡改');
}
```

### 5. 输入验证

所有API输入都应验证：

```php
// 验证店铺ID
if (!is_numeric($storeId) || $storeId <= 0) {
    throw new InvalidArgumentException('Invalid store ID');
}
```

### 6. 日志记录

记录重要操作：

```php
// 记录Token生成
error_log("Token generated for domain: {$domain}");

// 记录搜索任务
error_log("Search task created: task_id={$taskId}, store_id={$storeId}");
```
