# Weline_Websites 域名管理架构文档

## 概述

域名管理系统为 Weline_Websites 模块提供完整的域名全生命周期管理能力，包括域名商管理、域名购买和证书管理集成。

## 架构总览

```
域名管理页面（Tab 布局）
├── Tab1: 域名商管理 — 配置 API 服务商（AWS、阿里云、Azure 等）
├── Tab2: 域名购买 — 批量检查可用性 → 绑定站点 → 批量购买
├── Tab3: 证书管理 — 由 Server 模块通过 Hook 注入
└── Tab N: 其他扩展 — 其他模块通过 Hook 注入
```

## 数据模型

### DomainRegistrar（域名商渠道）

| 字段 | 类型 | 说明 |
|------|------|------|
| registrar_id | INT PK | 域名商 ID |
| code | VARCHAR(100) UNIQUE | 适配器标识（如 aws_route53） |
| name | VARCHAR(255) | 显示名称 |
| description | VARCHAR(500) | 描述 |
| status | VARCHAR(20) | active / disabled |

### DomainRegistrarAccount（域名商账号）

| 字段 | 类型 | 说明 |
|------|------|------|
| account_id | INT PK | 账号 ID |
| registrar_id | INT FK | 关联域名商 ID |
| account_name | VARCHAR(255) | 账号名称 |
| api_key | VARCHAR(500) | API Key（base64 编码） |
| api_secret | VARCHAR(500) | API Secret（base64 编码） |
| region | VARCHAR(100) | 区域 |
| extra_config | TEXT | 额外配置 JSON |
| status | VARCHAR(20) | active / disabled |

### DomainPurchaseOrder（购买订单）

| 字段 | 类型 | 说明 |
|------|------|------|
| order_id | INT PK | 订单 ID |
| account_id | INT FK | 域名商账号 ID |
| order_no | VARCHAR(64) UNIQUE | 订单号（系统生成） |
| total_count | INT | 域名总数 |
| success_count | INT | 成功数 |
| fail_count | INT | 失败数 |
| status | VARCHAR(20) | pending / processing / completed / failed |

### DomainPurchaseItem（购买条目）

| 字段 | 类型 | 说明 |
|------|------|------|
| item_id | INT PK | 条目 ID |
| order_id | INT FK | 订单 ID |
| domain | VARCHAR(255) | 域名 |
| years | INT | 购买年限 |
| website_id | INT | 绑定站点 ID（0=不绑定） |
| auto_create_site | VARCHAR(10) | yes/no 是否自动建站 |
| status | VARCHAR(20) | pending / success / failed |
| error_message | VARCHAR(500) | 失败原因 |
| price | DECIMAL(10,2) | 价格 |
| currency | VARCHAR(10) | 币种 |

## 适配器扩展点

域名商适配器采用 Extends 扩展模式（参照 CDN 适配器架构）。

### 接口

`Weline\Websites\Api\DomainRegistrarInterface`

核心方法：
- `getRegistrarCode()` / `getRegistrarName()` - 标识和名称
- `getConfigFields()` - 动态配置表单字段定义
- `testConnection()` - 测试 API 连通性
- `checkAvailability()` / `batchCheckAvailability()` - 域名可用性检查
- `purchaseDomain()` - 购买域名
- `getDomainList()` / `getDomainDetail()` - 域名列表和详情

### 内置适配器

| 文件 | 代码 | 说明 |
|------|------|------|
| Adapter/AwsRoute53Registrar.php | aws_route53 | AWS Route53 |
| Adapter/AliyunDomainRegistrar.php | aliyun_domain | 阿里云域名 |
| Adapter/AzureDnsRegistrar.php | azure_dns | Azure DNS |

### 第三方扩展

其他模块可通过 `extends/module/Weline_Websites/Registrar/{Name}.php` 扩展新的域名商。

## Hook 扩展点

| Hook 名称 | 说明 |
|-----------|------|
| `Weline_Websites::backend::partials::domain::tabs` | Tab 导航按钮注入 |
| `Weline_Websites::backend::partials::domain::tabs-content` | Tab 内容面板注入 |

Server 模块通过这两个 Hook 将 SSL 证书管理 UI 注入到域名管理页面的第三个 Tab。

## 事件

| 事件名称 | 说明 |
|---------|------|
| `Weline_Websites::domain::purchase_success` | 域名购买成功后触发 |

## 购买流程

```
用户输入域名列表
    ↓
选择域名商账号
    ↓
调用适配器 batchCheckAvailability()
    ↓
用户选择可用域名 + 绑定站点选项
    ↓
创建购买订单 (DomainPurchaseOrder)
    ↓
逐个调用适配器 purchaseDomain()
    ↓
购买成功 → 入域名池 (DomainPool) → 可选绑定站点 (WebsiteDomain)
    ↓
触发 Weline_Websites::domain::purchase_success 事件
```
