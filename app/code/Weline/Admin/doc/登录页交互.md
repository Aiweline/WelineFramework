# 后台登录页交互说明

## 回车提交

登录表单在 `submit` 时不得同步 `disabled` 提交按钮；否则部分浏览器会取消回车隐式提交，按钮会卡在「登录中」。

正确做法：

1. 先进入 loading 态
2. `setTimeout(0)` 后再 `button.disabled = true`
3. 输入框 Enter 优先走 `form.requestSubmit(button)`
4. `pageshow`（含 bfcache）时复位 loading

实现：`view/templates/Login/index.phtml`、`view/templates/Login/fast.phtml`

## 登录框透明度

默认卡片背景为 `transparent`（无毛玻璃、无白雾叠层），输入约 `rgba(255,255,255,0.14)`，背景山水可透过登录框。`:hover` / `:focus-within` 后卡片与输入变为不透明可读态。

## 水墨 hover

登录卡片角为**水墨淡笔**：每次随机 **1–2 笔**、仅落在四角、单笔弧线、无晕染块。类名：`.weline-admin-login-card-ink` / `.weline-admin-login-ink-piece`。
