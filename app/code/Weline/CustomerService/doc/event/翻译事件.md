# Weline_CustomerService::translate - 客服消息翻译

## 事件说明

客服模块通过此事件请求翻译服务。事件数据包含待翻译文本、源语言、目标语言等。翻译模块可以监听此事件并提供翻译服务。

## 触发时机

在 `TranslationService::translate()` 或 `TranslationService::batchTranslate()` 方法中，当需要翻译客服消息时触发。

## 数据格式

### 单文本翻译

```php
[
    'text' => string,                 // 必需：待翻译文本
    'source_locale' => string,        // 必需：源语言（auto表示自动检测）
    'target_locale' => string,        // 必需：目标语言
    'context' => string,               // 可选：上下文（固定为'customer_service'）
    'session_id' => string|null,      // 可选：会话ID
]
```

### 批量翻译

```php
[
    'texts' => array,                 // 必需：待翻译文本数组
    'source_locale' => string,        // 必需：源语言
    'target_locale' => string,        // 必需：目标语言
    'context' => string,               // 可选：上下文（固定为'customer_service'）
    'session_id' => string|null,      // 可选：会话ID
]
```

## 可用数据

### 单文本翻译

- `text` (string) - 待翻译文本
- `source_locale` (string) - 源语言代码（auto表示自动检测）
- `target_locale` (string) - 目标语言代码
- `context` (string) - 上下文，固定为 `'customer_service'`
- `session_id` (string|null) - 会话ID（可选）

### 批量翻译

- `texts` (array) - 待翻译文本数组
- `source_locale` (string) - 源语言代码
- `target_locale` (string) - 目标语言代码
- `context` (string) - 上下文，固定为 `'customer_service'`
- `session_id` (string|null) - 会话ID（可选）

## 返回数据

### 单文本翻译

观察者应该设置以下字段：

- `translated_text` (string) - 翻译后的文本
- `success` (bool) - 是否成功
- `error` (string|null) - 错误信息（如果失败）

### 批量翻译

观察者应该设置以下字段：

- `translated_texts` (array) - 翻译后的文本数组
- `success` (bool) - 是否成功
- `error` (string|null) - 错误信息（如果失败）

## 使用场景

- 客服模块需要翻译客户消息
- 客服模块需要翻译客服回复
- 实现多语言客服支持

## 使用方法

### 在 event.xml 中注册观察者

```xml
<event name="Weline_CustomerService::translate">
    <observer name="Weline_TranslationService::customer_service_translate" 
              instance="Weline\TranslationService\Observer\CustomerServiceTranslationObserver" 
              disabled="false" 
              shared="true" 
              sort="10"/>
</event>
```

### 创建观察者类

```php
namespace Weline\TranslationService\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;
use Weline\TranslationService\Service\TranslationService;
use Weline\Framework\Manager\ObjectManager;

class CustomerServiceTranslationObserver implements ObserverInterface
{
    private TranslationService $translationService;
    
    public function __construct()
    {
        $this->translationService = ObjectManager::getInstance(TranslationService::class);
    }
    
    public function execute(Event &$event): void
    {
        $data = $event->getData();
        
        // 判断是单文本翻译还是批量翻译
        if (isset($data['text'])) {
            $this->handleSingleTranslation($event, $data);
        } elseif (isset($data['texts'])) {
            $this->handleBatchTranslation($event, $data);
        }
    }
    
    private function handleSingleTranslation(Event &$event, array $data): void
    {
        $text = $data['text'] ?? '';
        $sourceLocale = $data['source_locale'] ?? 'auto';
        $targetLocale = $data['target_locale'] ?? '';
        
        if (empty($text) || empty($targetLocale)) {
            $event->setData('success', false);
            $event->setData('error', '缺少必需参数');
            return;
        }
        
        try {
            // 调用翻译服务
            $translatedText = $this->translationService->translate(
                text: $text,
                targetLanguage: $targetLocale,
                sourceLanguage: $sourceLocale === 'auto' ? 'auto' : $sourceLocale
            );
            
            // 设置翻译结果
            $event->setData('translated_text', $translatedText);
            $event->setData('success', true);
        } catch (\Exception $e) {
            $event->setData('success', false);
            $event->setData('error', $e->getMessage());
        }
    }
    
    private function handleBatchTranslation(Event &$event, array $data): void
    {
        $texts = $data['texts'] ?? [];
        $sourceLocale = $data['source_locale'] ?? 'auto';
        $targetLocale = $data['target_locale'] ?? '';
        
        if (empty($texts) || empty($targetLocale)) {
            $event->setData('success', false);
            $event->setData('error', '缺少必需参数');
            return;
        }
        
        try {
            // 调用批量翻译服务
            $translatedTexts = $this->translationService->batchTranslate(
                texts: $texts,
                targetLanguage: $targetLocale,
                sourceLanguage: $sourceLocale === 'auto' ? 'auto' : $sourceLocale
            );
            
            // 设置翻译结果
            $event->setData('translated_texts', $translatedTexts);
            $event->setData('success', true);
        } catch (\Exception $e) {
            $event->setData('success', false);
            $event->setData('error', $e->getMessage());
        }
    }
}
```

## 使用示例

### 示例：客服模块使用翻译事件

```php
namespace Weline\CustomerService\Service;

use Weline\Framework\Event\EventsManager;
use Weline\Framework\Manager\ObjectManager;

class TranslationService
{
    private EventsManager $eventsManager;
    
    public function __construct(EventsManager $eventsManager)
    {
        $this->eventsManager = $eventsManager;
    }
    
    public function translate(string $text, string $targetLocale, string $sourceLocale = 'auto'): string
    {
        if (empty($text)) {
            return $text;
        }
        
        // 如果源语言和目标语言相同，不需要翻译
        if ($sourceLocale !== 'auto' && $sourceLocale === $targetLocale) {
            return $text;
        }
        
        // 准备事件数据
        $eventData = [
            'text' => $text,
            'source_locale' => $sourceLocale,
            'target_locale' => $targetLocale,
            'context' => 'customer_service',
        ];
        
        // 触发翻译事件
        $this->eventsManager->dispatch('Weline_CustomerService::translate', $eventData);
        
        // 从事件数据中获取翻译结果
        $translatedText = $eventData['translated_text'] ?? null;
        $success = $eventData['success'] ?? false;
        
        // 如果翻译成功，返回翻译结果；否则返回原文
        if ($success && !empty($translatedText)) {
            return $translatedText;
        }
        
        return $text;
    }
}
```

## 注意事项

- 翻译模块需要监听此事件并提供翻译服务
- 如果翻译失败，观察者应该设置 `success` 为 `false` 并设置 `error` 信息
- 如果翻译失败，客服模块会返回原文
- `context` 字段固定为 `'customer_service'`，可用于区分翻译场景
- `session_id` 可用于关联翻译记录和会话
