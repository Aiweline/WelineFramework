# 命令开发快速参考

本文档提供命令开发的快速参考，详细说明请参考 [命令开发指南.md](./命令开发指南.md)。

## 快速开始

### 1. 创建命令类

```php
<?php

namespace YourModule\Console\Your\Command;

use Weline\Framework\Console\CommandAbstract;
use Weline\Framework\Console\CommandHelper;
use Weline\Framework\Output\Cli\Printing;

class YourCommand extends CommandAbstract
{
    public function __construct(Printing $printer)
    {
        $this->printer = $printer;
    }

    public function execute(array $args = [], array $data = []): void
    {
        // 命令执行逻辑
        $this->printer->success('执行成功！');
    }

    public function tip(): string
    {
        return '命令简短描述';
    }

    public function help(): array|string
    {
        return CommandHelper::formatHelp(
            'your:command',
            '命令详细描述',
            ['-h, --help' => '显示帮助'],
            [],
            ['示例' => 'php bin/w your:command']
        );
    }
}
```

### 2. 文件位置

```
app/code/YourModule/
└── Console/
    └── Your/
        └── Command/
            └── YourCommand.php
```

### 3. 更新命令列表

```bash
php bin/w command:upgrade
```

## 命令别名

### 方式一：使用 aliases() 方法（推荐）

```php
public function aliases(): array
{
    return [
        'yc',           // 简短别名
        'your-cmd',     // 带连字符
        'y:cmd',        // 带命名空间
    ];
}
```

### 方式二：使用 ALIASES 常量

```php
public const ALIASES = [
    'yc',
    'your-cmd',
];
```

### 使用别名

```bash
# 原始命令
php bin/w your:command

# 使用别名
php bin/w yc
php bin/w your-cmd
```

## 参数处理

### 获取参数

```php
public function execute(array $args = [], array $data = []): void
{
    // 短选项
    $verbose = isset($args['v']) || isset($args['verbose']);
    
    // 长选项（带默认值）
    $name = $args['name'] ?? $args['n'] ?? 'default';
    
    // 位置参数
    $file = $data['file'] ?? null;
}
```

### 参数验证

```php
if (empty($args['name'])) {
    $this->printer->error('错误：必须指定 --name 参数');
    return;
}
```

## 帮助信息

### 使用 CommandHelper

```php
public function help(): array|string
{
    return CommandHelper::formatHelp(
        'command:name',              // 命令名
        '命令描述',                   // 描述
        [                            // 选项
            '-h, --help' => '显示帮助',
            '-v, --verbose' => '详细输出',
        ],
        [                            // 参数
            'file' => '文件路径',
        ],
        [                            // 示例
            '基本用法' => 'php bin/w command:name',
        ]
    );
}
```

## 输出方法

```php
// 成功消息
$this->printer->success('操作成功！');

// 错误消息
$this->printer->error('操作失败！');

// 警告消息
$this->printer->warning('请注意！');

// 提示消息
$this->printer->note('提示信息');

// 普通输出
$this->printer->print('普通文本');

// 打印列表
$this->printer->printList(['项1', '项2', '项3']);
```

## 命令命名规则

| 文件路径 | 命令名 |
|---------|--------|
| `Console/Server/Start.php` | `server:start` |
| `Console/Command/Upgrade.php` | `command:upgrade` |
| `Console/Cache/Clear.php` | `cache:clear` |

**规则**：
- 目录名 → 命令前缀（小写，冒号分隔）
- 类名 → 命令名（大写字母前加连字符，全小写）

## 常用命令

```bash
# 查看所有命令
php bin/w

# 查看命令帮助
php bin/w <command> --help

# 更新命令列表
php bin/w command:upgrade

# 启动开发服务器
php bin/w server:start

# 停止服务器
php bin/w server:stop
```

## 检查清单

创建命令时，请确保：

- [ ] 实现了 `CommandInterface` 或继承了 `CommandAbstract`
- [ ] 实现了 `execute()`, `tip()`, `help()` 方法
- [ ] 文件位置正确（模块的 `Console` 目录下）
- [ ] 命名空间正确
- [ ] （可选）实现了 `aliases()` 方法或定义了 `ALIASES` 常量
- [ ] 运行了 `php bin/w command:upgrade` 更新命令列表
- [ ] 测试了命令是否正常工作
