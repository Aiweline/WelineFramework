## 🌍 支持的语言

## 🔧 核心技术

### 1. 显示宽度计算

使用 PHP 的 `mb_strwidth()` 函数，正确计算东亚字符的显示宽度：

```php
protected function getDisplayWidth(string $text): int
{
    // 使用 mb_strwidth 获取东亚字符的显示宽度
    // 中文、日文等CJK字符占2个显示位置，英文占1个
    return mb_strwidth($text, 'UTF-8');
}
```

### 2. 字符宽度对照

| 字符类型 | 示例 | 字符数 | 显示宽度 |
|---------|------|--------|---------|
| 英文字母 | `Hello` | 5 | 5 |
| 中文汉字 | `你好` | 2 | 4 |
| 日文平假名 | `こんにちは` | 5 | 10 |
| 日文片假名 | `カタカナ` | 4 | 8 |
| 韩文 | `안녕하세요` | 5 | 10 |
| 数字 | `12345` | 5 | 5 |
| Emoji | `😀🎉` | 2 | 4 |

### 3. 智能截断

按照显示宽度进行截断，保证表格对齐：

```php
protected function truncateByDisplayWidth(string $text, int $width): string
{
    if ($this->getDisplayWidth($text) <= $width) {
        return $text;
    }
    
    // 使用 mb_strimwidth 按显示宽度截断
    return mb_strimwidth($text, 0, $width, '', 'UTF-8');
}
```

## 📋 关键改进点

### 1. 宽度计算方法改变

```php
// ❌ 修复前
$len = mb_strlen($text);  // 只计算字符数量，中文=1

// ✅ 修复后
$width = mb_strwidth($text, 'UTF-8');  // 计算显示宽度，中文=2
```

### 2. 截断方法改变

```php
// ❌ 修复前
$truncated = mb_substr($text, 0, $maxLen);  // 按字符数截断

// ✅ 修复后
$truncated = mb_strimwidth($text, 0, $maxWidth, '', 'UTF-8');  // 按显示宽度截断
```

### 3. 填充计算改变

```php
// ❌ 修复前
$padding = $columnWidth - mb_strlen($text);  // 可能计算错误

// ✅ 修复后
$padding = $columnWidth - mb_strwidth($text, 'UTF-8');  // 精确计算
```

## 🔧 使用的 PHP 函数

### mb_strwidth()
```php
mb_strwidth(string $string, ?string $encoding = null): int
```
- 返回字符串的显示宽度
- 中文、日文等 CJK 字符返回 2
- 英文、数字返回 1
- 完美支持终端显示

### mb_strimwidth()
```php
mb_strimwidth(
    string $string,
    int $start,
    int $width,
    string $trim_marker = "",
    ?string $encoding = null
): string
```
- 按显示宽度截断字符串
- 自动处理多字节字符
- 不会截断字符中间

### mb_substr()
```php
mb_substr(
    string $string,
    int $start,
    ?int $length = null,
    ?string $encoding = null
): string
```
- 按字符数截取
- 用于辅助处理
