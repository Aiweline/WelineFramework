# 表格自适应宽度说明

## 📏 自适应原则

表格宽度**根据内容自动调整**，而不是填满整个终端宽度。

### 核心逻辑

```
表格宽度 = 内容最大宽度 + 边距
```

- ✅ **优先**：根据实际内容宽度自适应
- ✅ **限制**：不超过终端宽度
- ✅ **截断**：超长内容智能截断

## 🎯 三种场景

### 场景 1：内容较短 → 紧凑显示

```
┌──────────────────┐
│   服务器信息     │
├──────┬───────────┤
│ PID  │ 12345     │  ← 表格宽度刚好够显示内容
├──────┼───────────┤
│ 端口 │ 9981      │
└──────┴───────────┘
```

**特点**：
- 表格宽度：约 20 字符
- 不会填满终端（假设终端宽度 120）
- 紧凑、清晰

### 场景 2：内容适中 → 自适应扩展

```
┌───────────────────────────────────┐
│          访问地址                 │
├──────────┬────────────────────────┤
│ 前端首页 │ http://localhost:9981/ │  ← 根据URL长度自动扩展
├──────────┼────────────────────────┤
│ 后端管理 │ http://localhost/admin │
└──────────┴────────────────────────┘
```

**特点**：
- 表格宽度：约 40 字符
- 根据最长内容（URL）自动调整
- 不浪费空间

### 场景 3：内容超长 → 智能截断

```
┌──────────────────────────────────────────────────────┐
│                    访问地址                          │
├──────────┬───────────────────────────────────────────┤
│ 后端管理 │ http://127.0.0.1:9981/U@Ma5p...1Tg1E8... │  ← 超过终端宽度时才截断
└──────────┴───────────────────────────────────────────┘
```

**特点**：
- 表格宽度：终端宽度（120）
- URL 被智能截断
- 保留关键信息（域名、路径开头/结尾）

## 💻 代码实现

### 1. 计算内容宽度

```php
// 计算所有内容的最大显示宽度
$maxKeyWidth = $this->getDisplayWidth($title);
$maxValueWidth = 0;

foreach ($data as $row) {
    $keyWidth = $this->getDisplayWidth((string)$row[0]);
    $valueWidth = $this->getDisplayWidth((string)$row[1]);
    if ($keyWidth > $maxKeyWidth) $maxKeyWidth = $keyWidth;
    if ($valueWidth > $maxValueWidth) $maxValueWidth = $valueWidth;
}
```

### 2. 根据内容设置列宽

```php
// 优先使用内容的实际宽度
$keyWidth = $maxKeyWidth + 2;      // 左右各留1个空格
$valueWidth = $maxValueWidth + 2;  // 左右各留1个空格
$totalWidth = $keyWidth + $valueWidth + 3;  // +3是边框宽度
```

### 3. 检查是否超出终端

```php
// 只有超出终端宽度时才进行调整
if ($totalWidth > $maxWidth) {
    // 按比例缩减
    $keyWidth = max(15, min($keyWidth, (int)($maxWidth * 0.3)));
    $valueWidth = $maxWidth - $keyWidth - 5;
    $totalWidth = $maxWidth;
}
```

## 📐 宽度计算公式

```
表格总宽度 = max(内容宽度, 标题宽度) + 边距

其中：
- 内容宽度 = 键列最大宽度 + 值列最大宽度 + 4（左右空格）
- 标题宽度 = 标题文字宽度 + 4（左右空格）
- 边距 = 3（三个边框字符 │）

限制条件：
if (表格总宽度 > 终端宽度) {
    表格总宽度 = 终端宽度
    内容进行智能截断
}
```
