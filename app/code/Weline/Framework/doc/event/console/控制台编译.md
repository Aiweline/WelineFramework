# Weline Framework 模块 - 事件文档

## 概述

本文档详细说明了 Weline Framework 模块提供的 `Weline_Framework_Console::compile` 事件及其使用方法。该事件在控制台编译时触发，允许其他模块执行编译相关操作。

## 事件列表

### 1. Weline_Framework_Console::compile - 控制台编译事件

#### 基本信息

- **事件名称**：`Weline_Framework_Console::compile`
- **事件类型**：控制台事件
- **触发时机**：在执行控制台编译命令时
- **触发位置**：`app/code/Weline/Framework/Setup/Console/Setup/Di/Compile.php` 第 25 行
- **配置文件**：`app/code/Weline/Framework/etc/event.xml`

#### 功能说明

`Weline_Framework_Console::compile` 事件在控制台编译时触发，允许其他模块执行编译相关操作。

该事件主要用于：
- 执行编译相关操作
- 生成编译文件
- 清理编译缓存
- 编译后处理

#### 触发时机

```php
// app/code/Weline/Framework/Setup/Console/Setup/Di/Compile.php
$evenManager->dispatch('Weline_Framework_Console::compile');
```

#### 使用场景

- **编译处理**：执行模块特定的编译操作
- **生成文件**：生成编译后的文件
- **清理缓存**：清理编译相关的缓存
- **编译后处理**：执行编译后的处理逻辑

#### 使用方法

##### 基本用法

在模块的 `etc/event.xml` 文件中注册观察者：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="Weline_Framework_Console::compile">
        <observer name="Your_Module::compile_handler"
                  instance="Your\Module\Observer\CompileHandlerObserver"
                  disabled="false"
                  shared="true"
                  sort="100"/>
    </event>
</config>
```

创建观察者类：

```php
<?php

namespace Your\Module\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class CompileHandlerObserver implements ObserverInterface
{
    /**
     * 执行观察者逻辑
     *
     * @param Event $event
     * @return void
     */
    public function execute(Event &$event): void
    {
        // 执行编译操作
        // ...
    }
}
```

#### 事件数据

`Weline_Framework_Console::compile` 事件不传递额外的数据。

#### 注意事项

1. **执行顺序**：观察者按照 `sort` 属性排序执行，数值越小越先执行
2. **性能考虑**：该事件在编译时触发，可以执行耗时操作

- **事件定义**：`app/code/Weline/Framework/event.php`
