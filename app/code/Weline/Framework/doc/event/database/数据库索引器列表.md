# Weline Framework 模块 - 事件文档

## 概述

本文档详细说明了 Weline Framework 模块提供的 `Framework_Database::indexer_listing` 事件及其使用方法。该事件在获取数据库索引器列表时触发，允许其他模块添加自定义索引器。

## 事件列表

### 1. Framework_Database::indexer_listing - 数据库索引器列表事件

#### 基本信息

- **事件名称**：`Framework_Database::indexer_listing`
- **事件类型**：数据库事件
- **触发时机**：在获取数据库索引器列表时
- **触发位置**：`app/code/Weline/Framework/Database/Console/Index/Listing.php` 第 47 行
- **配置文件**：`app/code/Weline/Framework/etc/event.xml`

#### 功能说明

`Framework_Database::indexer_listing` 事件在获取数据库索引器列表时触发，允许其他模块添加自定义索引器。可以通过设置 `break` 为 `true` 来阻止框架默认的索引器列表操作。

该事件主要用于：
- 添加自定义索引器
- 修改索引器列表
- 索引器列表处理

#### 触发时机

```php
// app/code/Weline/Framework/Database/Console/Index/Listing.php
$params = new DataObject(['args' => $args, 'break' => false]);
$eventManager->dispatch('Framework_Database::indexer_listing', $params);
if ($params->getData('break')) {
    return;
}
```

#### 使用场景

- **添加索引器**：添加自定义索引器到列表
- **修改列表**：修改索引器列表
- **列表处理**：处理索引器列表

#### 使用方法

##### 基本用法

在模块的 `etc/event.xml` 文件中注册观察者：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="Framework_Database::indexer_listing">
        <observer name="Your_Module::add_custom_indexers"
                  instance="Your\Module\Observer\AddCustomIndexersObserver"
                  disabled="false"
                  shared="true"
                  sort="100"/>
    </event>
</config>
```

创建观察者类：

```php
<?php

namespace Your\Module\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class AddCustomIndexersObserver implements ObserverInterface
{
    /**
     * 执行观察者逻辑
     *
     * @param Event $event
     * @return void
     */
    public function execute(Event &$event): void
    {
        $args = $event->getData('args');
        
        // 添加自定义索引器
        // ...
    }
}
```

#### 事件数据

`Framework_Database::indexer_listing` 事件传递的数据：

| 字段 | 类型 | 说明 |
|------|------|------|
| args | array | 命令参数 |
| break | bool | 是否阻止框架默认操作，设置为 true 可阻止 |

**获取和修改数据**：

```php
$args = $event->getData('args');
$break = $event->getData('break');

// 设置 break 来阻止框架默认操作
$event->setData('break', true);
```

#### 注意事项

1. **执行顺序**：观察者按照 `sort` 属性排序执行，数值越小越先执行
2. **break设置**：如果设置 `break` 为 `true`，框架默认的索引器列表操作将被跳过
