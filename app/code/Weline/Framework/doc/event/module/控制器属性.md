# Weline Framework 模块 - 事件文档

## 概述

本文档详细说明了 Weline Framework 模块提供的 `Weline_Framework_Module::controller_attributes` 事件及其使用方法。该事件在解析控制器属性时触发，允许其他模块修改控制器属性。

## 事件列表

### 1. Weline_Framework_Module::controller_attributes - 控制器属性事件

#### 基本信息

- **事件名称**：`Weline_Framework_Module::controller_attributes`
- **事件类型**：模块事件
- **触发时机**：在解析控制器属性时
- **触发位置**：`app/code/Weline/Framework/Module/Helper/Data.php` 第 177 行等多处
- **配置文件**：`app/code/Weline/Framework/etc/event.xml`

#### 功能说明

`Weline_Framework_Module::controller_attributes` 事件在解析控制器属性时触发，允许其他模块修改控制器属性。

该事件主要用于：
- 修改控制器属性
- 添加控制器属性
- 控制器属性验证
- 控制器属性处理

#### 触发时机

```php
// app/code/Weline/Framework/Module/Helper/Data.php
$data->setData('controller_data', $ctl_data);
$data->setData('params', $params);
$this->getEvenManager()->dispatch('Weline_Framework_Module::controller_attributes', $data);
```

#### 使用场景

- **属性修改**：修改控制器属性
- **属性添加**：添加控制器属性
- **属性验证**：验证控制器属性
- **属性处理**：处理控制器属性

#### 使用方法

##### 基本用法

在模块的 `etc/event.xml` 文件中注册观察者：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="Weline_Framework_Module::controller_attributes">
        <observer name="Your_Module::modify_controller_attributes"
                  instance="Your\Module\Observer\ModifyControllerAttributesObserver"
                  disabled="false"
                  shared="true"
                  sort="100"/>
    </event>
</config>
```

创建观察者类：

```php
<?php

namespace Your\Module\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class ModifyControllerAttributesObserver implements ObserverInterface
{
    /**
     * 执行观察者逻辑
     *
     * @param Event $event
     * @return void
     */
    public function execute(Event &$event): void
    {
        $controllerData = $event->getData('controller_data');
        $params = $event->getData('params');
        
        // 修改控制器属性
        // ...
    }
}
```

#### 事件数据

`Weline_Framework_Module::controller_attributes` 事件传递的数据：

| 字段 | 类型 | 说明 |
|------|------|------|
| controller_data | array | 控制器数据 |
| params | array | 参数数组 |

**获取和修改数据**：

```php
$controllerData = $event->getData('controller_data');
$params = $event->getData('params');

// 修改数据
$event->setData('controller_data', $modifiedData);
```

#### 注意事项

1. **执行顺序**：观察者按照 `sort` 属性排序执行，数值越小越先执行
2. **数据修改**：可以直接修改事件数据
