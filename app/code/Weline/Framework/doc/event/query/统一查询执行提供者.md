# Weline_Framework_Query::provider_execute（已废弃）

> **已废弃**：查询由 extends 注册的查询器（`QueryProviderInterface`）执行，不再通过此事件分发。
>
> 请实现 `QueryProviderInterface` 并在 `extends/module/Weline_Framework/Query/` 下注册。

## 迁移指南

旧方式（已删除）：

```xml
<event name="Weline_Framework_Query::provider_execute">
    <observer name="..." instance="..." />
</event>
```

新方式：

1. 在模块目录下创建 `extends/module/Weline_Framework/Query/XxxQueryProvider.php`
2. 实现 `QueryProviderInterface`（getProviderName、execute、getDescriptor）
3. 执行 `php bin/w extends:rebuild`
