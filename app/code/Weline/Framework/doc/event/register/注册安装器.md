# Weline Framework 模块 - 事件文档

## 概述

本文档详细说明了 Weline Framework 模块提供的 `Weline_Framework_Register::register_installer` 事件及其使用方法。该事件在注册模块安装器时触发，允许其他模块自定义安装器路径。

## 事件列表

### 1. Weline_Framework_Register::register_installer - 注册安装器事件

#### 基本信息

- **事件名称**：`Weline_Framework_Register::register_installer`
- **事件类型**：注册事件
- **触发时机**：在注册模块安装器时
- **触发位置**：`app/code/Weline/Framework/Register/Register.php` 第 71 行
- **配置文件**：`app/code/Weline/Framework/etc/event.xml`

#### 功能说明

`Weline_Framework_Register::register_installer` 事件在注册模块安装器时触发，允许其他模块自定义安装器路径。可以修改 `installer` 字段来改变安装器类名。

该事件主要用于：
- 自定义安装器路径
- 安装器类名映射
- 安装器验证
- 安装器扩展

#### 触发时机

```php
// app/code/Weline/Framework/Register/Register.php
$installerPathData = ObjectManager::getInstance(DataObject::class);
$installerPathData
    ->setData('installer', self::NAMESPACE . ucfirst($type) . '\Handle')
    ->setData('register_arguments', $install_params);
$eventsManager->dispatch('Weline_Framework_Register::register_installer', $installerPathData);
$installer_class = $installerPathData->getData('installer');
```

#### 使用场景

- **自定义安装器**：使用自定义安装器类
- **安装器映射**：将安装器类名映射到其他类
- **安装器验证**：验证安装器类是否存在
- **安装器扩展**：扩展安装器功能

#### 使用方法

##### 基本用法

在模块的 `etc/event.xml` 文件中注册观察者：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="Weline_Framework_Register::register_installer">
        <observer name="Your_Module::custom_installer"
                  instance="Your\Module\Observer\CustomInstallerObserver"
                  disabled="false"
                  shared="true"
                  sort="100"/>
    </event>
</config>
```

创建观察者类：

```php
<?php

namespace Your\Module\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class CustomInstallerObserver implements ObserverInterface
{
    /**
     * 执行观察者逻辑
     *
     * @param Event $event
     * @return void
     */
    public function execute(Event &$event): void
    {
        $installer = $event->getData('installer');
        $registerArguments = $event->getData('register_arguments');
        
        // 自定义安装器类名
        $customInstaller = $this->getCustomInstaller($installer);
        if ($customInstaller) {
            $event->setData('installer', $customInstaller);
        }
    }
    
    private function getCustomInstaller(string $installer): ?string
    {
        // 安装器映射逻辑
        return null;
    }
}
```

#### 事件数据

`Weline_Framework_Register::register_installer` 事件传递的数据：

| 字段 | 类型 | 说明 |
|------|------|------|
| installer | string | 安装器类名，可以修改此值来改变安装器类 |
| register_arguments | array | 注册参数 |

**获取和修改数据**：

```php
$installer = $event->getData('installer');
$registerArguments = $event->getData('register_arguments');

// 修改安装器类名
$event->setData('installer', $customInstaller);
```

#### 注意事项

1. **执行顺序**：观察者按照 `sort` 属性排序执行，数值越小越先执行
2. **覆盖行为**：后执行的观察者可以覆盖先执行的观察者的设置
3. **类名格式**：安装器类名必须是完整的类名（包含命名空间）
