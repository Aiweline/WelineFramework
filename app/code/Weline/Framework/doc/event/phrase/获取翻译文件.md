# Weline Framework 模块 - 事件文档

## 概述

本文档详细说明了 Weline Framework 模块提供的 `Weline_Framework::get_words_file` 事件及其使用方法。该事件在获取国际化翻译文件时触发，允许其他模块自定义翻译文件路径。

## 事件列表

### 1. Weline_Framework::get_words_file - 获取翻译文件事件

#### 基本信息

- **事件名称**：`Weline_Framework::get_words_file`
- **事件类型**：国际化事件
- **触发时机**：在 `Parser::getWords()` 方法中，获取翻译文件时
- **触发位置**：`app/code/Weline/Framework/Phrase/Parser.php` 第 113 行
- **配置文件**：`app/code/Weline/Framework/etc/event.xml`

#### 功能说明

`Weline_Framework::get_words_file` 事件在获取国际化翻译文件时触发，此时：
- 翻译缓存未命中或需要实时翻译
- 默认翻译文件路径已设置
- 可以通过事件修改翻译文件路径
- 允许模块提供自定义的翻译文件

该事件主要用于：
- 自定义翻译文件路径
- 提供多语言翻译文件
- 动态加载翻译文件
- 集成第三方翻译服务

#### 触发时机

```php
// app/code/Weline/Framework/Phrase/Parser.php
public static function getWords()
{
    // ... 缓存检查 ...
    
    # 事件分配
    $eventsManager = ObjectManager::getInstance(EventsManager::class);
    $file_data = new DataObject(['file_path' => Env::path_TRANSLATE_DEFAULT_FILE]);
    $eventsManager->dispatch('Weline_Framework::get_words_file', $file_data);
    $words_file = $file_data->getData('file_path');
    
    # 实时翻译
    if (is_file($words_file)) {
        self::$words = (array)include $words_file;
        // ...
    }
}
```

#### 使用场景

- **自定义翻译文件路径**：根据模块或区域提供不同的翻译文件
- **多语言支持**：为不同语言提供不同的翻译文件
- **动态翻译加载**：根据用户设置动态加载翻译文件
- **第三方翻译集成**：集成外部翻译服务

#### 使用方法

##### 基本用法

在模块的 `etc/event.xml` 文件中注册观察者：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="Weline_Framework::get_words_file">
        <observer name="Your_Module::custom_translate_file"
                  instance="Your\Module\Observer\CustomTranslateFileObserver"
                  disabled="false"
                  shared="true"
                  sort="100"/>
    </event>
</config>
```

创建观察者类：

```php
<?php

namespace Your\Module\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;
use Weline\Framework\App\Env;

class CustomTranslateFileObserver implements ObserverInterface
{
    /**
     * 执行观察者逻辑
     *
     * @param Event $event
     * @return void
     */
    public function execute(Event &$event): void
    {
        // 获取当前翻译文件路径
        $filePath = $event->getData('file_path');
        
        // 根据条件修改翻译文件路径
        $customPath = $this->getCustomTranslateFilePath();
        
        if ($customPath && is_file($customPath)) {
            $event->setData('file_path', $customPath);
        }
    }
    
    private function getCustomTranslateFilePath(): ?string
    {
        // 返回自定义翻译文件路径
        // 注意：框架默认使用 zh_Hans_CN 作为中文语言代码
        return BP . 'app/code/Your/Module/i18n/zh_Hans_CN.php';
    }
}
```

#### 事件数据

`Weline_Framework::get_words_file` 事件传递的数据：

| 字段 | 类型 | 说明 |
|------|------|------|
| file_path | string | 翻译文件路径，可以修改此值来改变翻译文件 |

**获取和修改数据**：

```php
// 获取当前翻译文件路径
$filePath = $event->getData('file_path');

// 修改翻译文件路径
$event->setData('file_path', $customPath);
```

#### 使用示例

##### 示例 1：根据语言自定义翻译文件

```php
<?php

namespace Your\Module\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;
use Weline\Framework\Http\Cookie;

class LanguageTranslateFileObserver implements ObserverInterface
{
    public function execute(Event &$event): void
    {
        $defaultPath = $event->getData('file_path');
        // 框架默认使用 zh_Hans_CN 作为中文语言代码
        $lang = Cookie::getLangLocal() ?: 'zh_Hans_CN';
        
        // 根据语言选择翻译文件
        $langPath = BP . "app/code/Your/Module/i18n/{$lang}.php";
        
        if (is_file($langPath)) {
            $event->setData('file_path', $langPath);
        }
    }
}
```

##### 示例 2：模块级翻译文件

```php
<?php

namespace Your\Module\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;
use Weline\Framework\App\Env;

class ModuleTranslateFileObserver implements ObserverInterface
{
    public function execute(Event &$event): void
    {
        // 获取当前模块的翻译文件
        $modulePath = Env::getInstance()->getModulePath('Your_Module');
        // 框架默认使用 zh_Hans_CN 作为中文语言代码
        $lang = Cookie::getLangLocal() ?: 'zh_Hans_CN';
        $translateFile = $modulePath . "/i18n/{$lang}.php";
        
        if (is_file($translateFile)) {
            // 合并模块翻译文件和默认翻译文件
            $defaultPath = $event->getData('file_path');
            $defaultWords = is_file($defaultPath) ? (array)include $defaultPath : [];
            $moduleWords = (array)include $translateFile;
            
            // 合并翻译（模块翻译优先）
            $mergedWords = array_merge($defaultWords, $moduleWords);
            
            // 创建临时合并文件
            $tempFile = sys_get_temp_dir() . '/merged_translate_' . uniqid() . '.php';
            file_put_contents($tempFile, '<?php return ' . var_export($mergedWords, true) . ';');
            
            $event->setData('file_path', $tempFile);
        }
    }
}
```

#### 注意事项

1. **文件路径格式**：翻译文件必须是有效的 PHP 文件，返回数组格式
2. **文件存在性**：修改文件路径前，应检查文件是否存在
3. **性能考虑**：该事件在每次翻译时都可能触发，应避免执行耗时操作
4. **文件格式**：翻译文件必须返回数组，格式为 `['原文' => '译文']`
5. **优先级**：后执行的观察者可以覆盖先执行的观察者的修改
6. **缓存机制**：翻译结果会被缓存，修改文件路径后可能需要清除缓存

#### 系统行为说明

1. **默认行为**：
   - 默认使用 `Env::path_TRANSLATE_DEFAULT_FILE` 作为翻译文件路径
   - 如果事件未修改路径，使用默认路径

2. **执行顺序**：
   - 观察者按照 `sort` 属性从小到大排序执行
   - 后执行的观察者可以覆盖先执行的观察者的修改

3. **文件加载**：
   - 事件执行后，系统会加载 `file_path` 指定的文件
   - 文件必须返回数组格式的翻译数据

#### 故障排查

##### 问题：翻译文件未生效

**可能原因：**
1. 文件路径不正确
2. 文件不存在
3. 文件格式不正确
4. 翻译缓存未清除

**解决方法：**
1. 检查文件路径是否正确
2. 确认文件是否存在
3. 检查文件格式是否为有效的 PHP 数组
4. 清除翻译缓存

##### 问题：翻译文件加载失败

**可能原因：**
1. 文件语法错误
2. 文件权限问题
3. 文件路径包含特殊字符

**解决方法：**
1. 检查 PHP 语法错误
2. 检查文件权限
3. 使用绝对路径
