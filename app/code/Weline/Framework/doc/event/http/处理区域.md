# Weline Framework 模块 - 事件文档

## 概述

本文档详细说明了 Weline Framework 模块提供的 `WelineFramework_Http::process_area` 事件及其使用方法。该事件在处理HTTP请求区域时触发，允许其他模块自定义区域处理逻辑。

## 事件列表

### 1. WelineFramework_Http::process_area - 处理区域事件

#### 基本信息

- **事件名称**：`WelineFramework_Http::process_area`
- **事件类型**：HTTP请求事件
- **触发时机**：在处理HTTP请求区域时
- **触发位置**：`app/code/Weline/Framework/Http/Request/RequestAbstract.php` 第 189 行
- **配置文件**：`app/code/Weline/Framework/etc/event.xml`

#### 功能说明

`WelineFramework_Http::process_area` 事件在处理HTTP请求区域时触发，允许其他模块自定义区域处理逻辑。可以修改 `area` 字段来改变处理区域。

该事件主要用于：
- 自定义区域处理逻辑
- 区域映射
- 区域验证
- 区域扩展

#### 触发时机

```php
// app/code/Weline/Framework/Http/Request/RequestAbstract.php
$eventData = ['area' => $area, 'path' => $this->area_router];
$eventManager->dispatch('WelineFramework_Http::process_area', $eventData);
return $area;
```

#### 使用场景

- **区域处理**：自定义区域处理逻辑
- **区域映射**：将区域映射到其他区域
- **区域验证**：验证区域是否有效
- **区域扩展**：扩展区域功能

#### 使用方法

##### 基本用法

在模块的 `etc/event.xml` 文件中注册观察者：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="WelineFramework_Http::process_area">
        <observer name="Your_Module::custom_area_process"
                  instance="Your\Module\Observer\CustomAreaProcessObserver"
                  disabled="false"
                  shared="true"
                  sort="100"/>
    </event>
</config>
```

创建观察者类：

```php
<?php

namespace Your\Module\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class CustomAreaProcessObserver implements ObserverInterface
{
    /**
     * 执行观察者逻辑
     *
     * @param Event $event
     * @return void
     */
    public function execute(Event &$event): void
    {
        $area = $event->getData('area');
        $path = $event->getData('path');
        
        // 自定义区域处理逻辑
        $customArea = $this->processArea($area, $path);
        if ($customArea) {
            $event->setData('area', $customArea);
        }
    }
    
    private function processArea(string $area, string $path): ?string
    {
        // 区域处理逻辑
        return null;
    }
}
```

#### 事件数据

`WelineFramework_Http::process_area` 事件传递的数据：

| 字段 | 类型 | 说明 |
|------|------|------|
| area | string | 区域名称，可以修改此值来改变处理区域 |
| path | string | 区域路径 |

**获取和修改数据**：

```php
$area = $event->getData('area');
$path = $event->getData('path');

// 修改区域
$event->setData('area', $customArea);
```

#### 注意事项

1. **执行顺序**：观察者按照 `sort` 属性排序执行，数值越小越先执行
2. **覆盖行为**：后执行的观察者可以覆盖先执行的观察者的设置
