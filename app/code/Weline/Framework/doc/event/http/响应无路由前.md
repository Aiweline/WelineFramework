# Weline Framework 模块 - 事件文档

## 概述

本文档详细说明了 Weline Framework 模块提供的 `Weline_Framework_http_response_no_router_before` 事件及其使用方法。该事件在响应无路由请求前触发，允许其他模块处理无路由请求。

## 事件列表

### 1. Weline_Framework_http_response_no_router_before - 响应无路由前事件

#### 基本信息

- **事件名称**：`Weline_Framework_http_response_no_router_before`
- **事件类型**：HTTP响应事件
- **触发时机**：在响应无路由请求前
- **触发位置**：`app/code/Weline/Framework/Http/Response.php` 第 96 行
- **配置文件**：`app/code/Weline/Framework/etc/event.xml`

#### 功能说明

`Weline_Framework_http_response_no_router_before` 事件在响应无路由请求前触发，允许其他模块处理无路由请求。可以通过设置响应或退出来阻止框架默认的404响应。

该事件主要用于：
- 处理无路由请求
- 自定义404页面
- 无路由请求重定向
- 无路由请求日志记录

#### 触发时机

```php
// app/code/Weline/Framework/Http/Response.php
$this->getEvenManager()->dispatch('Weline_Framework_http_response_no_router_before');
if (DEV || CLI) {
    // 输出错误信息
}
```

#### 使用场景

- **自定义404**：显示自定义404页面
- **请求重定向**：将无路由请求重定向到其他页面
- **日志记录**：记录无路由请求日志
- **请求处理**：处理无路由请求

#### 使用方法

##### 基本用法

在模块的 `etc/event.xml` 文件中注册观察者：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="Weline_Framework_http_response_no_router_before">
        <observer name="Your_Module::handle_no_router"
                  instance="Your\Module\Observer\HandleNoRouterObserver"
                  disabled="false"
                  shared="true"
                  sort="100"/>
    </event>
</config>
```

创建观察者类：

```php
<?php

namespace Your\Module\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class HandleNoRouterObserver implements ObserverInterface
{
    /**
     * 执行观察者逻辑
     *
     * @param Event $event
     * @return void
     */
    public function execute(Event &$event): void
    {
        // 处理无路由请求
        $this->handleNoRouter();
    }
    
    private function handleNoRouter(): void
    {
        // 可以输出自定义404页面或重定向
        // echo 'Custom 404 Page';
        // exit;
    }
}
```

#### 事件数据

`Weline_Framework_http_response_no_router_before` 事件不传递额外的数据。

#### 注意事项

1. **执行顺序**：观察者按照 `sort` 属性排序执行，数值越小越先执行
2. **退出控制**：如果需要阻止框架默认的404响应，可以在观察者中使用 `exit` 退出
