# Weline Framework 模块 - 事件文档

## 概述

本文档详细说明了 Weline Framework 模块提供的 `Weline_Framework_Url::detect_language` 事件及其使用方法。该事件在URL解析时检测语言时触发，允许其他模块自定义语言检测逻辑。

## 事件列表

### 1. Weline_Framework_Url::detect_language - 检测语言事件

#### 基本信息

- **事件名称**：`Weline_Framework_Url::detect_language`
- **事件类型**：URL解析事件
- **触发时机**：在URL解析时检测语言时
- **触发位置**：`app/code/Weline/Framework/Http/Url.php` 第 48 行
- **配置文件**：`app/code/Weline/Framework/etc/event.xml`

#### 功能说明

`Weline_Framework_Url::detect_language` 事件在URL解析时检测语言时触发，允许其他模块自定义语言检测逻辑。通过设置 `result` 为 `true` 表示检测成功，系统会使用该语言代码。

该事件主要用于：
- 自定义语言检测逻辑
- 支持多语言URL
- 语言代码验证
- 语言映射

#### 触发时机

```php
// app/code/Weline/Framework/Http/Url.php
$data = new DataObject([
    'result' => false,
    'uri' => &$uri,
    'code' => $code,
]);
$eventManager->dispatch('Weline_Framework_Url::detect_language', $data);
if ($data->getData('result')) {
    // 语言检测成功
    self::$parserServer['WELINE_USER_LANG'] = $code;
    return true;
}
```

#### 使用场景

- **自定义语言检测**：根据URL、Cookie、Header等自定义语言检测
- **语言代码映射**：将URL中的语言代码映射到系统语言代码
- **语言验证**：验证语言代码是否有效
- **多语言支持**：支持多语言URL结构

#### 使用方法

##### 基本用法

在模块的 `etc/event.xml` 文件中注册观察者：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="Weline_Framework_Url::detect_language">
        <observer name="Your_Module::custom_language_detect"
                  instance="Your\Module\Observer\CustomLanguageDetectObserver"
                  disabled="false"
                  shared="true"
                  sort="100"/>
    </event>
</config>
```

创建观察者类：

```php
<?php

namespace Your\Module\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class CustomLanguageDetectObserver implements ObserverInterface
{
    /**
     * 执行观察者逻辑
     *
     * @param Event $event
     * @return void
     */
    public function execute(Event &$event): void
    {
        $code = $event->getData('code');
        $uri = $event->getData('uri');
        
        // 自定义语言检测逻辑
        if ($this->isValidLanguage($code)) {
            $event->setData('result', true);
        }
    }
    
    private function isValidLanguage(string $code): bool
    {
        // 验证语言代码
        $validLanguages = ['zh_Hans_CN', 'en_US', 'ja_JP'];
        return in_array($code, $validLanguages);
    }
}
```

#### 事件数据

`Weline_Framework_Url::detect_language` 事件传递的数据：

| 字段 | 类型 | 说明 |
|------|------|------|
| result | bool | 检测结果，设置为 true 表示检测成功 |
| uri | string | URI字符串（引用传递，可以修改） |
| code | string | 语言代码 |

**获取和修改数据**：

```php
// 获取数据
$code = $event->getData('code');
$uri = $event->getData('uri');

// 设置检测结果
$event->setData('result', true);

// 修改URI（如果需要）
$event->setData('uri', $modifiedUri);
```

#### 使用示例

##### 示例 1：语言代码映射

```php
<?php

namespace Your\Module\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class LanguageCodeMappingObserver implements ObserverInterface
{
    public function execute(Event &$event): void
    {
        $code = $event->getData('code');
        
        // 语言代码映射
        $languageMap = [
            'zh' => 'zh_Hans_CN',
            'en' => 'en_US',
            'ja' => 'ja_JP',
        ];
        
        if (isset($languageMap[$code])) {
            $event->setData('code', $languageMap[$code]);
            $event->setData('result', true);
        }
    }
}
```

##### 示例 2：根据Cookie检测语言

```php
<?php

namespace Your\Module\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;
use Weline\Framework\Http\Cookie;

class CookieLanguageDetectObserver implements ObserverInterface
{
    public function execute(Event &$event): void
    {
        // 如果URL中没有语言代码，从Cookie获取
        $code = $event->getData('code');
        
        if (empty($code)) {
            $cookieLang = Cookie::get('WELINE_USER_LANG');
            if ($cookieLang && $this->isValidLanguage($cookieLang)) {
                $event->setData('code', $cookieLang);
                $event->setData('result', true);
            }
        }
    }
    
    private function isValidLanguage(string $code): bool
    {
        $validLanguages = ['zh_Hans_CN', 'en_US', 'ja_JP'];
        return in_array($code, $validLanguages);
    }
}
```

#### 注意事项

1. **result设置**：必须设置 `result` 为 `true` 才能表示检测成功
2. **URI修改**：可以修改URI来移除语言代码前缀
3. **执行顺序**：观察者按照 `sort` 属性排序执行，数值越小越先执行
4. **性能考虑**：该事件在每次URL解析时都可能触发，应避免执行耗时操作

#### 系统行为说明

1. **默认行为**：
   - 默认 `result` 为 `false`
   - 如果所有观察者都未设置 `result` 为 `true`，语言检测失败

2. **执行顺序**：
   - 观察者按照 `sort` 属性从小到大排序执行
   - 第一个设置 `result` 为 `true` 的观察者会生效
