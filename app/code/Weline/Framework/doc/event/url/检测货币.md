# Weline Framework 模块 - 事件文档

## 概述

本文档详细说明了 Weline Framework 模块提供的 `Weline_Framework_Url::detect_currency` 事件及其使用方法。该事件在URL解析时检测货币时触发，允许其他模块自定义货币检测逻辑。

## 事件列表

### 1. Weline_Framework_Url::detect_currency - 检测货币事件

#### 基本信息

- **事件名称**：`Weline_Framework_Url::detect_currency`
- **事件类型**：URL解析事件
- **触发时机**：在URL解析时检测货币时
- **触发位置**：`app/code/Weline/Framework/Http/Url.php` 第 73 行
- **配置文件**：`app/code/Weline/Framework/etc/event.xml`

#### 功能说明

`Weline_Framework_Url::detect_currency` 事件在URL解析时检测货币时触发，允许其他模块自定义货币检测逻辑。通过设置 `result` 为 `true` 表示检测成功，系统会使用该货币代码。

该事件主要用于：
- 自定义货币检测逻辑
- 支持多货币URL
- 货币代码验证
- 货币映射

#### 触发时机

```php
// app/code/Weline/Framework/Http/Url.php
$detect_currency_data = new DataObject([
    'result' => false,
    'uri' => $uri,
    'code' => $code,
]);
$eventManager->dispatch('Weline_Framework_Url::detect_currency', $detect_currency_data);
$uri_ = $detect_currency_data->getData('uri');
```

#### 使用场景

- **自定义货币检测**：根据URL、Cookie、Header等自定义货币检测
- **货币代码映射**：将URL中的货币代码映射到系统货币代码
- **货币验证**：验证货币代码是否有效
- **多货币支持**：支持多货币URL结构

#### 使用方法

##### 基本用法

在模块的 `etc/event.xml` 文件中注册观察者：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="Weline_Framework_Url::detect_currency">
        <observer name="Your_Module::custom_currency_detect"
                  instance="Your\Module\Observer\CustomCurrencyDetectObserver"
                  disabled="false"
                  shared="true"
                  sort="100"/>
    </event>
</config>
```

创建观察者类：

```php
<?php

namespace Your\Module\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class CustomCurrencyDetectObserver implements ObserverInterface
{
    /**
     * 执行观察者逻辑
     *
     * @param Event $event
     * @return void
     */
    public function execute(Event &$event): void
    {
        $code = $event->getData('code');
        
        // 自定义货币检测逻辑
        if ($this->isValidCurrency($code)) {
            $event->setData('result', true);
        }
    }
    
    private function isValidCurrency(string $code): bool
    {
        // 验证货币代码
        $validCurrencies = ['CNY', 'USD', 'EUR', 'JPY'];
        return in_array($code, $validCurrencies);
    }
}
```

#### 事件数据

`Weline_Framework_Url::detect_currency` 事件传递的数据：

| 字段 | 类型 | 说明 |
|------|------|------|
| result | bool | 检测结果，设置为 true 表示检测成功 |
| uri | string | URI字符串（可以修改） |
| code | string | 货币代码 |

**获取和修改数据**：

```php
// 获取数据
$code = $event->getData('code');
$uri = $event->getData('uri');

// 设置检测结果
$event->setData('result', true);

// 修改URI（如果需要）
$event->setData('uri', $modifiedUri);
```

#### 使用示例

##### 示例 1：货币代码映射

```php
<?php

namespace Your\Module\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class CurrencyCodeMappingObserver implements ObserverInterface
{
    public function execute(Event &$event): void
    {
        $code = $event->getData('code');
        
        // 货币代码映射
        $currencyMap = [
            'rmb' => 'CNY',
            'dollar' => 'USD',
            'euro' => 'EUR',
        ];
        
        if (isset($currencyMap[$code])) {
            $event->setData('code', $currencyMap[$code]);
            $event->setData('result', true);
        }
    }
}
```

##### 示例 2：根据Cookie检测货币

```php
<?php

namespace Your\Module\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;
use Weline\Framework\Http\Cookie;

class CookieCurrencyDetectObserver implements ObserverInterface
{
    public function execute(Event &$event): void
    {
        // 如果URL中没有货币代码，从Cookie获取
        $code = $event->getData('code');
        
        if (empty($code)) {
            $cookieCurrency = Cookie::get('WELINE_USER_CURRENCY');
            if ($cookieCurrency && $this->isValidCurrency($cookieCurrency)) {
                $event->setData('code', $cookieCurrency);
                $event->setData('result', true);
            }
        }
    }
    
    private function isValidCurrency(string $code): bool
    {
        $validCurrencies = ['CNY', 'USD', 'EUR', 'JPY'];
        return in_array($code, $validCurrencies);
    }
}
```

#### 注意事项

1. **result设置**：必须设置 `result` 为 `true` 才能表示检测成功
2. **URI修改**：可以修改URI来移除货币代码前缀
3. **执行顺序**：观察者按照 `sort` 属性排序执行，数值越小越先执行
4. **性能考虑**：该事件在每次URL解析时都可能触发，应避免执行耗时操作

#### 系统行为说明

1. **默认行为**：
   - 默认 `result` 为 `false`
   - 如果所有观察者都未设置 `result` 为 `true`，货币检测失败

2. **执行顺序**：
   - 观察者按照 `sort` 属性从小到大排序执行
   - 第一个设置 `result` 为 `true` 的观察者会生效
