## 模块注册

### 1. 创建register.php

**位置**: `app/code/Vendor/ModuleName/register.php`

```php
<?php
\Weline\Framework\Register\Register::register(__DIR__, \Weline\Framework\Register\Register::MODULE);
```

**要点**:
- ✅ 必须使用 `Weline\Framework\Register\Register::register()`
- ✅ 第一个参数必须是 `__DIR__`
- ✅ 第二个参数必须是 `Register::MODULE`
- ❌ 不要添加任何其他代码

### 2. 创建module.xml

**位置**: `app/code/Vendor/ModuleName/etc/module.xml`

```xml
<?xml version="1.0"?>
<config xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" 
        xsi:noNamespaceSchemaLocation="urn:weline:framework:module:etc/module.xsd">
    <module name="Vendor_ModuleName" setup_version="1.0.0">
        <sequence>
            <module name="Weline_Framework"/>
            <!-- 其他依赖模块 -->
        </sequence>
    </module>
</config>
```

**要点**:
- ✅ 模块名格式: `Vendor_ModuleName`（驼峰命名）
- ✅ 必须包含 `setup_version`
- ✅ 通常依赖 `Weline_Framework`
- ✅ `<sequence>` 标签定义模块依赖顺序

### 3. 安装模块

```bash
php bin/w setup:upgrade
```

**验证模块安装**:
```bash
# 查看模块列表
php bin/w module:list | grep "ModuleName"

# 应该显示: Vendor_ModuleName    # 开启
```

## 数据库操作

### Setup/Install.php

**位置**: `app/code/Vendor/ModuleName/Setup/Install.php`

```php
<?php

declare(strict_types=1);

namespace Vendor\ModuleName\Setup;

use Weline\Framework\Setup\InstallInterface;
use Weline\Framework\Setup\Setup;
use Weline\Framework\Setup\Model\Context;

class Install implements InstallInterface
{
    public function setup(Setup $setup, Context $context)
    {
        // 创建表1
        $setup->getConnection()->createTable(
            'your_table_name',
            function (\Weline\Framework\Setup\Schema\Table $table) {
                $table->addColumn('id', 'int', 10, 'PRIMARY KEY AUTO_INCREMENT', 'ID')
                      ->addColumn('name', 'varchar', 255, 'NOT NULL', '名称')
                      ->addColumn('status', 'varchar', 50, 'NOT NULL DEFAULT \'active\'', '状态')
                      ->addColumn('created_at', 'timestamp', 0, 'NOT NULL DEFAULT CURRENT_TIMESTAMP', '创建时间')
                      ->addColumn('updated_at', 'timestamp', 0, 'NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP', '更新时间');
            },
            '表描述'
        );
        
        // 创建表2...
    }
}
```

**要点**:
- ✅ 必须实现 `InstallInterface`
- ✅ 必须实现 `setup(Setup $setup, Context $context)` 方法
- ✅ 使用 `$setup->getConnection()->createTable()` 创建表
- ✅ 表名使用小写+下划线命名
- ✅ 主键通常命名为 `id`
- ✅ 建议添加 `created_at` 和 `updated_at` 时间戳

**列类型参考**:
```php
// 整数
->addColumn('field', 'int', 10, 'NOT NULL', '描述')

// 字符串
->addColumn('field', 'varchar', 255, 'NOT NULL', '描述')
->addColumn('field', 'text', 0, 'NULL', '描述')

// 时间
->addColumn('field', 'timestamp', 0, 'NOT NULL DEFAULT CURRENT_TIMESTAMP', '描述')
->addColumn('field', 'date', 0, 'NOT NULL', '描述')

// 小数
->addColumn('field', 'decimal', '10,2', 'NOT NULL DEFAULT 0', '描述')

// 布尔（使用tinyint）
->addColumn('field', 'tinyint', 1, 'NOT NULL DEFAULT 0', '描述')
```

## Model实体开发

### 基础Model

**位置**: `app/code/Vendor/ModuleName/Model/Entity.php`

```php
<?php

declare(strict_types=1);

namespace Vendor\ModuleName\Model;

use Weline\Framework\Database\Model;
use Weline\Framework\Setup\Db\ModelSetup;
use Weline\Framework\Setup\Data\Context;

/**
 * Entity Model
 * 
 * @package Vendor_ModuleName
 */
class Entity extends Model
{
    // 状态常量
    public const STATUS_ACTIVE = 'active';
    public const STATUS_INACTIVE = 'inactive';
    
    /**
     * 初始化模型
     */
    public function _init(): void
    {
        $this->_table = 'your_table_name';
        $this->_id_field_name = 'id';
    }
    
    /**
     * 模型设置（必需实现）
     */
    public function setup(ModelSetup $setup, Context $context): void
    {
        // 表创建在 Setup/Install.php 中处理
    }
    
    /**
     * 模型升级（必需实现）
     */
    public function upgrade(ModelSetup $setup, Context $context): void
    {
        // 升级逻辑
    }
    
    /**
     * 模型安装（必需实现）
     */
    public function install(ModelSetup $setup, Context $context): void
    {
        // 安装逻辑
    }
    
    /**
     * 检查是否活跃
     */
    public function isActive(): bool
    {
        return $this->getData('status') === self::STATUS_ACTIVE;
    }
    
    /**
     * 验证数据
     */
    public function validate(): bool
    {
        if (empty($this->getData('name'))) {
            throw new \InvalidArgumentException('Name is required');
        }
        return true;
    }
    
    /**
     * 保存前钩子
     */
    public function beforeSave(): self
    {
        $this->validate();
        return parent::beforeSave();
    }
}
```

**关键要点**:

#### 1. 必须继承 `Weline\Framework\Database\Model`
```php
use Weline\Framework\Database\Model;

class YourModel extends Model
```

#### 2. 必须实现 `_init()` 方法
```php
public function _init(): void
{
    $this->_table = 'your_table_name';      // 数据库表名
    $this->_id_field_name = 'id';           // 主键字段名
}
```

#### 3. 必须实现 ModelInterface 的三个方法
```php
use Weline\Framework\Setup\Db\ModelSetup;
use Weline\Framework\Setup\Data\Context;

public function setup(ModelSetup $setup, Context $context): void {}
public function upgrade(ModelSetup $setup, Context $context): void {}
public function install(ModelSetup $setup, Context $context): void {}
```

**注意**: 
- ⚠️ 参数类型必须是 `ModelSetup` 和 `Context`，不是 `Setup`
- ⚠️ 必须有返回类型声明 `: void`
- ⚠️ 通常这三个方法留空，表创建在 `Setup/Install.php` 中

#### 4. 常用Model方法

```php
// 基础CRUD
$model->load($id);                   // 加载记录
$model->load($value, 'field');       // 通过字段加载
$model->save();                      // 保存
$model->delete();                    // 删除
$model->getId();                     // 获取ID

// 数据操作
$model->getData('field');            // 获取字段值
$model->getData();                   // 获取所有数据
$model->setData('field', $value);    // 设置字段值
$model->setData([...]);              // 批量设置

// 查询
$model->where('field', $value)->find()->fetch();  // 单条查询
$model->where('field', $value)->select()->fetch(); // 多条查询

// JSON字段处理
$config = $model->getData('config');
if (is_string($config)) {
    $config = json_decode($config, true);
}

// 保存JSON字段
if (is_array($this->getData('config'))) {
    $this->setData('config', json_encode($this->getData('config')));
}
```

## Service服务层

### 基础Service

**位置**: `app/code/Vendor/ModuleName/Service/EntityService.php`

```php
<?php

declare(strict_types=1);

namespace Vendor\ModuleName\Service;

use Vendor\ModuleName\Model\Entity;

/**
 * Entity Service
 * 
 * 业务逻辑处理层
 * 
 * @package Vendor_ModuleName
 */
class EntityService
{
    public function __construct(
        private readonly Entity $entity
    ) {
    }
    
    /**
     * 通过ID获取实体
     *
     * @param int $id
     * @return Entity
     * @throws \RuntimeException
     */
    public function getById(int $id): Entity
    {
        $entity = clone $this->entity;
        $entity->load($id);
        
        if (!$entity->getId()) {
            throw new \RuntimeException("Entity with ID {$id} not found");
        }
        
        return $entity;
    }
    
    /**
     * 创建实体
     *
     * @param array $data
     * @return Entity
     */
    public function create(array $data): Entity
    {
        $entity = clone $this->entity;
        $entity->setData($data);
        $entity->save();
        
        return $entity;
    }
    
    /**
     * 更新实体
     *
     * @param int $id
     * @param array $data
     * @return Entity
     */
    public function update(int $id, array $data): Entity
    {
        $entity = $this->getById($id);
        $entity->setData($data);
        $entity->save();
        
        return $entity;
    }
    
    /**
     * 删除实体
     *
     * @param int $id
     * @return bool
     */
    public function delete(int $id): bool
    {
        $entity = $this->getById($id);
        return $entity->delete();
    }
}
```

**要点**:
- ✅ 使用 `private readonly` 依赖注入
- ✅ Service处理业务逻辑，不直接操作数据库
- ✅ 通过Model进行数据访问
- ✅ 使用 `clone $this->entity` 创建新实例
- ✅ 抛出有意义的异常

## Controller控制器

### REST API Controller

**位置**: `app/code/Vendor/ModuleName/Controller/Api/Entity.php`

```php
<?php

declare(strict_types=1);

namespace Vendor\ModuleName\Controller\Api;

use Weline\Framework\App\Controller\FrontendRestController;
use Vendor\ModuleName\Service\EntityService;

/**
 * Entity API Controller
 * 
 * @package Vendor_ModuleName
 */
class Entity extends FrontendRestController
{
    public function __construct(
        private readonly EntityService $entityService
    ) {
    }
    
    /**
     * GET /api/v1/entity/{id}
     */
    public function get(): array
    {
        try {
            $id = (int) $this->request->getParam('id');
            
            if (!$id) {
                return $this->error('ID is required', 400);
            }
            
            $entity = $this->entityService->getById($id);
            
            return $this->success('请求成功', [
                'id' => $entity->getId(),
                'name' => $entity->getData('name'),
                'status' => $entity->getData('status'),
            ]);
            
        } catch (\RuntimeException $e) {
            return $this->error($e->getMessage(), 404);
        } catch (\Exception $e) {
            return $this->error('Internal server error', 500);
        }
    }
    
    /**
     * POST /api/v1/entity
     */
    public function post(): array
    {
        try {
            $data = $this->request->getBodyParams();
            
            // 验证必需字段
            if (empty($data['name'])) {
                return $this->error('Name is required', 400);
            }
            
            $entity = $this->entityService->create($data);
            
            return $this->success('创建成功', [
                'id' => $entity->getId(),
            ]);
            
        } catch (\InvalidArgumentException $e) {
            return $this->error($e->getMessage(), 400);
        } catch (\Exception $e) {
            return $this->error('Internal server error', 500);
        }
    }
}
```

**关键要点**:

#### 1. 继承正确的基类
```php
// REST API控制器
use Weline\Framework\App\Controller\FrontendRestController;
class Entity extends FrontendRestController {}

// 或后台REST API
use Weline\Framework\App\Controller\BackendRestController;
class Entity extends BackendRestController {}
```

#### 2. 使用父类的 success() 和 error() 方法
```php
// ✅ 正确
return $this->success('消息', $data);
return $this->error('错误消息', $code);

// ❌ 错误 - 不要自定义这些方法
private function success() {} // 会导致访问级别冲突
```

#### 3. 获取请求数据
```php
// GET参数
$id = $this->request->getParam('id');

// POST数据
$data = $this->request->getBodyParams();

// Headers
$token = $this->request->getHeader('Authorization');
```

#### 4. HTTP状态码
```php
return $this->error('Not found', 404);
return $this->error('Bad request', 400);
return $this->error('Unauthorized', 401);
return $this->error('Server error', 500);
```

## 路由配置

### routes.xml

**位置**: `app/code/Vendor/ModuleName/etc/routes.xml`

```xml
<?xml version="1.0"?>
<config xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" 
        xsi:noNamespaceSchemaLocation="urn:weline:framework:router:etc/routes.xsd">
    <router area="frontend_rest_api">
        <!-- GET /api/v1/entity/{id} -->
        <route path="/api/v1/entity/:id" method="GET">
            <module>Vendor_ModuleName</module>
            <controller>Api\Entity</controller>
            <action>get</action>
        </route>
        
        <!-- POST /api/v1/entity -->
        <route path="/api/v1/entity" method="POST">
            <module>Vendor_ModuleName</module>
            <controller>Api\Entity</controller>
            <action>post</action>
        </route>
        
        <!-- PUT /api/v1/entity/{id} -->
        <route path="/api/v1/entity/:id" method="PUT">
            <module>Vendor_ModuleName</module>
            <controller>Api\Entity</controller>
            <action>put</action>
        </route>
        
        <!-- DELETE /api/v1/entity/{id} -->
        <route path="/api/v1/entity/:id" method="DELETE">
            <module>Vendor_ModuleName</module>
            <controller>Api\Entity</controller>
            <action>delete</action>
        </route>
    </router>
</config>
```

**要点**:
- ✅ `area` 可选值: `frontend_rest_api`, `backend_rest_api`, `frontend_pc`, `backend_pc`
- ✅ 路径参数使用 `:paramName` 格式
- ✅ `method` 支持: GET, POST, PUT, DELETE, PATCH
- ✅ `controller` 路径相对于 `Controller/` 目录
- ✅ `action` 对应控制器的方法名

## 常见问题

### 1. 模块不显示或未激活

**问题**: 运行 `php bin/w module:list` 看不到模块

**解决方案**:
```bash
# 1. 检查register.php
cat app/code/Vendor/ModuleName/register.php

# 2. 检查module.xml
cat app/code/Vendor/ModuleName/etc/module.xml

# 3. 重新运行安装
php bin/w setup:upgrade

# 4. 清除缓存
php bin/w cache:clear
```

### 2. 数据库表未创建

**问题**: Setup/Install.php 执行了但表没创建

**常见原因**:
1. ❌ `createTable()` 方法调用不正确
2. ❌ 表名已存在
3. ❌ SQL语法错误

**解决方案**:
```php
// 在 Setup/Install.php 中添加调试
public function setup(Setup $setup, Context $context)
{
    echo "Creating tables...\n";
    
    try {
        $setup->getConnection()->createTable(
            'your_table',
            function ($table) {
                // 表结构
            },
            '表描述'
        );
        echo "Table created successfully\n";
    } catch (\Exception $e) {
        echo "Error: " . $e->getMessage() . "\n";
    }
}
```

### 3. Model接口方法签名错误

**错误**: `Declaration of Model::setup() must be compatible with...`

**原因**: 方法参数类型不匹配

**正确写法**:
```php
use Weline\Framework\Setup\Db\ModelSetup;  // ✅ 使用 ModelSetup
use Weline\Framework\Setup\Data\Context;

public function setup(ModelSetup $setup, Context $context): void {}
public function upgrade(ModelSetup $setup, Context $context): void {}
public function install(ModelSetup $setup, Context $context): void {}
```

**错误写法**:
```php
use Weline\Framework\Setup\Setup;  // ❌ 不要用 Setup

public function setup(Setup $setup, Context $context) {} // ❌ 缺少返回类型
```

### 4. Controller方法访问级别错误

**错误**: `Access level to Controller::success() must be protected`

**原因**: 重写了父类的protected方法

**解决方案**:
```php
// ❌ 错误 - 不要自定义success/error方法
private function success() {}
private function error() {}

// ✅ 正确 - 直接使用父类方法
return $this->success('消息', $data);
return $this->error('错误', $code);
```

### 5. 路由不工作

**问题**: API端点404

**检查清单**:
```bash
# 1. 确认routes.xml路径正确
ls app/code/Vendor/ModuleName/etc/routes.xml

# 2. 重新安装以注册路由
php bin/w setup:upgrade

# 3. 检查路由是否注册
grep -r "your-route-path" generated/routers/

# 4. 测试端点
php bin/w http:request GET /api/v1/entity/1
```

### 6. 依赖注入失败

**错误**: `Class does not exist`

**解决方案**:
```php
// ✅ 正确 - 使用完整命名空间
public function __construct(
    private readonly \Vendor\ModuleName\Service\EntityService $service
) {}

// ✅ 或使用use导入
use Vendor\ModuleName\Service\EntityService;

public function __construct(
    private readonly EntityService $service
) {}
```

## 最佳实践

### 1. 命名规范

```php
// 模块名: 大写驼峰 + 下划线
Vendor_ModuleName

// 类名: 大写驼峰
class EntityService

// 方法名: 小写驼峰
public function getById()

// 常量: 大写 + 下划线
public const STATUS_ACTIVE = 'active';

// 数据库表: 小写 + 下划线
your_table_name

// 数据库字段: 小写 + 下划线
created_at, user_id
```

### 2. 类型声明

```php
// ✅ 使用严格类型
declare(strict_types=1);

// ✅ 参数类型提示
public function create(int $id, string $name): Entity

// ✅ 返回类型声明
public function getData(): array

// ✅ 属性类型声明
private readonly EntityService $service;
```

### 3. 文档注释

```php
/**
 * 获取实体
```
