## 🎨 表格字符说明

### 标准表格
```
┌─┬─┐  ╔═╦═╗  
│ │ │  ║ ║ ║  
├─┼─┤  ╠═╬═╣  
└─┴─┘  ╚═╩═╝  
```

- **┌ ─ ┐ │ ├ ┼ ┤ └ ┴ ┘** - 标准边框（用于普通表格）
- **╔ ═ ╗ ║ ╠ ╬ ╣ ╚ ╩ ╝** - 加粗边框（用于重要信息框）

## 💡 TablePrinter Trait 功能

### 1. printTable($title, $data, $showBorder)
```php
$this->printTable('服务器信息', [
    ['进程ID', '4532'],
    ['监听地址', '127.0.0.1:9981'],
    ['运行时间', '0小时23分钟49秒'],
]);
```

**特点**：
- ✅ 自动计算列宽
- ✅ 支持中文对齐
- ✅ 可选边框显示
- ✅ 美观的分隔线

### 2. printList($title, $items)
```php
$this->printList('访问地址', [
    'http://localhost:8080/',
    'http://localhost:8080/api/rest',
]);
```

**特点**：
- ✅ 简洁的列表格式
- ✅ 统一的边框样式
- ✅ 自动宽度调整

### 3. printBox($title, $message, $type)
```php
$this->printBox('清理完成', '服务器进程和配置已清理完毕', 'success');
```

**类型**：
- `info` - 普通信息（默认）
- `success` - 成功信息（加粗边框）
- `warning` - 警告信息（加粗边框）
- `error` - 错误信息（加粗边框）

## 🚀 使用示例

### 在其他命令中使用 TablePrinter

```php
namespace Your\Console\Command;

use Weline\Framework\Console\Console\Server\TablePrinter;
use Weline\Framework\Console\CommandInterface;

class YourCommand implements CommandInterface
{
    use TablePrinter;
    
    public function execute(array $args = [], array $data = [])
    {
        // 打印表格
        $this->printTable('用户信息', [
            ['用户名', 'admin'],
            ['邮箱', 'admin@example.com'],
            ['状态', '在线'],
        ]);
        
        // 打印列表
        $this->printList('权限列表', [
            'user.create',
            'user.edit',
            'user.delete',
        ]);
        
        // 打印信息框
        $this->printBox('操作成功', '用户信息已更新', 'success');
    }
}
```
