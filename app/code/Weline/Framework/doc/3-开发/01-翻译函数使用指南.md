# 翻译函数使用指南

## 摘要

本文档介绍如何在 WelineFramework 中使用翻译函数 `__()` 实现国际化支持，包括在 PHP 代码、JavaScript 代码和模板文件中的使用方法，以及占位符的使用技巧。

## 什么是翻译函数

翻译函数 `__()` 是 WelineFramework 框架提供的国际化翻译函数，用于将代码中的文本转换为当前用户语言环境对应的翻译文本。通过翻译函数，开发者可以轻松实现多语言支持，无需修改业务逻辑代码。

## 为什么需要翻译函数

在开发多语言应用时，翻译函数提供了以下优势：

- **代码与翻译分离**：业务逻辑代码与翻译文本分离，便于维护和更新
- **统一管理**：所有翻译文本集中在 CSV 文件中，便于翻译人员协作
- **动态切换**：用户可以根据需要切换语言，无需修改代码
- **前后端统一**：PHP、JavaScript 和模板使用相同的翻译函数，保持一致性
- **占位符支持**：支持参数化翻译，动态插入变量值

## 目录文件结构

翻译文件应放在模块的 `i18n/` 目录下：

```
app/code/Weline/YourModule/
├── i18n/
│   ├── zh_Hans_CN.csv     # 中文翻译文件（默认，简体中文）
│   └── en_US.csv          # 英文翻译文件
├── Controller/
│   └── Index.php
└── ...
```

### 目录说明

- **i18n/**: 翻译文件目录，存放模块的翻译词条
- **zh_Hans_CN.csv**: 中文翻译文件（默认），源文本和翻译文本相同。框架默认使用 `zh_Hans_CN` 作为中文语言代码
- **en_US.csv**: 英文翻译文件，提供英文翻译

**注意**：框架默认使用 `zh_Hans_CN` 作为中文语言代码。虽然也支持 `zh_CN`，但建议使用 `zh_Hans_CN` 以保持一致性。

### CSV 文件格式

翻译文件使用 CSV 格式，每行一个翻译对：

```csv
"源文本","翻译文本"
"用户管理","用户管理"
"操作成功","操作成功"
"用户 %{name} 有 %{count} 条消息","用户 %{name} 有 %{count} 条消息"
```

**中文翻译文件（zh_Hans_CN.csv）**：
```csv
"用户管理","用户管理"
"操作成功","操作成功"
"用户 %{name} 有 %{count} 条消息","用户 %{name} 有 %{count} 条消息"
```

**英文翻译文件（en_US.csv）**：
```csv
"用户管理","User Management"
"操作成功","Operation Successful"
"用户 %{name} 有 %{count} 条消息","User %{name} has %{count} messages"
```

**重要说明**：
- CSV 文件使用双引号包裹文本
- 逗号分隔两列：第一列是源文本，第二列是翻译文本
- 中文翻译文件中，源文本和翻译文本通常相同
- 占位符（如 `%{name}`）在翻译文件中保持不变

## 使用方法

### 1. PHP 中使用翻译函数

#### 基本用法

```php
<?php

// 简单翻译
echo __('用户管理');
// 输出：用户管理（中文环境）或 User Management（英文环境）

// 在控制器中使用
class Index extends FrontendController
{
    public function index()
    {
        $this->assign('title', __('用户管理'));
        return $this->fetch();
    }
}
```

#### 带参数的翻译

```php
<?php

// 通用占位符（单个参数）
echo __('欢迎 %{}!', $username);
// 输出：欢迎 John!（中文）或 Welcome John!（英文）

// 数字占位符（多个参数）
echo __('用户 %{1} 有 %{2} 条消息', [$username, $count]);
// 输出：用户 John 有 5 条消息

// 命名占位符（推荐，语义清晰）
echo __('用户 %{name} 有 %{count} 条消息', [
    'name' => $username,
    'count' => $messageCount
]);
// 输出：用户 John 有 5 条消息
```

#### 在控制器中使用

```php
<?php

namespace Weline\YourModule\Controller;

use Weline\Framework\App\Controller\FrontendController;

class Index extends FrontendController
{
    public function index()
    {
        // 设置页面标题
        $this->assign('title', __('用户管理'));
        
        // 返回消息
        $this->messageManager->addSuccess(__('操作成功'));
        $this->messageManager->addError(__('操作失败：%{1}', [$error]));
        
        return $this->fetch();
    }
    
    public function postSave()
    {
        // 保存数据
        $result = $this->saveData();
        
        if ($result) {
            return $this->fetchJson([
                'code' => 200,
                'msg' => __('保存成功'),
                'data' => $result
            ]);
        } else {
            return $this->fetchJson([
                'code' => 400,
                'msg' => __('保存失败：%{error}', ['error' => $this->getError()])
            ]);
        }
    }
}
```

#### 在模型中使用

```php
<?php

namespace Weline\YourModule\Model;

use Weline\Framework\Database\AbstractModel;

class User extends AbstractModel
{
    public function validate()
    {
        if (empty($this->getData('name'))) {
            throw new \Exception(__('用户名不能为空'));
        }
        
        if (strlen($this->getData('password')) < 8) {
            throw new \Exception(__('密码长度不能少于 %{min} 位', ['min' => 8]));
        }
        
        return true;
    }
}
```

### 2. JavaScript 中使用翻译函数

框架会自动在页面中注入 `__()` 函数，可以在 JavaScript 代码中直接使用。

#### 基本用法

```javascript
// 简单翻译
console.log(__('用户管理'));
// 输出：用户管理（中文环境）或 User Management（英文环境）

// 在 DOM 操作中使用
document.getElementById('title').innerText = __('用户管理');
```

#### 带参数的翻译

```javascript
// 通用占位符（单个参数）
console.log(__('欢迎 %{}!', username));
// 输出：欢迎 John!

// 数组参数（数字占位符）
console.log(__('用户 %{1} 有 %{2} 条消息', [username, count]));
// 输出：用户 John 有 5 条消息

// 对象参数（命名占位符，推荐）
console.log(__('用户 %{name} 有 %{count} 条消息', {
    name: username,
    count: messageCount
}));
// 输出：用户 John 有 5 条消息
```

#### 在事件处理中使用

```javascript
// 按钮点击事件
button.addEventListener('click', function() {
    if (confirm(__('确定要删除 %{count} 项吗？', {
        count: selectedItems.length
    }))) {
        // 执行删除操作
    }
});

// AJAX 请求
$.ajax({
    url: '/api/users',
    success: function(data) {
        showMessage(__('成功加载 %{count} 个用户', {
            count: data.length
        }));
    },
    error: function() {
        showError(__('加载失败，请稍后重试'));
    }
});

// 表单验证
function validateForm() {
    let errors = [];
    
    if (!username) {
        errors.push(__('字段 %{field} 不能为空', {
            field: __('用户名')
        }));
    }
    
    if (password.length < 8) {
        errors.push(__('密码长度不能少于 %{min} 位', {
            min: 8
        }));
    }
    
    if (errors.length > 0) {
        alert(__('发现 %{count} 个错误：\n%{errors}', {
            count: errors.length,
            errors: errors.join('\n')
        }));
        return false;
    }
    
    return true;
}
```

### 3. 模板中使用翻译函数

在模板文件中，可以使用 PHP 的 `__()` 函数或 `<lang>` 标签。

#### 使用 PHP 函数

```php
<?php /** @var \Weline\Framework\View\Template $this */ ?>

<h1><?= __('用户管理') ?></h1>
<p><?= __('欢迎 %{}!', $username) ?></p>
<p><?= __('用户 %{name} 有 %{count} 条消息', [
    'name' => $username,
    'count' => $message_count
]) ?></p>

<button><?= __('保存') ?></button>
<button><?= __('取消') ?></button>
```

#### 使用 lang 标签

```html
<!-- 简单翻译 -->
<h1><lang>用户管理</lang></h1>
<p><lang>欢迎使用系统</lang></p>

<!-- 带参数翻译 - 字符串参数 -->
<p><lang args="$username">欢迎 %{}!</lang></p>

<!-- 带参数翻译 - 数组参数 -->
<p><lang args="[$username, $count]">用户 %{1} 有 %{2} 条消息</lang></p>

<!-- 带参数翻译 - 命名参数（推荐） -->
<p><lang args="['name' => $username, 'count' => $message_count]">
    用户 %{name} 有 %{count} 条消息
</lang></p>

<!-- 在属性中使用 -->
<input type="text" placeholder="<?= __('请输入用户名') ?>" />
<button title="<?= __('点击保存') ?>"><?= __('保存') ?></button>
```

#### 在 JavaScript 代码块中使用

```html
<script>
    // 简单翻译
    var title = '<?= __('用户管理') ?>';
    document.getElementById('page-title').innerText = title;
    
    // 带参数翻译
    function showWelcome(name) {
        alert('<?= __('欢迎 %{}!') ?>'.replace('%{}', name));
    }
    
    // 使用框架注入的 __() 函数
    function showUserInfo(name, count) {
        return __('用户 %{name} 有 %{count} 条消息', {
            name: name,
            count: count
        });
    }
</script>
```

## 占位符使用

翻译函数支持三种占位符格式：

### 1. 通用占位符 `%{}`

适用于单个参数的情况：

```php
// PHP
__('欢迎 %{}!', $username)

// JavaScript
__('欢迎 %{}!', username)
```

### 2. 数字占位符 `%{1}`, `%{2}`, ...

适用于多个参数，按顺序传递：

```php
// PHP
__('用户 %{1} 在 %{2} 登录', [$username, $loginTime])

// JavaScript
__('用户 %{1} 在 %{2} 登录', [username, loginTime])
```

### 3. 命名占位符 `%{name}`, `%{count}`, ...（推荐）

语义清晰，推荐使用：

```php
// PHP
__('用户 %{name} 有 %{count} 条消息', [
    'name' => $username,
    'count' => $messageCount
])

// JavaScript
__('用户 %{name} 有 %{count} 条消息', {
    name: username,
    count: messageCount
})
```

### 占位符格式对比

| 占位符格式 | 参数类型 | 适用场景 | 示例 |
|----------|---------|---------|------|
| `%{}` | 字符串/数字 | 单个参数 | `__('欢迎 %{}!', $name)` |
| `%{1}`, `%{2}` | 数组 | 多个参数，顺序固定 | `__('用户 %{1} 有 %{2} 条消息', [$name, $count])` |
| `%{name}`, `%{count}` | 对象/关联数组 | 多个参数，语义清晰（推荐） | `__('用户 %{name} 有 %{count} 条消息', ['name' => $name, 'count' => $count])` |

## 完整示例

### 控制器示例

```php
<?php

declare(strict_types=1);

namespace Weline\YourModule\Controller\Backend;

use Weline\Framework\App\Controller\BackendController;

class User extends BackendController
{
    public function index()
    {
        $this->assign('title', __('用户管理'));
        $this->assign('users', $this->getUserList());
        return $this->fetch();
    }
    
    public function postSave()
    {
        $data = $this->request->getPost();
        
        try {
            $user = $this->userModel->load($data['id']);
            $user->setData($data);
            $user->save();
            
            return $this->fetchJson([
                'code' => 200,
                'msg' => __('用户 %{name} 保存成功', ['name' => $data['name']])
            ]);
        } catch (\Exception $e) {
            return $this->fetchJson([
                'code' => 400,
                'msg' => __('保存失败：%{error}', ['error' => $e->getMessage()])
            ]);
        }
    }
    
    public function postDelete()
    {
        $ids = $this->request->getPost('ids', []);
        
        if (empty($ids)) {
            return $this->fetchJson([
                'code' => 400,
                'msg' => __('请选择要删除的用户')
            ]);
        }
        
        $count = $this->userModel->deleteByIds($ids);
        
        return $this->fetchJson([
            'code' => 200,
            'msg' => __('成功删除 %{count} 个用户', ['count' => $count])
        ]);
    }
}
```

### 模板示例

```php
<?php /** @var \Weline\Framework\View\Template $this */ ?>
<?php $users = $this->getData('users'); ?>

<div class="user-management">
    <h1><lang>用户管理</lang></h1>
    
    <div class="toolbar">
        <button onclick="addUser()"><?= __('添加用户') ?></button>
        <button onclick="batchDelete()"><?= __('批量删除') ?></button>
    </div>
    
    <table>
        <thead>
            <tr>
                <th><?= __('用户名') ?></th>
                <th><?= __('邮箱') ?></th>
                <th><?= __('创建时间') ?></th>
                <th><?= __('操作') ?></th>
            </tr>
        </thead>
        <tbody>
            <?php foreach ($users as $user): ?>
            <tr>
                <td><?= $this->escapeHtml($user->getName()) ?></td>
                <td><?= $this->escapeHtml($user->getEmail()) ?></td>
                <td><?= $user->getCreatedAt() ?></td>
                <td>
                    <button onclick="editUser(<?= $user->getId() ?>)">
                        <?= __('编辑') ?>
                    </button>
                    <button onclick="deleteUser(<?= $user->getId() ?>, '<?= $this->escapeJs($user->getName()) ?>')">
                        <?= __('删除') ?>
                    </button>
                </td>
            </tr>
            <?php endforeach; ?>
        </tbody>
    </table>
</div>

<script>
function addUser() {
    window.location.href = '/backend/your-module/user/add';
}

function editUser(id) {
    window.location.href = '/backend/your-module/user/edit?id=' + id;
}

function deleteUser(id, name) {
    if (confirm(__('确定要删除用户 %{name} 吗？', {name: name}))) {
        $.ajax({
            url: '/backend/your-module/user/delete',
            method: 'POST',
            data: {id: id},
            success: function(response) {
                if (response.code === 200) {
                    alert(__('删除成功'));
                    location.reload();
                } else {
                    alert(__('删除失败：%{error}', {error: response.msg}));
                }
            },
            error: function() {
                alert(__('请求失败，请稍后重试'));
            }
        });
    }
}

function batchDelete() {
    var selectedIds = getSelectedIds();
    
    if (selectedIds.length === 0) {
        alert(__('请选择要删除的用户'));
        return;
    }
    
    if (confirm(__('确定要删除 %{count} 个用户吗？', {count: selectedIds.length}))) {
        $.ajax({
            url: '/backend/your-module/user/batch-delete',
            method: 'POST',
            data: {ids: selectedIds},
            success: function(response) {
                if (response.code === 200) {
                    alert(__('成功删除 %{count} 个用户', {count: response.data.count}));
                    location.reload();
                } else {
                    alert(__('删除失败：%{error}', {error: response.msg}));
                }
            }
        });
    }
}
</script>
```

## 后续处理

### 1. 收集翻译词条

框架提供了命令行工具自动收集代码中的翻译词条：

```bash
# 收集所有模块的翻译词条
php bin/w i18n:collect

# 收集特定模块的翻译词条
php bin/w i18n:collect Weline_YourModule
```

**收集原理**：
- 框架会扫描 PHP 文件（`.php`）、模板文件（`.phtml`）和 JavaScript 文件（`.js`）
- 使用正则表达式匹配 `__()` 函数调用：`/__\(['"](.+?)['"]\)/`
- 自动生成或更新 `i18n/*.csv` 文件

### 2. 手动添加翻译词条

如果自动收集遗漏了某些词条，可以手动添加到 CSV 文件：

```csv
"新添加的文本","新添加的文本"
```

然后在英文翻译文件中添加对应的翻译：

```csv
"新添加的文本","Newly Added Text"
```

### 3. 更新翻译文件

修改翻译文件后，需要清理缓存使翻译生效：

```bash
# 清理缓存
php bin/w cache:clear
```

## 验证

### 验证翻译函数是否工作

1. **检查翻译文件是否存在**：
   ```bash
   ls app/code/Weline/YourModule/i18n/
   # 应该看到 zh_Hans_CN.csv 和 en_US.csv
   ```

2. **检查翻译词条是否完整**：
   - 打开 `i18n/zh_Hans_CN.csv`，确认所有使用的文本都已添加
   - 打开 `i18n/en_US.csv`，确认所有文本都有对应的英文翻译

3. **测试翻译效果**：
   - 切换到中文环境，查看页面显示是否为中文
   - 切换到英文环境，查看页面显示是否为英文
   - 检查带参数的翻译是否正确替换占位符

### 常见问题

1. **翻译不生效**：
   - 检查翻译文件是否存在
   - 检查 CSV 文件格式是否正确（双引号、逗号分隔）
   - 清理缓存：`php bin/w cache:clear`
   - 检查翻译词条是否在 CSV 文件中

2. **占位符未替换**：
   - 检查占位符格式是否正确（`%{}`, `%{1}`, `%{name}`）
   - 检查参数传递是否正确
   - 检查翻译文件中占位符是否保持一致

3. **JavaScript 中 `__()` 函数未定义**：
   - 确保页面加载了框架的 JavaScript 文件
   - 检查浏览器控制台是否有错误
   - 确认框架已正确初始化

4. **翻译文件格式错误**：
   - CSV 文件必须使用双引号包裹文本
   - 逗号分隔两列，不能有多余的逗号
   - 每行一个翻译对，不能换行

## 最佳实践

1. **使用命名占位符**：参数较多时，使用命名占位符更易读易维护
   ```php
   // ✅ 推荐
   __('用户 %{name} 在 %{time} 登录', ['name' => $name, 'time' => $time])
   
   // ❌ 不推荐（参数多时难以维护）
   __('用户 %{1} 在 %{2} 登录', [$name, $time])
   ```

2. **保持文本完整性**：使用完整的句子，不要拆分
   ```php
   // ✅ 推荐
   __('用户 %{name} 有 %{count} 条消息')
   
   // ❌ 不推荐
   __('用户') . ' ' . $name . ' ' . __('有') . ' ' . $count . ' ' . __('条消息')
   ```

3. **代码中使用中文**：代码中的文本使用中文，翻译文件提供其他语言翻译
   ```php
   // ✅ 正确
   __('操作成功')
   
   // ❌ 错误
   __('Operation Successful')
   ```

4. **及时更新翻译文件**：添加新的翻译文本后，及时运行 `i18n:collect` 命令更新翻译文件

5. **测试多语言**：开发完成后，测试不同语言环境下的显示效果
