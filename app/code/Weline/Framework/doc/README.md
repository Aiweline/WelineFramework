## 🎯 框架简介

`Weline_Framework` 是 WelineFramework 的核心模块，提供：

### 核心功能
1. **MVC架构** - 清晰的模型-视图-控制器分离
2. **模块化设计** - 支持独立模块开发、部署和维护
3. **ORM数据库** - 自定义ORM链式操作
4. **事件系统** - 观察者模式和事件驱动
5. **命令行工具** - 丰富的CLI命令支持
6. **国际化支持** - 完整的多语言系统
7. **缓存系统** - 多层次缓存支持

### 架构特点
- **依赖注入** - 自动依赖解析和注入
- **观察者模式** - 灵活的事件监听机制
- **插件系统** - 支持功能扩展
- **模板引擎** - 强大的视图渲染能力
