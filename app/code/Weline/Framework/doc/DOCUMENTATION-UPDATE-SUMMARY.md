## ✨ 主要特性说明

### JavaScript 增强功能
1. **新增 `%{}` 通用占位符** - 与 PHP 完全一致
2. **完善数字占位符** - `%{1}`, `%{2}`, ... 支持数组参数
3. **完善命名占位符** - `%{name}`, `%{count}`, ... 支持对象参数
4. **支持数字类型** - 不仅支持字符串，还支持数字参数
5. **空值处理** - 自动将 null/undefined 转换为空字符串

### Lang 标签增强功能
1. **新增 `args` 属性** - 支持传递占位符参数
2. **支持变量引用** - `args="$username"`
3. **支持数组参数** - `args="['John', 5]"`
4. **支持命名参数** - `args="['name' => 'John', 'count' => 5]"`
5. **完全向后兼容** - 所有旧代码无需修改
