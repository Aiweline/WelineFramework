# 自定义模型

## 摘要

本文档介绍如何在 WelineFramework 中创建自定义数据模型，包括模型的基本结构、数据库操作方法和使用示例。

## 什么是模型

模型（Model）是 MVC 架构中的数据层组件，负责与数据库交互，封装数据的增删改查操作。在 WelineFramework 中，模型继承自 `\Weline\Framework\Database\Model`，提供了强大的 ORM 功能。

## 为什么需要模型

模型将数据库操作封装为面向对象的方法，提供了以下优势：
- **类型安全**：通过字段常量定义，避免字段名拼写错误
- **链式操作**：支持流畅的链式查询语法
- **自动建表**：通过 `install()` 方法自动创建数据库表
- **数据验证**：内置数据验证机制

## 目录文件结构

模型文件应放在模组的 `Model` 目录下：

```
app/code/Weline/YourModule/
└── Model/
    └── YourModel.php
```

### 文件说明

- **Model/**: 模型目录，存放所有数据模型类
- **YourModel.php**: 模型类文件，继承 `\Weline\Framework\Database\Model`

## 模型结构

### 基础模型示例

```php
<?php

declare(strict_types=1);

namespace Weline\YourModule\Model;

use Weline\Framework\Database\Api\Db\TableInterface;
use Weline\Framework\Manager\ObjectManager;
use Weline\Framework\Setup\Data\Context;
use Weline\Framework\Setup\Db\ModelSetup;

class YourModel extends \Weline\Framework\Database\Model
{
    // 定义字段常量
    public const fields_ID = 'id';
    public const fields_NAME = 'name';
    public const fields_EMAIL = 'email';
    public const fields_CREATED_AT = 'created_at';

    /**
     * 安装数据表
     */
    public function install(ModelSetup $setup, Context $context): void
    {
        if (!$setup->tableExist()) {
            $setup->createTable('示例表')
                ->addColumn(self::fields_ID, TableInterface::column_type_INTEGER, 0, 'primary key auto_increment', 'ID')
                ->addColumn(self::fields_NAME, TableInterface::column_type_VARCHAR, 255, 'not null', '名称')
                ->addColumn(self::fields_EMAIL, TableInterface::column_type_VARCHAR, 255, '', '邮箱')
                ->addColumn(self::fields_CREATED_AT, TableInterface::column_type_DATETIME, 0, '', '创建时间')
                ->create();
        }
    }

    /**
     * 升级数据表
     */
    public function upgrade(ModelSetup $setup, Context $context): void
    {
        // 升级逻辑
    }

    /**
     * 设置方法
     */
    public function setName(string $name): YourModel
    {
        return $this->setData(self::fields_NAME, $name);
    }

    public function setEmail(string $email): YourModel
    {
        return $this->setData(self::fields_EMAIL, $email);
    }

    /**
     * 获取方法
     */
    public function getName(): ?string
    {
        return $this->getData(self::fields_NAME);
    }

    public function getEmail(): ?string
    {
        return $this->getData(self::fields_EMAIL);
    }
}
```

## 数据库操作

### 1. 保存数据

```php
// 创建新记录
$model = ObjectManager::getInstance(YourModel::class);
$model->setName('测试')
    ->setEmail('test@example.com')
    ->save();

// 更新记录（需要先加载）
$model = ObjectManager::getInstance(YourModel::class);
$model->load(1);  // 加载 ID 为 1 的记录
$model->setName('新名称')->save();
```

### 2. 查询数据

```php
// 根据主键加载
$model = ObjectManager::getInstance(YourModel::class);
$model->load(1);

// 条件查询
$models = ObjectManager::getInstance(YourModel::class)
    ->where(YourModel::fields_NAME, '测试')
    ->select()
    ->fetch();

// 链式查询
$models = ObjectManager::getInstance(YourModel::class)
    ->where(YourModel::fields_EMAIL, 'test@example.com', 'like')
    ->orderBy(YourModel::fields_CREATED_AT, 'DESC')
    ->limit(10)
    ->select()
    ->fetch();
```

### 3. 关联查询

```php
// JOIN 查询
$models = ObjectManager::getInstance(YourModel::class)
    ->joinModel(OtherModel::class, 'alias', 'main_table.id=alias.model_id')
    ->fields('main_table.*, alias.name as other_name')
    ->select()
    ->fetch();
```

### 4. 分页查询

```php
$models = ObjectManager::getInstance(YourModel::class)
    ->pagination($page, $pageSize, $params)
    ->select()
    ->fetch();

$pagination = $model->getPagination();  // 获取分页信息
```

### 5. 删除数据

```php
// 删除单条记录
$model = ObjectManager::getInstance(YourModel::class);
$model->load(1)->delete();

// 批量删除
ObjectManager::getInstance(YourModel::class)
    ->where(YourModel::fields_ID, [1, 2, 3], 'in')
    ->delete()
    ->fetch();
```

## 字段常量定义

模型必须定义字段常量，用于类型安全和避免拼写错误：

```php
public const fields_ID = 'id';
public const fields_NAME = 'name';
```

**注意**：主键字段必须定义为 `fields_ID`，框架会自动识别。

## 数据库表创建

### install() 方法

在 `install()` 方法中定义表结构：

```php
public function install(ModelSetup $setup, Context $context): void
{
    if (!$setup->tableExist()) {
        $setup->createTable('表名')
            ->addColumn(self::fields_ID, TableInterface::column_type_INTEGER, 0, 'primary key auto_increment', 'ID')
            ->addColumn(self::fields_NAME, TableInterface::column_type_VARCHAR, 255, 'not null', '名称')
            ->create();
    }
}
```

### 字段类型

常用的字段类型：
- `column_type_INTEGER`: 整数
- `column_type_VARCHAR`: 可变长度字符串
- `column_type_TEXT`: 文本
- `column_type_DATETIME`: 日期时间
- `column_type_DECIMAL`: 小数

## 后续处理

### 1. 安装数据表

创建模型后，运行模块升级命令创建数据表：

```bash
php bin/w module:upgrade
```

### 2. 在控制器中使用

```php
use Weline\Framework\Manager\ObjectManager;

public function index()
{
    $model = ObjectManager::getInstance(YourModel::class);
    $items = $model->select()->fetch();
    $this->assign('items', $items);
    return $this->fetch();
}
```

## 验证

### 验证数据表是否创建

1. 检查数据库表是否存在
2. 检查表结构是否正确

### 验证模型操作

1. **保存测试**：
```php
$model = ObjectManager::getInstance(YourModel::class);
$model->setName('测试')->save();
echo $model->getId();  // 应该输出新插入的 ID
```

2. **查询测试**：
```php
$model = ObjectManager::getInstance(YourModel::class);
$item = $model->load(1);
echo $item->getName();  // 应该输出名称
```

### 常见问题

1. **表未创建**：检查 `install()` 方法是否正确，确保运行了 `module:upgrade` 命令
2. **字段不存在**：检查字段常量定义是否正确，确保使用字段常量而不是字符串
3. **主键错误**：确保主键字段定义为 `fields_ID`
