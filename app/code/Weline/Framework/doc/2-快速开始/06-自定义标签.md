# 自定义标签

## 摘要

本文档介绍如何在 WelineFramework 中创建自定义模板标签，实现可复用的前端组件。

## 什么是自定义标签

自定义标签是 WelineFramework 提供的模板扩展机制，允许开发者创建可复用的模板标签，实现模块化、组件化的前端开发。标签可以在模板中像 HTML 标签一样使用，但具有自定义的处理逻辑。

## 为什么需要自定义标签

自定义标签提供了以下优势：
- **代码复用**：将重复的模板代码封装为标签，提高代码复用率
- **模块化开发**：将功能拆分为独立的标签组件，便于维护
- **统一管理**：标签集中管理，便于统一修改和升级
- **简化模板**：模板代码更简洁，可读性更好

## 目录文件结构

标签类应放在模组的 `Taglib` 目录下：

```
app/code/Weline/YourModule/
└── Taglib/
    └── YourTag.php
```

### 目录说明

- **Taglib/**: 标签目录，存放所有自定义标签类
- **YourTag.php**: 标签类文件，实现 `TaglibInterface` 接口

## 标签接口

标签类必须实现 `Weline\Taglib\TaglibInterface` 接口，包含以下方法：

### 必需方法

- `name()`: 返回标签名称
- `tag()`: 是否支持成对标签
- `callback()`: 处理回调函数

### 可选方法

- `attr()`: 属性定义
- `tag_start()`: 是否处理标签开始
- `tag_end()`: 是否处理标签结束
- `tag_self_close()`: 是否支持自闭合
- `parent()`: 依赖管理
- `document()`: 标签文档

## 创建自定义标签

### 基础标签示例

```php
<?php

declare(strict_types=1);

namespace Weline\YourModule\Taglib;

use Weline\Taglib\TaglibInterface;

class YourTag implements TaglibInterface
{
    /**
     * 标签名称
     */
    public static function name(): string
    {
        return 'your-tag';
    }

    /**
     * 是否支持成对标签
     */
    public static function tag(): bool
    {
        return true;  // true 支持 <tag></tag>，false 只支持 <tag/>
    }

    /**
     * 属性定义
     */
    public static function attr(): array
    {
        return [
            'title' => true,    // 必需属性
            'class' => false,   // 可选属性
            'style' => false    // 可选属性
        ];
    }

    /**
     * 处理回调函数
     */
    public static function callback(): callable
    {
        return function ($tag_key, $config, $tag_data, $attributes) {
            $title = $attributes['title'] ?? '';
            $class = $attributes['class'] ?? '';
            $content = $tag_data[1] ?? '';  // 标签内容
            
            return '<div class="' . htmlspecialchars($class ?? '') . '">' .
                   '<h3>' . htmlspecialchars($title ?? '') . '</h3>' .
                   $content .
                   '</div>';
        };
    }

    /**
     * 标签文档
     */
    public static function document(): string
    {
        return '自定义标签示例';
    }
}
```

### 自闭合标签示例

```php
public static function tag(): bool
{
    return false;  // 不支持成对标签
}

public static function tag_self_close(): bool
{
    return true;  // 支持自闭合
}
```

## 在模板中使用标签

标签在模板中使用 `w:` 前缀：

```php
<!-- 成对标签 -->
<w:your-tag title="标题" class="custom-class">
    标签内容
</w:your-tag>

<!-- 自闭合标签 -->
<w:your-tag title="标题" class="custom-class" />
```

## 标签注册

标签类创建后，框架会自动扫描并注册。确保：

1. 标签类位于 `Taglib` 目录下
2. 实现了 `TaglibInterface` 接口
3. 运行了 `module:upgrade` 命令

## 后续处理

### 1. 运行模块升级

创建标签后，运行模块升级命令：

```bash
php bin/w module:upgrade
```

### 2. 在模板中使用

在模板文件中使用自定义标签，确保使用 `w:` 前缀。

## 验证

### 验证标签是否注册成功

1. 检查标签是否在标签列表中
2. 在模板中使用标签，检查是否正常渲染

### 常见问题

1. **标签未识别**：检查标签类是否正确实现了接口，确保运行了 `module:upgrade` 命令
2. **属性错误**：检查属性定义是否正确，确保必需属性已提供
3. **渲染错误**：检查回调函数逻辑是否正确，确保返回值是字符串
