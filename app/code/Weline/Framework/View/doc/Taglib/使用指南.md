# Taglib 使用指南

## 基本用法

### 编译模板

```php
use Weline\Framework\View\Taglib;
use Weline\Framework\View\Template;

$taglib = new Taglib();
$template = new Template();

$content = '<block template="header.phtml"/>';
$compiled = $taglib->compile($template, $content, 'page.phtml');
```

### 注册自定义标签

```php
// 编译期标签
$taglib->registerTag('hello', function(array $params) {
    $name = $params['name'] ?? 'World';
    return "Hello, {$name}!";
});

// 使用
// <hello name="Weline"/>
// 输出: Hello, Weline!
```

### 注册运行期标签

```php
// 运行期标签（动态内容）
$taglib->registerRuntimeTag('current-time', function(array $params) {
    return date('Y-m-d H:i:s');
});

// 使用
// <current-time/>
// 输出: 2026-01-31 12:00:00
```

## 内置标签

### 控制流标签

```html
<!-- if 标签 -->
<if condition="$show">
    Content to show
</if>

<!-- foreach 标签 -->
<foreach items="$items" as="$item">
    <div><?= $item ?></div>
</foreach>
```

### 模板标签

```html
<!-- block 标签 -->
<block template="components/header.phtml"/>

<!-- lang 标签 -->
<lang>Hello World</lang>
```

## 属性中的 PHP 代码

Taglib v2 支持在标签属性中使用 PHP 代码：

```html
<!-- 动态属性值 -->
<block class="<?= $containerClass ?>"/>

<!-- 混合值 -->
<a href="/user/<?= $userId ?>">View Profile</a>

<!-- 复杂表达式 -->
<div title="<?= htmlspecialchars($user->getName()) ?>">
```

## 命名空间标签

支持带命名空间的标签：

```html
<!-- 格式: prefix:module:tag -->
<w:seo:account:select id="account"/>

<!-- 映射到: Weline\Seo\Taglib\AccountSelect::callback() -->
```

```php

// 获取编译后代码第 10 行对应的源位置

// ['file' => 'template.phtml', 'line' => 5]
```

### 启用调试模式

```php
$taglib->setDebug(true);
```

## 自定义编译优化

### 添加编译通道

```php
use Weline\Framework\View\Taglib\Compiler\Pass\CompilePassInterface;
use Weline\Framework\View\Taglib\Ast\ProgramNode;

class MyOptimizationPass implements CompilePassInterface
{
    public function process(ProgramNode $ast): ProgramNode
    {
        // 自定义优化逻辑
        return $ast;
    }
    
    public function getName(): string
    {
        return 'my-optimization';
    }
    
    public function getPriority(): int
    {
        return 100;
    }
}

$taglib->addCompilePass(new MyOptimizationPass());
```

## 最佳实践

1. **避免在循环中编译** - 编译是耗时操作，应该缓存结果
2. **使用命名空间标签** - 便于管理和查找标签实现
3. **静态内容优先** - 尽量使用静态属性，动态属性会延迟到运行期
4. **利用缓存** - 生产环境启用 APCu 获得最佳性能
