# Taglib v2 架构设计

## 概述

Taglib v2 是 Weline Framework 的模板标签编译器，采用极致性能架构设计。

## 设计原则

- **不向后兼容** - 完全重写，删除旧 3000+ 行代码
- **极致性能** - PHP 8.4+ 最新语法 + JIT 友好
- **多级缓存** - WeakMap + APCu + 文件缓存
- **编译管道** - 常量折叠、死代码消除等优化
- **SOLID 架构** - 职责单一，依赖抽象

## 编译流程

```
模板内容
    ↓
阶段1: PHP 预处理（PhpExtractor）
    - 使用 token_get_all() 精确识别 PHP 代码
    - 将 PHP 代码替换为占位符
    ↓
阶段2: 词法分析（Tokenizer）
    - 将清洁内容转换为 Token 流
    - Generator 流式处理，内存 O(1)
    ↓
阶段3: AST 构建（AstBuilder）
    - 构建抽象语法树
    - 使用 NodePool 复用对象
    ↓
阶段4: 编译管道（CompilePipeline）
    - 常量折叠
    - 死代码消除
    - 内联优化
    ↓
阶段5: 代码生成（CodeGenerator）
    - 生成 PHP 代码
    - 恢复 PHP 占位符
    ↓
编译结果（缓存到多级缓存）
```

## 多级缓存架构

| 层级 | 技术 | 访问速度 | 生命周期 |
|------|------|----------|----------|
| L0 | WeakMap | 0.001ms | 进程内，随 Template 对象销毁 |
| L1 | APCu | 0.01ms | 跨进程共享，PHP 重启后清空 |
| L3 | 文件 | 0.5ms | 持久化，OPcache 可缓存 |

├── Compiler/
│   ├── CompilePipeline.php      # 编译管道

├── Generator/

## PHP 8.4+ 特性应用

### Property Hooks

```php
class TagNode extends Node
{
    public bool $isDynamic {
        get => $this->hasDynamicAttrs || $this->hasDynamicChildren;
    }
}
```

### readonly 属性

```php
class Node
{
    public function __construct(
        public readonly int $line,
        public readonly int $column = 0,
    ) {}
}
```

## 性能优化

| 优化项 | 技术 | 预期收益 |
|--------|------|----------|
| PHP 识别 | token_get_all() | 100% 准确识别 |
| 多级缓存 | WeakMap + APCu + File | 10-100x 命中速度 |
| 对象池 | NodePool + Flyweight | 减少 GC 压力 |
| 字符串去重 | StringInterner | 减少内存 30%+ |
| 分支预测 | 按频率排序条件 | CPU 效率提升 |
| 流式处理 | Generator yield | 大模板 O(1) 内存 |

## 使用示例

```php
use Weline\Framework\View\Taglib;
use Weline\Framework\View\Template;

$taglib = new Taglib();
$template = new Template();

// 编译模板
$compiled = $taglib->compile($template, $content, 'template.phtml');

// 注册自定义标签
$taglib->registerTag('custom', function(array $params) {
    return 'Custom output: ' . ($params['value'] ?? '');
});

// 注册运行期标签
$taglib->registerRuntimeTag('dynamic', function(array $params) {
    return 'Dynamic: ' . $params['__content'];
});

// 获取统计信息
$stats = $taglib->stats();
```

## 向后兼容

以下方法保留但标记为废弃：

- `tagReplace()` - 使用 `compile()` 替代
- `varParser()` - 变量解析由 PhpExtractor 处理
- `getTags()` - 使用 TagRegistry 替代
- `checkFilter()` - 过滤器处理方式已改变
- `checkVar()` - 使用 `varParser()` 替代
