# Taglib 多级缓存

## 概述

Taglib v2 采用三级缓存架构，最大化缓存命中率和访问速度。

## 缓存层级

### L0: WeakMap（进程内缓存）

- **技术**: PHP WeakMap
- **访问速度**: ~0.001ms
- **生命周期**: 随 Template 对象销毁
- **特点**: 
  - 最快的访问速度
  - 自动内存管理
  - 不跨请求

### L1: APCu（共享内存缓存）

- **技术**: APCu 扩展
- **访问速度**: ~0.01ms
- **生命周期**: PHP-FPM 重启前持续有效
- **特点**:
  - 跨进程共享
  - 适合多进程环境
  - 需要 APCu 扩展

### L3: 文件缓存

- **技术**: 文件系统
- **访问速度**: ~0.5ms（热读取）
- **生命周期**: 持久化，直到手动清除
- **特点**:
  - 重启后保留
  - OPcache 可缓存字节码
  - 适合生产环境

## 缓存查询流程

```
请求编译模板
    ↓
检查 L0 (WeakMap)
    ├─ 命中 → 返回
    ↓ 未命中
检查 L1 (APCu)
    ├─ 命中 → 回填 L0 → 返回
    ↓ 未命中
检查 L3 (文件)
    ├─ 命中 → 回填 L0, L1 → 返回
    ↓ 未命中
执行编译 → 写入 L0, L1, L3 → 返回
```

## 缓存 Key 设计

### 哈希算法

使用 xxh3 算法，比 md5 快 10 倍：

```php
$hash = hash('xxh3', $content);
```

### Key 格式

```
L0: Template 对象引用
L1: taglib_v2_{base64(path)}_{hash}
L3: {cacheDir}/{pathHash[0:2]}/{contentHash}.php
```

## 使用示例

### 基本用法

```php
use Weline\Framework\View\Taglib;
use Weline\Framework\View\Template;

$taglib = new Taglib();
$template = new Template();

// 首次编译（写入所有缓存层）
$compiled = $taglib->compile($template, $content, 'template.phtml');

// 再次编译（从缓存获取）
$compiled = $taglib->compile($template, $content, 'template.phtml');
```

### 直接使用缓存组件

```php
use Weline\Framework\View\Taglib\Cache\MultiLevelCache;

$cache = new MultiLevelCache();

// 获取缓存
$result = $cache->getByPath('/path/to/template.phtml', $content);

// 设置缓存
$cache->setByPath('/path/to/template.phtml', $content, $compiled);

// 清除缓存
$cache->delete('/path/to/template.phtml');

// 清空所有
$cache->flush();
```

### 查看统计

```php
$stats = $cache->stats();
```

## APCu 配置

### php.ini 配置

```ini
apc.enabled = 1
apc.shm_size = 128M
apc.ttl = 0
apc.gc_ttl = 3600
apc.entries_hint = 10000
```

### 检查 APCu 状态

```php
use Weline\Framework\View\Taglib\Cache\ApcuCache;

$apcu = new ApcuCache();
if ($apcu->isEnabled()) {
    echo "APCu 已启用";
} else {
    echo "APCu 未启用，将回退到文件缓存";
}
```

## 文件缓存配置

### 自定义缓存目录

```php
use Weline\Framework\View\Taglib\Cache\FileCache;

$cache = new FileCache('/custom/cache/path');
```

### 默认目录

```
{项目根目录}/var/cache/taglib/
```

## 缓存失效

### 自动失效

- 模板内容变化（内容哈希不同）
- Template 对象销毁（L0）
- PHP 进程重启（L0, L1）

### 手动失效

```php
// 清除特定模板
$taglib->clearCache();

// 或使用缓存组件
$cache->delete('/path/to/template.phtml');
$cache->flush();
```

## 生产环境建议

1. **启用 APCu** - 跨进程共享编译结果
2. **启用 OPcache** - 缓存生成的 PHP 字节码
3. **预热缓存** - 部署后预编译常用模板
4. **监控命中率** - 确保缓存有效工作

### 预热脚本

```php
$taglib = new Taglib();
$templates = [
    'header.phtml' => file_get_contents('/path/to/header.phtml'),
    'footer.phtml' => file_get_contents('/path/to/footer.phtml'),
    // ...
];

foreach ($templates as $path => $content) {
    $taglib->compile(new Template(), $content, $path);
}
```
