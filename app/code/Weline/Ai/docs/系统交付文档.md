## ❌ 如果测试失败

### 问题1: 数据库表不存在

**症状**：测试页面显示红色 "表不存在"

**解决方案**：
```bash
# 进入项目目录
cd E:\WelineFramework\DEV-workspace

# 创建表
php bin/w ai:database:create-tables

# 验证
php bin/w ai:test:verify
```

然后重新刷新浏览器测试页面。

### 问题2: 找不到类

**症状**：Class not found 错误

**解决方案**：
```bash
# 清除缓存
php bin/w cache:clear

# 重新生成
php bin/w setup:upgrade
```

### 问题3: 权限错误

**症状**：Permission denied

**解决方案**：
```bash
# Linux/Mac
chmod -R 755 app/code/Weline/Ai/
chmod -R 777 var/

# Windows
# 右键文件夹 -> 属性 -> 安全 -> 编辑权限
```

## 🚀 部署指南

### 系统要求
- PHP >= 8.0
- WelineFramework >= 1.0
- 数据库：SQLite/MySQL/PostgreSQL
- Web服务器：Nginx/Apache

### 部署步骤

#### 1. 模块注册
```bash
php bin/w setup:upgrade
```

#### 2. 创建数据库表
```bash
php bin/w ai:database:create-tables
```

#### 3. 验证安装
```bash
php verify_ai_installation.php
```

#### 4. 填充测试数据（可选）
```bash
php bin/w ai:database:seed
```

#### 5. 运行完整测试
```bash
# Windows
run_ai_tests.bat

# Linux/Mac
bash run_ai_tests.sh
```

## 📝 使用说明

### 快速开始

1. **创建AI模型**
```php
$aiModel = ObjectManager::getInstance(AiModel::class);
$aiModel->setData([...]);
$aiModel->save();
```

2. **生成API密钥**
```bash
php bin/w ai:apikey:generate --name="My Key"
```

3. **使用AI服务**
```php
$response = AiService::generateText('你好，AI！');
```
