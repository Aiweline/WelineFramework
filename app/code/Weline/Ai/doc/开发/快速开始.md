# 快速开始指南

## 5分钟快速上手

### 1. 安装模块

模块已预装在系统中，无需额外安装。

### 2. 初始化数据

```bash
# 运行系统升级（自动创建表）
php bin/w setup:upgrade

# 生成测试数据
php bin/w test:seed-data
```

### 3. 访问后台

```bash
# 启动服务器
php bin/w s:sta

# 访问后台管理
# 地址将在终端显示，例如: http://127.0.0.1:9981/<ENC>/admin/login
```

### 4. 核心功能快速浏览

#### 4.1 AI 管理（聚合页）
- 路由: `ai/backend/manager`
- 功能: 模型 | 适配器 | 供应商账户 三 Tab 聚合
- 操作: 新增模型、编辑模型、切换状态、测试连接

#### 4.2 场景配置
- 路由: `ai/backend/adapter`
- 功能: 管理场景适配器
- 操作: 扫描适配器、启用/禁用、查看详情

#### 4.3 模型管理（直接访问 / iframe 嵌入）
- 路由: `ai/backend/model`
- 功能: 查看、添加、编辑 AI 模型
- 操作: 新增模型、设置默认模型、配置参数

#### 4.4 供应商账户
- 路由: `ai/backend/provider`
- 功能: 管理供应商 API 账户

#### 4.5 默认模型
- 路由: `ai/backend/defaultmodel`
- 功能: 为不同服务类型设置默认模型

### 5. API使用示例

#### 5.1 通过REST API调用

```bash
# 测试模型列表API
php bin/w http:request ai/rest/ai/models -b --login -n=1
```

#### 5.2 通过PHP代码调用

```php
use Weline\Ai\Service\AiService;
use Weline\Framework\Manager\ObjectManager;

// 获取AI服务
$aiService = ObjectManager::getInstance(AiService::class);

// 发送聊天请求
$response = $aiService->chat([
    'model' => 'gpt-3.5-turbo',
    'messages' => [
        ['role' => 'user', 'content' => 'Hello, AI!']
    ]
]);

echo $response['choices'][0]['message']['content'];
```

### 6. 开发自定义适配器

```php
<?php

namespace YourModule\Ai\Adapter;

use Weline\Ai\Api\ScenarioAdapterInterface;

class CustomAdapter implements ScenarioAdapterInterface
{
    public function getCode(): string
    {
        return 'custom_scenario';
    }

    public function getName(): string
    {
        return '自定义场景';
    }

    public function getDescription(): string
    {
        return '处理自定义场景的适配器';
    }

    public function adaptPrompt(array $params): array
    {
        return [
            'role' => 'system',
            'content' => '你是一个专业的助手...'
        ];
    }

    public function processResponse(array $response, array $params): array
    {
        return $response;
    }

    public function validate(array $params): bool
    {
        return isset($params['input']);
    }

    public function getSupportedModelTypes(): array
    {
        return ['chat', 'completion'];
    }

    public function getParamTemplate(): array
    {
        return [
            'input' => [
                'type' => 'string',
                'required' => true,
                'description' => '输入内容'
            ]
        ];
    }

    public function getExamples(): array
    {
        return [
            [
                'name' => '基础使用',
                'params' => ['input' => '你好']
            ]
        ];
    }

    public function getVersion(): string
    {
        return '1.0.0';
    }
}
```

### 7. 运行测试

```bash
# 运行所有单元测试
php bin/w phpunit:run --module=Weline_Ai -b

# 运行Model层测试
php bin/w phpunit:run --module=Weline_Ai --filter=Model -b

# 运行HTTP端点测试
php bin/w http:request ai/backend/model/index -b --login -n=1
```

### 8. 查看日志和监控

```bash
# 查看服务器启动日志
cat var/log/server-start.log

# 查看应用日志
tail -f var/log/system.log

# 查看单元测试报告
cat var/log/unit-test-report.md
```

## 常见问题

### Q1: 数据库表没有创建？
```bash
# 手动重建表
php bin/w model:rebuild -m=Weline_Ai -n=AiModel
php bin/w setup:upgrade
```

### Q2: 后台无法访问？
```bash
# 检查服务器状态
php bin/w s:sta

# 查看启动日志获取正确的访问地址
cat var/log/server-start.log | grep "后端地址"
```

### Q3: API请求失败？
- 确保已登录后台
- 检查API密钥是否有效
- 查看日志了解具体错误
