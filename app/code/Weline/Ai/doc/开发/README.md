## 🎯 模块简介

`Weline_Ai` 是一个企业级AI模型管理和服务平台，提供：

### 核心功能
1. **AI模型管理** - 支持多供应商、多模型的统一管理
2. **API密钥管理** - 安全的密钥生成、配额控制和使用追踪
3. **AI助手管理** - 可定制的AI助手，支持多租户隔离
4. **场景适配器** - 插件化的场景适配机制
5. **监控与洞察** - 实时监控、性能分析和业务洞察

### 技术栈
- **框架**: WelineFramework
- **语言**: PHP 8.4+
- **数据库**: SQLite / MySQL
- **测试**: PHPUnit 9.6
- **前端**: Bootstrap 5 + Vanilla JS

### 版本信息
- **当前版本**: 1.0.0
- **状态**: 稳定版
- **最后更新**: 2025-10-22
