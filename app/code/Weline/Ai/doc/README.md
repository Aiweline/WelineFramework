## 🎯 模块简介

`Weline_Ai` 是一个企业级AI模型管理和服务平台，提供：

### 核心功能
1. **AI 管理** - 模型 | 适配器 | 供应商账户 聚合管理
2. **场景配置** - 场景适配器管理
3. **模型管理** - 多供应商、多模型统一管理
4. **REST API** - Chat、Model、API Key 等接口

### 技术栈
- **框架**: WelineFramework
- **语言**: PHP 8.4+
- **数据库**: SQLite / MySQL
- **测试**: PHPUnit 9.6
