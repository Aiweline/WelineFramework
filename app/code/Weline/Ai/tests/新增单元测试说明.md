## 🐛 如果测试失败

### 常见问题和解决方案

#### 1. 类未找到错误

**错误信息**: `Class 'Weline\Ai\Model\AiModel' not found`

**解决方案**:
```bash
# 重新生成自动加载
composer dump-autoload

# 升级模块
php bin/w module:upgrade Weline_Ai

# 清理缓存
php bin/w cache:clear
```

#### 2. 字段常量未定义

**错误信息**: `Undefined constant: Weline\Ai\Model\AiModel::fields_IS_ACTIVE`

**解决方案**:
- 确保 `AiModel.php` 已经包含所有字段常量定义
- 检查文件是否保存
- 重新加载自动加载

#### 3. 数据库表缺少字段

**错误信息**: `Unknown column 'is_active' in 'field list'`

**解决方案**:
```bash
# 运行数据库升级
php bin/w module:upgrade Weline_Ai

# 检查数据库表结构
php bin/w db:describe ai_model
```

## 💡 最佳实践

### 测试命名规范

- ✅ `test_method_name_expected_behavior`
- ✅ `test_scenario_when_condition`
- ❌ `testCase1`, `testCase2`

### 测试结构

```php
public function test_descriptive_name()
{
    // Arrange: 准备测试环境和数据
    $model = new AiModel();
    $model->setData('field', 'value');
    
    // Act: 执行要测试的操作
    $result = $model->someMethod();
    
    // Assert: 验证结果
    $this->assertEquals('expected', $result);
}
```

### 断言选择

- 使用 `assertSame()` 比较类型和值
- 使用 `assertEquals()` 只比较值
- 使用 `assertTrue()/assertFalse()` 验证布尔值
- 使用 `assertInstanceOf()` 验证对象类型
- 使用 `assertArrayHasKey()` 验证数组结构
