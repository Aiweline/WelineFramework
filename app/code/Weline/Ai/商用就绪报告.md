## 📚 使用示例

### PHP静态方法调用
```php
use Weline\Ai\Service\AiService;

// 基础文本生成
$response = AiService::generateText(
    $prompt = "请介绍一下人工智能",
    $modelCode = null, // 使用默认模型
    $scenarioCode = null,
    $locale = null
);

// 使用翻译适配器
$response = AiService::generateText(
    $prompt = "Hello, how are you?",
    $modelCode = "gpt-3.5-turbo",
    $scenarioCode = "translation",
    $locale = "zh-CN",
    $params = [
        "target_language" => "中文",
        "source_language" => "英文"
    ]
);

// 流式生成
AiService::generateTextStream(
    $prompt = "写一篇关于AI的文章",
    $callback = function($chunk) {
        echo $chunk;
        flush();
    },
    $modelCode = "gpt-4"
);
```

### 实例化调用
```php
use Weline\Ai\Service\AiService;
use Weline\Framework\Manager\ObjectManager;

$aiService = ObjectManager::getInstance(AiService::class);

$response = $aiService->generate(
    $prompt = "你好",
    $modelCode = "gpt-4",
    $scenarioCode = null,
    $locale = "zh-CN"
);
```

### AJAX前端调用
```javascript
// 发送消息
fetch('/ai/frontend/chat/send', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/x-www-form-urlencoded',
    },
    body: `message=${encodeURIComponent(message)}&model_code=gpt-4`
})
.then(response => response.json())
.then(data => {
    console.log(data.data.response);
});

// 流式响应
const eventSource = new EventSource('/ai/frontend/chat/stream?message=你好&model_code=gpt-4');

eventSource.addEventListener('message', (e) => {
    const data = JSON.parse(e.data);
    console.log(data.chunk);
});

eventSource.addEventListener('done', () => {
    eventSource.close();
});
```

## 🚀 部署指南

### 环境要求
- **PHP**: 8.0+
- **扩展**: curl, json, mbstring, sqlite3/mysql
- **服务器**: Apache/Nginx
- **缓存**: Redis（推荐）或文件缓存

### 安装步骤
```bash
# 1. 更新系统
php bin/m setup:upgrade

# 2. 收集模型
php bin/m model-collect-command

# 3. 扫描适配器
php bin/m adapter-scan-command

# 4. 初始化默认模型
php bin/m default-model-command action=init

# 5. 清理缓存
php bin/m cache:clear
```

### 配置OpenAI API
编辑模型配置文件：`app/code/Weline/Ai/etc/models/openai_gpt-4.json`
```json
{
    "vendor": "openai",
    "model_code": "gpt-4",
    "model_name": "GPT-4",
    "config": {
        "api_url": "https://api.openai.com/v1/chat/completions",
        "api_key": "YOUR_OPENAI_API_KEY",
        "max_tokens": 8192,
        "temperature": 0.7
    }
}
```

### 生产环境优化
```bash
# 启用生产模式
php bin/m deploy:mode:set prod

# 编译资源
php bin/m resource:compiler

# 预热缓存
php bin/m cache:warm
```
