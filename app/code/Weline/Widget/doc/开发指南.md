## 🧩 创建自定义部件

### 前置条件

- 熟悉 Weline Framework 模块结构
- 了解 PHP 和模板语法
- 了解 HTML/CSS/JavaScript 基础

### 部件类型

系统支持以下部件类型（可扩展）：

- `header` - 页面头部
- `content` - 内容区域
- `footer` - 页面底部
- `sidebar` - 侧边栏
- `banner` - 横幅广告
- `form` - 表单
- `list` - 列表
- `card` - 卡片
- `general` - 通用部件

### 创建步骤

1. **选择部件类型**：确定部件属于哪个类型
2. **创建目录结构**：在模块的 `extends/Weline_Widget/` 目录下创建部件目录
3. **编写配置文件**：创建 `widget.php` 定义部件元数据
4. **创建模板文件**：编写部件模板或Block类
5. **测试部件**：在编辑器中测试部件功能
6. **完善文档**：编写使用文档（可选）

## 📁 部件目录结构

### 标准结构

```
app/code/Weline/YourModule/
└── extends/
    └── Weline_Widget/
        └── YourModule/
            └── {type}/
                └── {name}/
                    ├── widget.php          # 部件配置文件（必需）
                    ├── default.phtml      # 部件模板（必需，如果未提供block_class）
                    ├── doc.md              # 部件文档（可选）
                    └── assets/             # 静态资源（可选）
                        ├── css/
                        │   └── style.css
                        └── js/
                            └── script.js
```

### 示例：创建默认头部部件

```
app/code/Weline/Widget/
└── extends/
    └── Weline_Widget/
        └── Weline_Widget/
            └── header/
                └── default/
                    ├── widget.php
                    └── default.phtml
```

## 📝 widget.php 配置文件

### 基本结构

```php
<?php
declare(strict_types=1);
return [
    'name' => '部件显示名称',
    'description' => '部件功能描述',
    'type' => 'header',  // 部件类型
    'widget_name' => 'default',  // 部件名称（目录名）
    'module' => 'Weline_Widget',  // 来源模块
    'version' => '1.0.0',  // 版本号
    'template' => 'Weline_Widget::widgets/header/default.phtml',  // 模板路径
    // 或使用 Block 类
    // 'block_class' => 'Weline\Widget\Block\Header\Default',
    'params' => [
        // 参数定义
    ],
    'doc' => '部件使用说明文档'
];
```

### 完整示例

```php
<?php
declare(strict_types=1);
return [
    'name' => '默认头部',
    'description' => '标准网站头部部件，包含Logo和标题。',
    'type' => 'header',
    'widget_name' => 'default',
    'module' => 'Weline_Widget',
    'version' => '1.0.0',
    'template' => 'Weline_Widget::widgets/header/default.phtml',
    'params' => [
        'title' => [
            'type' => 'string',
            'label' => '网站标题',
            'default' => 'Weline Framework',
            'placeholder' => '请输入网站标题',
            'required' => true
        ],
        'logo_url' => [
            'type' => 'string',
            'label' => 'Logo图片URL',
            'default' => '',
            'placeholder' => '请输入Logo图片地址'
        ],
        'show_search' => [
            'type' => 'boolean',
            'label' => '显示搜索框',
            'default' => true
        ]
    ],
    'doc' => '这是一个默认的网站头部部件，可配置标题、Logo和搜索框。'
];
```

## 🎨 创建部件模板

### 模板位置

模板文件应放在模块的 `view/templates/widgets/` 目录下：

```
app/code/Weline/YourModule/
└── view/
    └── templates/
        └── widgets/
            └── {type}/
                └── {name}.phtml
```

### 模板示例

```html
<!-- app/code/Weline/Widget/view/templates/widgets/header/default.phtml -->
<header class="widget-header">
    <div class="container">
        <?php if (!empty($logo_url)): ?>
            <div class="logo">
                <img src="<?= htmlspecialchars($logo_url) ?>" 
                     alt="<?= htmlspecialchars($title) ?>">
            </div>
        <?php endif; ?>
        
        <h1 class="site-title"><?= htmlspecialchars($title) ?></h1>
        
        <?php if ($show_search): ?>
            <div class="search-box">
                <input type="search" 
                       class="form-control" 
                       placeholder="<?= __('搜索...') ?>">
            </div>
        <?php endif; ?>
    </div>
</header>
```

### 模板变量

模板中可以使用以下变量：

- `$params` - 所有参数数组
- `$type` - 部件类型
- `$name` - 部件名称
- 以及 `widget.php` 中定义的所有参数（作为独立变量）

### 安全注意事项

- **XSS防护**：使用 `htmlspecialchars()` 转义输出
- **URL验证**：验证URL参数的有效性
- **数据过滤**：过滤用户输入的数据

## 🔧 创建部件Block类

### 何时使用Block类

- 需要复杂的数据处理逻辑
- 需要访问数据库
- 需要调用服务类
- 需要动态生成内容

### Block类示例

```php
<?php
declare(strict_types=1);
namespace Weline\Widget\Block\Header;

use Weline\Framework\View\Block;

class Default extends Block
{
    protected string $_template = 'Weline_Widget::widgets/header/default.phtml';
    
    public function __init()
    {
        parent::__init();
        
        // 获取参数
        $title = $this->getData('title') ?: 'Weline Framework';
        $logoUrl = $this->getData('logo_url') ?: '';
        $showSearch = (bool)$this->getData('show_search');
        
        // 处理数据
        if (empty($logoUrl)) {
            $logoUrl = $this->getDefaultLogo();
        }
        
        // 设置数据
        $this->assign('title', $title);
        $this->assign('logo_url', $logoUrl);
        $this->assign('show_search', $showSearch);
    }
    
    private function getDefaultLogo(): string
    {
        // 返回默认Logo路径
        return '/static/default-logo.png';
    }
}
```

### 在widget.php中引用

```php
return [
    // ...
    'block_class' => 'Weline\Widget\Block\Header\Default',
    // 不需要 template，Block类会指定模板
];
```

## 📋 部件参数定义

### 参数类型

#### 1. string - 文本类型

```php
'title' => [
    'type' => 'string',
    'label' => '标题',
    'default' => '默认标题',
    'placeholder' => '请输入标题',
    'required' => true
]
```

#### 2. number - 数字类型

```php
'count' => [
    'type' => 'number',
    'label' => '显示数量',
    'default' => 10,
    'min' => 1,
    'max' => 100
]
```

#### 3. boolean - 布尔类型

```php
'show_title' => [
    'type' => 'boolean',
    'label' => '显示标题',
    'default' => true
]
```

#### 4. select - 选择类型

```php
'layout' => [
    'type' => 'select',
    'label' => '布局方式',
    'default' => 'grid',
    'options' => [
        'grid' => '网格布局',
        'list' => '列表布局',
        'card' => '卡片布局'
    ]
]
```

#### 5. url - URL类型

```php
'image_url' => [
    'type' => 'url',
    'label' => '图片地址',
    'default' => '',
    'placeholder' => 'https://example.com/image.jpg'
]
```

### 参数属性说明

| 属性 | 类型 | 说明 | 必需 |
|------|------|------|------|
| `type` | string | 参数类型 | 是 |
| `label` | string | 参数标签 | 是 |
| `default` | mixed | 默认值 | 否 |
| `placeholder` | string | 占位符文本 | 否 |
| `required` | boolean | 是否必填 | 否 |
| `min` | number | 最小值（数字类型） | 否 |
| `max` | number | 最大值（数字类型） | 否 |
| `options` | array | 选项列表（选择类型） | 否 |

## 🧪 测试部件

### 1. 注册部件

确保部件目录结构正确，系统会自动扫描并注册。

### 2. 在编辑器中测试

1. 访问 **内容管理 > 可视化编辑器**
2. 在部件库中查找您的部件
3. 拖放到画布中
4. 配置参数
5. 预览效果

### 3. 直接使用标签测试

在模板中直接使用 `w:widget` 标签：

```html
<w:widget type="header" name="default" params='{"title":"测试标题","logo_url":"/static/logo.png","show_search":true}' />
```

### 4. 检查错误

- 查看浏览器控制台
- 查看PHP错误日志
- 检查部件配置文件格式

## 📦 发布部件

### 1. 完善文档

创建 `doc.md` 文件说明部件使用方法：

```markdown
# 默认头部部件

## 功能说明

标准网站头部部件，包含Logo、标题和搜索框。

## 参数说明

- `title`: 网站标题（必填）
- `logo_url`: Logo图片地址（可选）
- `show_search`: 是否显示搜索框（默认：是）

## 使用示例

```html
<w:widget type="header" name="default" params='{"title":"我的网站"}' />
```

## 注意事项

- Logo图片建议使用PNG或SVG格式
- 标题长度建议不超过20个字符
```

### 2. 版本管理

在 `widget.php` 中更新版本号：

```php
'version' => '1.0.1',  // 修复bug或添加功能时更新
```

### 3. 提交代码

将部件代码提交到版本控制系统。

## 💡 最佳实践

### 1. 命名规范

- **部件名称**：使用小写字母和连字符，如 `default-header`
- **目录名**：与 `widget_name` 保持一致
- **类名**：使用PascalCase，如 `DefaultHeader`

### 2. 参数设计

- **提供默认值**：所有参数都应提供合理的默认值
- **参数验证**：在Block类中验证参数有效性
- **文档说明**：为每个参数添加清晰的说明

### 3. 模板编写

- **语义化HTML**：使用语义化的HTML标签
- **响应式设计**：确保部件在不同设备上正常显示
- **性能优化**：避免不必要的计算和查询

### 4. 错误处理

- **优雅降级**：参数缺失时使用默认值
- **错误提示**：在模板中显示友好的错误信息
- **日志记录**：记录关键错误信息
