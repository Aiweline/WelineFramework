## 📖 模块概述

Widget 模块是一个强大的后端可视化编辑器，允许您通过拖放方式组织页面，使用 `w:widget` 标签存储和渲染部件。

### 核心特性

- ✅ **可视化编辑** - 拖放式页面编辑器
- ✅ **部件系统** - 可扩展的部件架构
- ✅ **实时预览** - 即时查看编辑效果
- ✅ **参数配置** - 灵活的部件参数系统
- ✅ **标签渲染** - 使用 `w:widget` 标签存储和渲染

### 技术栈

- PHP 8.0+
- Weline Framework
- JavaScript (ES6+)
- HTML5 Drag & Drop API
- Bootstrap 5
