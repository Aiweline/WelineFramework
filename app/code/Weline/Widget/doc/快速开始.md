# Widget 模块快速开始指南

## 🚀 5分钟快速上手

本指南将帮助您在5分钟内开始使用可视化编辑器。

## 第一步：访问可视化编辑器

1. 登录后台管理系统
2. 导航到 **内容管理 > 可视化编辑器**
3. 您将看到编辑器界面

## 第二步：创建第一个页面

### 2.1 填写页面信息

在编辑器顶部，填写：
- **页面标题**：例如 "首页"
- **页面标识**：例如 "homepage"（用于URL路由）

### 2.2 添加部件

1. 在左侧 **部件库** 中找到您需要的部件
2. 将部件拖放到中间的 **画布** 区域
3. 部件会自动添加到页面中

### 2.3 配置部件属性

1. 点击画布中的部件
2. 右侧 **属性配置** 面板会显示该部件的可配置参数
3. 修改参数值，实时查看效果

### 2.4 保存页面

点击顶部的 **保存** 按钮，页面内容将保存到数据库。

## 第三步：预览页面

点击 **预览** 按钮，在新窗口中查看页面效果。

## 📝 示例：创建一个简单的首页

### 步骤1：添加头部部件

1. 从部件库中找到 **header** 类型的部件
2. 拖放到画布中
3. 在属性配置中设置：
   - **网站标题**：我的网站
   - **Logo图片URL**：/static/logo.png
   - **显示搜索框**：是

### 步骤2：添加内容部件

1. 添加一个 **content** 类型的部件
2. 配置内容文本

### 步骤3：保存并预览

1. 点击 **保存**
2. 点击 **预览** 查看效果

## 🎯 常用操作

### 删除部件

1. 点击画布中的部件
2. 点击部件右上角的删除按钮
3. 确认删除

### 调整部件顺序

直接拖放部件到新位置即可。

### 编辑代码

1. 点击画布中的部件
2. 点击 **编辑代码** 按钮
3. 直接编辑 `w:widget` 标签代码

## 📋 页面内容格式

编辑器保存的页面内容使用 `w:widget` 标签格式：

```html
<w:widget type="header" name="default" params='{"title":"我的网站","logo_url":"/static/logo.png","show_search":true}' />
<w:widget type="content" name="text" params='{"text":"欢迎访问我的网站"}' />
```

## 🔍 查看已保存的页面

1. 在编辑器URL中添加 `?page_id=1` 参数
2. 页面会自动加载已保存的内容

## ⚠️ 注意事项

1. **页面标识必须唯一**：每个页面的标识（handle）必须唯一
2. **保存前填写信息**：页面标题和标识是必填项
3. **部件参数格式**：参数使用JSON格式，注意引号转义
