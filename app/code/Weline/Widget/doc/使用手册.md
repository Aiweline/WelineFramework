## 🖥️ 编辑器界面介绍

### 界面布局

编辑器分为三个主要区域：

```
┌─────────────┬──────────────────┬─────────────┐
│             │                  │             │
│  部件库     │      画布        │  属性配置   │
│  (左侧)     │     (中间)       │   (右侧)    │
│             │                  │             │
└─────────────┴──────────────────┴─────────────┘
```

### 顶部工具栏

- **保存**：保存当前页面到数据库
- **预览**：在新窗口预览页面效果
- **页面标题**：输入框，设置页面标题
- **页面标识**：输入框，设置页面唯一标识（用于URL路由）

### 左侧部件库

- 按类型分组显示所有可用部件
- 支持搜索过滤
- 每个部件显示名称和描述

### 中间画布

- 显示当前页面的所有部件
- 支持拖放排序
- 点击部件进行选择

### 右侧属性配置

- 显示选中部件的可配置参数
- 实时更新预览
- 支持各种输入类型（文本、数字、布尔值等）

## 📄 创建和管理页面

### 创建新页面

1. 访问 **内容管理 > 可视化编辑器**
2. 填写页面标题和标识
3. 添加部件
4. 点击保存

### 编辑已有页面

1. 访问编辑器时在URL中添加 `?page_id=1` 参数
2. 页面内容会自动加载
3. 进行修改后保存

### 页面标识规则

- **必须唯一**：每个页面的标识在系统中必须唯一
- **URL友好**：建议使用小写字母、数字和连字符
- **示例**：`homepage`, `about-us`, `product-list`

## 🧩 使用部件

### 添加部件

**方法1：拖放**
1. 在部件库中找到目标部件
2. 按住鼠标左键
3. 拖到画布区域
4. 释放鼠标

**方法2：代码编辑**
1. 点击画布中的任意部件
2. 点击 **编辑代码** 按钮
3. 在代码编辑器中添加新的 `w:widget` 标签
4. 保存代码

### 删除部件

1. 点击画布中的部件
2. 点击部件右上角的 **删除** 按钮（×）
3. 确认删除

### 调整部件顺序

1. 在画布中直接拖放部件
2. 部件会自动重新排序

### 复制部件

1. 点击部件
2. 点击 **编辑代码** 按钮
3. 复制 `w:widget` 标签代码
4. 粘贴到新位置

## ⚙️ 配置部件参数

### 基本操作

1. **选择部件**：点击画布中的部件
2. **查看参数**：右侧属性配置面板显示所有可配置参数
3. **修改参数**：直接在输入框中修改
4. **应用更改**：参数修改后自动更新预览

### 参数类型

#### 文本类型 (string)

```html
参数名：网站标题
类型：文本
输入：我的网站
```

#### 数字类型 (number)

```html
参数名：显示数量
类型：数字
输入：10
```

#### 布尔类型 (boolean)

```html
参数名：显示搜索框
类型：是/否
选择：是
```

#### URL类型 (url)

```html
参数名：Logo图片URL
类型：URL
输入：/static/logo.png
```

### 参数配置示例

以默认头部部件为例：

```json
{
  "title": "我的网站",
  "logo_url": "/static/logo.png",
  "show_search": true
}
```

## 👁️ 预览和保存

### 实时预览

- 修改部件参数后，预览区域会自动更新
- 无需手动刷新

### 保存页面

1. 确保填写了页面标题和标识
2. 点击顶部 **保存** 按钮
3. 系统会显示保存结果

**保存成功**：页面ID会显示在页面标识输入框下方

**保存失败**：检查错误提示，常见原因：
- 页面标识已存在
- 必填字段未填写
- 网络错误

### 预览页面

1. 点击顶部 **预览** 按钮
2. 在新窗口中查看完整页面效果
3. 可以测试交互功能

## 🔧 部件管理

### 查看所有部件

访问 **内容管理 > 可视化编辑器 > 部件管理**

### 部件列表功能

- **搜索**：按名称搜索部件
- **筛选**：按模块或类型筛选
- **详情**：查看部件的完整信息

### 部件信息

每个部件显示：
- **部件名称**：显示名称
- **类型**：部件类型（header, content, footer等）
- **来源模块**：定义该部件的模块
- **描述**：部件功能说明
- **版本**：部件版本号

### 查看部件详情

1. 在部件列表中点击 **详情** 按钮
2. 查看：
   - 基本信息
   - 参数配置说明
   - 使用文档

## 🚀 高级功能

### 代码编辑模式

1. 点击画布中的部件
2. 点击 **编辑代码** 按钮
3. 直接编辑 `w:widget` 标签

**代码格式**：
```html
<w:widget type="header" name="default" params='{"title":"我的网站"}' />
```

### 批量操作

目前不支持批量操作，需要逐个添加和配置部件。

### 导入/导出

- **导出**：复制画布中的代码
- **导入**：在代码编辑器中粘贴代码

### 页面模板

可以创建常用页面作为模板：
1. 创建页面并保存
2. 复制页面内容代码
3. 创建新页面时粘贴代码
4. 修改参数即可

## 💡 最佳实践

### 1. 页面规划

在开始编辑前，先规划：
- 页面需要哪些部件
- 部件的排列顺序
- 每个部件的参数配置

### 2. 命名规范

- **页面标题**：使用有意义的名称，如"首页"、"关于我们"
- **页面标识**：使用URL友好的格式，如"homepage"、"about-us"

### 3. 部件选择

- 优先使用系统提供的标准部件
- 需要特殊功能时，考虑开发自定义部件

### 4. 参数配置

- 使用有意义的参数值
- 注意参数类型（文本、数字、布尔值）
- 保存前检查参数是否正确

### 5. 测试验证

- 保存前预览页面
- 检查所有部件是否正常显示
- 测试交互功能

## 🆘 常见问题

### Q: 保存时提示"页面标识已存在"？

A: 修改页面标识为其他唯一值，或编辑已有页面时使用正确的 page_id。

### Q: 部件不显示？

A: 检查：
1. 部件类型和名称是否正确
2. 部件是否已正确注册
3. 查看浏览器控制台错误信息

### Q: 参数修改不生效？

A: 确保：
1. 参数格式正确（JSON格式）
2. 参数类型匹配（文本、数字、布尔值）
3. 点击了应用按钮（如果存在）

### Q: 如何删除页面？

A: 目前需要通过数据库或后台管理删除，编辑器界面暂不支持删除功能。
