# 翻译服务模块 - 翻译事件文档

## 概述

本文档详细说明了翻译服务模块提供的核心翻译事件及其使用方法。这些事件允许其他模块通过事件系统使用翻译服务，实现完全解耦的翻译功能调用。

## 事件列表

### 1. Weline_TranslationService::translate - 翻译文本事件

#### 基本信息

- **事件名称**：`Weline_TranslationService::translate`
- **事件类型**：核心翻译事件
- **触发模块**：`Weline_TranslationService`
- **观察者**：`Weline\TranslationService\Observer\TranslationObserver`

#### 功能说明

用于翻译单个文本，其他模块可以通过触发此事件来使用翻译服务。观察者会调用 TranslationService 执行翻译，并将结果设置到事件数据的 `translated_text` 字段中。

#### 使用场景

- 翻译单个文本内容
- 动态内容翻译
- 用户输入翻译
- 内容管理系统的翻译功能

#### 使用方法

##### 基本用法

```php
use Weline\Framework\Event\EventsManager;
use Weline\Framework\Manager\ObjectManager;

// 获取事件管理器实例
/** @var EventsManager $eventsManager */
$eventsManager = ObjectManager::getInstance(EventsManager::class);

// 触发翻译事件
$eventData = [
    'text' => 'Hello, World!',
    'target_language' => 'zh',
    'source_language' => 'auto',
    'provider_code' => null,  // 可选，指定渠道代码
    'options' => [],           // 可选，额外选项
    'module_name' => 'Weline_YourModule'  // 可选，调用模块名称
];
$eventsManager->dispatch('Weline_TranslationService::translate', $eventData);

// 获取翻译结果
$translatedText = $eventData['translated_text'] ?? '';
$success = $eventData['success'] ?? false;
$error = $eventData['error'] ?? null;
```

#### 数据格式说明

| 字段 | 类型 | 必填 | 说明 | 默认值 |
|------|------|------|------|--------|
| text | string | 是 | 要翻译的文本 | - |
| target_language | string | 是 | 目标语言代码（ISO 639-1、ISO 639-2或BCP 47格式） | - |
| source_language | string | 否 | 源语言代码（auto表示自动检测） | 'auto' |
| provider_code | string\|null | 否 | 指定渠道代码（null表示自动选择） | null |
| options | array | 否 | 额外选项 | [] |
| module_name | string | 否 | 调用模块名称（用于记录和统计） | null |

#### 返回数据说明

| 字段 | 类型 | 说明 |
|------|------|------|
| translated_text | string | 翻译后的文本（观察者设置） |
| success | bool | 是否成功 |
| error | string\|null | 错误信息（如果失败） |

### 2. Weline_TranslationService::batch_translate - 批量翻译事件

#### 基本信息

- **事件名称**：`Weline_TranslationService::batch_translate`
- **事件类型**：核心翻译事件
- **触发模块**：`Weline_TranslationService`
- **观察者**：`Weline\TranslationService\Observer\BatchTranslationObserver`

#### 功能说明

用于批量翻译多个文本，其他模块可以通过触发此事件来批量使用翻译服务。观察者会调用 TranslationService 执行批量翻译，并将结果设置到事件数据的 `translated_texts` 字段中。

#### 使用场景

- 批量翻译多个文本
- 翻译列表内容
- 批量处理翻译任务
- 内容导入时的批量翻译

#### 使用方法

##### 基本用法

```php
use Weline\Framework\Event\EventsManager;
use Weline\Framework\Manager\ObjectManager;

// 获取事件管理器实例
/** @var EventsManager $eventsManager */
$eventsManager = ObjectManager::getInstance(EventsManager::class);

// 触发批量翻译事件
$eventData = [
    'texts' => ['Hello', 'World', 'Translation'],
    'target_language' => 'zh',
    'source_language' => 'auto',
    'provider_code' => null,
    'options' => [],
    'module_name' => 'Weline_YourModule'
];
$eventsManager->dispatch('Weline_TranslationService::batch_translate', $eventData);

// 获取翻译结果
$translatedTexts = $eventData['translated_texts'] ?? [];
$success = $eventData['success'] ?? false;
$error = $eventData['error'] ?? null;
```

#### 数据格式说明

| 字段 | 类型 | 必填 | 说明 | 默认值 |
|------|------|------|------|--------|
| texts | array | 是 | 要翻译的文本数组 | - |
| target_language | string | 是 | 目标语言代码 | - |
| source_language | string | 否 | 源语言代码 | 'auto' |
| provider_code | string\|null | 否 | 指定渠道代码 | null |
| options | array | 否 | 额外选项 | [] |
| module_name | string | 否 | 调用模块名称 | null |

#### 返回数据说明

| 字段 | 类型 | 说明 |
|------|------|------|
| translated_texts | array | 翻译后的文本数组（观察者设置） |
| success | bool | 是否成功 |
| error | string\|null | 错误信息（如果失败） |

### 3. Weline_TranslationService::detect_language - 语言检测事件

#### 基本信息

- **事件名称**：`Weline_TranslationService::detect_language`
- **事件类型**：核心翻译事件
- **触发模块**：`Weline_TranslationService`
- **观察者**：`Weline\TranslationService\Observer\DetectLanguageObserver`

#### 功能说明

用于检测文本的语言，其他模块可以通过触发此事件来使用语言检测功能。观察者会调用 TranslationService 执行语言检测，并将结果设置到事件数据的 `detected_language` 字段中。

#### 使用场景

- 自动检测用户输入的语言
- 内容语言识别
- 多语言内容分类
- 语言路由判断

#### 使用方法

##### 基本用法

```php
use Weline\Framework\Event\EventsManager;
use Weline\Framework\Manager\ObjectManager;

// 获取事件管理器实例
/** @var EventsManager $eventsManager */
$eventsManager = ObjectManager::getInstance(EventsManager::class);

// 触发语言检测事件
$eventData = [
    'text' => 'Hello, World!',
    'provider_code' => null  // 可选，指定渠道代码
];
$eventsManager->dispatch('Weline_TranslationService::detect_language', $eventData);

// 获取检测结果
$detectedLanguage = $eventData['detected_language'] ?? null;
$success = $eventData['success'] ?? false;
$error = $eventData['error'] ?? null;
```

#### 数据格式说明

| 字段 | 类型 | 必填 | 说明 | 默认值 |
|------|------|------|------|--------|
| text | string | 是 | 要检测的文本 | - |
| provider_code | string\|null | 否 | 指定渠道代码 | null |

#### 返回数据说明

| 字段 | 类型 | 说明 |
|------|------|------|
| detected_language | string | 检测到的语言代码（ISO 639-1格式，观察者设置） |
| success | bool | 是否成功 |
| error | string\|null | 错误信息（如果失败） |

## 语言代码格式

所有事件都支持以下语言代码格式，模块会自动转换为ISO 639-1格式：

- **ISO 639-1**：`zh`、`en`、`ja` 等（2位字母）
- **ISO 639-2**：`zho`、`eng`、`jpn` 等（3位字母）
- **BCP 47**：`zh-CN`、`en-US`、`zh_Hans_CN` 等（语言标签）

## 完整示例

### 示例1：在控制器中使用翻译事件

```php
namespace Weline\YourModule\Controller\Backend;

use Weline\Framework\Event\EventsManager;
use Weline\Framework\Manager\ObjectManager;

class YourController extends \Weline\Admin\Controller\BaseController
{
    public function translateContent(): string
    {
        $content = $this->request->getParam('content');
        $targetLang = $this->request->getParam('target_language', 'zh');
        
        // 触发翻译事件
        $eventManager = ObjectManager::getInstance(EventsManager::class);
        $eventData = [
            'text' => $content,
            'target_language' => $targetLang,
            'source_language' => 'auto',
            'module_name' => 'Weline_YourModule'
        ];
        $eventManager->dispatch('Weline_TranslationService::translate', $eventData);
        
        if ($eventData['success'] ?? false) {
            $translated = $eventData['translated_text'] ?? '';
            return $this->json(['success' => true, 'translated' => $translated]);
        } else {
            return $this->json(['success' => false, 'error' => $eventData['error'] ?? '翻译失败']);
        }
    }
}
```

### 示例2：批量翻译内容列表

```php
$texts = ['Title 1', 'Title 2', 'Title 3'];
$eventManager = ObjectManager::getInstance(EventsManager::class);
$eventData = [
    'texts' => $texts,
    'target_language' => 'en',
    'source_language' => 'zh',
    'module_name' => 'Weline_YourModule'
];
$eventManager->dispatch('Weline_TranslationService::batch_translate', $eventData);

if ($eventData['success'] ?? false) {
    $translatedTexts = $eventData['translated_texts'] ?? [];
    // 使用翻译结果
    foreach ($translatedTexts as $index => $translated) {
        echo "原文: {$texts[$index]}, 翻译: {$translated}\n";
    }
}
```

### 示例3：检测用户输入语言

```php
$userInput = $_POST['content'] ?? '';
$eventManager = ObjectManager::getInstance(EventsManager::class);
$eventData = [
    'text' => $userInput
];
$eventManager->dispatch('Weline_TranslationService::detect_language', $eventData);

if ($eventData['success'] ?? false) {
    $detectedLang = $eventData['detected_language'] ?? 'unknown';
    echo "检测到的语言: {$detectedLang}";
}
```

## 注意事项

1. **事件是异步的**：事件触发后，观察者会立即执行，但结果会同步设置到事件数据中
2. **错误处理**：始终检查 `success` 字段和 `error` 字段来判断翻译是否成功
3. **语言代码**：建议使用ISO 639-1格式（如 `zh`、`en`），模块会自动转换其他格式
4. **模块名称**：建议设置 `module_name` 字段，便于在翻译记录中追踪调用来源
5. **性能考虑**：批量翻译比多次单次翻译更高效，建议批量处理时使用 `batch_translate` 事件
