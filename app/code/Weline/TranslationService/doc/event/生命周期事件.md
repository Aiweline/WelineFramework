# 翻译服务模块 - 生命周期事件文档

## 概述

本文档详细说明了翻译服务模块提供的生命周期事件及其使用方法。这些事件允许其他模块在翻译过程的各个阶段进行干预，实现参数修改、结果处理、错误处理等功能。

## 事件列表

### 1. Weline_TranslationService::translate_before - 翻译前事件

#### 基本信息

- **事件名称**：`Weline_TranslationService::translate_before`
- **事件类型**：生命周期事件
- **触发时机**：在调用 TranslationService::translate() 方法时，执行翻译前
- **触发位置**：`Service/TranslationService.php` 的 `translate()` 方法

#### 功能说明

在翻译执行前触发，允许其他模块修改翻译参数，如源语言、目标语言、渠道代码等。这对于实现动态渠道选择、语言映射等功能非常有用。

#### 使用场景

- 根据内容类型自动选择翻译渠道
- 实现语言映射（如将 `zh-CN` 映射为 `zh`）
- 添加翻译选项（如专业术语库、领域设置等）
- 实现翻译缓存检查

#### 使用方法

##### 在模块的 etc/event.xml 中注册观察者

```xml
<event name="Weline_TranslationService::translate_before">
    <observer name="Weline_YourModule::translate_before" 
              instance="Weline\YourModule\Observer\TranslationBeforeObserver" 
              disabled="false" 
              shared="true" 
              sort="10"/>
</event>
```

##### 创建观察者类

```php
namespace Weline\YourModule\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class TranslationBeforeObserver implements ObserverInterface
{
    public function execute(Event &$event): void
    {
        // 获取事件数据
        $text = $event->getData('text');
        $targetLanguage = $event->getData('target_language');
        $sourceLanguage = $event->getData('source_language');
        $providerCode = $event->getData('provider_code');
        $options = $event->getData('options') ?? [];
        
        // 根据业务逻辑修改参数
        // 例如：根据内容长度选择不同的渠道
        if (mb_strlen($text) > 1000) {
            $event->setData('provider_code', 'google');  // 长文本使用Google
        }
        
        // 例如：添加专业术语选项
        if (strpos($text, 'API') !== false) {
            $options['domain'] = 'technical';
            $event->setData('options', $options);
        }
        
        // 例如：实现语言映射
        if ($targetLanguage === 'zh-CN') {
            $event->setData('target_language', 'zh');
        }
    }
}
```

#### 数据格式说明

| 字段 | 类型 | 说明 | 可修改 |
|------|------|------|--------|
| text | string | 要翻译的文本 | 是 |
| target_language | string | 目标语言代码 | 是 |
| source_language | string | 源语言代码 | 是 |
| provider_code | string\|null | 渠道代码 | 是 |
| options | array | 额外选项 | 是 |

### 2. Weline_TranslationService::translate_after - 翻译后事件

#### 基本信息

- **事件名称**：`Weline_TranslationService::translate_after`
- **事件类型**：生命周期事件
- **触发时机**：在翻译成功后，返回结果前
- **触发位置**：`Service/TranslationService.php` 的 `translate()` 方法

#### 功能说明

在翻译完成后触发，允许其他模块修改翻译结果、执行后处理操作、记录日志等。这对于实现翻译结果优化、术语替换、格式调整等功能非常有用。

#### 使用场景

- 翻译结果后处理（如术语替换、格式调整）
- 翻译结果缓存
- 翻译质量检查
- 翻译结果日志记录
- 翻译结果统计

#### 使用方法

##### 创建观察者类

```php
namespace Weline\YourModule\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class TranslationAfterObserver implements ObserverInterface
{
    public function execute(Event &$event): void
    {
        // 获取事件数据
        $translatedText = $event->getData('translated_text');
        $sourceText = $event->getData('text');
        $targetLanguage = $event->getData('target_language');
        
        // 后处理：替换专业术语
        $translatedText = str_replace('API', '应用程序接口', $translatedText);
        
        // 后处理：首字母大写
        $translatedText = ucfirst($translatedText);
        
        // 设置修改后的翻译结果
        $event->setData('translated_text', $translatedText);
        
        // 记录日志
        error_log("翻译完成: {$sourceText} -> {$translatedText}");
    }
}
```

#### 数据格式说明

| 字段 | 类型 | 说明 | 可修改 |
|------|------|------|--------|
| text | string | 源文本 | 否 |
| translated_text | string | 翻译结果 | **是** |
| target_language | string | 目标语言代码 | 否 |
| source_language | string | 源语言代码 | 否 |
| provider_code | string | 使用的渠道代码 | 否 |
| character_count | int | 字符数 | 否 |
| cost | float | 翻译成本 | 否 |
| response_time | int | 响应时间（毫秒） | 否 |
| options | array | 额外选项 | 否 |

### 3. Weline_TranslationService::translate_error - 翻译错误事件

#### 基本信息

- **事件名称**：`Weline_TranslationService::translate_error`
- **事件类型**：生命周期事件
- **触发时机**：在翻译失败时
- **触发位置**：`Service/TranslationService.php` 的 `translate()` 方法的 catch 块

#### 功能说明

在翻译失败时触发，允许其他模块处理错误、实现降级方案、记录错误日志等。这对于实现自动重试、渠道切换、错误通知等功能非常有用。

#### 使用场景

- 实现自动重试机制
- 渠道降级（主渠道失败时切换到备用渠道）
- 错误通知（发送错误消息给管理员）
- 错误日志记录
- 实现翻译缓存降级

#### 使用方法

##### 创建观察者类

```php
namespace Weline\YourModule\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;
use Weline\Framework\Event\EventsManager;
use Weline\Framework\Manager\ObjectManager;

class TranslationErrorObserver implements ObserverInterface
{
    public function execute(Event &$event): void
    {
        // 获取错误信息
        $errorMessage = $event->getData('error_message');
        $providerCode = $event->getData('provider_code');
        $text = $event->getData('text');
        $targetLanguage = $event->getData('target_language');
        
        // 记录错误日志
        error_log("翻译失败 [{$providerCode}]: {$errorMessage}");
        
        // 实现降级方案：如果主渠道失败，尝试备用渠道
        if ($providerCode === 'google') {
            // 尝试使用百度翻译
            $eventManager = ObjectManager::getInstance(EventsManager::class);
            $retryData = [
                'text' => $text,
                'target_language' => $targetLanguage,
                'source_language' => 'auto',
                'provider_code' => 'baidu',
                'module_name' => 'Weline_YourModule'
            ];
            $eventManager->dispatch('Weline_TranslationService::translate', $retryData);
            
            if ($retryData['success'] ?? false) {
                // 降级成功，设置翻译结果
                $event->setData('translated_text', $retryData['translated_text'] ?? '');
                $event->setData('fallback_used', true);
                $event->setData('fallback_provider', 'baidu');
            }
        }
        
        // 发送错误通知
        $this->sendErrorNotification($errorMessage, $text);
    }
    
    private function sendErrorNotification(string $error, string $text): void
    {
        // 实现错误通知逻辑
    }
}
```

#### 数据格式说明

| 字段 | 类型 | 说明 |
|------|------|------|
| text | string | 要翻译的文本 |
| target_language | string | 目标语言代码 |
| source_language | string | 源语言代码 |
| provider_code | string\|null | 失败的渠道代码 |
| error_message | string | 错误信息 |
| error_code | int | 错误代码 |
| options | array | 额外选项 |

### 4. Weline_TranslationService::batch_translate_before - 批量翻译前事件

#### 基本信息

- **事件名称**：`Weline_TranslationService::batch_translate_before`
- **事件类型**：生命周期事件
- **触发时机**：在调用 TranslationService::batchTranslate() 方法时，执行批量翻译前

#### 功能说明

在批量翻译执行前触发，允许其他模块修改批量翻译参数。

#### 使用方法

与 `translate_before` 类似，但数据格式包含 `texts` 数组而不是单个 `text`。

---

### 5. Weline_TranslationService::batch_translate_after - 批量翻译后事件

#### 基本信息

- **事件名称**：`Weline_TranslationService::batch_translate_after`
- **事件类型**：生命周期事件
- **触发时机**：在批量翻译成功后，返回结果前

#### 功能说明

在批量翻译完成后触发，允许其他模块修改批量翻译结果。

#### 使用方法

与 `translate_after` 类似，但数据格式包含 `translated_texts` 数组而不是单个 `translated_text`。

---

### 6. Weline_TranslationService::batch_translate_error - 批量翻译错误事件

#### 基本信息

- **事件名称**：`Weline_TranslationService::batch_translate_error`
- **事件类型**：生命周期事件
- **触发时机**：在批量翻译失败时

#### 功能说明

在批量翻译失败时触发，允许其他模块处理错误。

#### 使用方法

与 `translate_error` 类似，但数据格式包含 `texts` 数组。

## 事件执行顺序

```
1. translate_before (修改参数)
   ↓
2. TranslationService::translate() (执行翻译)
   ↓
3. translate_after (修改结果) 或 translate_error (处理错误)
```

## 完整示例

### 示例：实现翻译结果缓存

```php
namespace Weline\YourModule\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;
use Weline\Framework\Cache\CacheInterface;
use Weline\Framework\Manager\ObjectManager;

class TranslationCacheObserver implements ObserverInterface
{
    private CacheInterface $cache;
    
    public function __construct()
    {
        $this->cache = ObjectManager::getInstance(CacheInterface::class);
    }
    
    public function execute(Event &$event): void
    {
        $eventName = $event->getName();
        
        if ($eventName === 'Weline_TranslationService::translate_before') {
            // 翻译前：检查缓存
            $text = $event->getData('text');
            $targetLanguage = $event->getData('target_language');
            $cacheKey = "translation:{$text}:{$targetLanguage}";
            
            $cached = $this->cache->get($cacheKey);
            if ($cached !== false) {
                // 从缓存获取，跳过翻译
                $event->setData('translated_text', $cached);
                $event->setData('from_cache', true);
                // 可以设置一个标志，让观察者知道这是从缓存获取的
            }
        } elseif ($eventName === 'Weline_TranslationService::translate_after') {
            // 翻译后：保存到缓存
            $text = $event->getData('text');
            $translatedText = $event->getData('translated_text');
            $targetLanguage = $event->getData('target_language');
            
            // 如果不是从缓存获取的，保存到缓存
            if (!($event->getData('from_cache') ?? false)) {
                $cacheKey = "translation:{$text}:{$targetLanguage}";
                $this->cache->set($cacheKey, $translatedText, 3600); // 缓存1小时
            }
        }
    }
}
```

## 注意事项

1. **事件执行顺序**：观察者的执行顺序由 `sort` 参数决定，数字越小越先执行
2. **数据修改**：在 `before` 事件中可以修改输入参数，在 `after` 事件中可以修改输出结果
3. **错误处理**：在 `error` 事件中可以实现降级方案，但要注意避免无限循环
4. **性能考虑**：生命周期事件会在每次翻译时触发，注意观察者的性能影响
5. **数据一致性**：修改事件数据时要确保数据格式正确，避免导致后续处理失败
