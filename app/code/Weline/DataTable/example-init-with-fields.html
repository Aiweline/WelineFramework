数据表初始化时加载字段配置示例

功能说明

这个示例演示了数据表在初始化时自动加载字段配置的功能：

  • 初始化时加载字段：表格初始化时自动请求字段配置接口
  • 缓存机制：第一次请求后，字段配置会被缓存，后续点击字段设置不再重复请求
  • 立即渲染：获取到用户配置的展示字段和筛选字段后，立即渲染到数据表格上
  • 配置保存：保存字段配置后，表格会立即重新构建并加载数据

测试步骤

  1. 页面加载后，观察控制台输出，查看字段配置加载过程
  2. 点击"字段设置"按钮，观察是否使用了缓存的字段数据
  3. 修改字段配置并保存，观察表格是否立即重新渲染
  4. 再次点击"字段设置"，确认使用了缓存数据
