✨ 美化特性

🎨 现代化设计

采用渐变背景、圆角边框和阴影效果，提供现代化的视觉体验。按钮具有微妙的悬停动画和点击反馈。

🎯 智能交互

支持点击切换、点击外部关闭、ESC键关闭等多种交互方式。图标具有旋转动画效果。

🌈 颜色区分

不同类型的重置操作使用不同的颜色：表头重置（绿色）、筛选重置（紫色）、全部重置（红色）。

📱 响应式设计

完美适配移动端设备，在小屏幕上自动调整布局和尺寸，保持良好的用户体验。

🌙 深色模式

自动检测系统主题偏好，在深色模式下提供相应的颜色方案，确保良好的可读性。

♿ 无障碍支持

支持键盘导航和屏幕阅读器，提供完整的无障碍访问支持。

JavaScript 交互

// 初始化下拉菜单功能 initDropdowns: function() { // 绑定下拉菜单切换事件 $(document).on('click.w-dropdown-toggle', '[data-w-toggle="dropdown"]', function(e) { e.preventDefault(); e.stopPropagation(); const $toggle = $(this); const $dropdown = $toggle.siblings('.w-dropdown-menu'); const $otherDropdowns = $('.w-dropdown-menu.show').not($dropdown); // 关闭其他下拉菜单 $otherDropdowns.removeClass('show'); // 切换当前下拉菜单 $dropdown.toggleClass('show'); // 添加旋转动画到图标 const $icon = $toggle.find('i.fas.fa-undo'); if ($dropdown.hasClass('show')) { $icon.css('transform', 'rotate(180deg)'); } else { $icon.css('transform', 'rotate(0deg)'); } }); // 点击外部关闭下拉菜单 $(document).on('click.w-dropdown-close', function(e) { if (!$(e.target).closest('.w-dropdown').length) { $('.w-dropdown-menu.show').removeClass('show'); $('[data-w-toggle="dropdown"] i.fas.fa-undo').css('transform', 'rotate(0deg)'); } }); }
