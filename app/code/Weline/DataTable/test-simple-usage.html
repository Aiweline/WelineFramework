DataTable 最简单用法演示

最简单用法 - 只需要两个参数

开发者只需要指定 model 和 scope 两个参数，DataTable 就会自动生成完整的表格结构。

<w:d-table model="WeShop\Store\Model\Store" scope="store-listing"></w:d-table>
自动生成的功能：
  • 自动从模型中获取字段信息
  • 智能生成表格列和过滤器
  • 自动识别字段类型和宽度
  • 过滤掉敏感字段（password、token等）
  • 限制显示字段数量，避免表格过宽
  • 支持搜索、排序、分页功能
  • 用户可自定义显示列和筛选器
智能特性：
  • 状态字段自动生成选项（启用/禁用）
  • 日期字段自动使用日期选择器
  • 邮箱字段自动使用邮箱输入框
  • 电话字段自动使用电话输入框
  • ID字段使用较窄的列宽
  • 名称字段使用较宽的列宽

手动配置用法

如果需要更精确的控制，开发者可以手动配置字段和过滤器。

<w:d-table model="WeShop\Store\Model\Store" scope="store-listing"> <w:t-filter> <w:field belong="t-filter" name="name" type="text" placeholder="搜索名称"></w:field> <w:field belong="t-filter" name="status" type="select" options="1:启用,0:禁用"></w:field> </w:t-filter> <w:t-header> <w:field belong="t-header" name="store_id" sortable="true" width="80">ID</w:field> <w:field belong="t-header" name="name" sortable="true" width="200">名称</w:field> <w:field belong="t-header" name="status" sortable="true" width="100">状态</w:field> </w:t-header> </w:d-table>

使用说明

必需参数：
  • model: 数据模型类名（如：WeShop\Store\Model\Store）
  • scope: 数据作用域，用于存储用户配置
可选参数：
  • page-size: 每页显示数量（默认20）
  • searchable: 是否启用搜索（默认true）
  • sortable: 是否启用排序（默认true）
  • editable: 是否启用编辑（默认false）
