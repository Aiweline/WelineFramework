## 使用方式

### 1. 基本用法
```phtml
<w:d-table model="Weline\DataTable\Model\TestUser" scope="user-demo"></w:d-table>
```

### 2. 完整配置
```phtml
<w:d-table model="Weline\DataTable\Model\TestUser" scope="user-management">
    <w:d-form for="add,edit" title="用户管理" form-mode="modal">
        <w:field name="username" type="text" label="用户名" required="true"></w:field>
        <w:field name="email" type="email" label="邮箱" required="true"></w:field>
        <w:field name="status" type="select" label="状态" options="1:启用,0:禁用"></w:field>
    </w:d-form>
    <w:t-filter>
        <w:field belong="t-filter" name="username" type="text" placeholder="搜索用户名"></w:field>
    </w:t-filter>
    <w:t-header>
        <w:field belong="t-header" name="user_id" sortable="true">ID</w:field>
        <w:field belong="t-header" name="username" sortable="true">用户名</w:field>
        <w:field belong="t-header" name="email" sortable="true">邮箱</w:field>
        <w:field belong="t-header" name="status" sortable="true">状态</w:field>
    </w:t-header>
</w:d-table>
```

### 3. 控制台命令
```bash
# 创建测试数据
php bin/w datatable create-test-data

# 清空测试数据
php bin/w datatable clear-test-data --type=users

# 导出数据
php bin/w datatable export-data --table=users --format=csv

# 生成配置
php bin/w datatable generate-config
```

- 删除数据：`DELETE /datatable/rest/v1/data-table/deleteData`
