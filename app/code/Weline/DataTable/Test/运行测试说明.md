## 🚀 运行测试

### 方式1：使用框架命令（推荐）

```bash
# 设置开发环境（首次运行需要）
php bin/w d:m:set dev

# 运行 DataTable 模块的所有测试
php bin/w phpunit:run --module=Weline_DataTable -b

# 或使用简化命令
php bin/w p:r Weline_DataTable

# 前台运行（直接查看结果）
php bin/w phpunit:run -f --module=Weline_DataTable

# 运行特定测试文件
php bin/w phpunit:run --name=ApiTest -b

# 运行特定测试方法
php bin/w phpunit:run --name=ApiTest::testDataTablePostDataSuccess -b
```

### 方式2：直接使用 PHPUnit

```bash
# 进入测试目录
cd app/code/Weline/DataTable/Test

# 运行所有测试
php ../../../../vendor/phpunit/phpunit/phpunit --configuration phpunit.xml

# 运行特定测试文件
php ../../../../vendor/phpunit/phpunit/phpunit --configuration phpunit.xml Unit/ApiTest.php

# 运行特定测试方法
php ../../../../vendor/phpunit/phpunit/phpunit --configuration phpunit.xml --filter testDataTablePostDataSuccess

# 生成测试报告
php ../../../../vendor/phpunit/phpunit/phpunit --configuration phpunit.xml --coverage-html coverage-html
```

### 方式3：验证测试文件（快速检查）

```bash
# 运行验证脚本
php app/code/Weline/DataTable/Test/verify_test.php
```
