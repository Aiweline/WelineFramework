**测试控制器位置：** `app/code/Weline/DataTable/Controller/Backend/Test/Comprehensive.php`

#### 2.4 属性继承功能测试

**访问路径：** `/datatable/backend/test/comprehensive/inheritance`

**测试目的：** 测试子标签从父标签继承属性的机制

**测试内容：**
- model 属性继承
- sortable 属性继承
- searchable 属性继承
- scope 自动生成
- 属性覆盖机制

**使用的模型：** `Weline\DataTable\Model\TestUser`

**测试代码示例：**
```xml
<w:d-table model="Weline\DataTable\Model\TestUser" scope="parent-scope" sortable="true" searchable="true">
    <w:t-header>
        <!-- 子标签自动继承父标签的 model, sortable, searchable 属性 -->
        <w:field belong="t-header" name="id">ID</w:field>
        <w:field belong="t-header" name="name">姓名</w:field>
    </w:t-header>
    <w:t-filter>
        <!-- 子标签自动继承父标签的 model, searchable 属性 -->
        <w:field belong="t-filter" name="name" type="text"></w:field>
    </w:t-filter>
</w:d-table>
```

**预期结果：**
- ✅ model 属性正确继承
- ✅ sortable 属性正确继承
- ✅ searchable 属性正确继承
- ✅ scope 自动生成正确
- ✅ 子标签可以覆盖父标签属性

## 常见问题

### 1. 表格不显示数据
**可能原因：**
- 数据表不存在或为空
- 模型类路径错误
- scope 配置错误

**解决方法：**
- 检查数据表是否存在
- 确认模型类名正确
- 检查 scope 是否唯一

### 2. 表单提交失败
**可能原因：**
- API 路由未配置
- 字段验证失败
- 数据格式错误

**解决方法：**
- 检查 API 路由配置
- 查看浏览器控制台错误
- 检查字段验证规则

### 3. 多模型查询失败
**可能原因：**
- JOIN 条件错误
- 表别名使用错误
- 字段名冲突

**解决方法：**
- 检查 JOIN 语法
- 确认表别名正确
- 使用完整字段名（表别名.字段名）
