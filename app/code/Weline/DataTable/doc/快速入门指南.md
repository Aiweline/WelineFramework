# DataTable 模块快速入门指南

## 🚀 快速开始

### 第一步：创建基础表格

最简单的用法，只需传入模型和范围参数：

```html
<d-table model="App\Models\User" scope="user-list">
    <t-header>
        <field belong="t-header" name="id" sortable="true" width="80">
            ID
        </field>
        <field belong="t-header" name="username" sortable="true" width="150">
            用户名
        </field>
        <field belong="t-header" name="email" sortable="true" width="200">
            邮箱
        </field>
    </t-header>
</d-table>
```

### 第二步：添加搜索过滤器

```html
<d-table model="App\Models\User" scope="user-search">
    <t-header>
        <field belong="t-header" name="id" sortable="true" width="80">
            ID
        </field>
        <field belong="t-header" name="username" sortable="true" width="150">
            用户名
        </field>
    </t-header>
    <t-filter>
        <field belong="t-filter" name="username" type="text" 
               placeholder="搜索用户名">
        </field>
    </t-filter>
</d-table>
```

### 第三步：添加表单功能

```html
<d-form 
    model="App\Models\User" 
    scope="user-form" 
    mode="add"
    form-mode="modal"
    title="添加用户"
    show-trigger-button="true">
    <field belong="d-form" name="username" type="text" required="true">
        用户名
    </field>
    <field belong="d-form" name="email" type="email" required="true">
        邮箱
    </field>
</d-form>
```

## 📚 核心标签

| 标签 | 用途 | 示例 |
|------|------|------|
| `d-table` | 数据表格容器 | `<d-table model="User" scope="table1">` |
| `t-header` | 表格表头定义 | `<t-header>` |
| `t-filter` | 搜索过滤工具栏 | `<t-filter>` |
| `field` | 字段定义 | `<field name="username" type="text">` |
| `d-form` | 表单容器 | `<d-form model="User" mode="add">` |

## 🔧 常用属性

### d-table 属性
- `model` (必需): 数据模型类名
- `scope` (必需): 表格作用域
- `join`: 多表关联查询
- `sortable`: 启用排序 (默认: true)
- `searchable`: 启用搜索 (默认: true)

### field 属性
- `belong` (必需): 所属标签 (t-header/t-filter/d-form)
- `name` (必需): 字段名称
- `type`: 字段类型 (text,email,number,select,date等)
- `sortable`: 是否可排序
- `width`: 列宽
- `placeholder`: 占位符文本
- `required`: 是否必填
- `options`: 选择项 (用于select,radio,checkbox)

## 🎯 常见字段类型

```html
<!-- 基础输入 -->
<field belong="d-form" name="username" type="text" required="true">
    用户名
</field>

<field belong="d-form" name="email" type="email" required="true">
    邮箱地址
</field>

<field belong="d-form" name="age" type="number" min="18" max="100">
    年龄
</field>

<!-- 选择类型 -->
<field belong="d-form" name="status" type="select" 
       options="active:激活,inactive:禁用">
    状态
</field>

<field belong="d-form" name="gender" type="radio" 
       options="male:男,female:女">
    性别
</field>

<!-- 日期时间 -->
<field belong="d-form" name="birthday" type="date">
    生日
</field>

<field belong="d-form" name="meeting_time" type="datetime">
    会议时间
</field>

<!-- 文件上传 -->
<field belong="d-form" name="avatar" type="image" accept="image/*">
    头像
</field>
```

## 🔗 多模型关联

```html
<!-- 用户订单关联表 -->
<d-table 
    model="User as u, Order as o" 
    join="left u.id = o.user_id"
    scope="user-orders">
    <t-header>
        <field belong="t-header" name="u.username" sortable="true">
            用户名
        </field>
        <field belong="t-header" name="o.order_no" sortable="true">
            订单号
        </field>
        <field belong="t-header" name="o.total_amount" sortable="true">
            订单金额
        </field>
    </t-header>
</d-table>
```

## 🎨 自动生成

当标签内容为空时，系统会自动生成结构：

```html
<!-- 自动生成表格 -->
<d-table model="App\Models\User" scope="auto-table">
    <!-- 自动生成表头和过滤器 -->
</d-table>

<!-- 自动生成表单 -->
<d-form 
    model="App\Models\User" 
    scope="auto-form" 
    auto-generate="true">
    <!-- 自动生成所有字段 -->
</d-form>
```

## 🔄 属性继承

子标签自动继承父标签的属性：

```html
<d-table 
    model="App\Models\User" 
    scope="demo"
    sortable="true"
    searchable="true">
    
    <!-- 自动继承: model, scope, sortable, searchable -->
    <t-header>
        <field belong="t-header" name="username">
            用户名
        </field>
    </t-header>
</d-table>
```
