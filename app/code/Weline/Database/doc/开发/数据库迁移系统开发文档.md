# Weline_Database 数据库迁移系统开发文档

## 概述

Weline_Database模块是一个企业级的数据库迁移管理系统，支持版本控制、数据备份、安全回滚，确保即使在删除表结构的情况下也能完整恢复数据和表结构。

## 核心特性

### 1. 企业级稳定性
- **事务安全**: 所有迁移操作都在事务中执行，确保数据一致性
- **依赖管理**: 自动检查迁移依赖关系，确保执行顺序
- **错误恢复**: 失败时自动回滚，支持重试机制
- **状态跟踪**: 详细的执行状态记录和日志

### 2. 数据安全保障
- **智能备份**: 自动备份受影响的数据
- **安全回滚**: 支持表结构删除后的完整数据恢复
- **数据验证**: 迁移前后数据完整性验证
- **版本控制**: 基于模块版本号的迁移管理

### 3. 语义化命名
- **清晰结构**: `{action}__{description}_{date}-{version}.php`
- **示例**: `create_table__users_20250101-v1.0.0.php`
- **示例**: `drop_column__raw_data_20250102-v1.0.1.php`

## 迁移脚本开发

### 1. 创建迁移脚本

在模块的 `Setup/Db/Migration/` 目录下创建迁移文件：

```php
<?php
/**
 * 删除用户表的raw_data字段迁移
 * 
 * @author WelineFramework
 * @package Weline\YourModule\Setup\Db\Migration
 */

namespace Weline\YourModule\Setup\Db\Migration;

use Weline\Database\Interface\MigrationInterface;
use Weline\Framework\Database\ConnectionFactory;
use Weline\Framework\Database\Api\Db\Ddl\TableInterface;

class DropColumnRawData20250102V101 implements MigrationInterface
{
    private ConnectionFactory $connectionFactory;
    private array $backupData = [];
    
    public function __construct(ConnectionFactory $connectionFactory)
    {
        $this->connectionFactory = $connectionFactory;
    }
    
    /**
     * 执行迁移升级
     */
    public function install(): bool
    {
        try {
            $connection = $this->connectionFactory->getConnection();
            
            // 1. 备份数据
            $this->backupData = $this->backupColumnData('users', 'raw_data');
            
            // 2. 删除字段
            $connection->deleteColumn('users', 'raw_data');
            
            return true;
            
        } catch (\Exception $e) {
            throw new \Exception("删除字段失败: " . $e->getMessage());
        }
    }
    
    /**
     * 执行迁移回滚
     */
    public function uninstall(): bool
    {
        try {
            $connection = $this->connectionFactory->getConnection();
            
            // 1. 恢复字段
            $connection->addColumn(
                'users',
                'raw_data',
                TableInterface::column_type_TEXT,
                0,
                'null',
                '原始数据JSON'
            );
            
            // 2. 恢复数据
            $this->restoreColumnData('users', 'raw_data', $this->backupData);
            
            return true;
            
        } catch (\Exception $e) {
            throw new \Exception("恢复字段失败: " . $e->getMessage());
        }
    }
    
    /**
     * 备份列数据
     */
    private function backupColumnData(string $table, string $column): array
    {
        $connection = $this->connectionFactory->getConnection();
        $query = $connection->select()
            ->from($table, ['id', $column])
            ->where("{$column} IS NOT NULL");
        
        $data = $connection->fetchAll($query);
        
        // 保存备份到文件
        $backupFile = $this->getBackupFilePath($table, $column);
        file_put_contents($backupFile, json_encode($data, JSON_UNESCAPED_UNICODE));
        
        return $data;
    }
    
    /**
     * 恢复列数据
     */
    private function restoreColumnData(string $table, string $column, array $data): void
    {
        if (empty($data)) {
            return;
        }
        
        $connection = $this->connectionFactory->getConnection();
        
        foreach ($data as $row) {
            $connection->update(
                $table,
                [$column => $row[$column]],
                ['id = ?' => $row['id']]
            );
        }
    }
    
    /**
     * 获取备份文件路径
     */
    private function getBackupFilePath(string $table, string $column): string
    {
        $backupDir = 'var/backup/migrations/';
        if (!is_dir($backupDir)) {
            mkdir($backupDir, 0755, true);
        }
        
        return $backupDir . "{$table}_{$column}_" . date('YmdHis') . '.json';
    }
    
    public function getInfo(): array
    {
        return [
            'name' => 'Drop Raw Data Column',
            'description' => '删除用户表的raw_data字段',
            'version' => '1.0.1',
            'date' => '2025-01-02',
            'author' => 'WelineFramework'
        ];
    }
    
    public function validate(): bool
    {
        $connection = $this->connectionFactory->getConnection();
        $columns = $connection->describeTable('users');
        
        return isset($columns['raw_data']);
    }
    
    public function getDependencies(): array
    {
        return [];
    }
    
    public function getDescription(): string
    {
        return '删除用户表的raw_data字段，支持数据备份和恢复';
    }
    
    public function getVersion(): string
    {
        return '1.0.1';
    }
    
    public function getDate(): string
    {
        return '2025-01-02';
    }
}
```

### 3. 数据备份和恢复策略

#### 自动备份机制
```php
/**
 * 智能数据备份
 */
private function smartBackup(string $table, string $operation, array $params = []): array
{
    $backupData = [];
    
    switch ($operation) {
        case 'drop_column':
            // 备份要删除的列数据
            $backupData = $this->backupColumnData($table, $params['column']);
            break;
            
        case 'drop_table':
            // 备份整个表数据
            $backupData = $this->backupTableData($table);
            break;
            
        case 'modify_column':
            // 备份修改前的列数据
            $backupData = $this->backupColumnData($table, $params['column']);
            break;
    }
    
    return $backupData;
}
```

#### 数据恢复机制
```php
/**
 * 智能数据恢复
 */
private function smartRestore(string $table, string $operation, array $backupData, array $params = []): void
{
    switch ($operation) {
        case 'drop_column':
            // 恢复列数据
            $this->restoreColumnData($table, $params['column'], $backupData);
            break;
            
        case 'drop_table':
            // 恢复整个表数据
            $this->restoreTableData($table, $backupData);
            break;
            
        case 'modify_column':
            // 恢复修改前的列数据
            $this->restoreColumnData($table, $params['column'], $backupData);
            break;
    }
}
```
