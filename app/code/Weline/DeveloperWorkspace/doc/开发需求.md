# DeveloperWorkspace 开发需求

## 需求记录

### 2026-01-31 - Framework/View/doc 扫描支持

**背景**
文档导入命令目前只扫描 `Framework/doc`，导致 `Framework/view/doc` 中的文档无法被收集与检索。

**功能需求**
- 文档导入命令需支持扫描 `Framework/view/doc` 下的文档。
- 所有来自 `Framework/view/doc` 的文档必须归属到 `Weline_Framework` 模块。
- 原有其他模块的文档扫描规则保持不变。
- 扫描统计需正确计入新增/更新数量。
- `Framework/view/doc` 不存在时，导入命令应跳过并继续执行。

**技术方案（概要）**
- 在文档扫描服务中为 `Weline_Framework` 增加额外的 `view/doc` 扫描入口。
- 扫描结果仍以 `Weline_Framework` 模块归属写入。

**测试计划**
- 单元/集成测试：新增覆盖 `view/doc` 扫描的用例，验证文档路径以 `view/doc/` 前缀保存且模块归属正确。
- 命令行验证：运行 `php bin/w doc:import` 确认扫描统计包含 `view/doc` 文档。
