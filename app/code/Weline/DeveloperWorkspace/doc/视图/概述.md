# Weline 框架视图标签使用说明

本说明涵盖 Weline 框架下 view 层常用标签及语法，适用于所有 module 的 phtml 文件。

---

## 1. 条件与循环标签（建议优先使用 w: 前缀）

- **w:if / w:else / w:foreach**
  ```phtml
  <w:if condition="$var == 1">内容</w:if>
  <w:else/>  <!-- 可选 -->
  <w:foreach name="list" item="item">内容</w:foreach>
  ```

## 2. 结构与模板标签

- **w:block**
  ```phtml
  <w:block class="Weline\Admin\Block\Backend\Page\Topbar" />
  ```
- **w:template**
  ```phtml
  <w:template>Weline_Admin::common/head.phtml</w:template>
  ```

## 3. 资源标签

- **w:js**
  ```phtml
  <w:js>Weline_Backend::/backend/lib/vue/vue2.6.11.js</w:js>
  ```
- **w:css**
  ```phtml
  <w:css>Weline_Eav::libs/bootstrap-colorpicker/bootstrap-colorpicker.min.css</w:css>
  ```

## 4. 作用域标签

- **w:scope**
  ```phtml
  <w:scope url="@backend-url('backend/user-data')" container-id="WebsitePage" event="input change click" />
  ```

## 5. DataTable 相关标签

- **w:d-table** 及子标签
  ```phtml
  <w:d-table model="ModelClass" scope="scope-name">
    <w:t-filter>...</w:t-filter>
    <w:t-header>...</w:t-header>
    <w:t-body></w:t-body>
    <w:t-footer></w:t-footer>
  </w:d-table>
  ```

## 6. 其它标签

- **dd**（原生 HTML 标签，建议优先使用 w: 前缀自定义标签）
  ```phtml
  <dd>内容</dd>
  ```
- 目前未发现 `w:url` 等标签的实际用例。
- 可根据实际项目扩展自定义 w: 前缀标签。

---

**建议：**
- 标签属性支持变量、表达式、函数调用。
- 标签可嵌套使用，便于复杂视图逻辑。
- 推荐结合 block/template 实现模块化和复用。
- 框架推荐优先使用 w: 前缀标签，统一风格、增强可维护性。

---

**典型用法补充：**
- 标签可嵌套使用，如 block/template 可组合实现页面布局：
  ```phtml
  <w:block class="Weline\Admin\Block\Backend\Page\Topbar" />
  <w:template>Weline_Admin::common/head.phtml</w:template>
  <w:block>
    <w:if condition="$showSidebar">
      <w:template>Weline_Admin::common/left-sidebar.phtml</w:template>
    </w:if>
  </w:block>
  ```
- 属性可传递变量、表达式，支持动态渲染：
  ```phtml
  <w:js>{{ jsPath }}</w:js>
  <w:css>{{ cssPath }}</w:css>
  <w:scope url="@backend-url('{{ apiUrl }}')" container-id="{{ containerId }}" />
  ```
- 推荐将页面结构拆分为多个 block/template，提升复用性和可维护性。
