│   └── Frontend                  # 前台控制器子目录

## 目录说明

### 核心目录

**etc/** - 配置文件目录
- `env.php` - 环境配置，定义模块的环境变量和配置参数
- `db.php` - 数据库配置，当模块使用独立数据库时的连接配置
- `db.sqlite` - SQLite数据库文件，用于开发测试的轻量级数据库
- `plugin.xml` - 插件配置，定义模块的插件机制和依赖关系
- `event.xml` - 事件配置，定义模块的事件监听和触发
- `xsd/` - XML Schema定义目录，用于XML配置文件的验证
- `backend/menu.xml` - 后台菜单配置，定义后台管理界面的菜单结构

**Api/** - API接口目录
- `Rest/V1/Demo.php` - REST API接口实现，支持版本管理

**Controller/** - 控制器目录
- `Backend/` - 后台控制器，处理后台管理功能
- `Frontend/` - 前台控制器，处理前台用户功能

**Model/** - 模型目录
- `Demo.php` - 主模型类，处理核心业务数据

**Block/** - 区块目录
- `Header/` - 头部区块，处理页面头部内容
- `Footer/` - 底部区块，处理页面底部内容

### 辅助目录

**Helper/** - 辅助类目录，提供工具方法
**Setup/** - 安装脚本目录，处理模块安装和升级
**Console/** - 控制台命令目录，提供命令行工具
**Plugin/** - 插件目录，扩展系统功能
**Observer/** - 观察者目录，处理系统事件
**Session/** - 会话管理目录，处理用户会话
**Config/** - 配置管理目录，处理配置文件
**Cache/** - 缓存管理目录，处理数据缓存

### 开发工具目录

**git/** - Git相关工具目录，提供Git操作脚本
**dev/** - 开发工具目录，存放开发工具和脚本
**test/** - 测试目录，存放测试文件
**example/** - 示例目录，存放使用示例

### 视图和文档目录

**view/** - 视图目录
- `statics/` - 静态资源（CSS、JS、图片）
- `templates/` - 模板文件
- `blocks/` - 区块视图
- `hooks/` - 钩子视图
- `tpl/` - 模板目录（开发缓存目录）

**Taglib/** - 标签库目录，存放自定义模板标签
**doc/** - 文档目录，存放模块文档

这些特殊目录结构体现了Weline框架的模块化设计理念，通过合理的目录分类和命名，使模块结构更加清晰和易于维护。
