

## 新建模块
### 1 创建目录
我们以创建Weline_Demo模块为例。创建目录：app/code/Weline/Demo 
### 2 创建安装文件
app/code/Weline/Demo/register.php
```php
<?php
use Weline\Framework\Register\Register;

Register::register(
    Register::MODULE,
    'Weline_Demo',
    __DIR__,
    '1.0.0',
    '测试模块'
);
```
这样就能完成最简单的模块了，没有任何其他代码的空模块。
## 模块安装
php bin/w setup:upgrade # 命令将收集模块，并安装模块，然后根据依赖关系安装必要的模型，最后更新路由
