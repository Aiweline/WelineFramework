# 模块控制器路由
### 1 先指定模块路由
app/code/Weline/Demo/etc/env.php
```php
<?php
return [
    'router'=>'weline/demo'
];
```
app/code/Weline/Demo/Controller/Frontend/Index.php
```php
<?php
namespace Weline\Demo\Controller\Frontend;
use Weline\Framework\App\Controller\FrontendController;

class Index extends FrontendController{
    public function index(){
        return $this->fetch(); # 默认读取app/code/Weline/Demo/view/templates/frontend/index/index.phtml
    }
}
```
### 2 配置视图
app/code/Weline/Demo/view/templates/Frontend/Index/index.phtml  # Frontend/Index是控制器路径，index是方法名。注意大小写要与控制器目录或者文件的大小写一致。
```php
<?php
echo 'Hello World';
```
### 3 更新路由
php bin/w setup:upgrade # 路由资源文件会自动生成
### 4 访问路由
#### 1 如果你没有配置nginx等服务器软件，框架内置了一个简单的服务器，你可以直接访问路由。如果有有nginx或者apache等配置，不用关注这个步骤
```bash
php bin/w server:start # 启动开发服务器
```
