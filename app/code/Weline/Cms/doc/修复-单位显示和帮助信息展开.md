# 修复：单位显示和帮助信息展开

## 问题描述

用户反馈了两个关键问题：

1. **单位信息缺失**：在可视化配置界面中，配置项的单位没有显示
   - 例如："内容最大宽度"应该显示单位"%/px"，但实际没有显示
   - "头部高度"、"左右内边距"等响应式配置项的单位"px"也没有显示

2. **组级别帮助信息无法展开**：分组的帮助说明点击后无反应，无法查看详细内容
   - 例如："尺寸"分组下的"📱响应式配置说明"无法展开

## 根本原因

### 1. 单位信息缺失的原因

在 `renderVisualConfig()` 函数中，`switch` 语句缺少关键类型的处理：

- ❌ **缺少 `responsive` 类型**：响应式配置（如 `60/80`）没有被正确处理
- ❌ **默认 `text` 类型没有单位**：即使配置项有 `unit` 属性，也不会显示

对比 `renderStyleConfig()` 函数，它有完整的类型处理，包括单位显示。

### 2. 帮助信息无法展开的原因

虽然 `toggleGroupHelp()` 函数存在，但实现不够健壮：
- ID 判断逻辑过于复杂
- 缺少对初始状态（`display: ''`）的处理
- 缺少调试信息

## 解决方案

### 修改文件

`app/code/GuoLaiRen/PageBuilder/view/templates/Backend/Page/form.phtml`

### 具体修改

#### 1. 添加 `responsive` 类型处理（约第3118行）

**新增代码**：
```javascript
case 'responsive':
    // 响应式配置：支持 60/80 或 60/70/80 格式，使用 text 输入框
    html += `
        <div class="input-group">
            <input type="text" 
                   class="form-control auto-save-field" 
                   id="${fieldId}"
                   data-config-key="${configKey}"
                   value="${currentValue}"
                   placeholder="${escapeHtml(config.default || '')}"
                   title="<?= __('支持单一值(如: 80)或多端值(如: 60/80 或 60/70/80)') ?>">
            ${config.unit ? `<span class="input-group-text">${escapeHtml(config.unit)}</span>` : ''}
        </div>
    `;
    break;
```

**效果**：
- 响应式配置项现在会显示单位（如 `px`、`%/px`）
- 支持工具提示说明输入格式

#### 4. 优化 `toggleGroupHelp()` 函数（约第1793行）

**修改前**：
```javascript
window.toggleGroupHelp = function(helpId) {
    const helpContent = document.getElementById(helpId);
    const icon = document.getElementById('help_icon_' + helpId.replace('help_', ''));
    
    if (helpContent) {
        const isHidden = helpContent.style.display === 'none';
        helpContent.style.display = isHidden ? 'block' : 'none';
        
        if (icon) {
            icon.className = isHidden ? 'mdi mdi-chevron-up' : 'mdi mdi-chevron-down';
        }
    }
};
```

**修改后**：
```javascript
/**
 * 切换分组说明的显示/隐藏
 * @param {string} helpContentId - 帮助内容的DOM元素ID
 */
window.toggleGroupHelp = function(helpContentId) {
    const helpContent = document.getElementById(helpContentId);
    if (!helpContent) {
        console.error('帮助内容元素未找到:', helpContentId);
        return;
    }
    
    // 构建图标ID：从 help_visual_xxx 推导出 help_icon_visual_xxx
    const helpIconId = helpContentId.replace('help_', 'help_icon_');
    const icon = document.getElementById(helpIconId);
    
    // 切换显示状态（处理初始空值情况）
    const isHidden = helpContent.style.display === 'none' || helpContent.style.display === '';
    helpContent.style.display = isHidden ? 'block' : 'none';
    
    // 切换图标方向
    if (icon) {
        icon.className = isHidden ? 'mdi mdi-chevron-up' : 'mdi mdi-chevron-down';
    } else {
        console.warn('帮助图标元素未找到:', helpIconId);
    }
};
```

**改进**：
- 添加 JSDoc 文档注释
- 添加错误处理和调试信息
- 修复初始状态判断（`display: ''` 也视为隐藏）
- 简化 ID 推导逻辑

## 支持的配置类型

现在 `renderVisualConfig()` 函数支持以下所有类型：

| 类型 | 单位显示 | 说明 |
|------|---------|------|
| `color` | ❌ | 颜色选择器，不需要单位 |
| `select` | ❌ | 下拉选择器，不需要单位 |
| `responsive` | ✅ | 响应式配置，显示单位（如 `px`、`%/px`） |
| `number` | ✅ | 数字输入，显示单位 |
| `textarea` | ❌ | 多行文本，不需要单位 |
| `text` | ✅ | 文本输入，如果有 `unit` 则显示 |

## 技术要点

### 1. 类型处理顺序

```javascript
switch (config.type) {
    case 'color':      // 特殊：颜色选择器
    case 'select':     // 特殊：下拉选择
    case 'responsive': // 新增：响应式配置（带单位）
    case 'number':     // 数字（带单位）
    case 'textarea':   // 多行文本
    case 'text':       // 文本（带单位）
    default:           // 默认同 text
}
```

### 2. 单位显示逻辑

```javascript
${config.unit ? `<span class="input-group-text">${escapeHtml(config.unit)}</span>` : ''}
```

- 条件判断：仅当 `config.unit` 存在时显示
- HTML 转义：使用 `escapeHtml()` 防止 XSS
- Bootstrap 样式：使用 `input-group-text` 类

### 3. ID 管理策略

```javascript
// 明确定义 ID
const helpContentId = `help_visual_${fileKey}_${groupKey}`;
const helpIconId = `help_icon_visual_${fileKey}_${groupKey}`;

// 在函数中推导
const helpIconId = helpContentId.replace('help_', 'help_icon_');
```

### 4. 初始状态处理

```javascript
// 错误：只判断 'none'
const isHidden = helpContent.style.display === 'none';

// 正确：同时判断 'none' 和 ''
const isHidden = helpContent.style.display === 'none' || helpContent.style.display === '';
```

## 常见问题

### Q1: 单位不显示？

**可能原因**：
1. 配置项的 `unit` 属性为空
2. 模板文件中没有定义单位
3. 类型不匹配

**解决方法**：
1. 检查模板文件配置：`size.height => 头部高度:responsive:60/80|px`
2. 确保 `|px` 部分存在
3. 清理缓存：`php bin/w cache:clear -f`

### Q2: 帮助信息点击后无反应？

**可能原因**：
1. JavaScript 错误
2. ID 不匹配
3. `toggleGroupHelp` 函数未加载

**解决方法**：
1. 打开浏览器控制台（F12）查看错误
2. 检查是否有红色错误信息
3. 刷新页面并重试

### Q3: 帮助内容显示乱码？

**可能原因**：
- 换行符处理问题
- HTML 转义问题

**解决方法**：
- 使用 `white-space: pre-line` 保留换行
- 使用 `escapeHtml()` 转义特殊字符
