# PageBuilder 静态资源本地化更新日志

## 更新时间
2025-10-21

## 更新内容

### 概述
将 PageBuilder 模块中使用的第三方 JavaScript 库从 CDN 加载改为本地文件加载，以提高加载速度、稳定性和离线可用性。

### 变更详情

#### 1. 新增文件
创建了 `view/statics/libs/` 目录，包含以下第三方库：

```
view/statics/libs/
├── ckeditor5/
│   ├── ckeditor.js          (1,122.94 KB)
│   └── zh-cn.js             (7.74 KB)
├── html2canvas/
│   └── html2canvas.min.js   (194.03 KB)
├── whatwg-fetch/
│   └── fetch.umd.js         (17.95 KB)
└── README.md
```

**总大小**: 约 1.34 MB

#### 2. 修改文件

**文件**: `view/templates/Backend/Page/form.phtml`

**变更前**:
```html
<!-- CKEditor 5 CDN -->
<script src="https://cdn.ckeditor.com/ckeditor5/39.0.1/classic/ckeditor.js"></script>
<script src="https://cdn.ckeditor.com/ckeditor5/39.0.1/classic/translations/zh-cn.js"></script>

<!-- html2canvas for screenshot功能 -->
<script src="https://cdn.jsdelivr.net/npm/html2canvas@1.4.1/dist/html2canvas.min.js"></script>
```

**变更后**:
```html
<!-- CKEditor 5 本地文件 -->
<script src="@static(GuoLaiRen_PageBuilder::libs/ckeditor5/ckeditor.js)"></script>
<script src="@static(GuoLaiRen_PageBuilder::libs/ckeditor5/zh-cn.js)"></script>

<!-- html2canvas for screenshot功能 -->
<script src="@static(GuoLaiRen_PageBuilder::libs/html2canvas/html2canvas.min.js)"></script>
```

**使用 @static 标签的原因**：
- 框架推荐的标准方式
- 自动处理模块路径
- 支持版本控制和缓存优化

### 优势

1. **性能提升**
   - 减少外部 CDN 请求
   - 利用浏览器缓存
   - 减少 DNS 查询时间

2. **稳定性提升**
   - 不依赖外部 CDN 可用性
   - 避免 CDN 被墙或不稳定导致的加载失败
   - 版本锁定，避免 CDN 更新导致的兼容性问题

3. **离线可用**
   - 支持内网环境部署
   - 支持离线开发和测试

4. **安全性提升**
   - 减少对第三方服务的依赖
   - 避免 CDN 被劫持或篡改的风险
   - 遵守数据安全和隐私政策

### 保留的外部加载

以下追踪代码保持从外部加载，不建议本地化：

1. **Google Tag Manager (GTM)**
   - 原因：需要实时更新追踪逻辑，官方要求从其服务器加载
   - 文件：`view/templates/base/tracking.phtml`
   - 文件：各样式模板的 `header.phtml`

2. **Facebook Pixel**
   - 原因：需要实时数据同步，官方要求从其服务器加载
   - 文件：`view/templates/base/tracking.phtml`

3. **Google Analytics 4 (GA4)**
   - 原因：需要实时数据收集，官方要求从其服务器加载

### 使用方式

在模板中引用本地静态文件：

**方式1：@static 标签（推荐）**
```php
<script src="@static(GuoLaiRen_PageBuilder::libs/ckeditor5/ckeditor.js)"></script>
```

**方式2：getStaticFile() 方法**
```php
<script src="<?= $this->getStaticFile('libs/ckeditor5/ckeditor.js') ?>"></script>
```

这些方法会自动生成正确的 URL 路径，支持：
- 模块静态文件路由
- 浏览器缓存控制
- 版本管理
- 路径自动处理

### 未来维护

#### 更新第三方库版本

1. 访问对应库的官方网站
2. 下载新版本文件
3. 替换 `view/statics/libs/` 中的对应文件
4. 更新 `view/statics/libs/README.md` 中的版本号
5. 测试功能是否正常
6. 更新本文档

#### 添加新的第三方库

1. 在 `view/statics/libs/` 下创建库的目录
2. 下载相关文件
3. 更新 `view/statics/libs/README.md`
4. 在模板中使用 `getStaticFile()` 引用
5. 更新本文档

### 回滚方案

如果本地化后出现问题，可以快速回滚：

1. 修改 `view/templates/Backend/Page/form.phtml`
2. 将静态文件引用改回 CDN URL
3. 重启服务或清理缓存
