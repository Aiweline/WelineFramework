# Cadrenal Therapeutics Landing 样式模板

这是一个专为生物制药公司、医疗健康类企业设计的企业官网落地页模板，风格参考 StockVerse/Cadrenal Therapeutics 网站设计。

## 特点

### 🎯 核心功能
- **Logo展示**：支持文字 logo 或图片 logo，可灵活配置位置
- **Hero区域**：大标题和副标题，支持背景图片和遮罩效果
- **关于我们**：公司简介和核心信息展示
- **产品展示**：产品卡片网格布局，展示临床阶段资产
- **使命愿景**：企业使命和愿景展示
- **行动号召**：CTA按钮，支持事件跟踪
- **响应式设计**：完美适配移动端和桌面端

### 🎨 设计风格
- 专业的企业级设计风格
- 清晰的层次结构和信息架构
- 温和的色彩方案，适合医疗健康行业
- 卡片式布局，突出关键信息

### 📱 响应式配置
模板内所有关键尺寸都支持移动端/桌面端独立配置：
- 字体大小：移动端/桌面端双值配置（如 `28/48` 表示移动端28px，桌面端48px）
- 间距边距：所有 padding、margin 都支持响应式
- 布局自动适配：产品卡片在移动端自动堆叠

### 🔧 可配置项

#### 头部 (header.phtml)
- Logo 显示/隐藏、地址、位置、宽度
- Hero 主标题和副标题内容、颜色、大小
- Hero 背景色、背景图片、遮罩效果
- 容器最大宽度和内边距
- **页面背景**：全屏背景图配置

#### 内容 (content.phtml)
1. **关于我们**
   - 标题和内容文字
   - 颜色、字号、行高
   - 显示/隐藏开关

2. **产品展示**
   - 标题配置
   - 产品卡片样式（背景、边框、内边距）
   - 产品名称、描述、状态
   - 网格布局自动适配

3. **使命愿景**
   - 标题和内容
   - 颜色和排版配置

4. **CTA按钮**
   - 按钮文字、链接、颜色
   - 字体大小、内边距、圆角
   - 事件跟踪配置

#### 页脚 (footer.phtml)
- 公司信息（名称、地址、电话、邮箱、网站）
- 快速导航链接
- 免责声明
- 版权信息
- 页脚样式（背景、文字、链接颜色）

## 使用场景

适用于以下类型的落地页：
- 🏥 生物制药公司官网
- 💊 医疗健康企业介绍
- 🔬 医药研发机构展示
- 📊 临床试验和产品推广
- 🏢 企业信息展示和品牌建设

- ✅ 纯 HTML/CSS/JavaScript，无需外部依赖
- ✅ 移动优先的响应式设计
- ✅ 支持 Google Analytics 和 GTM 追踪
- ✅ 完整的统计代码集成
- ✅ 平滑动画和过渡效果
- ✅ 语义化HTML结构

## 配置提示

1. **Hero背景**：建议使用高质量的企业形象图片，配合遮罩效果提升文字可读性
2. **产品卡片**：保持描述简洁明了，突出核心特点
3. **CTA按钮**：使用品牌主色调，确保与整体风格一致
4. **页脚信息**：确保公司联系信息准确完整
5. **免责声明**：根据实际业务需求修改法律声明文本
