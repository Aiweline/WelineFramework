# Weline 模块加载系统使用指南

## 概述

Weline 模块加载系统允许你通过配置文件声明JS模块，然后使用 `Weline.load()` 方法按需加载这些模块。这样可以：

1. **统一管理**：所有模块路径集中配置
2. **CDN Fallback**：支持多个路径，自动回退
3. **路径解析**：支持 `Weline_Module::path/to/file.js` 格式
4. **翻译分析**：加载时可以分析JS文件中的翻译词

## 配置文件

### 位置

`app/code/Weline/Frontend/view/theme/config/modules.json`

### 配置格式

```json
{
    "version": "1.0.0",
    "description": "Weline Frontend 模块配置文件",
    "modules": {
        "jquery": {
            "paths": [
                "https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js",
                "https://code.bdstatic.com/npm/jquery@3.6.0/dist/jquery.min.js",
                "Weline_Frontend::libs/jquery/3.6.0/jquery.min.js"
            ],
            "globalVar": "jQuery",
            "description": "jQuery库"
        },
        "vue": {
            "paths": [
                "Weline_Frontend::libs/vue/vue2.6.11.js"
            ],
            "globalVar": "Vue",
            "description": "Vue.js框架"
        }
    },
    "moduleAliases": {
        "jq": "jquery",
        "$": "jquery"
    }
}
```

### 配置说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `modules` | object | 模块配置对象 |
| `modules[moduleName].paths` | string[] | 模块路径数组（支持CDN fallback） |
| `modules[moduleName].globalVar` | string | 全局变量名（用于验证加载成功） |
| `modules[moduleName].description` | string | 模块描述 |
| `moduleAliases` | object | 模块别名映射 |

### 路径格式

支持以下路径格式：

1. **完整URL**：`https://cdn.example.com/lib.js`
2. **绝对路径**：`/static/path/to/file.js`
3. **模块路径**：`Weline_Module::path/to/file.js`（推荐）

模块路径会自动解析为：`/static/Weline_Module/path/to/file.js`

## 使用方法

### 1. 声明模块（按需加载或立即加载）

在页面底部声明需要使用的模块。可以选择按需加载（默认）或立即加载。

#### 1.1 按需加载（默认，推荐）

```javascript
// 在页面底部声明模块，按需加载
Weline.declare('vue');
Weline.declare('jquery');

// 后续代码中，当第一次访问 Vue 或 jQuery 时，会自动加载
// 例如：
new Vue({...});  // 此时会触发 Vue 的加载
$('#app');       // 此时会触发 jQuery 的加载
```

**优势**：
- 不阻塞页面加载
- 只加载实际使用的模块
- 代码更简洁

#### 1.2 立即加载（带回调）

```javascript
// 声明模块，立即加载
Weline.declare('vue', true);
Weline.declare(['jquery', 'vue'], true);

// 声明模块，立即加载，加载完成后执行回调
Weline.declare(['jquery', 'vue'], true, function() {
    // 所有模块加载完成后执行
    new Vue({
        el: '#app',
        data: {
            message: 'Hello Vue!'
        }
    });
    $('#app').show();
});

// 模块会立即开始加载，不等待使用
// 适合需要立即使用的场景
```

**适用场景**：
- 模块在页面加载后立即需要使用
- 需要预加载关键模块
- 避免首次使用时的加载延迟
- 需要在所有模块加载完成后执行初始化代码

### 2. 立即加载模块（带回调）

如果需要立即加载模块（不等待使用），可以使用 `Weline.load()`：

```javascript
// 加载单个模块
await Weline.load('jquery');

// 加载多个模块
await Weline.load(['jquery', 'vue']);

// 加载多个模块，加载完成后执行回调
Weline.load(['jquery', 'vue'], function() {
    // 所有模块加载完成后执行
    new Vue({
        el: '#app',
        data: {
            message: 'Hello Vue!'
        }
    });
    $('#app').show();
});

// 使用别名
await Weline.load('jq');  // 等同于 'jquery'
await Weline.load('$');   // 等同于 'jquery'
```

### 3. 确保模块已加载

使用 `Weline.use()` 确保模块已加载（如果已声明则按需加载，否则立即加载）：

```javascript
// 确保Vue已加载
const Vue = await Weline.use('vue');
new Vue({...});
```

### 2. 使用自定义路径

```javascript
// 使用单个自定义路径
await Weline.load('myModule', 'Weline_Demo::lib/my-module.js');

// 使用路径数组（CDN fallback）
await Weline.load('myModule', [
    'https://cdn.example.com/my-module.js',
    'Weline_Demo::lib/my-module.js'
]);
```

### 3. 在模板中使用

```php
<?php
// 在 head.phtml 中配置模块
?>
<script>
    window.WelineConfig = window.WelineConfig || {};
    window.WelineConfig.modulesConfig = {
        modules: {
            vue: {
                paths: ['Weline_Demo::lib/vue/vue2.6.11.js'],
                globalVar: 'Vue'
            }
        }
    };
</script>
<js>Weline_Theme::theme/frontend/assets/js/theme.js</js>

<script>
    // 页面加载后加载模块
    (async function() {
        await Weline.load('vue');
        // 现在可以使用 Vue 了
        new Vue({...});
    })();
</script>
```

### 4. 在HTML中使用 data 属性

#### 4.1 声明模块（按需加载或立即加载）

```html
<!-- 在页面底部声明模块，按需加载（默认） -->
<script>
    Weline.declare('vue');
    Weline.declare('jquery');
</script>

<!-- 声明模块，立即加载 -->
<script>
    Weline.declare('vue', true);
    Weline.declare('jquery', true);
</script>

<!-- 或者使用 data 属性（按需加载） -->
<div data-weline-declare="vue,jquery">
    <!-- 内容 -->
</div>
```

#### 4.2 立即加载模块

```html
<!-- 自动加载模块 -->
<div data-weline-load="jquery,vue">
    <!-- 内容 -->
</div>

<script>
    // 监听模块加载完成事件
    document.querySelector('[data-weline-load]').addEventListener('weline-modules-loaded', function(e) {
        console.log('模块加载完成:', e.detail.modules);
        // 现在可以使用这些模块了
        // jQuery 和 Vue 都已加载
    });
    
    // 监听加载错误事件
    document.querySelector('[data-weline-load]').addEventListener('weline-modules-error', function(e) {
        console.error('模块加载失败:', e.detail.error);
    });
</script>
```

**区别**：
- `data-weline-declare`：声明模块，按需加载（推荐）
- `data-weline-load`：立即加载模块

## API 参考

### `Weline.declare(moduleNames, loadImmediatelyOrCustomPath, customPath)`

声明模块（支持按需加载或立即加载）。

**参数**：
- `moduleNames` (string|string[]): 模块名称或模块名称数组
- `loadImmediatelyOrCustomPath` (boolean|string|string[]): 
  - 如果是 `boolean`：是否立即加载（`true`=立即加载，`false`=按需加载）
  - 如果是 `string|string[]`：自定义路径，按需加载
- `customPath` (string|string[]|boolean): 
  - 如果第二个参数是 `boolean`：自定义路径（可选）
  - 如果第二个参数是路径：是否立即加载（`boolean`，可选）

**返回**：`Promise<void>`

**示例**：
```javascript
// 声明单个模块，按需加载（默认）
Weline.declare('vue');

// 声明多个模块，按需加载
Weline.declare(['jquery', 'vue']);

// 声明模块，立即加载
Weline.declare('vue', true);
Weline.declare(['jquery', 'vue'], true);

// 声明模块，立即加载，加载完成后执行回调
Weline.declare(['jquery', 'vue'], true, function() {
    // 所有模块加载完成后执行
    new Vue({...});
    $('#app').show();
});

// 使用自定义路径，按需加载
Weline.declare('myModule', 'Weline_Demo::lib/my-module.js');

// 使用自定义路径，立即加载
Weline.declare('myModule', 'Weline_Demo::lib/my-module.js', true);
Weline.declare('myModule', true, 'Weline_Demo::lib/my-module.js');

// 使用自定义路径，立即加载，执行回调
Weline.declare('myModule', 'Weline_Demo::lib/my-module.js', true, function() {
    // 模块加载完成后执行
});

// 后续代码中，当第一次访问全局对象时，会自动加载（如果未立即加载）
new Vue({...});  // 触发 Vue 的加载
```

### `Weline.load(moduleNames, customPathOrCallback, callback)`

立即加载JS模块（支持回调）。

**参数**：
- `moduleNames` (string|string[]): 模块名称或模块名称数组
- `customPathOrCallback` (string|string[]|Function): 自定义路径（可选），或回调函数（可选）
- `callback` (Function): 回调函数（可选，当所有模块加载完成后执行）

**返回**：`Promise<any|any[]>`

**示例**：
```javascript
// 加载jQuery
const jQuery = await Weline.load('jquery');

// 加载多个模块
const [jQuery, Vue] = await Weline.load(['jquery', 'vue']);

// 加载多个模块，加载完成后执行回调
Weline.load(['jquery', 'vue'], function() {
    // 所有模块加载完成后执行
    new Vue({...});
    $('#app').show();
});

// 使用自定义路径
await Weline.load('myModule', 'Weline_Demo::lib/my-module.js');

// 使用自定义路径，加载完成后执行回调
Weline.load('myModule', 'Weline_Demo::lib/my-module.js', function() {
    // 模块加载完成后执行
});
```

### `Weline.use(moduleName)`

确保模块已加载（如果已声明则按需加载，否则立即加载）。

**参数**：
- `moduleName` (string): 模块名称

**返回**：`Promise<any>`

**示例**：
```javascript
// 确保Vue已加载
const Vue = await Weline.use('vue');
new Vue({...});
```

### `Weline.staticResourceResolver.resolve(modulePath)`

解析模块路径。

**参数**：
- `modulePath` (string): 模块路径

**返回**：`string` - 解析后的URL

**示例**：
```javascript
// 解析模块路径
const url = Weline.staticResourceResolver.resolve('Weline_Frontend::libs/vue/vue2.6.11.js');
// DEV: /Weline/Frontend/view/statics/libs/vue/vue2.6.11.js
// PROD: /static/Weline/Frontend/libs/vue/vue2.6.11.js

// 完整URL直接返回
const cdnUrl = Weline.staticResourceResolver.resolve('https://cdn.example.com/lib.js');
// 返回: https://cdn.example.com/lib.js
```

### `Weline.configManager.getModuleConfig(moduleName)`

获取模块配置。

**参数**：
- `moduleName` (string): 模块名称

**返回**：`Promise<Object|null>`

**示例**：
```javascript
const config = await Weline.configManager.getModuleConfig('jquery');
// 返回: { paths: [...], globalVar: 'jQuery', description: '...' }
```

## 路径解析规则

### 1. 完整URL

```javascript
'https://cdn.example.com/lib.js'
// 返回: https://cdn.example.com/lib.js
```

### 2. 绝对路径

```javascript
'/static/path/to/file.js'
// 返回: /static/path/to/file.js
```

### 3. 模块路径（推荐）

```javascript
'Weline_Frontend::libs/vue/vue2.6.11.js'
// 返回: /static/Weline_Frontend/libs/vue/vue2.6.11.js

'Weline_Demo::lib/my-module.js'
// 返回: /static/Weline_Demo/lib/my-module.js
```

## CDN Fallback 机制

当配置了多个路径时，系统会按顺序尝试加载，直到成功：

```json
{
    "modules": {
        "jquery": {
            "paths": [
                "https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js",  // 优先尝试CDN
                "https://code.bdstatic.com/npm/jquery@3.6.0/dist/jquery.min.js",     // CDN备用
                "Weline_Frontend::libs/jquery/3.6.0/jquery.min.js"                   // 本地备用
            ],
            "globalVar": "jQuery"
        }
    }
}
```

加载流程：
1. 尝试第一个CDN
2. 如果失败，尝试第二个CDN
3. 如果都失败，使用本地路径
4. 验证全局变量 `jQuery` 是否存在

## 翻译词分析

通过统一使用 `Weline.load()` 加载模块，可以在加载时分析JS文件中的翻译词：

```javascript
// 在加载模块时，可以拦截并分析
const originalLoad = Weline.load;
Weline.load = async function(moduleNames, customPath) {
    const result = await originalLoad.call(this, moduleNames, customPath);
    
    // 分析已加载的脚本中的翻译词
    // 这里可以添加翻译词提取逻辑
    
    return result;
};
```

## 最佳实践

### 1. 使用模块路径格式

```javascript
// ✅ 推荐
await Weline.load('vue', 'Weline_Demo::lib/vue/vue2.6.11.js');

// ❌ 不推荐（硬编码路径）
await Weline.load('vue', '/static/Weline_Demo/lib/vue/vue2.6.11.js');
```

### 2. 配置CDN Fallback

```json
{
    "modules": {
        "vue": {
            "paths": [
                "https://cdn.jsdelivr.net/npm/vue@2.6.14/dist/vue.min.js",
                "Weline_Frontend::libs/vue/vue2.6.11.js"
            ],
            "globalVar": "Vue"
        }
    }
}
```

### 3. 使用别名简化调用

```json
{
    "moduleAliases": {
        "jq": "jquery",
        "$": "jquery",
        "vue2": "vue"
    }
}
```

```javascript
// 使用别名
await Weline.load('jq');  // 加载jQuery
await Weline.load('$');   // 加载jQuery
```

### 4. 错误处理

```javascript
try {
    await Weline.load('vue');
    // 使用 Vue
} catch (error) {
    console.error('Vue加载失败:', error);
    // 降级处理
}
```

## 完整示例

### 示例1：在页面中声明并使用Vue（推荐）

```php
<!-- 在模板中 -->
<script>
    // 配置模块（可选，如果已在 modules.json 中配置）
    window.WelineConfig = window.WelineConfig || {};
    window.WelineConfig.modulesConfig = {
        modules: {
            vue: {
                paths: ['Weline_Demo::lib/vue/vue2.6.11.js'],
                globalVar: 'Vue'
            }
        }
    };
</script>
<js>Weline_Theme::theme/frontend/assets/js/theme.js</js>

<!-- 页面内容 -->
<div id="app">
    {{ message }}
</div>

<!-- 在页面底部声明模块 -->
<script>
    // 声明Vue，按需加载（默认）
    Weline.declare('vue');
    
    // 或者立即加载
    // Weline.declare('vue', true);
</script>

<script>
    // 当第一次使用 Vue 时，会自动加载
    // 如果需要在加载完成后执行，可以使用 Weline.use()
    (async function() {
        try {
            // 确保Vue已加载
            const Vue = await Weline.use('vue');
            
            // 使用Vue
            new Vue({
                el: '#app',
                data: {
                    message: 'Hello Vue!'
                }
            });
        } catch (error) {
            console.error('Vue加载失败:', error);
        }
    })();
</script>
```

### 示例1b：直接使用（更简洁）

<!-- 在页面底部声明 -->
<script>
