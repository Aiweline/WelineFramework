## 🔄 工作流程

### 登录保护流程
```
用户访问 /twofa (未登录)
    ↓
CheckUserLogin Observer 触发
    ↓
检查 FrontendUserSession::isLogin()
    ↓
未登录 → 保存当前URL → 重定向到登录页
    ↓
用户输入账号密码 → 登录成功
    ↓
从Session读取来源URL → 自动返回 /twofa
    ↓
正常使用TwoFactorAuth功能
```

### 注册登录流程
```
访问注册页 → 填写信息 → 实时验证
    ↓
提交表单 → 后端验证
    ↓
创建用户 → 自动登录
    ↓
跳转到个人中心
```

```sql
CREATE TABLE `frontend_user` (
  `user_id` INT AUTO_INCREMENT PRIMARY KEY,    -- 用户ID
  `username` VARCHAR(60) NOT NULL,             -- 用户名
  `password` VARCHAR(255) NOT NULL,            -- 密码（加密）
  `avatar` VARCHAR(255),                       -- 头像URL
  `login_ip` VARCHAR(16),                      -- 最后登录IP

  `attempt_times` INT DEFAULT 0,               -- 登录尝试次数
  `attempt_ip` VARCHAR(16)                     -- 尝试登录IP
);
```

## 🚀 使用示例

### 1. 在控制器中检查登录
```php
use Weline\Frontend\Session\FrontendUserSession;

class MyController extends FrontendController
{
    private FrontendUserSession $session;
    
    public function getIndex()
    {
        if (!$this->session->isLogin()) {
            $currentUrl = $this->request->getUrlBuilder()->getCurrentUrl();
            $this->redirect('/frontend/account/login?referer=' . urlencode($currentUrl));
            return;
        }
        
        $user = $this->session->getLoginUser();
        // 已登录，继续业务逻辑
    }
}
```

### 2. 在视图中显示用户信息
```php
<?php if ($this->session->isLogin()): ?>
    <?php $user = $this->session->getLoginUser(); ?>
    <div class="user-info">
        <span>欢迎, <?php echo htmlspecialchars($user->getUsername()); ?></span>
        <a href="/frontend/account">个人中心</a>
        <a href="/frontend/account/logout">退出</a>
    </div>
<?php else: ?>
    <a href="/frontend/account/login">登录</a>
    <a href="/frontend/account/register">注册</a>
<?php endif; ?>
```

### 3. AJAX登录
```javascript
$.ajax({
    url: '/frontend/account/login',
    method: 'POST',
    data: {
        username: 'myuser',
        password: 'mypass',
        remember: 1
    },
    success: function(response) {
        if (response.success) {
            window.location.href = response.redirect;
        }
    }
});
```
