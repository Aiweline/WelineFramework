# Weline Queue 模块 - 事件文档

## 概述

本文档详细说明了 Weline Queue 模块提供的 `Weline_Queue::stop` 事件及其使用方法。该事件在停止队列时触发。

## 事件列表

### 1. Weline_Queue::stop - 队列停止事件

#### 基本信息

- **事件名称**：`Weline_Queue::stop`
- **事件类型**：队列操作事件
- **触发时机**：在停止队列时
- **触发位置**：`app/code/Weline/Queue/Controller/Backend/Queue.php` 第 484 行
- **配置文件**：`app/code/Weline/Queue/etc/event.xml`

#### 功能说明

`Weline_Queue::stop` 事件在停止队列时触发，允许其他模块监听队列停止操作。

该事件主要用于：
- 队列停止日志记录
- 停止前验证
- 停止后处理
- 停止通知

#### 触发时机

```php
// app/code/Weline/Queue/Controller/Backend/Queue.php
$this->getEventManager()->dispatch('Weline_Queue::stop', $data);
```

#### 使用场景

- **日志记录**：记录队列停止的详细信息
- **停止验证**：在停止前验证是否可以停止
- **后处理**：停止后执行相关处理
- **通知发送**：发送停止通知

#### 使用方法

##### 基本用法

在模块的 `etc/event.xml` 文件中注册观察者：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="Weline_Queue::stop">
        <observer name="Your_Module::queue_stop"
                  instance="Your\Module\Observer\QueueStopObserver"
                  disabled="false"
                  shared="true"
                  sort="100"/>
    </event>
</config>
```

#### 事件数据

`Weline_Queue::stop` 事件传递的数据根据具体实现而定。
