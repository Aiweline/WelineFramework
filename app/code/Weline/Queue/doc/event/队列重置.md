# Weline Queue 模块 - 事件文档

## 概述

本文档详细说明了 Weline Queue 模块提供的 `Weline_Queue::reset` 事件及其使用方法。该事件在重置队列时触发。

## 事件列表

### 1. Weline_Queue::reset - 队列重置事件

#### 基本信息

- **事件名称**：`Weline_Queue::reset`
- **事件类型**：队列操作事件
- **触发时机**：在重置队列时
- **触发位置**：`app/code/Weline/Queue/Controller/Backend/Queue.php` 第 447 行
- **配置文件**：`app/code/Weline/Queue/etc/event.xml`

#### 功能说明

`Weline_Queue::reset` 事件在重置队列时触发，允许其他模块监听队列重置操作。

该事件主要用于：
- 队列重置日志记录
- 重置前验证
- 重置后处理
- 重置通知

#### 触发时机

```php
// app/code/Weline/Queue/Controller/Backend/Queue.php
$this->getEventManager()->dispatch('Weline_Queue::reset', $data);
```

#### 使用场景

- **日志记录**：记录队列重置的详细信息
- **重置验证**：在重置前验证是否可以重置
- **后处理**：重置后执行相关处理
- **通知发送**：发送重置通知

#### 使用方法

##### 基本用法

在模块的 `etc/event.xml` 文件中注册观察者：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="Weline_Queue::reset">
        <observer name="Your_Module::queue_reset"
                  instance="Your\Module\Observer\QueueResetObserver"
                  disabled="false"
                  shared="true"
                  sort="100"/>
    </event>
</config>
```

#### 事件数据

`Weline_Queue::reset` 事件传递的数据根据具体实现而定。
