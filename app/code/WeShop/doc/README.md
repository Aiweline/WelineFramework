## 子模块文档

WeShop 各业务模块继续在各自模块内维护 `doc/` 目录，例如：

- `app/code/WeShop/Address/doc/`
- `app/code/WeShop/Analytics/doc/`
- `app/code/WeShop/B2B/doc/`
- `app/code/WeShop/Cart/doc/`
- `app/code/WeShop/Checkout/doc/`
- `app/code/WeShop/Filters/doc/`
- `app/code/WeShop/Order/doc/`
- `app/code/WeShop/Payment/doc/`
- `app/code/WeShop/Product/doc/`

其余子模块按同一约定维护，不再与当前根 `doc/` 混放。
