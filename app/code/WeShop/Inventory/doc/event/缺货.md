# 缺货事件 (out_of_stock)

## 事件名称
`WeShop_Inventory::out_of_stock`

## 触发时机
产品库存归零时触发。

## 事件参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| product_id | int | 产品ID |
| source_id | int | 库存源ID |

## 使用场景

- 发送缺货通知
- 更新产品状态为缺货
- 记录缺货日志
- 触发补货流程

## 注册观察者

在模块的 `etc/event.xml` 中注册：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="WeShop_Inventory::out_of_stock">
        <observer name="Your_Module::out_of_stock"
                  instance="Your\Module\Observer\OutOfStockObserver"
                  disabled="false"
                  shared="true"
                  sort="0"/>
    </event>
</config>
```
