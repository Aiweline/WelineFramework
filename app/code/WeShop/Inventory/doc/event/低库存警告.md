# 低库存警告事件 (stock_low)

## 事件名称
`WeShop_Inventory::stock_low`

## 触发时机
库存低于阈值时触发。

## 事件参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| product_id | int | 产品ID |
| source_id | int | 库存源ID |
| current_quantity | int | 当前库存数量 |
| threshold | int | 低库存阈值 |

## 使用场景

- 发送低库存通知
- 记录低库存日志
- 触发自动补货流程
- 更新产品状态

namespace YourModule\Observer;

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

class StockLowObserver implements ObserverInterface
{
    public function execute(Event &$event): void
    {
        $data = $event->getData();
        $productId = $data['product_id'] ?? 0;

        $threshold = $data['threshold'] ?? 0;
        
        // 发送低库存通知
        $this->sendLowStockNotification($productId, $sourceId, $currentQuantity, $threshold);
        
        // 触发自动补货流程

}

## 注册观察者

在模块的 `etc/event.xml` 中注册：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="WeShop_Inventory::stock_low">
        <observer name="Your_Module::stock_low"
                  instance="Your\Module\Observer\StockLowObserver"
                  disabled="false"
                  shared="true"
                  sort="0"/>
    </event>
</config>
```
