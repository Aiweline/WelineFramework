# 库存变更事件 (stock_change)

## 事件名称
`WeShop_Inventory::stock_change`

## 触发时机
库存数量变更时触发（增加/减少）。

## 事件参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| product_id | int | 产品ID |
| source_id | int | 库存源ID |
| old_quantity | int | 变更前数量 |
| new_quantity | int | 变更后数量 |
| change_quantity | int | 变更数量（正数为增加，负数为减少） |

## 使用场景

- 记录库存变更日志
- 更新库存统计信息
- 触发低库存警告
- 同步库存信息到其他系统

## 观察者示例

use Weline\Framework\Event\Event;
use Weline\Framework\Event\ObserverInterface;

        $data = $event->getData();
        $productId = $data['product_id'] ?? 0;

        $oldQuantity = $data['old_quantity'] ?? 0;
        $newQuantity = $data['new_quantity'] ?? 0;
        $changeQuantity = $data['change_quantity'] ?? 0;
        
        // 记录库存变更日志
        $this->logStockChange($productId, $sourceId, $oldQuantity, $newQuantity, $changeQuantity);
        
        // 检查是否需要触发低库存警告
        if ($newQuantity < $this->getLowStockThreshold($productId)) {
            $this->triggerLowStockWarning($productId, $newQuantity);
        }
    }

    {

    }
    
    private function getLowStockThreshold(int $productId): int
    {
        // 获取低库存阈值
        return 10;
    }

## 注册观察者

在模块的 `etc/event.xml` 中注册：

```xml
<?xml version="1.0"?>
<config xmlns:xs="http://www.w3.org/2001/XMLSchema-instance"
        xs:noNamespaceSchemaLocation="urn:Weline_Framework::Event/etc/xsd/event.xsd"
        xmlns="urn:Weline_Framework::Event/etc/xsd/event.xsd">
    <event name="WeShop_Inventory::stock_change">
        <observer name="Your_Module::stock_change"
                  instance="Your\Module\Observer\StockChangeObserver"
                  disabled="false"
                  shared="true"
                  sort="0"/>
    </event>
</config>
```
