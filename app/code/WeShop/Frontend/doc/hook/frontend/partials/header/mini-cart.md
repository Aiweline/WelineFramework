# WeShop_Frontend::frontend::partials::header::mini-cart - Header 迷你购物车

## 概述
用于在站点头部迷你购物车入口区域插入扩展内容。

## Hook 信息
- Hook 名称：`WeShop_Frontend::frontend::partials::header::mini-cart`
- 显示名称：Header 迷你购物车
- 定义模块：WeShop_Frontend
- 区域：frontend
- 类型：partials
- 组件：header
- 位置：mini-cart

## 触发时机
站点头部渲染迷你购物车入口时触发。

## 使用方法
在 `view/hooks/WeShop_Frontend/frontend/partials/header/mini-cart.phtml` 添加实现。

## 典型场景
- 自定义入口样式与图标
- 显示购物车数量徽标

- `app/code/WeShop/Frontend/hook.php`
