# WeShop_Cart::frontend::partials::mini-cart::footer-after - 迷你购物车底部之后

## 概述
用于在迷你购物车底部渲染后插入扩展内容。

## Hook 信息
- Hook 名称：`WeShop_Cart::frontend::partials::mini-cart::footer-after`
- 显示名称：迷你购物车底部之后
- 定义模块：WeShop_Cart
- 区域：frontend
- 类型：partials
- 组件：mini-cart
- 位置：footer-after

## 触发时机
迷你购物车底部区域输出后触发。

## 使用方法
在 `view/hooks/WeShop_Cart/frontend/partials/mini-cart/footer-after.phtml` 添加实现。

## 典型场景
- 插入额外营销信息
- 添加帮助或客服入口
