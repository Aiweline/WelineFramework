## 概述
用于在迷你购物车头部渲染前插入扩展内容。

## Hook 信息
- Hook 名称：`WeShop_Cart::frontend::partials::mini-cart::header-before`
- 显示名称：迷你购物车头部之前
- 定义模块：WeShop_Cart
- 区域：frontend
- 类型：partials
- 组件：mini-cart
- 位置：header-before

## 触发时机
迷你购物车头部区域输出前触发。

## 使用方法
在 `view/hooks/WeShop_Cart/frontend/partials/mini-cart/header-before.phtml` 添加实现。

- 插入促销提示或活动入口
