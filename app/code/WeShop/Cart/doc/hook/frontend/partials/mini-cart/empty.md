# WeShop_Cart::frontend::partials::mini-cart::empty - 迷你购物车空状态

## 概述
用于扩展迷你购物车为空时的输出内容。

## Hook 信息
- Hook 名称：`WeShop_Cart::frontend::partials::mini-cart::empty`
- 显示名称：迷你购物车空状态
- 定义模块：WeShop_Cart
- 区域：frontend
- 类型：partials
- 组件：mini-cart
- 位置：empty

## 触发时机
迷你购物车为空时渲染空状态区域触发。

## 使用方法
在 `view/hooks/WeShop_Cart/frontend/partials/mini-cart/empty.phtml` 添加实现。

## 典型场景
- 提示用户去购物
- 插入推荐商品入口

## 相关文件
- `app/code/WeShop/Cart/hook.php`
- `app/code/WeShop/Cart/view/hooks/WeShop_Cart/frontend/partials/mini-cart/empty.phtml`
