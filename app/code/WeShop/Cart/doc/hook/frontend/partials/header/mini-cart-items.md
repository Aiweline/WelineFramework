## 概述

## Hook 信息
- Hook 名称：`WeShop_Cart::frontend::partials::header::mini-cart-items`
- 显示名称：Header 迷你购物车商品列表
- 定义模块：WeShop_Cart
- 区域：frontend
- 类型：partials
- 组件：header
- 位置：mini-cart-items

## 触发时机

## 使用方法
在 `view/hooks/WeShop_Cart/frontend/partials/header/mini-cart-items.phtml` 添加实现。

## 典型场景
- 自定义列表样式
- 插入列表顶部/底部提示
