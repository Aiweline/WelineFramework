# WeShop_Cart::frontend::partials::cart::sidebar - 购物车侧边栏扩展

## 概述
用于在购物车页面侧边栏插入扩展内容。

## Hook 信息
- Hook 名称：`WeShop_Cart::frontend::partials::cart::sidebar`
- 显示名称：购物车侧边栏扩展
- 定义模块：WeShop_Cart
- 区域：frontend
- 类型：partials
- 组件：cart
- 位置：sidebar

## 触发时机
购物车页面侧边栏渲染时触发。

## 使用方法
在 `view/hooks/WeShop_Cart/frontend/partials/cart/sidebar.phtml` 添加实现。

## 典型场景
- 展示促销信息
- 推荐商品展示
- 显示购物指南或帮助信息

## 相关文件
- `app/code/WeShop/Cart/hook.php`
- `app/code/WeShop/Cart/view/hooks/WeShop_Cart/frontend/partials/cart/sidebar.phtml`
