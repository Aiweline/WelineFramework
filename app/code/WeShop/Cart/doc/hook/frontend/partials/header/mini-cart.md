# WeShop_Cart::frontend::partials::header::mini-cart - WeShop Header 迷你购物车

## 概述
用于扩展站点头部迷你购物车入口区域的输出内容。

## Hook 信息
- Hook 名称：`WeShop_Cart::frontend::partials::header::mini-cart`
- 显示名称：WeShop Header 迷你购物车
- 定义模块：WeShop_Cart
- 区域：frontend
- 类型：partials
- 组件：header
- 位置：mini-cart

## 触发时机
站点头部渲染迷你购物车入口时触发。

## 使用方法
在 `view/hooks/WeShop_Cart/frontend/partials/header/mini-cart.phtml` 添加实现。

## 典型场景
- 自定义入口样式
- 插入购物车数量提示

## 相关文件
- `app/code/WeShop/Cart/hook.php`
- `app/code/WeShop/Cart/view/hooks/WeShop_Cart/frontend/partials/header/mini-cart.phtml`
