# WeShop_Cart::frontend::partials::cart::continue-shopping - 继续购物按钮

## 概述
用于扩展购物车页面“继续购物”按钮区域的输出内容。

## Hook 信息
- Hook 名称：`WeShop_Cart::frontend::partials::cart::continue-shopping`
- 显示名称：继续购物按钮
- 定义模块：WeShop_Cart
- 区域：frontend
- 类型：partials
- 组件：cart
- 位置：continue-shopping

## 触发时机
购物车页面渲染继续购物按钮时触发。

## 使用方法
在 `view/hooks/WeShop_Cart/frontend/partials/cart/continue-shopping.phtml` 添加实现。

## 典型场景
- 替换或增强按钮样式
- 添加返回上级入口
