# WeShop_Cart::frontend::partials::cart::empty - 空购物车状态

## 概述
用于扩展空购物车状态的输出内容。

## Hook 信息
- Hook 名称：`WeShop_Cart::frontend::partials::cart::empty`
- 显示名称：空购物车状态
- 定义模块：WeShop_Cart
- 区域：frontend
- 类型：partials
- 组件：cart
- 位置：empty

## 触发时机
购物车为空时渲染空状态区域触发。

## 使用方法
在 `view/hooks/WeShop_Cart/frontend/partials/cart/empty.phtml` 添加实现。

## 典型场景
- 插入推荐商品或活动入口
- 添加返回购物引导

## 相关文件
- `app/code/WeShop/Cart/hook.php`
- `app/code/WeShop/Cart/view/hooks/WeShop_Cart/frontend/partials/cart/empty.phtml`
