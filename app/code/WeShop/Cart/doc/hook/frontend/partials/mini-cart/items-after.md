# WeShop_Cart::frontend::partials::mini-cart::items-after - 迷你购物车商品列表后

## 概述
用于在迷你购物车商品列表渲染后插入扩展内容。

## Hook 信息
- Hook 名称：`WeShop_Cart::frontend::partials::mini-cart::items-after`
- 显示名称：迷你购物车商品列表后
- 定义模块：WeShop_Cart
- 区域：frontend
- 类型：partials
- 组件：mini-cart
- 位置：items-after

## 触发时机
迷你购物车商品列表输出后触发。

## 使用方法
在 `view/hooks/WeShop_Cart/frontend/partials/mini-cart/items-after.phtml` 添加实现。

## 典型场景
- 推荐商品展示
- 显示关联商品
- 添加快捷操作入口

## 相关文件
- `app/code/WeShop/Cart/hook.php`
- `app/code/WeShop/Cart/view/hooks/WeShop_Cart/frontend/partials/mini-cart/items-after.phtml`
