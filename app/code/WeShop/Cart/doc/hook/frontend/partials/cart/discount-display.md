# WeShop_Cart::frontend::partials::cart::discount-display - 折扣显示

## 概述
用于扩展购物车页面折扣显示区域的输出内容。

## Hook 信息
- Hook 名称：`WeShop_Cart::frontend::partials::cart::discount-display`
- 显示名称：折扣显示
- 定义模块：WeShop_Cart
- 区域：frontend
- 类型：partials
- 组件：cart
- 位置：discount-display

## 触发时机
购物车页面渲染折扣信息时触发。

## 使用方法
在 `view/hooks/WeShop_Cart/frontend/partials/cart/discount-display.phtml` 添加实现。

## 典型场景
- 显示折扣规则说明
- 插入额外优惠提示

## 相关文件
- `app/code/WeShop/Cart/hook.php`
- `app/code/WeShop/Cart/view/hooks/WeShop_Cart/frontend/partials/cart/discount-display.phtml`
