# WeShop Product 价格计算系统使用指南（WASM版本）

## 概述

WeShop Product 模块提供了一个基于 WebAssembly (WASM) 的高性能价格计算系统，允许其他模块（如配送费、税费、折扣等）通过中间件参与价格计算，实现模块化的价格管理。

### 核心特性

- **WASM 高性能计算**: 使用 WebAssembly 进行价格计算，性能更高
- **中间件架构**: 采用中间件模式，支持模块化扩展
- **标准中间件名称**: 由 Product 模块定义标准中间件名称，确保一致性
- **自动降级**: WASM 加载失败时自动降级到 JavaScript 计算

## 核心概念

### 1. PriceContext（价格上下文）

价格上下文包含：
- **productInfo**: 产品信息（价格、重量、体积等）
- **quantity**: 数量
- **basePrice**: 基础价格
- **components**: 价格组成部分（基础、配送费、税费、折扣、其他）
- **metadata**: 元数据（配送位置、客户ID、优惠券等）

### 2. PriceCalculator（价格计算器）

管理所有价格计算中间件，按优先级顺序执行。使用 WASM 进行高性能计算。

### 3. 中间件命名规范

**重要**: 中间件名称必须使用 Product 模块定义的标准名称，不能随意命名。

标准中间件名称（通过 `WeShop.Product.MIDDLEWARE_NAMES` 获取）：
- `'shipping'`: 配送费中间件
- `'tax'`: 税费中间件
- `'discount'`: 折扣中间件
- `'other'`: 其他费用中间件

### 4. 价格计算流程

```
基础价格 × 数量
    ↓
[配送费计算器] → 添加配送费
    ↓
[税费计算器] → 添加税费
    ↓
[折扣计算器] → 减去折扣
    ↓
[其他费用计算器] → 添加其他费用
    ↓
最终总价
```

## 使用方法

### 1. 基础使用：计算产品价格

```javascript
// 产品信息
const productInfo = {
    product_id: 123,
    price: 100.00,
    weight: 1.5, // 公斤
    volume: 0.01, // 立方米
    name: '示例产品'
};

// 计算价格
const context = await WeShop.Product.calculatePrice(productInfo, {
    quantity: 2,
    currency: 'CNY',
    metadata: {
        location: {
            countryCode: 'CN',
            province: '北京市',
            city: '北京市',
            district: '朝阳区'
        }
    }
});

// 获取价格明细
const breakdown = context.getPriceBreakdown();
console.log('基础价格:', breakdown.base);
console.log('配送费:', breakdown.shipping);
console.log('税费:', breakdown.tax);
console.log('折扣:', breakdown.discount);
console.log('总价:', breakdown.total);
```

### 2. 注册价格计算中间件

**重要**: 中间件名称必须使用 Product 模块定义的标准名称。

#### 方式一：通过 Weline.Theme.PriceCalculator（推荐）

```javascript
// 获取标准中间件名称常量
const MIDDLEWARE_NAMES = WeShop.Product.MIDDLEWARE_NAMES;
// 或
const MIDDLEWARE_NAMES = Weline.Theme.PriceCalculator.getMiddlewareNames();

// 注册配送费中间件
Weline.Theme.PriceCalculator.register(
    MIDDLEWARE_NAMES.SHIPPING, // 使用标准名称
    async function(context) {
        // 计算配送费
        const shippingFee = await calculateShippingFee(context);
        
        return {
            amount: shippingFee,
            description: '配送费'
            // 注意：不再需要返回 type，因为已在注册时指定
        };
    },
    {
        name: 'MyShippingCalculator',
        priority: 10 // 优先级（数字越小越先执行）
    }
);
```

#### 方式二：直接使用 WeShop.Product.PriceCalculator

```javascript
// 获取标准中间件名称常量
const MIDDLEWARE_NAMES = WeShop.Product.MIDDLEWARE_NAMES;

// 注册税费中间件
WeShop.Product.PriceCalculator.register(
    MIDDLEWARE_NAMES.TAX, // 使用标准名称
    async function(context) {
        // 计算税费
        const taxRate = 0.13; // 13%税率
        const tax = context.components.base * taxRate;
        
        return {
            amount: tax,
            description: '税费 (13%)'
        };
    },
    {
        name: 'TaxCalculator',
        priority: 20
    }
);
```

#### 错误示例（不要这样做）

```javascript
// ❌ 错误：使用自定义中间件名称
Weline.Theme.PriceCalculator.register('my_custom_shipping', ...);

// ❌ 错误：拼写错误
Weline.Theme.PriceCalculator.register('shiping', ...);

// ✅ 正确：使用标准名称
Weline.Theme.PriceCalculator.register(MIDDLEWARE_NAMES.SHIPPING, ...);
```

### 3. 更新价格显示

```javascript
// 计算价格
const context = await WeShop.Product.calculatePrice(productInfo, options);

// 更新页面显示
WeShop.Product.updatePrice(context, {

        shipping: '.product-price-shipping',

    }

```

### 4. 监听价格更新事件

```javascript
document.addEventListener('weline:product:price-updated', function(event) {
    const { context, breakdown } = event.detail;
    console.log('价格已更新:', breakdown);
    
    // 自定义更新逻辑

```

## 自定义价格计算中间件示例

### 示例1：会员折扣中间件

```javascript
const MIDDLEWARE_NAMES = WeShop.Product.MIDDLEWARE_NAMES;

Weline.Theme.PriceCalculator.register(
    MIDDLEWARE_NAMES.DISCOUNT, // 使用标准名称
    async function(context) {
    const customerId = context.metadata.customerId;
    if (!customerId) {
        return null; // 未登录，不计算会员折扣
    }
    
    // 获取会员等级和折扣率
    const memberLevel = await getMemberLevel(customerId);
    const discountRate = memberLevel.discount_rate || 0;
    
    if (discountRate > 0) {
        const discount = context.components.base * discountRate;
        return {
            amount: -discount, // 负数表示折扣
            description: `会员折扣 (${discountRate * 100}%)`
        };
    }
    
    return null;
}, {
    name: 'MemberDiscountCalculator',
    priority: 50 // 在配送费和税费之后计算
});
```

### 示例2：优惠券中间件

```javascript
const MIDDLEWARE_NAMES = WeShop.Product.MIDDLEWARE_NAMES;

Weline.Theme.PriceCalculator.register(
    MIDDLEWARE_NAMES.DISCOUNT, // 使用标准名称
    async function(context) {
    const couponCode = context.metadata.couponCode;
    if (!couponCode) {
        return null;
    }
    
    // 验证优惠券
    const coupon = await validateCoupon(couponCode);
    if (!coupon || !coupon.is_valid) {
        return null;
    }
    
    let discount = 0;
    if (coupon.type === 'fixed') {
        // 固定金额折扣
        discount = coupon.value;
    } else if (coupon.type === 'percentage') {
        // 百分比折扣
        discount = context.components.base * (coupon.value / 100);
    }
    
    return {
        amount: -discount,
        description: `优惠券: ${couponCode}`
    };
}, {
    name: 'CouponCalculator',
    priority: 60
});
```

### 示例3：税费中间件

```javascript
const MIDDLEWARE_NAMES = WeShop.Product.MIDDLEWARE_NAMES;

Weline.Theme.PriceCalculator.register(
    MIDDLEWARE_NAMES.TAX, // 使用标准名称
    async function(context) {
    const location = context.metadata.location;
    if (!location) {
        return null;
    }
    
    // 根据地区获取税率
    const taxRate = await getTaxRate(location);
    if (!taxRate || taxRate <= 0) {
        return null;
    }
    
    const tax = context.components.base * taxRate;
    
    return {
        amount: tax,
        description: `税费 (${taxRate * 100}%)`
    };
}, {
    name: 'TaxCalculator',
    priority: 30 // 在配送费之后，折扣之前
});
```

## 优先级说明

价格计算器按优先级顺序执行，建议的优先级：

- **10-19**: 配送费（需要先计算，因为可能影响其他费用）
- **20-29**: 税费（基于基础价格+配送费）
- **30-39**: 其他费用
- **50-59**: 会员折扣（基于总价）
- **60-69**: 优惠券折扣（基于总价）
- **70-79**: 其他折扣

## 中间件返回值格式

中间件函数应返回以下格式的对象：

```javascript
{
    amount: number,        // 调整金额（正数表示增加，负数表示减少）
    description: string,   // 描述（用于显示）
    metadata: object       // 可选：额外元数据
}
```

**注意**: 
- 不再需要返回 `type`，因为中间件类型已在注册时通过中间件名称指定
- 返回 `null` 表示不进行任何调整
- `amount` 为负数时表示折扣（减少价格）

## API参考

### WeShop.Product.calculatePrice(productInfo, options)

计算产品价格。

**参数**:
- `productInfo` (Object): 产品信息
  - `product_id` (number): 产品ID
  - `price` (number): 基础价格
  - `weight` (number, 可选): 重量（公斤）
  - `volume` (number, 可选): 体积（立方米）
- `options` (Object, 可选): 选项
  - `quantity` (number): 数量，默认1
  - `currency` (string): 货币代码，默认'CNY'
  - `metadata` (Object): 元数据
    - `location` (Object): 配送位置
    - `customerId` (number): 客户ID
    - `couponCode` (string): 优惠券代码

**返回**: `Promise<PriceContext>`

### WeShop.Product.updatePrice(context, options)

更新价格显示。

**参数**:
- `context` (PriceContext): 价格上下文
- `options` (Object): 选项
  - `selectors` (Object): DOM选择器
    - `basePrice` (string): 基础价格选择器
    - `shipping` (string): 配送费选择器
    - `tax` (string): 税费选择器
    - `discount` (string): 折扣选择器
    - `total` (string): 总价选择器

### Weline.Theme.PriceCalculator.register(middlewareName, middleware, options)

注册价格计算中间件。

**参数**:
- `middlewareName` (string): 中间件名称（必须使用标准名称）
  - 使用 `WeShop.Product.MIDDLEWARE_NAMES.SHIPPING` 等常量
  - 或使用字符串 `'shipping'`, `'tax'`, `'discount'`, `'other'`
- `middleware` (Function): 中间件函数
- `options` (Object): 选项
  - `name` (string): 中间件实例名称（用于识别和调试）
  - `priority` (number): 优先级，默认100

### Weline.Theme.PriceCalculator.unregister(middlewareName, name)

移除价格计算中间件。

**参数**:
- `middlewareName` (string): 中间件名称
- `name` (string): 中间件实例名称

### Weline.Theme.PriceCalculator.getMiddlewareNames()

获取标准中间件名称常量。

**返回**: `Object` - 包含 `SHIPPING`, `TAX`, `DISCOUNT`, `OTHER` 常量

## WASM 支持

### 自动加载

WASM 模块会在 Product 模块加载时自动尝试加载。如果加载失败，系统会自动降级到 JavaScript 计算，不影响功能。

### 手动加载

```javascript
// 手动加载 WASM 模块
await WeShop.Product.loadWasm();
```

### WASM 编译

WASM 模块源码位于 `app/code/WeShop/Product/wasm/src/`，编译方法请参考 `wasm/README.md`。

## 最佳实践

1. **使用标准中间件名称**: 始终使用 `WeShop.Product.MIDDLEWARE_NAMES` 常量，避免拼写错误
2. **优先级设置**: 根据业务逻辑合理设置优先级，确保计算顺序正确
3. **错误处理**: 在中间件中妥善处理错误，返回 `null` 而不是抛出异常
4. **性能优化**: 使用 IndexedDB 缓存价格规则，避免重复请求
5. **异步处理**: 中间件可以是异步函数，支持从接口获取数据
6. **元数据传递**: 通过 `context.metadata` 传递额外信息，避免全局变量
7. **WASM 降级**: 确保中间件在 WASM 不可用时仍能正常工作
