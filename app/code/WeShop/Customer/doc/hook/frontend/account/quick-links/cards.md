# WeShop_Customer::frontend::account::quick-links::cards

Use this hook to inject additional quick-link cards inside the storefront customer account quick-actions grid.

- Host template: `app/design/WeShop/default/frontend/pages/customer/index.phtml`
- Placement: inside the `Jump Back In` card grid, after the built-in quick links.
- Expected output: one or more card-sized anchors or panels that visually match the quick-action tiles.
