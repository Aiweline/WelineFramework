# WeShop_Customer::frontend::account::recommendations::cards

Use this hook to inject additional recommendation-focused content into the storefront customer account recommendation section.

- Host template: `app/design/WeShop/default/frontend/pages/customer/index.phtml`
- Placement: inside the `Recommended For You` card before the built-in recommendation list or empty state.
- Expected output: cards, product strips, or recommendation summaries that fit the account-center merchandising area.
