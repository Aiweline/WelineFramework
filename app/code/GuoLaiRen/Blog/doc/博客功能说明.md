# GuoLaiRen_Blog 模块 - 博客功能说明

## 功能概述

`GuoLaiRen_Blog` 是一个完整的博客管理系统，提供独立的博客文章管理功能。

### 核心功能

- **独立的博客文章模型**：使用 `GuoLaiRen\Blog\Model\Post` 管理博客文章
- **后台管理界面**：完整的博客文章 CRUD 操作（创建、读取、更新、删除）
- **前台展示**：博客列表页和文章详情页
- **SEO 友好**：支持 URL 别名（slug）和文章摘要

## 数据模型

### 博客文章表（guolairen_blog_post）

| 字段 | 类型 | 说明 |
|------|------|------|
| post_id | INTEGER | 主键，自增 |
| title | VARCHAR(255) | 文章标题 |
| slug | VARCHAR(255) | URL别名（唯一） |
| summary | TEXT | 文章摘要 |
| content | TEXT | 文章内容（HTML） |
| cover_image | VARCHAR(500) | 封面图片URL |
| status | SMALLINT | 状态：0=草稿，1=已发布 |
| published_at | DATETIME | 发布时间 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

## 路由说明

### 后台路由

- **博客列表**：`/blog/backend/post/index`
- **新建博客**：`/blog/backend/post/create`
- **编辑博客**：`/blog/backend/post/edit?id={id}`
- **删除博客**：`/blog/backend/post/delete?id={id}`

### 前台路由

- **博客列表**：`/blog/frontend/index/index?page=1`
- **文章详情**：`/blog/frontend/post/view?slug={slug}`

## 使用方式

### 1. 安装模块

运行系统升级命令创建数据表：

```bash
php bin/w setup:upgrade --module GuoLaiRen_Blog
```

### 2. 后台管理

1. 登录后台，在左侧菜单找到 **"内容管理" → "博客管理"**
2. 点击 **"新建博客"** 创建文章
3. 填写：
   - **文章标题**：必填
   - **URL别名**：必填，用于生成文章URL（只能包含小写字母、数字和连字符）
   - **文章摘要**：可选，用于列表页显示
   - **文章内容**：支持HTML格式
   - **封面图片URL**：可选
   - **状态**：草稿或已发布
   - **发布时间**：可选，留空则使用当前时间
4. 点击 **"创建博客"** 或 **"更新博客"** 保存

### 3. 前台访问

- 访问 `/blog/frontend/index/index` 查看博客列表
- 点击文章标题或"阅读更多"按钮查看详情
- 详情页URL格式：`/blog/frontend/post/view?slug={文章slug}`

## 功能特性

### 自动生成URL别名

在编辑表单中，如果URL别名为空，系统会根据文章标题自动生成：
- 转换为小写
- 移除特殊字符
- 空格替换为连字符
- 多个连字符合并为一个

### 状态管理

- **草稿（0）**：文章未发布，前台不可见
- **已发布（1）**：文章已发布，前台可见

### 搜索功能

后台列表页支持按标题或URL别名搜索文章。

### 分页功能

前台列表页支持分页浏览，每页显示10篇文章。

## 注意事项

1. **URL别名唯一性**：系统会检查URL别名的唯一性，重复的别名无法保存
2. **发布时间**：如果文章状态为"已发布"但发布时间为空，系统会自动设置为当前时间
3. **路由注册**：添加新控制器后，需要运行 `php bin/w setup:upgrade --route` 更新路由配置
