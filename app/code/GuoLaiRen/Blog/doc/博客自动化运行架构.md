# GuoLaiRen_Blog 博客自动化运行架构

## 系统概述

博客自动化发文系统通过定时任务实现 **趋势监控 → 智能筛选 → AI 创作 → 自动发布** 的全流程自动化。

## 核心配置说明

| 配置项 | 配置键 | 说明 | 默认值 |
|-------|--------|------|--------|
| API 类型 | `guolairen_blog_trends_api_type` | none / serpapi / official | none |
| SerpApi Key | `guolairen_blog_trends_serpapi_key` | SerpApi 密钥 | - |
| 增长基准 | `guolairen_blog_trends_growth_comparison` | day / week / both | both |
| 增长阈值 | `guolairen_blog_trends_growth_threshold` | 增长率百分比 | 0 |
| 趋势地区 | `guolairen_blog_trends_region` | US / CN 等 | US |
| 默认语言 | `guolairen_blog_trends_default_language` | AI 生成语言 | en_US |
| 发布为草稿 | `guolairen_blog_trends_publish_as_draft` | 1=草稿, 0=直接发布 | 1 |

## 定时任务配置

| 任务名 | 执行名 | Cron 表达式 | 说明 |
|-------|--------|-------------|------|
| Blog 趋势同步 | `blog_trend_sync` | `0 6 * * *` | 每日 06:00 拉取趋势 |
| Blog AI 自动发文 | `blog_ai_publish` | `*/5 * * * *` | 每 5 分钟执行，按当日配额补量（适配单次仅 1 篇等场景） |

## 工作流程总结

1. **趋势监控** (每日 06:00)
   - 从 Google Trends 拉取关键词趋势数据
   - 计算日环比、周环比增长率
   - 超过阈值的词写入增长词记录表
   - **去重机制**：同一 `profile_id + keyword + trend_date` 不重复写入

2. **智能发文** (每 5 分钟执行，直至当日配额满)
   - 按站点+画像读取发文配额
   - 检查当日已发文数量，计算剩余配额
   - 从增长词记录中取未使用的词
   - 调用 AI 生成标题、摘要、正文
   - 创建博客文章（支持草稿/直接发布）
   - 标记增长词为已使用

3. **多站点支持**
   - 每个站点可配置不同的画像和每日发文数
   - 文章自动归入站点对应的默认分类

4. **增长词日志管理** ★
   - 后台可查看所有增长词记录
   - 支持按画像、使用状态筛选
   - 支持关键词搜索
   - 可手动标记为已使用/未使用
   - 可删除单条日志
   - 可批量清理旧日志（默认30天前已使用的）

5. **前台分类页面** ★
   - 访问 `/blog/category/{slug}` 查看分类下的文章列表
   - 支持 PageBuilder 渲染，继承首页样式配置
   - 提供分类列表、最近文章侧边栏
