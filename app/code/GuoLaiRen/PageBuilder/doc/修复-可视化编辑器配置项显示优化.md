# 可视化编辑器配置项显示优化

## 问题描述

在可视化编辑器的样式配置部分，配置项缺少以下信息：
1. 响应式图标徽章（📱📱💻）
2. 字段说明和描述信息
3. 设备兼容性提示

这导致用户无法直观地了解哪些配置项支持响应式，以及如何正确配置这些项。

## 解决方案

### 修改的文件

`app/code/GuoLaiRen/PageBuilder/view/templates/Backend/Page/form.phtml`

### 具体修改

#### 1. `renderVisualConfig()` 函数（可视化配置界面）

**位置**：约第 2960 行

**修改内容**：

#### A. 分组级别的修改
- 在分组标题后添加响应式图标徽章（当 `group.tag` 存在时）
- 在分组内容区域顶部添加可折叠的帮助信息（当 `group.help_title` 或 `group.help_content` 存在时）

**添加的代码（分组级别）**：
```javascript
// 分组标题上的响应式徽章
html += `<span><i class="mdi ${icon}"></i> ${group.label || groupKey}`;

if (group.tag) {
    let tagIcons = '';
    if (group.tag.includes('M')) tagIcons += '<i class="mdi mdi-cellphone" style="font-size: 0.85rem;"></i> ';
    if (group.tag.includes('T')) tagIcons += '<i class="mdi mdi-tablet" style="font-size: 0.85rem;"></i> ';
    if (group.tag.includes('D')) tagIcons += '<i class="mdi mdi-monitor" style="font-size: 0.85rem;"></i> ';
    
    html += ` <span class="badge bg-info ms-2" style="font-size: 0.65rem; vertical-align: middle;">
                ${tagIcons}
             </span>`;
}

html += `</span>`;

// 分组的帮助信息（可折叠）
if (group.help_title || group.help_content) {
    html += `
        <div class="alert alert-info alert-dismissible mb-3" role="alert" style="position: relative;">
            <div class="d-flex justify-content-between align-items-center" style="cursor: pointer;" onclick="toggleGroupHelp('help_visual_${fileKey}_${groupKey}')">
                <h6 class="alert-heading mb-0">
                    <i class="mdi mdi-information"></i> ${escapeHtml(group.help_title || '说明')}
                </h6>
                <i class="mdi mdi-chevron-down" id="help_icon_visual_${fileKey}_${groupKey}"></i>
            </div>
            <div id="help_visual_${fileKey}_${groupKey}" style="display: none; margin-top: 10px; white-space: pre-line; line-height: 1.6;">
                ${escapeHtml(group.help_content || '')}
            </div>
        </div>`;
}
```

#### B. 配置项级别的修改
- 在配置项标签后添加响应式图标徽章（当 `config.responsive` 为 true 时）
- 在输入框前添加描述信息（当 `config.description` 存在时）

**添加的代码（配置项级别）**：
```javascript
// 渲染标签和响应式徽章
html += `<label for="${fieldId}" class="form-label">${escapeHtml(config.label)}`;

// 添加响应式标记图标
if (config.responsive) {
    html += ` <span class="badge bg-info ms-1" 
                    title="支持响应式配置（移动端/平板/PC端）" 
                    style="font-size: 0.75rem; vertical-align: middle;">
                <i class="mdi mdi-cellphone" style="font-size: 0.85rem;"></i>
                <i class="mdi mdi-tablet" style="font-size: 0.85rem;"></i>
                <i class="mdi mdi-monitor" style="font-size: 0.85rem;"></i>
             </span>`;
}

html += `</label>`;

// 添加描述信息
if (config.description) {
    html += `<div class="form-text text-muted mb-2" 
                  style="margin-top: -8px; font-size: 0.85rem;">
                <i class="mdi mdi-information-outline"></i> 
                ${escapeHtml(config.description)}
             </div>`;
}
```

#### 2. `renderStyleConfig()` 函数（样式配置表单）

**位置**：约第 1480-1583 行

**修改内容**：
- 统一图标库：将 Font Awesome 图标替换为 Material Design Icons (mdi)
- 确保响应式徽章和描述信息的显示逻辑与可视化配置一致

**图标替换**：
- `fas fa-mobile-alt` → `mdi mdi-cellphone`
- `fas fa-tablet-alt` → `mdi mdi-tablet`
- `fas fa-desktop` → `mdi mdi-monitor`
- `fas fa-info-circle` → `mdi mdi-information`
- `fas fa-chevron-down` → `mdi mdi-chevron-down`

#### 3. `toggleGroupHelp()` 函数

**位置**：约第 1792 行

**修改内容**：
- 更新图标切换逻辑，使用 Material Design Icons

**修改前**：
```javascript
icon.className = isHidden ? 'fas fa-chevron-up' : 'fas fa-chevron-down';
```

**修改后**：
```javascript
icon.className = isHidden ? 'mdi mdi-chevron-up' : 'mdi mdi-chevron-down';
```

## 功能说明

### 1. 分组级别的响应式徽章

**显示条件**：分组的 `tag` 属性存在（如 `MD`、`MTD`）

**图标显示**：
- `M` → 📱 (`mdi-cellphone`) - 移动端
- `T` → 📱 (`mdi-tablet`) - 平板
- `D` → 💻 (`mdi-monitor`) - PC端

**徽章样式**：
- 背景色：浅蓝色 (`bg-info`)
- 显示位置：分组标题后
- 字体大小：0.65rem

### 2. 分组级别的帮助信息

**显示条件**：分组的 `help_title` 或 `help_content` 属性存在

**功能特性**：
- 可折叠/展开（点击标题区域）
- 默认收起状态
- 使用浅蓝色警告框样式 (`alert-info`)
- 支持多行文本显示（`white-space: pre-line`）

**样式**：
- 标题图标：`mdi-information`
- 折叠图标：`mdi-chevron-down` / `mdi-chevron-up`
- 内容区域：行高 1.6，便于阅读

### 3. 配置项级别的响应式徽章

**显示条件**：配置项的 `responsive` 属性为 `true`

**图标含义**：
- 📱 (`mdi-cellphone`) - 移动端
- 📱 (`mdi-tablet`) - 平板
- 💻 (`mdi-monitor`) - PC端

**徽章样式**：
- 背景色：浅蓝色 (`bg-info`)
- 显示位置：配置项标签后
- 鼠标悬停提示："支持响应式配置（移动端/平板/PC端）"
- 字体大小：0.75rem

### 4. 配置项级别的描述信息

**显示条件**：配置项的 `description` 属性存在

**样式**：
- 文字颜色：灰色 (`text-muted`)
- 字体大小：0.85rem（略小于正常文字）
- 位置：标签下方，输入框上方
- 带有信息图标前缀 (`mdi-information-outline`)

### 5. 配置项定义格式

在模板文件中定义配置项时，使用以下格式：

```php
/**
 * @fields_start
 * 
 * group:size => 尺寸[MD]:📱响应式配置说明:说明内容
 * size.height => 头部高度:responsive:60/80|px[MD]
 * size.max_width => 内容最大宽度:responsive:100/1200|%/px[移动端使用百分比，PC端使用像素]
 * 
 * @fields_end
 */
```

**格式说明**：
- `responsive` - 类型标记为响应式
- `60/80` - 默认值（移动端60/PC端80）
- `|px` - 单位
- `[MD]` - 响应式标记（M=移动端, D=桌面）
- `[描述文字]` - 自定义描述信息

**响应式标记**：
- `[M]` - 仅移动端
- `[MD]` - 移动端和PC端（推荐）
- `[MT]` - 移动端和平板
- `[MTD]` - 完整三端支持

## 后端解析

配置项的解析由 `GuoLaiRen\PageBuilder\Model\Style::parseStyleConfig()` 方法完成。

该方法会：
1. 扫描模板文件中的 `@fields_start` 到 `@fields_end` 区域
2. 解析配置项格式：`key => label:type:default|options[description]`
3. 提取 `description` 和 `responsive` 属性
4. 检测描述中的响应式标记（MTD字母组合）
5. 检测默认值中的斜杠（如 `60/80`）自动标记为响应式

返回的配置数组包含：
```php
[
    'key' => 'size.height',
    'label' => '头部高度',
    'type' => 'responsive',
    'default' => '60/80',
    'unit' => 'px',
    'description' => '支持响应式配置（移动端、PC端）',
    'responsive' => true,
]
```

## 前端渲染

前端的 `renderVisualConfig()` 和 `renderStyleConfig()` 函数会：
1. 检查 `config.responsive` 属性
2. 根据 `config.description` 中的设备关键词显示对应图标
3. 渲染描述信息到界面

## 测试方法

1. 启动后端服务器：
   ```bash
   php bin/w s:sta
   ```

2. 登录后台

3. 进入 PageBuilder > 页面管理 > 编辑页面

4. 点击"可视化配置"或查看"样式配置"区域

5. 验证以下内容：
   - 响应式配置项是否显示图标徽章（📱📱💻）
   - 描述信息是否显示在输入框上方
   - 鼠标悬停在徽章上是否显示提示

## 技术要点

1. **图标库统一**：全部使用 Material Design Icons (mdi)
2. **徽章样式一致**：使用 Bootstrap 的 `badge bg-info` 类
3. **描述信息格式**：使用 `form-text text-muted` 类
4. **响应式判断**：支持自动检测（斜杠分隔符）和显式标记（[MD]）
5. **HTML 转义**：使用 `escapeHtml()` 函数防止 XSS 攻击
