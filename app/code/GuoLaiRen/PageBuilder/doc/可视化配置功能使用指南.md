# 📱 PageBuilder 可视化配置功能使用指南

## ✅ 功能概述

**可视化配置**功能允许您边配置边查看页面效果，提供了直观的实时预览体验。

### 🎯 核心特性

1. **左右分栏布局** - 左侧配置，右侧实时预览
2. **自动保存** - 1秒防抖，修改即时保存
3. **实时刷新** - 保存成功后自动刷新预览
4. **多端预览** - 支持 PC、平板、手机视图切换
5. **分组配置** - 按功能分组（位置、颜色、布局、排版、间距等）
6. **全屏模式** - 提供更大的配置和预览空间
7. **快捷键支持** - ESC 退出全屏

## 🚀 使用步骤

### 1. 打开编辑页面
```
后台 > PageBuilder > 页面管理 > 编辑页面
```

### 2. 选择样式模板
在页面编辑表单中，选择一个样式模板（如 `default`）

### 3. 点击"可视化配置"按钮
点击表单中的 **"🎨 可视化配置"** 按钮（位于样式选择区域）

### 4. 进入可视化配置界面
- 界面自动进入全屏模式
- 左侧显示配置面板
- 右侧显示实时预览

### 5. 修改配置
- 展开任意配置分组（点击分组标题）
- 修改配置项（颜色、尺寸、文本等）
- **系统自动保存**（1秒防抖）
- 保存成功后**自动刷新预览**

### 6. 切换设备视图
点击顶部工具栏的设备按钮：
- 🖥️ **桌面** - 完整宽度
- 📱 **平板** - 768px 宽
- 📱 **手机** - 375px 宽

### 7. 手动刷新预览
如需手动刷新，点击顶部工具栏的 **"🔄 刷新"** 按钮

### 8. 退出可视化配置
- 点击左上角的 **"✖ 退出"** 按钮
- 或按键盘 **ESC** 键

## 🎨 配置技巧

### 颜色配置
- 点击颜色框弹出颜色选择器
- 可以输入十六进制颜色值（如 `#ff6b6b`）
- 支持透明度设置

### 数值配置
- 输入框支持直接输入数字
- 带单位的配置会自动显示单位（如 `px`）
- 可以使用上下键微调数值

### 选择配置
- 下拉选择框提供预设选项
- 直接点击选择即可

### 文本配置
- 直接在输入框中输入文本
- 支持多行文本（如版权信息）

## 💾 保存机制

### 自动保存
- **触发条件**：任何配置项修改后 1 秒
- **保存状态**：
  - 🟡 **未保存** - 刚修改，等待保存
  - 🟠 **保存中...** - 正在发送保存请求
  - 🟢 **已保存** - 保存成功
  - 🔴 **保存失败** - 保存出错

### 保存范围
- 只保存当前页面的样式配置
- 不影响其他页面
- 实时更新数据库

### 预览刷新
- 保存成功后自动刷新预览
- 刷新时显示加载动画
- 刷新完成后立即显示最新效果

## 🔧 技术实现

### 前端
- **框架**：原生 JavaScript
- **UI**：Bootstrap 5
- **图标**：Material Design Icons
- **通信**：Fetch API

### 后端
- **控制器**：`GuoLaiRen\PageBuilder\Controller\Backend\Preview`
- **方法**：
  - `full()` - 完整预览
  - `autoSave()` - 自动保存

### 数据流
```
用户修改配置
    ↓ (1秒防抖)
发送 AJAX 请求 → autoSave()
    ↓
保存到数据库 (style_settings 字段)
    ↓
返回成功响应
    ↓
刷新 iframe → full() 重新渲染
    ↓
显示最新预览
```

## 📊 API 接口

### 1. 获取样式配置组
```
GET /pagebuilder/backend/page/styleConfig?style_code=default
```

**响应示例**：
```json
{
  "success": true,
  "data": {
    "position": {
      "key": "position",
      "label": "位置",
      "icon": "mdi-crosshairs",
      "configs": {
        "position.logo_position": {
          "key": "position.logo_position",
          "label": "Logo位置",
          "type": "select",
          "default": "left",
          "options": ["left", "center", "right"]
        }
      }
    }
  }
}
```

### 2. 自动保存配置
```
POST /pagebuilder/backend/preview/autoSave
Content-Type: application/json

{
  "page_id": 1,
  "style_config": {
    "position.logo_position": "center",
    "color.background_color": "#ffffff",
    "size.height": "80"
  }
}
```

**响应示例**：
```json
{
  "success": true,
  "message": "配置已自动保存",
  "saved_at": "2024-01-11 15:30:45"
}
```

### 3. 完整预览
```
GET /pagebuilder/backend/preview/full?page_id=1
```

**响应**：完整的 HTML 页面（用于 iframe）

## ⚠️ 注意事项

### 1. 浏览器兼容性
- 推荐使用现代浏览器（Chrome、Firefox、Edge、Safari）
- 需要支持 ES6+、Fetch API、CSS Grid

### 2. 权限要求
- 需要登录后台
- 需要 `GuoLaiRen_PageBuilder::page_builder_preview` 权限
- 需要 `GuoLaiRen_PageBuilder::page_builder_auto_save` 权限

### 3. 数据限制
- 配置数据存储在 `style_settings` 字段（JSON 格式）
- 单个页面配置建议不超过 100 项
- 文本配置建议不超过 500 字符

### 4. 性能建议
- 避免频繁修改配置（自动保存有防抖机制）
- 大量修改后可以手动刷新预览
- 建议定期清理缓存

## 🐛 常见问题

### Q1: 可视化配置按钮点击无反应？
**A**: 检查以下几点：
1. 是否选择了样式模板
2. 是否在编辑模式（不是新建模式）
3. 浏览器控制台是否有错误
4. 清理浏览器缓存（Ctrl+Shift+R）

### Q2: 配置修改后预览没有变化？
**A**: 
1. 检查保存状态（是否显示"已保存"）
2. 手动点击刷新按钮
3. 检查浏览器控制台网络请求
4. 清理后端缓存：`php bin/w cache:clear -f`

### Q3: 预览显示空白或错误？
**A**: 
1. 检查页面数据是否完整
2. 检查样式模板文件是否存在
3. 查看后端日志：`var/log/error.log`
4. 重新运行：`php bin/w setup:upgrade -m GuoLaiRen_PageBuilder`

### Q4: 保存状态一直显示"保存中"？
**A**: 
1. 检查网络连接
2. 查看浏览器控制台错误
3. 检查后端接口是否正常
4. 验证登录状态是否过期

### Q5: 设备切换不生效？
**A**: 
- 这是正常的，设备切换只改变预览容器尺寸
- 刷新预览后会看到不同尺寸的效果
- 确保样式模板支持响应式设计

## 🎓 扩展开发

### 添加新的配置项
在样式模板的 PHP 文件顶部添加配置：

```php
/**
 * @fields_start
 * 
 * group:custom => 自定义分组
 * custom.my_field => 我的字段:text:默认值
 * custom.my_color => 我的颜色:color:#ff0000
 * custom.my_select => 我的选择:select:option1|option1,option2,option3
 * custom.my_number => 我的数字:number:100|px
 * 
 * @fields_end
 */
```

### 添加新的配置类型
修改 `form.phtml` 的 `renderVisualConfig()` 函数，添加新的 `case` 分支：

```javascript
case 'my_custom_type':
    html += `
        <input type="..." 
               class="form-control auto-save-field" 
               id="${fieldId}"
               data-config-key="${configKey}"
               value="${currentValue}">
    `;
    break;
```

### 自定义预览模板
修改 `app/code/GuoLaiRen/PageBuilder/view/templates/Backend/Page/Preview/full.phtml`
