# GuoLaiRen_PageBuilder - 智能体组件生成计划

## 一、概述

PageBuilder 模块实现一个专用的 AI 智能体（PageBuilderAgent），用于组件生成场景。
通过 Weline_Ai 模块的 extends 扩展点注册，前端默认选中该智能体。

## 二、架构

### 智能体注册

```
GuoLaiRen/PageBuilder/
├── extends/module/Weline_Ai/Agent/
│   └── PageBuilderAgent.php       # 实现 AgentInterface
└── Service/AI/Tool/
    ├── PreviewReferenceTool.php    # 预览参考组件
    ├── GetComponentFrameworkTool.php # 获取框架模板
    ├── ListComponentsTool.php      # 列出可用组件
    ├── ValidateCodeTool.php        # 验证代码规范
    └── GetPageLayoutTool.php       # 获取页面布局
```

- Agent Code: `pagebuilder_component`
- 场景码: `pagebuilder_component_generation`
- 最大迭代: 5 轮

### 智能体执行流程

```
用户输入描述 → buildAgentUserPrompt()
    → AgentScanner.getAgent('pagebuilder_component')
    → PageBuilderAgent.execute()
        → System Prompt（CSS变量/BEM/框架规约）
        → Plan → Tool 调用 → Observe → 循环
        → 生成 JSON（html_content, css_extra, js_content 等）
    → SSE 事件推送（tool_call, tool_result, chunk, complete）
    → 前端渲染预览
```

### 后端接口（已完成）

| 接口 | 方法 | 说明 |
|------|------|------|
| `/pagebuilder/backend/ai-generate/agents` | GET | 获取可用智能体列表 |
| `/pagebuilder/backend/ai-generate/agent-component-stream` | GET | 智能体流式生成组件 |

### 前端功能（已完成）

| 函数 | 说明 |
|------|------|
| `_loadAgentList()` | 加载智能体列表并渲染选择 UI |
| `_toggleAgent()` | 切换智能体选中状态 |
| `generateComponentWithAgent()` | 智能体模式 SSE 流式生成 |
| `_workshopStartGenerate()` | 根据是否选中智能体分流 |

## 四、5 个 Tool 说明

| Tool | 名称 | 用途 |
|------|------|------|
| PreviewReferenceTool | `preview_reference_component` | Agent 获取已有组件代码用于参考 |
| GetComponentFrameworkTool | `get_component_framework` | Agent 获取框架模板结构 |
| ListComponentsTool | `list_components` | Agent 发现可用组件列表 |
| ValidateCodeTool | `validate_code` | Agent 验证生成代码的规范性 |
| GetPageLayoutTool | `get_page_layout` | Agent 了解当前页面布局上下文 |
