# 可视化编辑器 - 响应式配置说明

## 界面展示

### 响应式配置项显示

在可视化编辑器中，支持响应式的配置项会显示特殊的图标和提示：

```
┌─────────────────────────────────────────────────┐
│ 头部高度 📱 📱 💻 🌐                            │
│ ↑标签    ↑响应式图标  ↑翻译按钮                 │
│ ℹ️ 支持响应式配置                               │
│ ↑描述信息                                       │
│ ┌─────────────────────────────────────┐        │
│ │ 60/80                      │ px │   │        │
│ └─────────────────────────────────────┘        │
│ ↑输入框（支持响应式格式）    ↑单位              │
└─────────────────────────────────────────────────┘
```

### 图标说明

| 图标 | 说明 |
|------|------|
| 📱 | 移动端设备图标（Cellphone） |
| 📱 | 平板设备图标（Tablet） |
| 💻 | PC端/桌面设备图标（Monitor） |
| 🌐 | 翻译按钮（用于多语言配置） |
| ℹ️ | 描述信息图标 |

### 徽章样式

**响应式徽章：**
- 背景色：浅蓝色（Bootstrap bg-info）
- 包含三个设备图标
- 鼠标悬停显示完整提示："支持响应式配置（移动端/平板/PC端）"

**描述信息：**
- 文字颜色：灰色
- 字体大小：0.85rem（略小于正常文字）
- 位置：标签下方，输入框上方
- 带有信息图标前缀

## 配置项类型

### 1. 响应式尺寸配置

**示例：头部高度**
```
┌─────────────────────────────────────────┐
│ 头部高度 📱📱💻                          │
│ ℹ️ 支持响应式配置                       │
│ ┌───────────────────────────┬─────┐    │
│ │ 60/80                     │ px  │    │
│ └───────────────────────────┴─────┘    │
└─────────────────────────────────────────┘
```

**输入格式：**
- 单一值：`80` （所有端80px）
- 双端值：`60/80` （移动60px，PC 80px）
- 三端值：`60/70/80` （移动60px，平板70px，PC 80px）

### 2. 自定义描述配置

**示例：内容最大宽度**
```
┌─────────────────────────────────────────┐
│ 内容最大宽度 📱📱💻                      │
│ ℹ️ 移动端使用百分比，PC端使用像素        │
│ ┌───────────────────────────┬─────┐    │
│ │ 100/1200                  │%/px │    │
│ └───────────────────────────┴─────┘    │
└─────────────────────────────────────────┘
```

**说明：**
- 描述信息会根据配置项定义自动显示
- 可以提供更详细的使用说明
- 帮助用户理解特殊配置项的用途

### 3. 普通配置项

**示例：Logo位置**
```
┌─────────────────────────────────────────┐
│ Logo位置                                │
│ ┌───────────────────────────────────┐  │
│ │ left                         ▼    │  │
│ └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

**说明：**
- 不支持响应式的配置项不显示设备图标
- 保持简洁的界面
- 仅显示必要的配置选项

## 使用流程

### 1. 打开页面编辑

1. 进入 PageBuilder 后台
2. 选择或创建页面
3. 点击"可视化配置"按钮

### 2. 配置响应式尺寸

1. 找到带有 📱📱💻 图标的配置项
2. 在输入框中输入响应式值
   - 双端值：`60/80`
   - 三端值：`60/70/80`
3. 系统会自动解析并应用

### 3. 实时预览

1. 配置会自动保存（Auto-save）
2. 右侧预览区域实时更新
3. 可以点击"刷新预览"按钮强制刷新

### 4. 测试响应式效果

1. 使用浏览器开发者工具
2. 切换到"响应式设计模式"
3. 调整屏幕宽度观察变化：
   - 375px（移动端）
   - 768px（平板）
   - 1280px（PC端）

## 常见问题

### Q1: 如何知道哪些配置项支持响应式？

**A:** 查看配置项标签右侧是否有 📱📱💻 图标。有这些图标的配置项支持响应式配置。

### Q2: 输入响应式值后没有效果？

**A:** 检查以下几点：
1. 值的格式是否正确（使用斜杠分隔）
2. 是否点击了刷新按钮
3. 浏览器是否缓存了旧样式（清除缓存）

### Q3: 可以混合使用不同单位吗？

**A:** 可以，但需要在配置定义中指定。例如：`100/1200|%/px` 表示移动端使用百分比，PC端使用像素。

### Q4: 如何添加自定义描述？

**A:** 在模板文件的配置定义中，使用方括号添加描述：
```php
size.height => 头部高度:number:60/80|px[自定义描述文字]
```

### Q5: 平板端的值如何确定？

**A:** 
- 如果使用双端值（60/80），平板端会自动计算中间值
- 如果需要精确控制平板端，使用三端值（60/70/80）

## 技术细节

### 自动识别响应式配置

系统会自动识别以下情况为响应式配置：
1. 默认值中包含斜杠（`60/80`）
2. 配置定义中包含 `[📱💻]` 标记
3. 描述信息中包含移动端/平板/PC等关键词

### 徽章渲染代码

```javascript
// 添加响应式标记图标
if (config.responsive) {
    html += ` <span class="badge bg-info ms-1" 
                    title="支持响应式配置（移动端/平板/PC端）" 
                    style="font-size: 0.75rem; vertical-align: middle;">
                <i class="mdi mdi-cellphone" style="font-size: 0.85rem;"></i>
                <i class="mdi mdi-tablet" style="font-size: 0.85rem;"></i>
                <i class="mdi mdi-monitor" style="font-size: 0.85rem;"></i>
             </span>`;
}

// 添加描述信息
if (config.description) {
    html += `<div class="form-text text-muted mb-2" 
                  style="margin-top: -8px; font-size: 0.85rem;">
                <i class="mdi mdi-information-outline"></i> 
                ${escapeHtml(config.description)}
             </div>`;
}
```

## 最佳实践

### 1. 合理使用响应式标记

**推荐：**
```php
* size.height => 头部高度:number:60/80|px[📱💻]
* size.padding => 内边距:number:16/40|px[📱💻]
```

**不推荐：**
```php
* style.color => 颜色:color:#000000|[📱💻]  // ❌ 颜色不适合响应式
* layout.position => 位置:select:left|left,right[📱💻]  // ❌ 选择器不需要响应式
```

### 2. 提供清晰的描述

**好的描述：**
```php
* size.max_width => 内容最大宽度:number:100/1200|%/px[移动端使用百分比，PC端使用像素]
```

**不好的描述：**
```php
* size.max_width => 内容最大宽度:number:100/1200|%/px[宽度]  // ❌ 描述过于简单
```

### 3. 保持配置一致性

在同一分组内的配置项，建议使用相同的单位和格式，便于管理和理解。
