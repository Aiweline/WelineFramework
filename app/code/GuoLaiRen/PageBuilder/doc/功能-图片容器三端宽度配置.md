# Marketing Landing 模板 - 图片容器三端宽度配置

## 功能概述
为 `marketing-landing` 模板的图片容器 (`.marketing-image-wrapper`) 添加移动端/平板/PC端三端独立的百分比宽度配置功能。

## 开发日期
2024-01-XX

## 需求背景
之前图片容器宽度只支持移动端和PC端两端配置，且在平板端(iPad)使用硬编码的 45% 宽度。为了提供更灵活的布局控制，需要支持三端独立配置。

## 功能特性

### 配置项
**配置键**: `image.container_width`  
**配置类型**: responsive(响应式)  
**默认值**: `100/45/50`  
**单位**: 百分比(%)

### 三端配置说明
```
格式：移动端/平板端/PC端
示例：100/45/50
```

| 设备 | 断点 | 默认宽度 | 说明 |
|------|------|----------|------|
| **移动端** | < 768px | 100% | 图片占满整个容器宽度 |
| **平板端** | 768px - 1024px | 45% | 图片占45%，表单占55% |
| **PC端** | > 1024px | 50% | 图片占50%，表单占50% |

## 技术实现

### Schema 定义
```php
/**
 * @fields_start
 * 
 * group:image => 图片设置[MTD]:📱三端响应式配置说明:控制图片容器在不同设备上的宽度\n• 移动端/平板/PC端三端配置\n• 单位：百分比(%)\n• 示例：100/45/50 表示移动端100%，平板45%，PC端50%
 * image.url => 图片地址:text:
 * image.alt => 图片描述:text:Money Calendar
 * image.border_radius => 圆角大小:number:8|px
 * image.container_width => 图片容器宽度:responsive:100/45/50|%[移动端/平板/PC]
 * 
 * @fields_end
 */
```

### PHP 代码解析
```php
// 图片容器宽度 - 三端配置（移动端/平板/PC端）
$imageContainerWidthConfig = parseResp(getContentConfig($styleSettings, 'image.container_width', '100/45/50') . '%');
$imageContainerWidthMobile = floatval($imageContainerWidthConfig['mobile']) . '%';
$imageContainerWidthTablet = floatval($imageContainerWidthConfig['tablet'] ?: $imageContainerWidthConfig['desktop']) . '%';
$imageContainerWidthDesktop = floatval($imageContainerWidthConfig['desktop']) . '%';
```

**解析逻辑**：
- 使用 `parseResp()` 函数解析响应式配置
- 移动端: 直接使用 `mobile` 值
- 平板端: 使用 `tablet` 值，如果未设置则回退到 `desktop` 值
- PC端: 使用 `desktop` 值

### CSS 应用

#### 移动端 (< 768px)
```css
.marketing-image-wrapper {
    flex: 0 0 auto;
    width: <?= $imageContainerWidthMobile ?>;  /* 默认 100% */
    order: -1; /* 图片在上 */
}
```

#### 平板端 (768px - 1024px)
```css
@media (min-width: 768px) and (max-width: 1024px) {
    .marketing-image-wrapper {
        width: <?= $imageContainerWidthTablet ?>;  /* 默认 45% */
        max-width: none;
        order: 0;
    }
    .marketing-form-wrapper {
        flex: 1 1 auto;
        width: <?= (100 - floatval($imageContainerWidthConfig['tablet'] ?: $imageContainerWidthConfig['desktop'])) ?>%;  /* 自动计算剩余空间 */
        max-width: none;
        min-width: 0;
    }
}
```

#### PC端 (> 1024px)
```css
@media (min-width: 1024px) {
    .marketing-image-wrapper {
        width: <?= $imageContainerWidthDesktop ?>;  /* 默认 50% */
        max-width: none;
        order: 0;
    }
    .marketing-form-wrapper {
        flex: 1 1 auto;
        width: <?= (100 - floatval($imageContainerWidthConfig['desktop'])) ?>%;  /* 自动计算剩余空间 */
        max-width: none;
        min-width: 0;
    }
}
```

## 使用示例

### 示例 1: 默认配置
```
配置: 100/45/50
```
- 移动端: 图片100%宽度，上下堆叠
- 平板端: 图片45%，表单55%，横向布局
- PC端: 图片50%，表单50%，横向布局

### 示例 2: 图片主导型
```
配置: 100/60/55
```
- 移动端: 图片100%宽度
- 平板端: 图片60%，表单40%（突出图片）
- PC端: 图片55%，表单45%

### 示例 3: 表单主导型
```
配置: 100/35/40
```
- 移动端: 图片100%宽度
- 平板端: 图片35%，表单65%（突出表单）
- PC端: 图片40%，表单60%

### 示例 4: 平衡型
```
配置: 100/50/50
```
- 移动端: 图片100%宽度
- 平板端: 图片50%，表单50%
- PC端: 图片50%，表单50%（完全平衡）

## 自动计算表单宽度

**重要特性**: 表单容器宽度会自动计算，确保图片和表单宽度之和为 100%

```php
// 表单宽度 = 100% - 图片宽度
$formWidth = 100 - $imageWidth;
```

例如：
- 图片40% → 表单60%
- 图片45% → 表单55%
- 图片50% → 表单50%
- 图片60% → 表单40%

## 配置建议

### 移动端 (< 768px)
**推荐**: `100%`  
**原因**: 移动端屏幕窄，上下堆叠布局，图片占满宽度效果最佳

### 平板端 (768px - 1024px)
**推荐范围**: `40% - 55%`
- `35-40%`: 强调表单转化
- `45-50%`: 平衡展示
- `50-60%`: 强调视觉吸引

**不推荐**:
- < 30%: 图片太小，视觉冲击力不足
- \> 65%: 表单空间过小，用户体验差

### PC端 (> 1024px)
**推荐范围**: `40% - 55%`
- `40-45%`: 表单主导，适合长表单
- `50%`: 完全平衡
- `50-55%`: 图片主导，适合视觉营销

**不推荐**:
- < 35%: 浪费大屏空间
- \> 60%: 表单过于拥挤

## 响应式配置格式

### 完整三端配置
```
100/45/50
移动端/平板/PC端
```

### 两端配置（自动补全）
```
100/50
移动端/PC端（平板自动使用PC端值50%）
```

### 单一配置（全端相同）
```
50
所有设备都使用50%
```

## 与其他配置的协同

### 布局方向 (`layout.direction`)
```
layout.direction: image-left   → 图片在左
layout.direction: image-right  → 图片在右（默认）
```

### 元素间距 (`layout.gap`)
```
layout.gap: 20/40  → 移动端20px，PC端40px
```

### 表单宽度
自动计算，无需单独配置：
```php
表单宽度 = 100% - 图片容器宽度
```
