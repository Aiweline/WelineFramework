# 响应式配置快速参考

## 📝 配置格式速查

### 基本格式

```
key => 标签:类型:默认值|单位[描述]
```

### 格式变体

| 用途 | 格式 | 示例 |
|------|------|------|
| 单一值 | `key => 标签:类型:值` | `size.height => 高度:number:80` |
| 带单位 | `key => 标签:类型:值\|单位` | `size.height => 高度:number:80\|px` |
| 响应式 | `key => 标签:类型:值1/值2` | `size.height => 高度:number:60/80` |
| 响应式带标记 | `key => 标签:类型:值1/值2[MTD]` | `size.height => 高度:number:60/80[MD]` |
| 自定义描述 | `key => 标签:类型:值[描述]` | `size.height => 高度:number:80[固定高度]` |

## 🏷️ MTD 标记说明

| 标记 | 含义 | 显示图标 | 说明 |
|------|------|---------|------|
| `[M]` | Mobile | 📱 | 仅移动端 |
| `[MD]` | Mobile + Desktop | 📱💻 | 移动端和PC端（推荐） |
| `[MT]` | Mobile + Tablet | 📱📱 | 移动端和平板 |
| `[MTD]` | Mobile + Tablet + Desktop | 📱📱💻 | 完整三端支持 |
| `[T]` | Tablet | 📱 | 仅平板 |
| `[TD]` | Tablet + Desktop | 📱💻 | 平板和PC端 |
| `[D]` | Desktop | 💻 | 仅PC端 |

## 💡 配置示例

### 尺寸配置

```php
* group:size => 尺寸[MD]
* size.height => 高度:number:60/80|px[MD]
* size.width => 宽度:number:100/1200|%/px[移动端百分比，PC端像素]
* size.padding => 内边距:number:16/40[MD]
* size.margin => 外边距:number:10/20/30|px[MTD]  // 三端独立配置
```

### 样式配置

```php
* group:style => 样式
* style.background => 背景色:color:#ffffff
* style.text_color => 文字颜色:color:#333333
* style.border => 边框:select:yes|yes,no
```

### 布局配置

```php
* group:layout => 布局
* layout.position => 位置:select:left|left,center,right
* layout.columns => 列数:number:3
* layout.gap => 间距:number:16/24[MD]
```

## 🔧 PHP使用代码

### 读取和解析

```php
// 1. 读取配置
$heightValue = getConfig($styleSettings, 'size.height', '60/80');

// 2. 解析配置
$height = parseResponsiveValue($heightValue . 'px');
// 返回: ['mobile' => '60', 'desktop' => '80', 'unit' => 'px']

// 3. 生成CSS
$heightCSS = generateResponsiveCSS($height);
// 返回: "clamp(60px, calc(60px + 20 * ((100vw - 375px) / 905)), 80px)"

// 4. 应用到模板
echo "<style>.element { height: $heightCSS; }</style>";
```

## ✅ 最佳实践

### 推荐做法

- ✅ 优先使用双端值：`60/80[MD]`
- ✅ 使用MTD标记：`[M]` `[MD]` `[MTD]`
- ✅ 提供清晰描述：`[移动端使用百分比]`
- ✅ 使用统一单位：尽量用 `px`
- ✅ 合理的差值：移动端和PC端差值不要过大

### 避免做法

- ❌ 颜色不用响应式：`color:number:60/80` ✗
- ❌ 选择器不用响应式：`position:select:left/right` ✗
- ❌ 过度细分：`60/65/70/75/80` ✗
- ❌ 差值过大：`20/200` ✗
- ❌ 使用emoji标记：`[📱💻]` ✗（已废弃，请用 `[MD]`）

## 🐛 故障排查

### 配置不生效

1. 检查格式是否正确
2. 清除缓存：`php bin/w cache:clear -f`
3. 点击可视化编辑器中的刷新按钮
4. 检查浏览器控制台是否有错误

### 图标不显示

1. 确认配置项包含 `[📱💻]` 标记
2. 或者确认默认值中包含斜杠 `/`
3. 检查解析器是否正确识别

### 描述信息不显示

1. 确认方括号语法正确：`[描述文字]`
2. 确认解析器已更新到最新版本
