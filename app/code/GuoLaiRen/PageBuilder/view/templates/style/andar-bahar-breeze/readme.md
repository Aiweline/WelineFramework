# Andar Bahar Breeze Theme

## Theme Overview

**Andar Bahar Breeze** is a light, fresh, motion-forward landing page theme for Andar Bahar and adjacent Indian card-game traffic pages. It keeps the same PageBuilder PHP config and fallback conventions used by the existing house themes, while shifting the visual language toward mint glass, sky ribbons, and breezy layered modules.

## Core Direction

- Clean airy palette with mint, sky, and soft sand accents
- Distinct module structures, not just recolored copies
- Responsive layouts designed mobile first
- Strong micro-motion and section-level ambient effects
- Compatible with the existing PageBuilder data/config contract used across this theme family

## Key Modules

- `header/nav.phtml`: floating glass nav with moving light band and compact mobile pills
- `content/hero-slider.phtml`: layered breeze-board hero with drifting info strips
- `content/advantages.phtml`: airflow lanes instead of static feature blocks
- `content/games.phtml`: fan-deck game selector with a live detail stage
- `content/testimonials.phtml`: floating review stream with lighter visual rhythm
- `content/faq.phtml`: strip accordion with inline answers and tap-first behavior
- `footer/links.phtml`: breeze-ribbon footer with centered social actions

## Design Notes

- Palette: mint, sky, warm sand, and sea-glass white
- Typography: `Sora` for headings and `Manrope` for body text
- Motion: drifting highlights, layered panels, and soft section ambience
- Assets stay theme-local, so future swaps do not affect PHP logic

## Layout Coverage

This theme includes homepage, about, contact, FAQ, games, blog list/category/detail, and legal/policy page layouts.

## Usage

1. Create a new page in PageBuilder.
2. Select `andar-bahar-breeze` as the style template.
3. Keep the existing config keys or override module fields in the editor.
4. Check mobile first, then fine-tune tablet and desktop spacing.

## Compatibility

- PHP 7.4+
- Modern browsers
- Existing PageBuilder component config format
