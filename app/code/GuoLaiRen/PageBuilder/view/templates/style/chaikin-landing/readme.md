# Chaikin Analytics Landing 样式模板

这是一个专为金融分析、股票投资类产品设计的高转化率落地页模板，风格参考 Chaikin Analytics 网站设计。

## 特点

### 🎯 核心功能
- **品牌标识区域**：支持文字 logo 或图片 logo，可灵活配置
- **视频播放器**：支持 YouTube 和 HTML5 视频，带自定义播放按钮
- **引导弹窗**：页面加载后延迟弹出，引导用户行动
- **表单收集**：电子邮件和电话号码收集，带完整的验证和提交逻辑
- **响应式设计**：完美适配移动端和桌面端

### 🎨 设计风格
- 深色背景配合亮色文字，营造专业感
- 高对比度颜色方案，突出重点内容
- 大字号标题，增强视觉冲击力
- 按钮抖动动画，吸引用户注意

### 📱 响应式配置
模板内所有关键尺寸都支持移动端/桌面端独立配置：
- 字体大小：移动端/桌面端双值配置（如 `28/56` 表示移动端28px，桌面端56px）
- 间距边距：所有 padding、margin 都支持响应式
- 布局方向：可选择图片在左/右/上/下

### 🔧 可配置项

#### 头部 (header.phtml)
- Logo 显示/隐藏、地址、位置、宽度
- 品牌文字内容、颜色、大小
- 头部背景色、内边距
- 容器最大宽度
- **页面背景图**：全屏背景图地址、位置、尺寸、重复方式
  - 默认背景图：`https://assets.chaikinanalytics.com/mkt/2025/cpgr/hidden-ai/bg-showcase-cpgr-hidden-ai-light.webp`

#### 内容 (content.phtml)
1. **布局设置**
   - 布局方向（图片位置）
   - 间距、最小高度、内边距

2. **容器样式**
   - 最大宽度、内边距、背景色

3. **标题/副标题**
   - 文本内容、颜色、字号、字重
   - 行高、对齐方式、外边距

4. **视频播放器**
   - 视频地址、封面图
   - 播放提示文字、按钮样式
   - 宽高比、最大宽度

5. **引导弹窗** ⭐
   - 启用/禁用开关
   - 弹出延迟时间（秒）
   - 标题和内容文本
   - 两个按钮的文字和颜色
   - 关闭链接文字和颜色
   - 弹窗尺寸和边框样式

6. **表单区域**
   - 表单标题、颜色、字号
   - 背景色、圆角、内边距
   - 输入框样式（背景、边框、圆角等）
   - 按钮样式（文字、颜色、抖动间隔）
   - 提示消息（必填、格式错误、提交成功等）

7. **图片区域**
   - 图片地址、描述文字
   - 边框圆角、容器宽度

8. **免责声明**
   - 文本内容、颜色、字号
   - 行高、外边距

#### 页脚 (footer.phtml)
- 通用样式（背景色、文字色、链接色）
- 免责声明和隐私政策链接
- 版权信息文字
- 排版（字号、行高）
- 布局（最大宽度、内边距）

## 使用场景

适用于以下类型的落地页：
- 📈 金融投资产品推广
- 📊 股票分析工具介绍
- 💼 专业咨询服务
- 🎓 在线课程招生
- 📧 线索收集和用户注册

## 技术特性

- ✅ 纯 HTML/CSS/JavaScript，无需外部依赖
- ✅ 移动优先的响应式设计
- ✅ 支持 Google Analytics 和 GTM 追踪
- ✅ 表单验证和错误提示
- ✅ ESC 键和背景点击关闭弹窗
- ✅ 平滑动画和过渡效果

## 配置提示

1. **弹窗延迟**：建议设置为 3-8 秒，给用户足够的浏览时间
2. **按钮颜色**：使用高对比度颜色，确保在深色背景下清晰可见
3. **标题文字**：简短有力，突出核心价值主张
4. **表单字段**：最小化必填字段，降低用户填写门槛
5. **免责声明**：根据实际业务需求修改法律声明文本
