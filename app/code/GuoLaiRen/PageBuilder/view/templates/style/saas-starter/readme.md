# SaaS Starter (saas-starter) 模板

## 模板概述

现代 SaaS 产品落地页模板，翡翠绿渐变深色主题。适合软件产品、数字服务、API 平台、开发者工具等场景。

### 适用场景

- SaaS 产品推广页
- 开发者工具落地页
- API 平台介绍
- 数字服务订阅
- 企业级软件官网

### 核心特性

1. **翡翠绿渐变主题** - 科技感强的视觉设计
2. **定价方案组件** - 3列布局，支持高亮推荐方案
3. **用户评价组件** - 网格布局客户证言
4. **CTA行动号召** - 醒目的转化引导区域
5. **完全可配置** - 所有文本、颜色、尺寸均可后台配置
6. **响应式设计** - 移动端、平板、桌面完美适配
