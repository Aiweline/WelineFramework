# RummyRoyal Template

Luxury gold + deep red dark theme for Indian Rummy gaming platforms.

## Design Inspiration

Inspired by mainstream Indian Rummy gaming apps: RummyCircle, Junglee Rummy, A23 Rummy.

## Color Palette

| Role | Color | Hex |
|------|-------|-----|
| Primary Gold | Gold | `#eab308` |
| Secondary Red | Deep Red | `#dc2626` |
| Body Background | Deep Purple Black | `#0a0514` |
| Card Background | Transparent White | `rgba(255,255,255,0.04)` |
| Text Primary | Warm White | `#fef3c7` |
| Text Secondary | Warm Beige | `#d4a574` |
| Text Muted | Brown | `#8b6f47` |

## Components (15)

### Header
- `header-nav` — Navigation with crown/card logo, gold gradient CTA

### Content (13)
- `hero-slider` — Full-width hero with stats and CTAs
- `games` — Game showcase grid (Points, Pool, Deals, Tournaments)
- `advantages` — 6-card advantages grid
- `app-download` — Download section with features and checklist
- `testimonials` — Player reviews with star ratings
- `faq` — FAQ accordion
- `cta-banner` — Call-to-action banner
- `blog-list` — Blog post grid with category filter
- `blog-detail` — Single blog post with sidebar
- `blog-category` — Blog category listing
- `legal-content` — Legal document with TOC
- `about-content` — About page with stats and values
- `contact-content` — Contact form and info

### Footer
- `footer-links` — 4-column footer with brand info

## CSS Prefix

All CSS classes use the `rr-` prefix.

- `rr_color($colors, $key, $default)` — Get color with fallback
- `rr_getConfig($settings, $key, $default)` — Get config with fallback
