# Algo Trading Landing 样式模板

这是一个专为算法交易、股票投资服务设计的高转化率落地页模板，参考了 StocksToTrade 风格的专业设计。

## 特点

### 🎯 核心功能
- **顶部警告条**：醒目的黄色横幅，吸引用户注意
- **主标题区域**：支持高亮关键词（如 "SIXTEEN 50%+"）
- **图表展示**：展示股票增长图表，增强说服力
- **多个CTA按钮**：主按钮、中间CTA、Why后CTA和最终CTA，引导用户行动
- **媒体Logo展示**：展示知名媒体品牌（如 NYT, Forbes, Bloomberg），增强可信度
- **三大优势说明**：带编号圆圈的要点展示，清晰易读
- **推荐人评价**：展示6位专业交易员和分析师，建立信任
- **完整免责声明**：符合金融服务的法律要求

### 🎨 设计风格
- 深蓝到紫到黑渐变背景（#091d56 → #331290 → #000），营造专业高端感
- 亮黄色警告条（#ffeb3b），突出重点信息
- 白色下划线高亮，突出重点数据（如 "SIXTEEN 50%+ Gainers"）
- 白色内容区域，分层清晰，带阴影和圆角
- 深蓝色编号圆圈（#1c1596），视觉引导明确
- 绿色CTA按钮（#278b00），引导用户行动
- 圆形头像展示，增强专业性

### 📱 响应式设计
模板完全响应式，在移动端和桌面端都有良好的显示效果：
- 字体大小自适应（使用 CSS clamp()）
- 布局自动调整
- 卡片网格响应式排列
- 移动端优化的间距和内边距

### 🔧 可配置项

#### 头部 (header.phtml)

**1. 顶部警告条**
- 启用/禁用
- 自定义文字、背景色、文字颜色
- 响应式字体大小和内边距
- 字体粗细设置

**2. 主标题**
- 自定义标题文字
- 高亮特定词汇（如 "SIXTEEN 50%+"）
- 颜色、大小、粗细、行高
- 响应式下边距

**3. 图表区域**
- 图表图片地址（标签中显示默认图片名称）
  - 字段留空时自动使用默认URL
  - 完整默认URL：`https://lp.stockstotrade.com/algo-trader/images/STT_ORACLE_WINNERS_SEPT.gif`
- 最大宽度设置（默认：600px）
- 响应式下边距（30-50px）

**4. CTA按钮**
- 按钮文字和链接
- 打开方式（当前页/新标签）
- 背景色、悬停色、文字颜色
- 响应式字体和内边距
- 边框圆角

**5. 媒体Logo**
- 显示/隐藏
- 标题文字（默认："As seen on..."）
- 支持两种显示方式：
  - **图片模式**：使用单张合成图片展示所有Logo（默认：https://lp.stockstotrade.com/algo-trader/images/img-brands.png）
  - **文字模式**：Logo列表（逗号分隔）：The New York Times, Forbes, Inc, Bloomberg, CBS, CNN, FOX
- 透明度设置
- 标题颜色和字体大小

**6. 页面样式**
- 背景颜色（body背景，默认：#000纯黑色）
- 启用/禁用渐变（Hero区域渐变背景）
- 渐变色1（默认：#091d56 深蓝色）
- 渐变色2（默认：#331290 紫色）
- 渐变色3（默认：#000 黑色）

**7. 容器设置**
- 最大宽度（默认：1280px）
- 响应式左右内边距（20-40px）
- 响应式顶部内边距（20-30px）
- 响应式底部内边距（40-80px）

**8. 自定义CSS**
- 启用/禁用自定义CSS
- 自定义CSS代码输入

#### 内容 (content.phtml)

**1. 介绍部分**
- 3个段落文字

  - **要点3**: Emotionless Trade Decisions

- 响应式下边距

**5. Why后CTA**

- 按钮文字和链接

## 配置提示

### 内容优化
1. **警告条文字**：保持简短有力，突出 "免费" 或 "限时" 等关键词
2. **主标题高亮**：选择数字或百分比作为高亮词，增强视觉冲击
3. **统计数据**：使用真实可验证的数据，增强可信度
4. **要点说明**：每个要点保持简洁，重点突出核心优势

### 视觉设计
5. **图表图片**：建议使用清晰的股票增长图表，尺寸 500px 宽度左右
6. **媒体Logo**：选择知名度高的媒体品牌，增强可信度
7. **推荐人信息**：使用真实的头像和可验证的信息
8. **颜色搭配**：保持与品牌色调一致，避免过多颜色干扰

### 法律合规
9. **免责声明**：根据当地金融服务法律要求调整内容
10. **链接配置**：确保 Privacy Statement 和 Terms of Service 链接有效

### 转化优化
11. **CTA按钮**：使用行动导向的文字，如 "立即获取"、"免费试用"
12. **多个CTA**：在不同位置放置CTA按钮，提高转化率
13. **社会证明**：推荐人和媒体Logo增强信任感
14. **紧迫感**：警告条和高亮数字创造紧迫感
