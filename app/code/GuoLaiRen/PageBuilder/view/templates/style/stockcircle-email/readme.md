# StockCircle Email Template - 邮件模板

## 📋 模板概述

这是一个基于邮件模板设计的落地页模板，完美复刻了 StockCircle 邮件模板的样式和布局。模板支持暗色和亮色两种主题模式，所有元素都可以通过可视化配置进行自定义。

### 🎯 适用场景

- 邮件内容展示页
- 营销邮件落地页
- 社区邀请页
- 产品介绍页
- 注册引导页

### ✨ 核心特性

1. **双主题支持** - 支持暗色（Dark）和亮色（Light）两种主题模式
2. **完全可配置** - 所有文本、颜色、尺寸都可以通过后台配置
3. **响应式设计** - 完美适配移动端和桌面端
4. **邮件友好** - 基于邮件模板设计，兼容性强
5. **简洁美观** - 清晰的层次结构和视觉引导

## 🏗️ 模板结构

### 文件组成

```
stockcircle-email/
├── header.phtml   - 头部区域（Logo）
├── content.phtml  - 内容区域（标题、段落、按钮、列表、图片）
├── footer.phtml   - 页脚区域（社区指南、版权信息）
└── readme.md      - 本文档
```

## ⚙️ 配置项说明

### Header 配置项

#### Logo 设置
- **显示Logo** - 是否显示 Logo（yes/no）
- **Logo地址(暗色主题)** - 暗色主题下的 Logo URL
- **Logo地址(亮色主题)** - 亮色主题下的 Logo URL
- **Logo宽度** - 响应式宽度（移动端/PC端，单位：px）
- **Logo链接地址** - Logo 点击跳转的 URL

#### 全局主题设置 ⚠️ 重要
- **主题模式** - dark（暗色）/ light（亮色）
  - **这是全局主题配置**，控制整个模板（Header、Content、Footer）的色系
  - 修改此配置后，三个文件（Header、Content、Footer）的主题会自动同步
  - 暗色（dark）：深色背景（#212a37）、浅色文字（#e0e0e0）、绿色标题（#03d453）
  - 亮色（light）：浅色背景（#ffffff）、深色文字（#666666）、深色标题（#333333）
  - **建议**：在 Header 配置中设置主题，Content 和 Footer 会自动继承

#### 头部样式设置
- **背景颜色(暗色主题)** - 暗色主题下的背景色（默认：#212a37）
- **背景颜色(亮色主题)** - 亮色主题下的背景色（默认：#ffffff）
- **上下内边距** - 响应式内边距（移动端/PC端，单位：px）
- **左右内边距** - 响应式内边距（移动端/PC端，单位：px）

#### 容器设置
- **最大宽度** - 内容区最大宽度（移动端/PC端，移动端使用%，PC端使用px）

## 📝 使用指南

### 1. 创建新页面

1. 进入后台：**页面构建器 → 新建页面**
2. 填写基本信息：
   - 页面句柄：例如 `stockcircle-invite`
   - 页面类型：选择合适的类型
   - 页面名称：运营用名称
   - 页面标题：显示在浏览器标签的标题

### 2. 选择样式模板

在**页面样式**区域，选择 **stockcircle-email** 模板

### 3. 配置 SEO 信息

在**SEO 设置**区域：
- **SEO 标题** - 页面 `<title>` 标签
- **SEO 描述** - 页面 `<meta description>`
- **SEO 关键词** - 页面 `<meta keywords>`

### 4. 上传 Logo（可选）

在**页面标识**区域上传 Logo 图片，如果不上传则使用默认 Logo

### 5. 添加跟踪代码（可选）

- **GA4 ID** - Google Analytics 4 跟踪 ID
- **GTM ID** - Google Tag Manager 容器 ID
- **FB Pixel ID** - Facebook Pixel ID

### 6. 可视化配置

点击右侧悬浮按钮**可视化配置**，打开配置面板：

#### 配置 Header
1. 展开 **stockcircle-email_header.phtml** 配置
2. **重要**：首先选择全局主题模式（暗色/亮色），此设置会同步到 Content 和 Footer
3. 配置 Logo 地址和链接
4. 配置头部样式和容器设置

#### 配置 Content
1. 展开 **stockcircle-email_content.phtml** 配置
2. 主题模式会自动继承 Header 的设置（无需重复配置）
3. 配置标题、段落文本、按钮
4. 配置列表项和展示图片
5. 设置分割线

#### 配置 Footer
1. 展开 **stockcircle-email_footer.phtml** 配置
2. 主题模式会自动继承 Header 的设置（无需重复配置）
3. 配置社区指南文本
4. 配置版权信息和链接

### 7. 保存并发布

1. 点击**保存页面**
2. 将状态改为**已发布**
3. 访问页面：`/pagebuilder/page/view?handle=your-handle`

## 📱 响应式说明

### 断点定义

- **移动端 (M)**: 375px
- **桌面端 (D)**: 1280px

### 响应式配置格式

- **单一值**: `60` - 所有端相同
- **双端值**: `60/80` - 移动端 60，PC端 80

### 布局变化

- **移动端**：
  - 纵向布局
  - 较小字号和内边距
  - 图片宽度 100%
  
- **PC端**：
  - 固定最大宽度（600px）
  - 较大字号和内边距
  - 保持邮件模板风格

## ⚠️ 注意事项

1. **主题配置** ⚠️ **重要**
   - Header、Content、Footer 三个文件共享同一个主题配置（`theme.mode`）
   - **建议在 Header 配置中设置主题模式**，Content 和 Footer 会自动继承
   - 三个文件的主题模式配置会自动同步，无需重复设置
   - 如果三个文件的主题不一致，页面会出现风格混乱
2. **Logo适配** - 暗色和亮色主题使用不同的 Logo，确保 Logo 在对应背景下清晰可见
3. **按钮颜色** - 按钮颜色需要与背景色形成对比，确保可读性
4. **链接替换** - 版权文本中的 "stockcircle.to" 和 "unsubscribe" 会自动替换为配置的链接
5. **段落处理** - 段落6会自动添加 `<strong>` 标签（用于签名）

## 🔧 高级定制

### 添加自定义字段

如需添加更多段落或列表项，编辑 `content.phtml` 文件，按照现有格式添加新的配置项和显示逻辑。

### 自定义样式

可以在可视化配置中添加更多配置项，例如：

```php
* group:custom => 自定义设置
* custom.font_family => 字体:select:Arial|Arial,Helvetica,sans-serif
```

### 修改主题颜色

所有颜色配置都支持暗色和亮色两种模式，可以在配置面板中分别设置。

## 📈 最佳实践

1. **内容简洁** - 保持邮件风格，内容简洁明了
2. **CTA突出** - 确保按钮颜色与背景对比明显
3. **图片优化** - 使用合适的图片尺寸，建议不超过 600px 宽度
4. **链接有效** - 确保所有链接地址正确有效
5. **测试主题** - 在两个主题下都测试页面效果

## 🚀 性能优化

1. **图片优化** - 使用 WebP 格式，压缩到合理大小
2. **字体加载** - 使用系统字体，减少外部资源加载
3. **CSS优化** - 使用响应式 clamp() 函数，减少媒体查询
