# IPL Sunflare Theme

## Theme Overview

**IPL Sunflare** is a warm, motion-first landing page theme for Indian gaming traffic pages. It keeps existing PageBuilder PHP/config conventions while introducing a new module choreography and stronger mobile-first readability.

## Core Direction

- Warm amber + ember color system
- New module structures (route-selector games, spotlight testimonials, split FAQ)
- Responsive-first spacing and typography
- Compatible with existing `dragon-tiger-sunrise` data/config patterns
- Tailwind-ready + custom CSS animation layers

## Key Modules

- `header/nav.phtml`: sticky glass nav with animated sweep and mobile drawer
- `content/hero-slider.phtml`: marquee headline + dual-stage hero composition
- `content/advantages.phtml`: lead-card narrative with support lane
- `content/games.phtml`: route selector with tab-driven mode panels
- `content/testimonials.phtml`: spotlight review + stream cards
- `content/faq.phtml`: split intro + timeline accordion cards
- `footer/links.phtml`: warm gradient footer with compact social cluster

## Design Notes

- Palette: cream, ember orange, amber gold
- Typography: `Sora` (headings) + `Manrope` (body)
- Motion: marquee, panel fade, hover lift, accent sweep
- Assets are theme-local and replaceable without touching PHP data flow

## Layout Coverage

Homepage, about, contact, FAQ, games, blog list/category/detail, and legal/policy page layouts are included.

## Usage

1. Create a new page in PageBuilder.
2. Select `ipl-sunflare` as the style template.
3. Keep existing keys or override module fields in visual editor.
4. Validate mobile first, then refine desktop spacing.

## Compatibility

- PHP 7.4+
- Modern browsers
- Existing PageBuilder component config format
