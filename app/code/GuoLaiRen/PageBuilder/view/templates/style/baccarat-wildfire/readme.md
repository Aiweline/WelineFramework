# Baccarat Wildfire Theme

## Theme Overview

**Baccarat Wildfire** is a bold landing page theme for Baccarat and related card-game traffic pages. It keeps the existing PageBuilder PHP config conventions while shifting the visual system toward higher contrast, sharper rhythm, and a louder ember-neon identity.

## Core Direction

- High-contrast ember, acid-lime, and electric-cyan palette with stronger section contrast
- Distinct module structures, not just recolored copies
- Mobile-first responsive spacing and typography
- Compatible with the existing `teen-patti-master`, `tpmst`, and `callbreak-legend` data/config patterns
- Tailwind-ready setup with custom CSS for richer layout control

## Key Modules

- `header/nav.phtml`: neon-ember navigation with animated sweep highlight
- `content/hero-slider.phtml`: centered stage hero with floating suit markers and card runway
- `content/advantages.phtml`: zigzag value rail with three asymmetrical signal cards
- `content/games.phtml`: swipeable table poster rail for room selection
- `content/testimonials.phtml`: compact summary strip plus half-revealed review carousel
- `content/faq.phtml`: balanced dual-column FAQ stack with inline expansion
- `footer/links.phtml`: ember footer with FAQ-blended transition background

## Design Notes

- Backgrounds alternate between bone-white, pale mint, and deep charcoal to avoid clone-like repetition
- Accent palette: ember orange, acid lime, electric cyan, and soft ivory
- Typography: `Outfit` for headings and `Plus Jakarta Sans` for body copy
- SVG assets are theme-local and can be swapped without changing PHP logic

## Layout Coverage

This theme includes homepage, about, contact, FAQ, games, blog list/category/detail, and legal/policy page layouts.

## Usage

1. Create a new page in PageBuilder.
2. Select `baccarat-wildfire` as the style template.
3. Keep the existing config keys or override module fields in the editor.
4. Preview on mobile first, then refine desktop spacing if needed.

## Compatibility

- PHP 7.4+
- Modern browsers
- Existing PageBuilder component config format
