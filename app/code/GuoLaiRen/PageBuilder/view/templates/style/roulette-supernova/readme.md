# Roulette Supernova

Roulette Supernova is a bold roulette landing theme built on the same PHP data-reading conventions as `tpmst`, while using a darker and more energetic presentation than the light-layout reference themes.

## Theme Direction

- High-contrast roulette palette with ember red, neon lime, and electric cyan accents
- Responsive-first sections that keep their structure clear on mobile
- Distinct homepage modules instead of simple visual recolors of earlier themes
- Inner pages rebuilt to avoid unsupported fixed configuration blocks
- Semantic HTML structure aimed at stable rendering and search-friendly page hierarchy

## Key Sections

- `content/hero-slider.phtml`: roulette hero with animated wheel theatre and mobile-friendly CTA staging
- `content/advantages.phtml`: contrast-heavy value cards with staggered placement
- `content/games.phtml`: roulette table showcase with a custom visual deck
- `content/testimonials.phtml`: player review rail with rating summary and swipe-friendly layout
- `content/faq.phtml`: compact question cards with inline reveal behavior
- `content/about-content.phtml`: brand story, values, and operations layout for about pages
- `content/contact-content.phtml`: support route page with cards, message dock, and optional map
- `content/legal-content.phtml`: policy layout with inline table of contents and dark reading surface
- `content/blog-list.phtml`, `content/blog-category.phtml`, `content/blog-detail.phtml`: roulette journal archive and article pages
- `footer/links.phtml`: compact roulette footer with seam crest and compliant navigation

## Compatibility Notes

- Header and footer follow the responsive document structure pattern used by the newer theme skeletons.
- JSON layouts avoid unsupported fixed body content so editors can adapt sites without editing template files.
- Default copy stays roulette-related and all frontend helper text is in English.
