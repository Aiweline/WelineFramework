# 营销落地页模板 - Marketing Landing Page Template

## 📋 模板概述

这是一个完整的营销落地页模板，专为高转化率的产品推广和线索收集设计。模板包含所有必要的元素，并且每个元素都可以通过可视化配置进行自定义。

### 🎯 适用场景

- 产品发布页
- 活动注册页
- 电子书/白皮书下载页
- 在线培训报名页
- Newsletter 订阅页
- 产品演示申请页

### ✨ 核心特性

1. **完全可配置** - 所有文本、颜色、尺寸都可以通过后台配置
2. **响应式设计** - 完美适配移动端、平板和桌面
3. **SEO 优化** - 从页面数据自动生成 Meta 信息
4. **跟踪集成** - 支持 GA4、GTM、Facebook Pixel
5. **表单收集** - 内置表单提交功能，数据保存到系统
6. **灵活布局** - 支持图片左右布局切换

## 📝 使用指南

### 1. 创建新页面

1. 进入后台：**页面构建器 → 新建页面**
2. 填写基本信息：
   - 页面句柄：例如 `money-calendar`
   - 页面类型：选择合适的类型
   - 页面名称：运营用名称
   - 页面标题：显示在浏览器标签的标题

### 2. 选择样式模板

在**页面样式**区域，选择 **marketing-landing** 模板

### 3. 配置 SEO 信息

在**SEO 设置**区域：
- **SEO 标题** - 页面 `<title>` 标签
- **SEO 描述** - 页面 `<meta description>`
- **SEO 关键词** - 页面 `<meta keywords>`

### 4. 上传 Logo（可选）

在**页面标识**区域上传 Logo 图片，如果不上传则使用默认 Logo

### 5. 添加跟踪代码（可选）

- **GA4 ID** - Google Analytics 4 跟踪 ID
- **GTM ID** - Google Tag Manager 容器 ID
- **FB Pixel ID** - Facebook Pixel ID

### 6. 可视化配置

点击右侧悬浮按钮**可视化配置**，打开配置面板：

#### 配置 Header
1. 展开 **marketing-landing_header.phtml** 配置
2. 配置 Logo、标题、副标题、样式
3. 实时预览效果

#### 配置 Content
1. 展开 **marketing-landing_content.phtml** 配置
2. 设置布局方向（图片左/右）
3. 配置表单样式和按钮
4. 上传展示图片
5. 设置免责声明

#### 配置 Footer
1. 展开 **marketing-landing_footer.phtml** 配置
2. 设置版权信息
3. 添加链接（格式：`Privacy Policy|/privacy`）
4. 设置免责声明

### 7. 保存并发布

1. 点击**保存页面**
2. 将状态改为**已发布**
3. 访问页面：`/pagebuilder/page/view?handle=your-handle`

## 📊 表单数据管理

### 查看提交数据

表单提交的数据保存在 `guolairen_page_builder_form_submission` 表中。

可以在后台查看：
1. 进入：**页面构建器 → 表单提交记录**
2. 查看所有提交的邮箱和电话
3. 可以导出为 CSV 或 Excel

### 数据字段

- `email` - 用户邮箱
- `phone` - 用户电话
- `page_id` - 提交来源页面 ID
- `ip_address` - 用户 IP 地址
- `user_agent` - 用户浏览器信息
- `referer` - 来源页面 URL
- `status` - 处理状态（新提交/已处理/已跟进）
- `create_time` - 提交时间

### 配色方案

#### 深色主题（推荐）
```
背景色：#020818
主标题：#ffffff
副标题：#e0e0e0
按钮色：#f39c12
悬停色：#e67e22
```

#### 浅色主题
```
背景色：#f8f9fa
主标题：#2c3e50
副标题：#7f8c8d
按钮色：#3498db
悬停色：#2980b9
```

### 文案建议

#### 主标题
- 简短有力（10-15 个字）
- 包含核心利益点
- 使用数字增加可信度
- 例如：`"3个月内收入翻倍的秘密"`

#### 副标题
- 扩展主标题信息
- 强调痛点或机会
- 3-5 行为宜
- 使用 `**粗体**` 突出关键词

#### 按钮文案
- 行动导向
- 表明价值
- 制造紧迫感
- 例如：`立即获取免费报告`、`开始我的财富之旅`

### 表单优化

1. **字段数量** - 保持在 2-3 个（邮箱、电话）
2. **占位符** - 清晰说明要填写的内容
3. **免责声明** - 必须透明告知用途
4. **按钮颜色** - 与背景形成对比
5. **移动优化** - 确保在手机上易于操作

## 🔧 高级定制

### 自定义字段

如需添加更多表单字段，编辑 `content.phtml`：

```php
<input type="text" 
       name="custom_field" 
       class="marketing-input" 
       placeholder="Your Custom Field">
```

同时更新控制器 `Controller/Frontend/Form.php` 接收该字段。

### 自定义样式

可以在可视化配置中添加更多配置项，例如：

```php
* group:custom => 自定义设置
* custom.font_family => 字体:select:Arial|Arial,Helvetica,sans-serif
```

### 集成 CRM

在 `Controller/Frontend/Form.php` 的 `submit()` 方法中添加 CRM API 调用：

```php
// 保存到系统后，同步到 CRM
$this->syncToCRM($email, $phone);
```

## 📱 响应式说明

### 断点定义

- **移动端 (M)**: 375px
- **平板 (T)**: 768px
- **桌面 (D)**: 1280px

### 响应式配置格式

- **单一值**: `80` - 所有端相同
- **双端值**: `60/80` - 移动端 60，PC端 80
- **三端值**: `60/70/80` - 移动端 60，平板 70，PC端 80

### 布局变化

- **移动端**：
  - 纵向堆叠
  - 表单在上，图片在下
  - 居中对齐
  
- **PC端**：
  - 横向布局
  - 可配置图片左/右
  - 表单和图片各占一半

## ⚠️ 注意事项

1. **Logo 颜色** - 默认 Logo 是白色，深色背景下使用
2. **图片尺寸** - 建议上传 1200x800 的横图
3. **表单验证** - 前端和后端都有验证，确保数据质量
4. **隐私合规** - 免责声明必须告知数据用途
5. **测试** - 发布前在不同设备测试表单提交

## 🚀 性能优化

1. **图片优化** - 使用 WebP 格式，压缩到 200KB 以下
2. **延迟加载** - 大图片使用 lazy loading
3. **缓存** - 静态资源设置长缓存
4. **CDN** - Logo 和图片使用 CDN

## 📈 转化率优化建议

1. **加载速度** - 页面应在 2 秒内加载完成
2. **信任元素** - 添加客户评价、证书、媒体报道
3. **视觉层次** - 确保 CTA 按钮突出
4. **社会证明** - 显示已有多少人注册
5. **紧迫感** - 限时优惠、倒计时
6. **A/B 测试** - 测试不同文案、颜色、布局
