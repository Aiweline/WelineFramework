# Blackjack Halo Theme

## Theme Overview

**Blackjack Halo** is a light, editorial-style landing page theme for blackjack and related table-game traffic pages. It keeps the existing PageBuilder PHP config conventions while introducing a warmer, more open visual system.

## Core Direction

- Light ivory blackjack palette instead of deep casino-dark backgrounds
- Distinct module structures, not just recolored copies
- Mobile-first responsive spacing and typography
- Compatible with the existing `teen-patti-master`, `tpmst`, and `callbreak-legend` data/config patterns
- Tailwind-ready setup with custom CSS for richer layout control

## Key Modules

- `header/nav.phtml`: light glass navigation with animated sweep highlight
- `content/hero-slider.phtml`: asymmetrical hero with halo-stage composition and felt cues
- `content/advantages.phtml`: editorial feature + stacked value cards
- `content/games.phtml`: table-route style showcase for game modes
- `content/testimonials.phtml`: review wall with spotlight and rating summary
- `content/faq.phtml`: warm accordion layout with sticky context panel
- `footer/links.phtml`: halo-rail footer with compact social actions

## Design Notes

- Backgrounds use cream, peach, and gold gradients instead of navy/green casino tones
- Accent palette: coral, gold, and soft jade details
- Typography: `Outfit` for headings and `Plus Jakarta Sans` for body copy
- SVG assets are theme-local and can be swapped without changing PHP logic

## Layout Coverage

This theme includes homepage, about, contact, FAQ, games, blog list/category/detail, and legal/policy page layouts.

## Usage

1. Create a new page in PageBuilder.
2. Select `blackjack-halo` as the style template.
3. Keep the existing config keys or override module fields in the editor.
4. Preview on mobile first, then refine desktop spacing if needed.

## Compatibility

- PHP 7.4+
- Modern browsers
- Existing PageBuilder component config format
