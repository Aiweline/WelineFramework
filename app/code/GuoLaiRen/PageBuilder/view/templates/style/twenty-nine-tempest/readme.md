# Twenty Nine Tempest Theme

Twenty Nine Tempest is a high-contrast, motion-driven landing-page theme built for 29 Card Game traffic pages.

## Highlights
- Full responsive layout across header, content modules, and footer
- Strong visual contrast between homepage sections for clearer pacing
- Animated navigation light sweep and wave-accented footer
- tpmst-compatible PHP config/data retrieval conventions

## Core Modules
- `header/nav.phtml`: sticky nav with hover sheen and CTA shine
- `content/hero-slider.phtml`: tactical hero with dual CTA and live stats
- `content/advantages.phtml`: feature split layout with signal cards
- `content/games.phtml`: game-mode grid with intro panel and badges
- `content/testimonials.phtml`: spotlight + rating board + review grid
- `content/faq.phtml`: two-column FAQ with brief and accordion
- `footer/links.phtml`: integrated wave layer + social action row

## Notes
- The theme keeps existing field keys and fallback behavior so page editor configuration remains compatible.
- Use `layouts/default/*.json` presets for home, blog, contact, legal, and policy pages.
