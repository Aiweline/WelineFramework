# 开发文档（表结构升级规范）

本文档记录表结构“编辑/升级”的推荐用法，示例参考 `app/code/Weline/Acl/Model/Acl.php` 行 231-241。

## 字段存在性检查

- 使用 `ModelSetup::hasField()` 检查字段是否存在（已提供便捷方法）。
- 支持两种调用形式：
  - `hasField('field_name')`（自动以当前模型表为表名）
  - `hasField('table_name', 'field_name')`

## 新增字段（推荐写法）

```php
// 例：若不存在排序字段，则新增
if (!$setup->hasField(self::fields_ORDER)) {
    $setup->alterTable()->addColumn(
        self::fields_ORDER,                // 新字段名
        self::fields_ACL_ID,               // 插入位置：位于哪个字段之后（为空则置于表首）
        TableInterface::column_type_INTEGER,
        11,
        'default 0',                       // 其他选项（NULL/NOT NULL/DEFAULT/UNIQUE 等）
        '排序'                              // 字段注释
    )->alter();                            // 提交变更
}
```

要点：
- 使用 `$setup->alterTable()->addColumn(...)->alter();` 完成新增字段。
- 不直接从业务层调用 `getConnection()->getConnector()`；统一通过 `ModelSetup` 暴露的方法进行操作。

## 示例来源

文件：`app/code/Weline/Acl/Model/Acl.php`

```php
public function upgrade(ModelSetup $setup, Context $context): void
{
    if (!$setup->hasField(self::fields_ORDER)) {
        $setup->alterTable()->addColumn(
            self::fields_ORDER,
            self::fields_ACL_ID,
            TableInterface::column_type_INTEGER,
            11, 'default 0', '排序'
        )->alter();
    }
}
```
