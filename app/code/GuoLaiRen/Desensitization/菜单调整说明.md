# 菜单顺序调整说明

## 调整时间
2025-10-27

## 调整前顺序
1. 脱敏规则管理 (order: 10)
2. 脱敏记录 (order: 20)
3. 敏感检测 (order: 30)
4. AI润色 (order: 40)
5. 脱敏测试 (order: 50)
6. 模块配置 (order: 60)

## 调整后顺序
1. **敏感检测** (order: 10) - 最常用的直接操作
2. **AI润色** (order: 20) - 常用的内容处理操作
3. **脱敏测试** (order: 30) - 用于验证和测试
4. **脱敏规则管理** (order: 40) - 初始设置和偶尔修改
5. **脱敏记录** (order: 50) - 查看历史记录
6. **模块配置** (order: 60) - 系统配置

## 调整原因
按照实际使用频率排序：
- 敏感检测和AI润色是最常用的直接操作，放在最前面
- 脱敏测试是验证功能，频率中等
- 规则管理是基础配置，设置完成后较少使用
- 记录和配置使用频率最低

## 修改文件
- `app/code/GuoLaiRen/Desensitization/etc/backend/menu.xml`

## 生效方法
菜单配置修改后需要重新加载：
1. 清除缓存：`php bin/w cache:flush`
2. 或者重新访问后台页面自动刷新菜单缓存

## 验证方法
访问任意脱敏模块页面，查看左侧菜单顺序是否为：
1. 敏感检测
2. AI润色
3. 脱敏测试
4. 脱敏规则管理
5. 脱敏记录
6. 模块配置
