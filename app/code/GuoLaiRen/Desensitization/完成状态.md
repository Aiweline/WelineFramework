## 功能特性

### 1. 三种脱敏方式
- **正则脱敏**：基于规则快速脱敏
- **AI智能脱敏**：使用AI理解上下文并脱敏
- **自定义脱敏**：使用自定义规则

### 2. 敏感内容检测
- 自动识别多种敏感信息
- 支持指定检测类型
- 提供详细的检测报告

### 3. AI润色
- 脱敏后自动润色
- 多种润色风格可选
- 保持原有格式

### 4. 模型选择
- 支持选择AI模型
- 自动获取可用模型列表
- 配置默认模型

## 使用说明

### 访问后台页面
1. 脱敏规则管理: `/desensitization/backend/rule/index`
2. 脱敏记录: `/desensitization/backend/log/index`
3. 敏感检测: `/desensitization/backend/detect/index`
4. AI润色: `/desensitization/backend/rewrite/index`
5. 脱敏测试: `/desensitization/backend/test/index`

### 使用API
```bash
# 脱敏处理
POST /desensitization/api/process
{
  "content": "测试内容",
  "method": "regex",
  "options": {
    "rule_types": ["email", "phone"]
  }
}

# 敏感检测
POST /desensitization/api/detect
{
  "content": "测试内容",
  "model_code": "gpt-4"
}

# AI润色
POST /desensitization/api/rewrite
{
  "content": "测试内容",
  "model_code": "gpt-4",
  "rewrite_style": "natural"
}
```

## 注意事项

1. **AI模型配置** - 需要先配置AI模型才能使用AI相关功能
2. **数据库升级** - 运行 `php bin/w setup:upgrade` 确保数据库表已创建
3. **模块启用** - 确保模块已启用 `php bin/w module:enable GuoLaiRen_Desensitization`
